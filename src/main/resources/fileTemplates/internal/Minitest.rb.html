Встроенный шаблон для тестов Minitest. Этот шаблон используется каждый раз при создании нового теста. Выберите Создать | Ruby тест из контекстного меню в одном из представлений проекта. Шаблон доступен, когда выбрана папка тестовых исходников. Помимо операторов Ruby, выражений и комментариев, вы можете использовать предопределённые переменные, перечисленные ниже. Каждая переменная расширяется как макрос до соответствующего значения. С помощью директивы #parse можно включить шаблоны из вкладки Включения, указав полное имя нужного шаблона в качестве параметра в кавычках.
Предопределённые переменные принимают следующие значения:
${FILE_NAME}   Имя текущего файла
${USER}   Системное имя текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PRODUCT_NAME}   Название текущей ИСР
${PROJECT_NAME}   Название текущего проекта
