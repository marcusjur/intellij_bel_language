Встроенный шаблон для класса RBS. Этот шаблон используется каждый раз при создании нового класса RBS. Выберите Создать | RBS файл/класс из контекстного меню в одном из представлений проекта. Помимо операторов RBS, выражений и комментариев, вы можете использовать предопределённые переменные, перечисленные ниже. Каждая переменная разворачивается как макрос в соответствующее значение. С помощью директивы #parse вы можете включить шаблоны со вкладки Include, указав полное имя нужного шаблона в качестве параметра в кавычках.
Предопределённые переменные принимают следующие значения:
${FILE_NAME}   Имя текущего файла
${USER}   Системное имя текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PRODUCT_NAME}   Название текущей ИСР
${PROJECT_NAME}   Название текущего проекта
