Встроенный шаблон для PHPSpec спецификации. Этот шаблон используется каждый раз при создании новой PHPSpec спецификации. Выберите Создать | PHP тест из контекстного меню в одном из представлений проекта, затем выберите "PHPSpec" как тестовую библиотеку.
Шаблон можно редактировать. Помимо PHP-выражений, операторов и комментариев, вы можете использовать предопределённые переменные, перечисленные ниже. Каждая переменная разворачивается как макрос в соответствующее значение.
С помощью директивы #parse можно включать шаблоны со вкладки Includes, указав полное имя нужного шаблона в кавычках как параметр.
Предопределённые переменные принимают следующие значения:
${TESTED_NAME}   Имя тестируемого класса
${TESTED_NAMESPACE}   Пространство имён тестируемого класса
${NAME}   Имя тестового класса
${NAMESPACE}   Пространство имён тестового класса
${FILE_NAME}   Имя текущего файла
${USER}   Системное имя текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PRODUCT_NAME}   Название текущей ИСР
${DS}   Знак доллара, вычисляется как обычный символ "$"
${PROJECT_NAME}   Название текущего проекта
