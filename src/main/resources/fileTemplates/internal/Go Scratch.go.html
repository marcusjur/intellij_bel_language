Go Scratch - это встроенный шаблон, используемый при создании новых временных файлов Go. Временные файлы - это полнофункциональные, работоспособные и отлаживаемые файлы, которые можно использовать для набросков ваших идей программования. Чтобы создать временный файл, нажмите Файл | Создать | Временный файл. В диалоговом окне Создать временный файл найдите и выберите Go.

Этот шаблон можно редактировать. В шаблоне можно использовать предопределённые переменные, перечисленные ниже. Эти переменные позже будут развернуты в соответствующие значения в редакторе.

Вы также можете указать пользовательские переменные. Пользовательские переменные используют следующий формат: ${VARIABLE_NAME}, где VARIABLE_NAME - это имя переменной (например, ${MY_CUSTOM_FUNCTION_NAME}). Перед созданием нового файла с пользовательскими переменными ИСР покажет диалоговое окно, где вы сможете определить значения пользовательских переменных из шаблона.

Используя директиву #parse, вы можете включать шаблоны из вкладки Include. Чтобы включить шаблон, укажите его полное имя в кавычках в качестве параметра (например, #parse("File Header")).
Список предопределённых переменных
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX}   Имя пакета, в котором создаётся новый тестовый файл. Файл должен иметь суффикс _test. Формат имени следующий: package_test.
${GO_PACKAGE_NAME}   Имя пакета, в котором создаётся новый файл.
${DATE}   Текущая системная дата.
${DAY}   День месяца.
${DAY_NAME_SHORT}   Первые 3 буквы названия текущего дня недели (например, Mon, Tue и т.д.).
${DAY_NAME_FULL}   Полное название текущего дня недели (например, Monday, Tuesday и т.д.).
${DIR_PATH}   Путь к директории нового файла. Этот путь относительно корня проекта.
${DS}   Знак доллара ($). Эта переменная используется для экранирования символа доллара, чтобы он не рассматривался как префикс переменной шаблона.
${FILE_NAME}   Имя нового файла.
${HOUR}   Текущий час.
${MINUTE}   Текущая минута.
${SECOND}   Текущая секунда.
${MONTH}   Текущий месяц.
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца (например, Jan, Feb и т.д.).
${MONTH_NAME_FULL}   Полное название текущего месяца (например, January, February и т.д.).
${NAME}   Имя нового элемента: файла, типа, интерфейса и т.д.
${PRODUCT_NAME}   Название ИСР.
${PROJECT_NAME}   Название текущего проекта.
${TIME}   Текущее системное время.
${USER}   Системное имя входа текущего пользователя.
${YEAR}   Текущий год.
