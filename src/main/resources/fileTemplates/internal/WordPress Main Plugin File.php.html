Встроенный шаблон основного файла плагина WordPress. Он используется каждый раз при создании нового проекта типа WordPress Plugin.
Шаблон можно редактировать. Помимо операторов PHP, выражений и комментариев, вы можете использовать предопределённые переменные, перечисленные ниже. Каждая переменная разворачивается как макрос в соответствующее значение.
С помощью директивы #parse можно включать шаблоны из вкладки Includes, указав полное имя нужного шаблона в качестве параметра в кавычках.
Предопределённые переменные принимают следующие значения:
${PLUGIN_NAME}   Текущее имя плагина, полученное из имени директории
${FILE_NAME}   Текущее имя файла
${USER}   Системное имя текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PRODUCT_NAME}   Название текущей ИСР
${PROJECT_NAME}   Название текущего проекта
${DS}   Знак доллара, вычисляется как обычный символ "$"
