Применяется для создания нового Java-интерфейса через Создать | Java класс | Интерфейс в окне инструмента Проект.

Этот встроенный шаблон можно редактировать. Помимо Java-выражений и комментариев, вы можете использовать предопределённые переменные (перечислены ниже), которые затем будут развернуты как макросы в соответствующие значения.

Вы также можете указать пользовательские переменные. Пользовательские переменные используют следующий формат: ${VARIABLE_NAME}, где VARIABLE_NAME - это имя переменной (например, ${MY_CUSTOM_FUNCTION_NAME}). Перед тем как ИСР создаст новый файл с пользовательскими переменными, появится диалоговое окно, где вы сможете определить значения пользовательских переменных в шаблоне.

Используя директиву #parse, вы можете включать шаблоны из вкладки Include. Чтобы включить шаблон, укажите его полное имя в кавычках в качестве параметра (например, #parse("File Header.java")).
Предопределённые переменные принимают следующие значения:
${PACKAGE_NAME}   Имя пакета, в котором создаётся новый интерфейс
${NAME}   Имя нового интерфейса, указанное в диалоговом окне Создать новый класс
${USER}   Системное имя входа текущего пользователя
${DATE}   Системное имя входа текущего пользователя
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца (Jan, Feb и т.д.)
${MONTH_NAME_FULL}   Полное название текущего месяца (January, February и т.д.)
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PROJECT_NAME}   Название текущего проекта
