Встроенный шаблон PHP-файла. Этот шаблон используется каждый раз, когда вы создаете новый PHP-файл, выбирая Создать | PHP файл из контекстного меню в окне проекта.
Шаблон можно редактировать. Помимо операторов PHP, выражений и комментариев, вы можете использовать предопределённые переменные, перечисленные ниже. Каждая переменная разворачивается как макрос в соответствующее значение.
С помощью директивы #parse можно включать шаблоны со вкладки Includes, указав полное имя нужного шаблона в кавычках в качестве параметра.
Предопределённые переменные принимают следующие значения:
${FILE_NAME}   Имя текущего файла
${USER}   Системное имя текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PRODUCT_NAME}   Название текущей ИСР
${PROJECT_NAME}   Название текущего проекта
