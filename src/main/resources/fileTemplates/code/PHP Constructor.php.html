Встроенный шаблон конструктора PHP класса.
Предопределённые переменные принимают следующие значения:
${PARAM_LIST}   Список параметров конструктора через запятую, соответствующий каждому выбранному свойству.
${PARAM_DOC}   PHPDoc комментарии для параметров.
Генерируется как серия строк '* @param type name'. Если параметров нет, возвращает пустое значение.
${THROWS_DOC}   PHPDoc комментарии для исключений.
Генерируется как серия строк '* @throws type'. Если исключения не выбрасываются, возвращает пустое значение.
${CLASS_NAME}   Имя класса.
${BODY}   Тело конструктора, содержащее присваивания вида '$this->property=$property;'.
