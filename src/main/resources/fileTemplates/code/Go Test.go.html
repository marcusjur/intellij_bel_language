Go Test — это встроенный шаблон для создания тестовых функций Go в тестовых файлах. Этот шаблон доступен только для тестовых файлов с суффиксом _test. Чтобы использовать шаблон, выберите Код | Сгенерировать и выберите Тест.
Список предопределённых переменных
${NAME}   Имя тестовой функции
${PARAM_NAME}   Имя параметра функции
${PARAM_TYPE}   Тип параметра функции
