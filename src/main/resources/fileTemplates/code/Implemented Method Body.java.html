Заполняет тело метода при генерации реализуемых методов (например, при применении операций Реализовать абстрактный класс или Реализовать абстрактные методы, или при вызове функции Код | Реализовать методы...).
Этот встроенный шаблон можно редактировать. Помимо выражений Java и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${METHOD_NAME}   Имя реализуемого метода
${CLASS_NAME}   Полное имя класса, реализующего метод
${SIMPLE_CLASS_NAME}   Краткое имя класса, реализующего метод
Все предопределённые переменные из шаблона Заголовок файла (вкладка Включения) также доступны (например, ${USER}, ${DATE} и т.д.).
