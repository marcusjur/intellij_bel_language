Этот встроенный шаблон используется для заполнения тела метода при каждой генерации переопределённого метода (например, при вызове функции Код | Переопределить методы...).
Шаблон можно редактировать. Помимо выражений Javascript и комментариев, вы можете использовать предопределённые переменные. При применении шаблона эти переменные будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${CALL_SUPER}   вызов метода super, для методов типа void - super.method_name(); с параметрами или без; для других методов - return super.method_name(); с параметрами или без
${PLAIN_CALL_SUPER}   вызов метода super, super.method_name(); с параметрами или без;
${RETURN_TYPE}   тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   значение по умолчанию, возвращаемое методом
${METHOD_NAME}   имя переопределяемого метода
Также доступны все предопределённые переменные из шаблона заголовка файла (вкладка Include). (Например: ${USER}, ${DATE} и т.д.)
