Заменяет строковые литералы (например, String s = "Hello";) на интернационализованные Java-выражения (например, String s = myResourceBundle.getString("hello");).
Этот встроенный шаблон можно редактировать. Помимо Java-выражений и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${RESOURCE_BUNDLE}   Выражение типа java.util.ResourceBundle, экземпляр которого доступен в данном контексте.
${PROPERTY_KEY}   Имя ключа свойства, указанного в соответствующем файле свойств. Обычно это значение используется в качестве параметра метода ResourceBundle.getString().
${PROPERTY_VALUE}   Значение свойства, определённое в соответствующем файле свойств. Это исходное значение строкового литерала Java.
