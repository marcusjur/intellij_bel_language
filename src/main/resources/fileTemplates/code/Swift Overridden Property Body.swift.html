Заполняет тело свойства при генерации переопределённых свойств (например, при использовании действия Код | Генерировать | Переопределить методы).
Шаблон можно редактировать. Помимо выражений Swift и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${CALL_SUPER}   Вызов свойства super: super.propertyName
${RETURN_TYPE}   Возвращаемый тип создаваемого свойства
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое свойством
${SIGNATURE}   Сигнатура свойства
${HAS_SET}   Значение "true", если свойство имеет сеттер
${HAS_SUPER_SET}   Значение "true", если переопределяемое свойство имеет сеттер
${GETTER_EFFECTS}   Эффекты геттера свойства (async, throws)
${GET_ATTRIBUTES}   Атрибуты геттера свойства
${SET_ATTRIBUTES}   Атрибуты сеттера свойства
Также доступны все предопределённые переменные из шаблона Заголовка файла Swift (вкладка Include), например ${USER}, ${DATE} и другие.
