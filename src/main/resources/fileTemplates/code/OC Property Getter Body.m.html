Создает тело переопределённого метода getter для свойства при генерации (например, при вызове операции Код | Переопределить методы).
Шаблон доступен для редактирования. Помимо выражений Objective-C и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${CALL_SUPER}   Вызов метода super, для методов типа void — [super methodName], с параметрами или без; для остальных методов — return [super methodName], с параметрами или без
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${CONTAINING_CLASS}   Имя класса, в котором переопределяется метод
${IVAR_IS_AVAILABLE}   Значение "true", когда свойство имеет переменную экземпляра
${IVAR}   Имя переменной экземпляра
${SETTER_SEMANTICS}   В режиме ARC: "copyArc", если установлен атрибут copy, иначе "assign"
В режиме без ARC: "copy", если установлен атрибут copy, "retain", если установлен атрибут retain, иначе "assign"
Все предопределённые переменные из шаблона Заголовка C-файла (вкладка Include) также доступны (например, ${USER}, ${DATE} и т.д.)
