Заполняет тело инициализатора при генерации реализованных инициализаторов (например, при использовании действия Код | Генерировать | Реализовать методы).
Шаблон доступен для редактирования. Помимо выражений Swift и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${SIGNATURE}   Сигнатура инициализатора
Все предопределённые переменные из шаблона Заголовка файла Swift (вкладка Include) также доступны (например, ${USER}, ${DATE} и т.д.).
