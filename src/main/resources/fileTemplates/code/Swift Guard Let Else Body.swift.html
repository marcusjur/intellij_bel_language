После развертывания опционального значения с помощью оператора guard, заполните блок else. Этот шаблон можно редактировать. Помимо выражений Swift и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения.
Предопределённые переменные примут следующие значения:
${RETURN_TYPE}   Тип возвращаемого значения окружающей функции (если существует)
${DEFAULT_RETURN_VALUE}   Предлагаемое значение по умолчанию для возвращаемого значения (может быть пустым)
Все предопределённые переменные из шаблона заголовка Swift-файла (вкладка Include) также доступны (например, ${USER}, ${DATE} и т.д.).
