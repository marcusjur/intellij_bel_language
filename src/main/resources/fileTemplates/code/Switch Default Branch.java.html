Генерирует ветку default для выражения или оператора switch, используемого в быстрых исправлениях, таких как вставка ветки 'default'.
Этот встроенный шаблон можно редактировать. Например, вы можете изменить выбрасываемое исключение или вызов логования.
Предопределённые переменные принимают следующие значения:
${EXPRESSION}   Текст выражения селектора switch
${EXPRESSION_TYPE}   Тип выражения селектора switch
