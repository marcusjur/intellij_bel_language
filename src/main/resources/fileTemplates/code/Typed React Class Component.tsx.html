Этот встроенный шаблон используется для создания классовых компонентов React (например, при рефакторинге через Extract Component). Вы можете настроить этот шаблон, например использовать Component вместо PureComponent или изменить определение типов prop. Вы также можете использовать предопределённые переменные. При применении шаблона эти переменные будут заменены соответствующими значениями. Этот шаблон используется в TypeScript и Flow.
Предопределённые переменные примут следующие значения:
${NAME}   Имя компонента
${COMPONENT_BODY}   Тело компонента
${PROP_TYPES}   Определение типов props
${PROP_TYPES}   React будет использовать значения props по умолчанию, если не предоставлены другие значения.
