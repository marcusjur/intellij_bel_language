Встроенный шаблон для PHP setter-метода. Используется, например, при вызове быстрого исправления "Добавить геттеры и сеттеры".
Предопределённые переменные принимают следующие значения:
${NAME}   Имя метода в стиле camelCase без префикса "set", первый символ в верхнем регистре. Например, имя свойства '_my_property' преобразуется в 'MyProperty'.
${CLASS_NAME}   Содержит имя класса.
${SCALAR_TYPE_HINT}   Содержит подсказку скалярного типа, если уровень языка PHP 7.0 или выше
${RETURN_TYPE}   Указывает, является ли уровень языка PHP 7.0 или выше, что позволяет указать возвращаемый тип для функции/метода.
${FIELD_NAME}   Исходное имя свойства без изменений.
${PARAM_NAME}   Имя формального параметра сеттера. Практически идентично ${FIELD_NAME}, но без первого подчеркивания (если оно присутствует). Например, имя свойства '_myProperty' преобразуется в 'myProperty'.
${TYPE_HINT}   Подсказка типа свойства, пустая строка, если не указана.
${DS}   Знак доллара, вычисляется как обычный символ "$".
