Создает тело переопределяемого метода при генерации (например, при вызове операции Код | Переопределить методы).
Шаблон доступен для редактирования. Помимо выражений и комментариев Objective-C, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${CALL_SUPER}   вызов метода super, для методов типа void — [super methodName], с параметрами или без; для остальных методов — return [super methodName], с параметрами или без
${RETURN_TYPE}   тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   значение по умолчанию, возвращаемое методом
${CONTAINING_CLASS}   имя класса, в котором переопределяется метод
Также доступны все предопределённые переменные из шаблона заголовка C-файла (вкладка Включает) (например, ${USER}, ${DATE} и т.д.)
