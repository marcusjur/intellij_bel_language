Это шаблон для создания метода тестования WebSocket в тестовом классе JUnit 4 для Ktor 2.x.x.
Предопределённые переменные примут следующие значения:
${NAME}   Имя создаваемого метода
${ROUTE}   Маршрут относительно текущего тестируемого сервиса
${CONFIGURE_SERVER}   Метод расширения маршрутизации для настройки дерева маршрутов тестируемой части сервера
${BODY}   Тело сгенерированного метода
