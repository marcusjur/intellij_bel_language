Этот встроенный шаблон используется для заполнения тела метода при каждой генерации метода (например, через интенцию создать метод из использования).
Шаблон можно редактировать. Помимо выражений Javascript и комментариев, вы можете использовать предопределённые переменные. При применении шаблона эти переменные будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${IS_ASSIGNMENT_TARGET}   Возвращает 'true', если создаваемый метод является целью присваивания.
${IS_NEW_CALL}   Возвращает 'true', если создаваемый метод находится под вызовом 'new'.
${HAS_EXPECTED_TYPE}   Возвращает 'true', если создаваемый метод находится в позиции, где ожидается тип.
${DEFAULT_RETURN_VALUE}   Значение по умолчанию для ожидаемого типа создаваемого метода (или 'undefined', если ожидаемый тип отсутствует)
${METHOD_NAME}   Имя создаваемого метода
${SEMICOLON}   Точка с запятой или пусто, в зависимости от настроек стиля кода
Также доступны все предопределённые переменные из шаблона заголовка файла (вкладка Include). (Например: ${USER}, ${DATE} и т.д.)
