Заполняет тело свойства при генерации реализованных свойств (например, при использовании операции Код | Генерировать | Реализовать методы).
Шаблоны доступны для редактирования. Помимо выражений Swift и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип создаваемого свойства
${DEFAULT_RETURN_VALUE}   Предлагаемое значение по умолчанию для возвращаемого значения (может быть пустым)
${SIGNATURE}   Сигнатура свойства
${HAS_SET}   Имеет значение "true", когда у свойства есть сеттер
${GETTER_EFFECTS}   Эффекты геттера свойства (async, throws)
${GET_ATTRIBUTES}   Атрибуты геттера свойства
${SET_ATTRIBUTES}   Атрибуты сеттера свойства
Все предопределённые переменные из шаблона Заголовка файла Swift (вкладка Include) также доступны (например, ${USER}, ${DATE} и т.д.).
