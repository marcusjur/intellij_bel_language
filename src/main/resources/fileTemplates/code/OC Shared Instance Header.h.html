Генерирует объявление метода +sharedInstance в заголовке класса, например при вызове операции Код | Сгенерировать | +sharedInstance.
Шаблон можно редактировать. Помимо выражений и комментариев Objective-C, вы можете использовать предопределённые переменные, которые затем будут развернуты в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${$CLASS_TYPE}   Имя класса, в котором реализуется метод
Все предопределённые переменные из шаблона заголовка C-файла (вкладка Include) также доступны (например, ${USER}, ${DATE})
