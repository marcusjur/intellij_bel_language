Заполняет тело метода при генерации переопределённых методов (например, при использовании действия Код | Генерировать | Переопределить методы).
Шаблон можно редактировать. Помимо выражений Swift и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${CALL_SUPER}   Вызов метода super для методов типа Void: super.methodName() с параметрами или без них; для других методов: return super.methodName() с параметрами или без них
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${SIGNATURE}   Сигнатура метода
Также доступны все предопределённые переменные из шаблона заголовка файла Swift (вкладка Include) (например, ${USER}, ${DATE} и т.д.).
