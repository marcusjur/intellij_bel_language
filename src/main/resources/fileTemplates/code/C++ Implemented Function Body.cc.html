Создает тело реализуемой функции при генерации (например, при использовании операции Код | Реализовать методы).
При редактировании шаблона вы можете использовать выражения C++, комментарии и следующие предопределённые переменные:
${RETURN_TYPE}   Тип возвращаемого значения функции.
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое функцией.
Все предопределённые переменные из шаблона Заголовок C-файла (вкладка Включить), такие как ${USER} и ${DATE}, также доступны.
