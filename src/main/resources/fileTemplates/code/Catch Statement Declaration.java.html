Заполняет параметр catch во время генерации, например при вызове функции Код | Окружить....
Этот встроенный шаблон можно редактировать. Например, вы можете сделать сгенерированный параметр final или использовать предопределённое имя независимо от полного имени перехваченного исключения.
Предопределённые переменные принимают следующие значения:
${EXCEPTION}   Переменная Exception задается как параметр catch, производный от типа перехваченного исключения e, если полное имя заканчивается на Exception.
${EXCEPTION_TYPE}   Тип параметра catch.
