Go Example - это встроенный шаблон, который используется для создания функций Go Example в тестовых файлах. Этот шаблон доступен только для тестовых файлов с суффиксом _test. Чтобы использовать этот шаблон, выберите Код | Сгенерировать и выберите Example.
Список предопределённых переменных
${NAME}   Имя функции Example
