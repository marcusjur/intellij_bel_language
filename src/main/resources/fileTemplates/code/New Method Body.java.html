Заполняет тело метода при генерации метода (например, при применении операции намерения Создать метод из использования).
Этот встроенный шаблон можно редактировать. Помимо выражений Java и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${METHOD_NAME}   Имя создаваемого метода
${CLASS_NAME}   Полное имя класса, в котором создаётся метод
${SIMPLE_CLASS_NAME}   Краткое имя класса, в котором реализуется метод
Также доступны все предопределённые переменные из шаблона заголовка файла (вкладка Include). (Например: ${USER}, ${DATE} и т.д.)
