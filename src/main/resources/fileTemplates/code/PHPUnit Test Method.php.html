Встроенный шаблон метода тестования PHPUnit. Используется, например, когда флажок для генерации тестового метода установлен в диалоговом окне "Создать новый тест".
Предопределённые переменные принимают следующие значения:
${NAME}   Название метода.
${CAPITALIZED_NAME}   Название метода с заглавной первой буквой.
${TESTED_NAME}   Название тестируемого класса.
