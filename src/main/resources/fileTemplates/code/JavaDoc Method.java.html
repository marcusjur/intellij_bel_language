Определяет шаблоны для генерации JavaDoc для методов, например при применении операции Добавить Javadoc.

Помимо выражений Java и комментариев, вы можете использовать предопределённые переменные, которые затем будут развернуты как макросы в соответствующие значения.

С помощью директивы #parse можно включать шаблоны, определённые на вкладке Включения. Чтобы включить шаблон, укажите его полное имя в кавычках в качестве параметра (например, #parse("File Header.java")).
Предопределённые переменные принимают следующие значения:
${PARAMS}   Список имен параметров
${TYPE_PARAMS}   Список имен параметров типа
${THROWS}   Список выбрасываемых исключений
${RETURN_TYPE}   Тип, возвращаемый методом
${ELEMENT_NAME}   Имя члена класса, для которого генерируется JavaDoc
${CONTAINING_CLASS}   Имя родительского класса
${PACKAGE_NAME}   Имя пакета, в котором создаётся новое перечисление
${NAME}   Имя нового перечисления, указанное в диалоге Создать новый класс
${USER}   Системное имя входа текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца (Jan, Feb и т.д.)
${MONTH_NAME_FULL}   Полное название текущего месяца (January, February и т.д.)
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PROJECT_NAME}   Имя текущего проекта
