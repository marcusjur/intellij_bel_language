Генерирует метод +sharedInstance в части реализации класса, например при вызове операции Код | Сгенерировать | +sharedInstance.
Шаблон можно редактировать. Помимо выражений Objective-C и комментариев, вы можете использовать предопределённые переменные, которые затем будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${$CLASS_TYPE}   Имя класса, в котором реализуется метод
Все предопределённые переменные из шаблона заголовка C-файла (вкладка Include) также доступны (например, ${USER}, ${DATE})
