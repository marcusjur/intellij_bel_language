Встроенный шаблон для PHP getter-метода. Используется, например, при вызове быстрого исправления "Добавить геттеры и сеттеры".
Предопределённые переменные принимают следующие значения:
${STATIC}   Принимает значение 'static', если свойство статическое, иначе - пустая строка.
${GET_OR_IS}   Префикс 'get/is'. Если свойство имеет логический тип (type hint), возвращает 'is', в противном случае возвращает строку 'get'.
${NAME}   Имя метода в стиле camelCase без префикса "get", первый символ в верхнем регистре. Например, имя свойства '_my_property' преобразуется в 'MyProperty'.
${CLASS_NAME}   Содержит имя класса.
${RETURN_TYPE}   Содержит тип возвращаемого значения, если уровень PHP 7.0 или выше
${FIELD_NAME}   Исходное имя свойства без изменений.
${TYPE_HINT}   Type hint свойства, пустая строка, если не указан.
${DS}   Знак доллара, возвращает обычный символ "$".
