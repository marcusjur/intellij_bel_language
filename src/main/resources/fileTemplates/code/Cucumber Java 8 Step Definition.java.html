Создание определений шагов для Java-файлов в Cucumber.
Предопределённые переменные принимают следующие значения:
${STEP_KEYWORD}   Аннотация определения шага. Обычно это ключевое слово, используемое для вызова шага (Given, Then, And и т.д.).
${STEP_REGEXP}   Регулярное выражение, сгенерированное для принятия шага.
${PARAMETERS}   Формальные параметры сгенерированного метода.
${BODY}   Тело сгенерированного метода.
