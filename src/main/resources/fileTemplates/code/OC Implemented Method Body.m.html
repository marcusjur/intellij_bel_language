Создает тело метода реализации при генерации (например, при вызове операции Код | Реализовать методы).
Шаблон доступен для редактирования. Помимо выражений и комментариев Objective-C, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${CONTAINING_CLASS}   Имя класса, в котором реализуется метод
Все предопределённые переменные, доступные в шаблоне заголовка C-файла (вкладка Include), также могут использоваться здесь (${USER}, ${DATE} и т.д.)
