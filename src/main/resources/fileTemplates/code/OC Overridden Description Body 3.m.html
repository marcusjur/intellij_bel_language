Тело метода -description, создаваемого при генерации для класса без переменных экземпляра (например, при вызове операции Код | Создать | -description).
Шаблон доступен для редактирования. Помимо выражений Objective-C и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${CONTAINING_CLASS}   Имя класса, в котором реализуется метод
Все предопределённые переменные из шаблона заголовка C-файла (вкладка Include) также доступны (например, ${USER}, ${DATE})
