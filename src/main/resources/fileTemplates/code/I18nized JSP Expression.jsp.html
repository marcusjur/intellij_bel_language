Замена текста в JSP, например:
  <html><body>
    Hello!
  </body></html>
с интернационализованным пользовательским тегом JSP, например:
  <html><body>
    <bean:message key="HelloKey"/>
  </body></html>
Этот встроенный шаблон можно редактировать. Помимо выражений, допустимых в контексте JSP, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${PROPERTY_KEY}   Имя ключа свойства, указанного в соответствующем файле свойств.
${PROPERTY_VALUE}   Значение свойства, указанное в соответствующем файле свойств. Это исходное значение выбранного текста JSP.
