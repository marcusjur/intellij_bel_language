Это встроенный шаблон для замены текста в GSP Например:
     <html> <body> Hello! </body> </html>
на интернационализованные GSP пользовательские теги, например:
     <html> <body> <g:message code="HelloKey"/> </body> </html> .

Этот шаблон можно редактировать. Помимо выражений, разрешённых в контексте GSP, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения.

Предопределённые переменные принимают следующие значения:
${PROPERTY_KEY}   Имя ключа свойства, указанного в соответствующем файле свойств.
${PROPERTY_VALUE}   Значение свойства, указанное в соответствующем файле свойств. Это исходное значение выбранного текста GSP.
