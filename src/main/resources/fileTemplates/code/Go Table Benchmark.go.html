Go Table Benchmark — это встроенный шаблон для создания табличных бенчмарк-функций в тестовых файлах Go. Этот шаблон доступен только в тестовых файлах с суффиксом _test. Чтобы использовать шаблон, выберите Код | Сгенерировать и выберите Табличный бенчмарк.
Список предопределённых переменных
${NAME}   Название табличной бенчмарк-функции
${PARAM_NAME}   Имя формального параметра функции
${PARAM_TYPE}   Тип формального параметра функции
