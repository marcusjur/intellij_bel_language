Этот встроенный шаблон используется для заполнения тела метода при его создании (например, через интент-действие создать метод из использования).
Шаблон можно редактировать. Помимо выражений TypeScript и комментариев, вы можете использовать предопределённые переменные. При применении шаблона эти переменные будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип возвращаемого значения создаваемого метода
${DEFAULT_RETURN_VALUE}   Значение по умолчанию, возвращаемое методом
${METHOD_NAME}   Имя создаваемого метода
${SEMICOLON}   Точка с запятой или пусто, в зависимости от настроек стиля кода
Также доступны все предопределённые переменные из шаблона заголовка файла (вкладка Include). (Например: ${USER}, ${DATE} и т.д.)
