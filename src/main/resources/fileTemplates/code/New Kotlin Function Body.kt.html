Заполняется каждый раз при программной генерации тела функции Kotlin, например при использовании операции намерения Создать функцию на основе использования.
Шаблоны доступны для редактирования. Помимо выражений и комментариев Kotlin, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения.
Предопределённые переменные примут следующие значения:
${RETURN_TYPE}   Возвращаемый тип создаваемой функции
${FUNCTION_NAME}   Имя создаваемой функции
${CLASS_NAME}   Полное имя класса, в котором создаётся функция
${SIMPLE_CLASS_NAME}   Краткое имя класса, в котором реализуется функция
