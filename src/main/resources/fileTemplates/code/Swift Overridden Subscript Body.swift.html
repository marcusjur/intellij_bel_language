Заполняет тело индекса при генерации переопределённых индексов (например, при использовании операции Код | Генерировать | Переопределить методы).
Шаблон доступен для редактирования. Помимо выражений Swift и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут заменены соответствующими значениями.
Предопределённые переменные принимают следующие значения:
${CALL_SUPER}   вызов super индекса - return super[] с параметрами
${RETURN_TYPE}   тип возвращаемого значения создаваемого индекса
${DEFAULT_RETURN_VALUE}   значение по умолчанию, возвращаемое индексом
${SIGNATURE}   сигнатура индекса
${HAS_SET}   "true", когда у индекса есть сеттер
${HAS_SUPER_SET}   "true", когда у переопределяемого индекса есть сеттер
${GETTER_EFFECTS}   эффекты геттера свойства (async, throws)
${GET_ATTRIBUTES}   атрибуты геттера индекса
${SET_ATTRIBUTES}   атрибуты сеттера индекса
Также доступны все предопределённые переменные из шаблона Заголовка файла Swift (вкладка Include), например ${USER}, ${DATE} и другие.
