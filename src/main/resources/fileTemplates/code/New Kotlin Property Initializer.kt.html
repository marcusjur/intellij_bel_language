Заполняет инициализатор свойства Kotlin каждый раз, когда он генерируется программой, например при использовании операции Создать свойство на основе использования.
Шаблоны можно редактировать. Помимо выражений Kotlin и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${RETURN_TYPE}   Тип создаваемого свойства
${PROPERTY_NAME}   Имя создаваемого свойства
${CLASS_NAME}   Полное имя класса, в котором создаётся свойство
${SIMPLE_CLASS_NAME}   Короткое имя класса, в котором создаётся свойство
