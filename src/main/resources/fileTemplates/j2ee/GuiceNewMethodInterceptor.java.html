Это встроенный шаблон, используемый при создании нового класса слушателя методов Guice.
Используйте директиву #parse для включения шаблона, указав его полное имя в кавычках как параметр на вкладке включить. Например:
#parse("File Header.java")
Предопределённые переменные будут иметь следующие значения:
${PACKAGE_NAME}   Имя пакета, в котором создаётся новый класс
${NAME}   Имя нового класса, указанное в диалоговом окне создания нового класса
${USER}   Текущее системное имя пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца. Например: Jan, Feb и т.д.
${MONTH_NAME_FULL}   Полное название текущего месяца. Например: January, February и т.д.
${DAY}   Текущий день
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PROJECT_NAME}   Название текущего проекта
