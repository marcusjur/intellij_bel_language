Встроенный шаблон PHPDoc-комментария для реализации хука Drupal. Он содержит сниппет кода, который может быть включен в шаблон файла (тег Templates) с помощью директивы #parse.
Шаблон доступен для редактирования. Помимо статического текста, кода и комментариев, вы можете использовать предопределённые переменные, которые будут раскрыты как макросы в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${HOOK_NAME}   Название хука.
