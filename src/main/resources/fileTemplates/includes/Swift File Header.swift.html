Содержит фрагмент кода, который можно включить в шаблон файла (вкладка Шаблон) с помощью директивы #parse.
Шаблон доступен для редактирования. Помимо статического текста, кода и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты в соответствующие значения как макросы.
Предопределённые переменные принимают следующие значения:
${FILE_NAME}   Имя текущего файла
${USER}   Системное имя входа текущего пользователя
${USER_NAME}   Полное имя текущего пользователя
${ORGANIZATION_NAME}   Название организации проекта Xcode
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца. Например: Jan, Feb и т.д.
${MONTH_NAME_FULL}   Полное название текущего месяца. Например: January, February и т.д.
${DAY}   Текущий день
${DAY_NAME_SHORT}   Первые 3 буквы названия дня недели. Например: Mon, Tue и т.д.
${DAY_NAME_FULL}   Полное название дня недели. Например: Monday, Tuesday и т.д.
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PROJECT_NAME}   Название текущего проекта
${COMPILER_VERSION_MAJOR}   Основная версия компилятора Swift
${COMPILER_VERSION_MINOR}   Дополнительная версия компилятора Swift
${PREFER_EXPLICIT_RETURN}   Значение опции Предпочитать явный возврат (Настройки | Редактор | Стиль кода | Swift | Генерация кода)
