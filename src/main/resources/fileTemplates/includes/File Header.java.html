Содержит фрагмент кода, который может быть включен в шаблон файла (вкладка Templates) с помощью директивы #parse.
Этот встроенный шаблон можно редактировать. Помимо статического текста, кода и комментариев, вы можете использовать предопределённые переменные, которые впоследствии будут развернуты как макросы в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${PACKAGE_NAME}   Имя пакета, в котором создаётся новый файл
${USER}   Системное имя текущего пользователя
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца. Например: Jan, Feb и т.д.
${MONTH_NAME_FULL}   Полное название текущего месяца. Например: January, February и т.д.
${DAY}   День месяца
${DAY_NAME_SHORT}   Первые 3 буквы названия текущего дня недели. Например: Mon, Tue и т.д.
${DAY_NAME_FULL}   Полное название текущего дня недели. Например: Monday, Tuesday и т.д.
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${PROJECT_NAME}   Название текущего проекта
