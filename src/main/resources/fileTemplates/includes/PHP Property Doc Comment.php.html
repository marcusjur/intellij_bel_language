Встроенный шаблон PHPDoc-комментария для свойства PHP-класса. Он содержит фрагмент кода, который может быть включен в шаблон файла (вкладка Templates) с помощью директивы #parse.
Шаблон можно редактировать. Помимо статического текста, кода и комментариев, вы можете использовать предопределённые переменные, которые будут раскрываться как макросы в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${NAME}   Имя свойства.
${CLASS_NAME}   Имя класса.
${STATIC}   Значение 'static', если свойство статическое, иначе пустая строка.
${NAMESPACE}   Полное имя пространства имён свойства без начального слеша.
${DS}   Знак доллара, вычисляется как обычный символ "$".
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца. Например: Jan, Feb и т.д.
${MONTH_NAME_FULL}   Полное название текущего месяца. Например: January, February и т.д.
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${SECOND}   Текущая секунда
${PROJECT_NAME}   Название текущего проекта
${USER}   Текущий пользователь
${CARET}   Отмечает позицию, куда должен быть перемещен текстовый курсор после добавления комментария.
