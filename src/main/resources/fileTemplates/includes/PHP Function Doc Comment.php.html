Встроенный шаблон PHPDoc-комментария для PHP-функции. Он содержит сниппет кода, который может быть включен в файловые шаблоны (вкладка Templates) с помощью директивы #parse.
Шаблон можно редактировать. Помимо статического текста, кода и комментариев, вы можете использовать предопределённые переменные, которые будут раскрываться как макросы в соответствующие значения.
Предопределённые переменные принимают следующие значения:
${NAME}   Имя функции.
${TYPE_HINT}   Подсказка типа возвращаемого значения функции. Если тип возвращаемого значения не определен, возвращается 'void'.
${PARAM_DOC}  

PHPDoc-комментарии для параметров.

Генерируется как серия строк '* @param type name'. Если параметров нет, возвращается пустое значение.

${THROWS_DOC}  

Документация для исключений.

Генерируется как серия строк '* @throws type'. Если исключения не выбрасываются, возвращается пустое значение.

${STATIC}   Принимает значение 'static', если функция/метод статический, иначе пустая строка. Например: 
      #if (${STATIC} == "static") * @static
      #end
${DS}   Знак доллара, возвращает обычный символ "$".
${DATE}   Текущая системная дата
${TIME}   Текущее системное время
${YEAR}   Текущий год
${MONTH}   Текущий месяц
${MONTH_NAME_SHORT}   Первые 3 буквы названия текущего месяца. Например: Jan, Feb и т.д.
${MONTH_NAME_FULL}   Полное название текущего месяца. Например: January, February и т.д.
${DAY}   День месяца
${HOUR}   Текущий час
${MINUTE}   Текущая минута
${SECOND}   Текущая секунда
${PROJECT_NAME}   Название текущего проекта
${USER}   Текущий пользователь
${CARET}   Отмечает позицию, куда должен быть перемещен текстовый курсор после добавления комментария.
