Создание шаблонного кода для идиоматической обработки ошибок.

Может быть вызвано для переменных и выражений типа error.

  • Заменяет переменную err следующей конструкцией:

    if err != nil {
    return value1, value2, ...
  }

  • Заменяет выражение expr следующей конструкцией:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Каждое значение value представляет собой либо переменную err, либо значение по умолчанию соответствующего возвращаемого типа включающей функции. Этот шаблон можно использовать для выражений с несколькими значениями.
