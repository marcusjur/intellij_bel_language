Создание шаблонного кода для идиоматичной обработки ошибок.

Вы можете вызвать шаблон для любого вызова, возвращающего error, или любого выражения типа error. Шаблон введет новые переменные, присвоит им выражения и проверит, не является ли ошибка nil с помощью следующего кода: 

if err != nil {
  return value1, value2, ...
}

Каждое значение value представляет собой либо переменную err, либо значение по умолчанию для соответствующего возвращаемого типа включающей функции.
