Создание шаблонного кода для идиоматической обработки ошибок.

Может быть вызвано на переменных и выражениях типа error.

  • Заменяет переменную err следующей конструкцией:

    if err != nil {
    return value1, value2, ...
  }

  • Заменяет выражение expr следующей конструкцией:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Каждое значение value является либо переменной err, либо значением по умолчанию соответствующего возвращаемого типа включающей функции. Этот шаблон можно вызывать на выражениях с несколькими значениями.
