Внедрить диалект и связать его с выбранным типом SQL. Применить этот диалект ко всем совпадениям того же типа (например, query).

В примере Before вы можете вызвать операцию Внедрить по типу для значения 'SELECT * FROM foo', затем внедрить этот тип объекта и связать его с любым диалектом SQL. Тип объекта для этого значения — query (определено как q query DEFAULT 'SELECT * FROM foo'). Таким образом, когда вы в следующий раз используете тип query в коде, выбранный диалект будет внедрен автоматически.
