Сделать класс final, если у него нет подклассов.

Если есть подклассы, предлагается преобразовать класс в sealed класс и указать список его подклассов в условии permits (для подклассов в других единицах компиляции). Если подклассы не отмечены как sealed / non-sealed / final, модификатор non-sealed будет применен к ним автоматически.
