Преобразует тело метода в форму с единственным оператором return. После применения этого действия все промежуточные операторы return будут удалены, а поток управления будет изменен так, чтобы выход осуществлялся через последний оператор тела метода. Обратите внимание, что хотя преобразование возможно в любое время, полученный код может быть не самым элегантным.
