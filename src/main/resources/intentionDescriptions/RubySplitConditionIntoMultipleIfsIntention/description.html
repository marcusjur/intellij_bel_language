Разделить условие конъюнкции или дизъюнкции на несколько операторов if.

Условие конъюнкции разделяется на два вложенных оператора if, а условие дизъюнкции разделяется на последовательность операторов if с повторяющимися блоками.

Разделение условия на несколько операторов if может быть полезно для улучшения читаемости, когда условие становится слишком большим, или для упрощения редактирования при необходимости изменения условий.
