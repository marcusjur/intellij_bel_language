Удалить квалификатор идентификатора. Это намерение выполняет действие, противоположное квалифицированному идентификатору.

Квалифицированное имя включает как минимум один квалификатор для уточнения контекста. Это может быть полезно, например, когда у вас есть две таблицы с похожими именами, но находящиеся в разных схемах.
