Можно добавить аннотацию @Range или изменить её значение. Аннотация @Range может быть добавлена к параметрам метода, полям или возвращаемым значениям метода и используется инструментами статического анализа (например, проверками) для валидации кода.

Диапазон можно указать для значений типа long, int, short, char или byte. Диапазон определяется указанием нижней границы from (включительно) и верхней границы to (включительно).
