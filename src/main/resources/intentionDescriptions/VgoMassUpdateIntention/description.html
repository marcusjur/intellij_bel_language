Обновить несколько зависимостей в выбранном файле go.mod.

Доступны следующие варианты:

  • Обновить все зависимости до последней патч-версии.
  • Обновить все зависимости до последней версии.
  • Обновить прямые зависимости до последней патч-версии.
  • Обновить прямые зависимости до последней версии.

Прямые зависимости — это модули, которые явно импортованы в исходных файлах проекта.

Подробную информацию о командах обновления зависимостей см. в разделе go-get на go.dev.
