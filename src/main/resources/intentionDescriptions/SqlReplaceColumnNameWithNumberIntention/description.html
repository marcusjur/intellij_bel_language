Замена позиционных ссылок на имена столбцов.

Некоторые СУБД позволяют ссылаться на столбцы в ORDER BY и GROUP BY по их порядковым номерам. Все диалекты, кроме Cassandra, ClickHouse, Hive, Spark и HSQLDB, поддерживают ORDER BY. MySQL, Maria, PostgreSQL (и его поддиалекты), Exasol, SQLite и Snowflake поддерживают GROUP BY.

Пример (PostgreSQL):

CREATE TABLE tab(a INT, b INT);
SELECT * FROM tab ORDER BY 1;

Вы можете использовать это намерение для замены 1 в SELECT-запросе на имя столбца. После применения этого намерения запрос будет выглядеть следующим образом.

SELECT * FROM tab ORDER BY a;
