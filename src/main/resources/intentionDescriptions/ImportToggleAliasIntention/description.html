Позволяет добавлять или удалять псевдонимы из элементов import с соответствующим переименованием ссылок. Применимо как к конструкциям import ... as ..., так и к from ... import ... as.
Использование псевдонимов в import-выражениях особенно полезно, когда вы часто ссылаетесь на модули с длинными именами.
