Развернуть вызов in_array($needle, $haystack, $strict) в соответствующую дизъюнкцию.

Если сгенерированная дизъюнкция является частью логического выражения, она будет заключена в круглые скобки. Если $haystack содержит элемент с побочными эффектами, этот элемент будет помещен в первое выражение равенства. Если $haystack содержит несколько элементов с побочными эффектами, использование этого намерения не рекомендуется.
