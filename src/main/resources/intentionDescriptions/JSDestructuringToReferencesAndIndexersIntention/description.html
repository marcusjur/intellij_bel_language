Преобразование шаблона деструктуризации в последовательность ссылок или индексных выражений на стороне доступа. В следующих случаях, которые могут вызвать побочные эффекты, для каждого элемента деструктуризации будет введена новая переменная.
  • Шаблон имеет значение по умолчанию
  • Есть использование для записи
  • Правосторонние выражения имеют побочные эффекты
  • Правостороннее выражение не является ссылкой
Эта операция противоположна "распространению на объявление деструктуризации".
