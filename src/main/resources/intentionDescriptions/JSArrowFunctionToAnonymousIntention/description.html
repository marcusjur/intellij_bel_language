Преобразовать стрелочную функцию в анонимное функциональное выражение.

Обратите внимание, что это преобразование не рекомендуется для стрелочных функций, использующих this, поскольку значение this может измениться для анонимной функции.
Подробнее см. https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Functions/Arrow_functions#Lexical_this.
