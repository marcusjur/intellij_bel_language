Это намерение заменит относительный from импорт на абсолютный импорт.
Обратите внимание, что относительные импорты работают только в пределах текущего корневого каталога исходников. Вы не можете использовать относительный импорт для пакетов из другого корневого каталога исходников. Поэтому, если расположение импорта может измениться, рекомендуется использовать абсолютный импорт.
