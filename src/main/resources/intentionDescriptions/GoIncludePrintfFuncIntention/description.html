Отметить функцию как функцию форматования, подобную fmt.Printf:
  • Будет проверять форматную строку на наличие неправильных спецификаторов формата, таких как %s, %d, %v и других
  • Будет подсвечивать спецификаторы формата в форматной строке
  • При включенном сворачивании кода будет доступна интерполяция (сворачивание) итоговой форматной строки
