Переместить класс ECMAScript или класс, интерфейс или перечисление TypeScript в новый или существующий файл, где имя файла соответствует имени типа.

Соответствие имени файла может быть специфичным для определённого фреймворка. Например, для компонентов Angular будет применяться соответствующее руководство по стилю.
