Заменить ссылки на позиции столбцов их именами.

Некоторые СУБД позволяют ссылаться на столбцы в ORDER BY и GROUP BY по их порядковым номерам. Все диалекты, кроме Cassandra, ClickHouse, Hive, Spark и HSQLDB, поддерживают ORDER BY. MySQL, Maria, PostgreSQL (и его поддиалекты), Exasol, SQLite и Snowflake поддерживают GROUP BY.
