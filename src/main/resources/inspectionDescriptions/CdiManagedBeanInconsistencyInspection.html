Сообщает о неправильных правилах определения Bean.

Класс Java верхнего уровня может быть простым Bean, если:

  • Он не является параметризованным типом.
  • Он не является non-static внутренним классом.
  • Это конкретный класс или класс, аннотированный с помощью @Decorator.
  • Имеет подходящий конструктор: класс имеет конструктор без параметров или конструктор, объявленный с аннотацией @Initializer.
  • Он не реализует ни один из следующих интерфейсов: javax.servlet.Servlet, javax.servlet.Filter, javax.servlet.ServletContextListener, javax.servlet.http.HttpSessionListener, javax.servlet.ServletRequestListener, javax.ejb.EnterpriseBean.
  • Он не расширяет javax.faces.component.UIComponent.
