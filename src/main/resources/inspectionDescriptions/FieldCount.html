Сообщает о классах, в которых количество полей превышает указанное максимальное значение.

Классы с большим количеством полей, как правило, пытаются выполнять слишком много функций. Рекомендуется разделить такой класс на несколько меньших классов.

Настройка проверки:

  • Используйте поле Лимит количества полей, чтобы указать максимально допустимое количество полей в классе.
  • Используйте опцию Включать константные поля в подсчет, чтобы указать, следует ли учитывать константные поля.
  • По умолчанию только неизменяемые объекты static final считаются константами. Используйте опцию Считать поля 'static final' константами, чтобы учитывать любые поля static final как константы.
  • Используйте опцию Включать константы перечислений в подсчет, чтобы указать, следует ли учитывать константы enum в классах типа enum.
