Сообщает о параметрах метода подкласса, которые имеют то же имя, что и параметры super-метода, но принадлежат к разным пакетам. В таких случаях метод подкласса не может переопределить super-метод.

Пример:


public class A {
 public void method(Object o) {}
}
public class B extends A {
 public void method(Object o) {} // предупреждение о типе параметра
 class Object {}
}

После применения быстрого исправления:


public class A {
 public void method(Object o) {}
}
public class B extends A {
 public void method(java.lang.Object o) {} // новый тип параметра
 class Object {}
}
