Сообщает о ненужных локальных переменных, которые не улучшают понимание метода, включая:
  • Локальные переменные, которые сразу возвращаются.
  • Локальные переменные, которые сразу присваиваются другой переменной и больше не используются.
  • Локальные переменные, которые всегда имеют то же значение, что и другая локальная переменная или параметр.

Пример:


  boolean yes() {
    boolean b = true;
    return b;
  }

После применения быстрого исправления:


  boolean yes() {
    return true;
  }

Настройка проверки:

  • Используйте опцию Игнорировать переменные с немедленным возвратом или выбросом, чтобы игнорировать переменные, которые сразу возвращаются или выбрасываются. Некоторые стили кодования рекомендуют использовать такие переменные для ясности и удобства отладки.
  • Используйте опцию Игнорировать переменные с аннотациями, чтобы игнорировать аннотированные переменные.
