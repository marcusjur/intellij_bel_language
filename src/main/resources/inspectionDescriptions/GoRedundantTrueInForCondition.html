Сообщает о избыточном литерале true в условии цикла for.

В Go цикл for имеет форму с неявным условием. Это означает, что вы можете создать бесконечный цикл, написав for { }. Поэтому в случае с условием true (for true { }), true считается избыточным и может быть опущено.

Подробную информацию о циклах for в Go смотрите в разделе For на go.dev.

Пример:

for true {
  println("ping")
}
