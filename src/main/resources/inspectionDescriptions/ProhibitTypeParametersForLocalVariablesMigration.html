Сообщать о локальных переменных с параметрами типа.

Параметры типа для локальных переменных не имеют смысла, так как они не могут быть специализованы.

Пример:


  fun main() {
      val <T> x = ""
  }

После применения быстрого исправления:


  fun main() {
      val x = ""
  }

Эта проверка выполняется только если уровень языка Kotlin для проекта или модуля 1.4 или выше.
