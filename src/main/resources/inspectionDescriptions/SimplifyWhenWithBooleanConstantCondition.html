Сообщает о выражениях when с ветвями, содержащими константы true или false.

Упростить "when" доступно как быстрое исправление для автоматического исправления кода.

Пример:


  fun redundant() {
      when { // <== Избыточно, быстрое исправление упростит выражение when до "println("true")"
          true -> println("true")
          else -> println("false")
      }
  }
