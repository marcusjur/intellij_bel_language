Сообщает о public методах, принимающих параметры типа boolean.

Если public метод (часть API) не является сеттером, то добавление параметра типа boolean почти всегда является плохой практикой. При чтении кода, использующего такой метод, сложно понять значение параметра boolean без просмотра исходного кода или документации.

Эта проблема также известна как булева ловушка. Параметры типа boolean обычно можно заменить на enum.

Пример:


  // Предупреждение: при просмотре вызовов
  // этого метода сложно понять
  // значение булевых параметров
  public boolean setPermission(File f,
                               int access,
                               boolean enable,
                               boolean ownerOnly) {
    // ...
  }

Используйте опцию сообщать только о методах с несколькими булевыми параметрами, чтобы получать предупреждения только когда метод содержит несколько параметров типа boolean.
