Обнаружена некорректная перегрузка метода (...). Перегрузка метода действительна только тогда, когда существует ровно одно другое неперегруженное определение этого метода.

Некорректную перегрузку метода можно удалить с помощью быстрого исправления.

Пример:

class Example
  # Не найдено неперегруженное объявление метода 'call' для класса 'Example'
  def call: -> untyped | ...
end

После применения быстрого исправления:

class Example
  def call: -> untyped
end
