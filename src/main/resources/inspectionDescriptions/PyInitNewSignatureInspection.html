Сообщение о несовместимых сигнатурах методов __new__ и __init__.

Пример:


class MyClass(object):
    def __new__(cls, arg1):
        return super().__new__(cls)
    def __init__(self):
        pass

Невозможно создать экземпляр MyClass, если методы __new__ и __init__ имеют разные аргументы.

ИСР предлагает исправить это с помощью рефакторинга "Изменить сигнатуру".
