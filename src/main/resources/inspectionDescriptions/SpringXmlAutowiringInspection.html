Сообщение о некорректной автоматической привязке (autowiring) Spring Bean в контексте XML-приложения.

Пример:


public interface FooInterface {...}
@Component public class FooBean implements FooInterface {...}
@Component public class OtherBean implements FooInterface {...}
@Component
public class MyComponent {
	 public void setFooInterface(FooInterface foo) {...}
}


<beans>
	<bean class="beans.OtherBean" id="otherBean"/>
	<bean class="beans.FooBean" id="fooBean"/>
	<bean autowire="byType" class="beans.MyComponent" id="foo"/> <!- Сообщение "Невозможно выполнить автоматическую привязку". Существует более одного bean-компонента типа 'FooInterface'. -->
                                                                   <!--          Beans: otherBean,fooBean. Свойства: 'fooInterface' " -->
</beans>
