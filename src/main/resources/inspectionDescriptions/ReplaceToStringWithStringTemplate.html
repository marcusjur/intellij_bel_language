Отчёт может быть заменен вызовом функции toString с помощью строкового шаблона.

Использование строковых шаблонов делает код более лаконичным.

Это быстрое исправление заменит toString на строковой шаблон.

Пример:


  fun test(): String {
      val x = 1
      return x.toString()
  }

После применения быстрого исправления:


  fun test(): String {
      val x = 1
      return "$x"
  }
