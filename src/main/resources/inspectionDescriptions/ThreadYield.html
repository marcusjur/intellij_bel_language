Сообщает о вызовах Thread.yield().

Поведение yield() недетерминировано и зависит от платформы, поэтому этот метод редко бывает полезен. Его использование должно сопровождаться тщательным анализом и тестованием производительности, чтобы убедиться в достижении желаемого эффекта.

Пример:


  public static void main(String[] args) {
    Runnable r = () -> {
      for (int i = 0; i < 10; i++) {
        System.out.println(i);
        Thread.yield();
      }
    };
    new Thread(r).start();
    new Thread(r).start();
  }
