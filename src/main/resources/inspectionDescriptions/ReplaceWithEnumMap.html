Сообщается, что функцию hashMapOf или конструктор HashMap можно заменить конструктором EnumMap.

Использование конструктора EnumMap делает код более лаконичным.

Это быстрое исправление заменит вызов функции на вызов конструктора EnumMap.

Пример:


  enum class E {
      A, B
  }
  fun getMap(): Map<E, String> = hashMapOf()

После применения быстрого исправления:


  enum class E {
      A, B
  }
  fun getMap(): Map<E, String> = EnumMap(E::class.java)
