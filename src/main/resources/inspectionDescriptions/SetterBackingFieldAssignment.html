Сообщать о сеттерах свойств, которые не обновляют поддерживающее поле.

Быстрое исправление добавит присваивание в поддерживающее поле.

Пример:


  class Test {
      var foo: Int = 1
          set(value) {
          }
  }

После применения быстрого исправления:


  class Test {
      var foo: Int = 1
          set(value) {
              field = value
          }
  }
