Сообщает о присваивании или модификации параметров lambda-выражения. Хотя иногда это делается намеренно, такая структура может вызывать путаницу и обычно является результатом опечатки или использования неправильной переменной.

Быстрое исправление добавит объявление новой переменной.

Пример:


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

После применения быстрого исправления:


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

Используйте опцию Игнорировать, если присваивание является преобразованием исходного параметра, чтобы игнорировать присваивания, которые изменяют значение параметра на основе его предыдущего значения.
