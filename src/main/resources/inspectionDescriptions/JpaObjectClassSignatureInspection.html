Сообщать об ошибках определения классов персистентных объектов в следующих случаях:
  • Класс не является классом верхнего уровня
  • Класс объявлен как final
  • Класс аннотирован несколько раз
  • Класс определен одновременно как персистентная сущность и слушатель сущности

Пример:



  @Entity
  @EntityListeners(value = JavaEntity.class) // Ошибка: аннотации @Entity и @EntityListener указывают на один и тот же класс
  public final class JavaEntity { // Ошибка: персистентная сущность не должна быть final
    ...
  }
