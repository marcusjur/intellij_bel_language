Сообщает об использовании цикла for..in для массивов. Рекомендуется заменить его циклом for..of.
for..of цикл (представленный в ECMAScript 6) выполняет итерацию по объектам типа iterable. Для массивов эта конструкция предпочтительнее for..in, так как она обрабатывает только значения массива, а не свойства объекта массива.
