Сообщение о цепочке операторов сравнения на равенство.

Такие сравнения могут вводить в заблуждение: a == b == c интерпретируется как (a == b) == c, хотя возможно предполагалось a == b && a == c.

Пример:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return a == b == c;
  }

Вы можете использовать круглые скобки, чтобы сделать сравнение более понятным:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return (a == b) == c;
  }
