Сообщать о вызовах System.exit(), Runtime.exit() и Runtime.halt().

Вызов System.exit() или Runtime.exit() активирует процедуры завершения работы и останавливает текущую Java виртуальную машину. Вызов Runtime.halt() принудительно останавливает JVM без запуска процедур завершения работы. Все эти методы следует использовать с особой осторожностью. Вызовы этих методов делают код непереносимым для большинства серверов приложений.

Используйте эту опцию, чтобы игнорировать вызовы в главном методе.
