Сообщает об утилитных классах, которые можно преобразовать в перечисления.

Некоторые правила оформления кода требуют реализации утилитных классов в виде перечислений, чтобы избежать проблем с покрытием кода в private конструкторах.

Пример:


  class StringUtils {
    public static final String EMPTY = "";
  }

После применения быстрого исправления:


  enum StringUtils {
    ;
    public static final String EMPTY = "";
  }
