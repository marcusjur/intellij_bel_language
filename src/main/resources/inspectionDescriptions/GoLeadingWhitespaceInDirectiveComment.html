Обнаружены пробелы перед директивой Go в комментарии.

Директива Go не будет распознана, если между // и директивой есть пробелы.

Это быстрое исправление удалит начальные пробелы перед директивой Go.

Пример:


//  go:embed file.txt
var File string

После применения быстрого исправления:


//go:embed file.txt
var File string
