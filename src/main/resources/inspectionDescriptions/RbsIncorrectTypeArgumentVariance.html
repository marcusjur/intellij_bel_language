Сообщает об использовании типовых аргументов с неправильной вариантностью в выражениях наследования (в предложениях суперкласса, собственного типа или включения). Типовые аргументы, передаваемые в выражения наследования, должны соответствовать вариантности объявленных типовых переменных.

Пример:

module Enumerable[unchecked out A]
end
class Example[in A]
  # Контравариантная типовая переменная 'A' класса 'Example' используется как ковариантный аргумент
  include Enumerable[A]
end
