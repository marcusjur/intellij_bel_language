Сообщает о классах с модификатором abstract, которые не содержат abstract методов. В большинстве случаев классы с модификатором abstract без абстрактных методов не имеют смысла, и модификатор abstract может быть удален из этого класса. Если класс объявлен как abstract для предотвращения создания экземпляров, лучше использовать private конструктор.

Пример:


  abstract class Example {
    public String getName() {
      return "IntelliJ ИСРA";
    }
  }

Используйте эту опцию, чтобы исключить служебные классы.
