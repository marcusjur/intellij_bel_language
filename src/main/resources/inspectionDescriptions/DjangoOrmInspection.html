Сообщает о методах, которые могут быть недоступны для вызова из-за определённых ограничений ORM.

Пример:


class Choice:
    question = ForeignKey(Question, null=False)
choice_set = Question.objects.get(id=1).choice_set.remove()

Если внешний ключ имеет параметр null=False, функция remove не может быть вызвана.
