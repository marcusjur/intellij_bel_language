Сообщает о классах, не реализующих Serializable, но содержащих поле serialVersionUID. Наличие поля serialVersionUID в таком контексте обычно указывает на ошибку, так как это поле будет проигнорировано, а класс не будет сериализован.

Пример:


  public class IWantToSerializeThis {
    private static final long serialVersionUID = 2669293150219020249L;
  }
