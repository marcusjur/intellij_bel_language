Сообщает о сравнениях с вызовом String.indexOf(), которые могут быть заменены вызовом метода String.contains().

Пример:


  boolean b = "abcd".indexOf('e') >= 0;

После применения быстрого исправления:


  boolean b = "abcd".contains('e');

Эта проверка срабатывает только если уровень языка проекта или модуля установлен на 5 или выше.
