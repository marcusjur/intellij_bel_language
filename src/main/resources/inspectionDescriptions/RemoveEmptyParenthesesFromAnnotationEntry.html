Обнаружены избыточные круглые скобки в элементах аннотаций.

Используйте быстрое исправление "Удалить ненужные скобки" для очистки кода.

Пример:


  annotation class MyAnnotationA
  annotation class MyAnnotationB(val x: Int)
  annotation class MyAnnotationC(val x: Int = 10) // имеет значение по умолчанию
  @MyAnnotationA() // <== избыточные скобки
  fun testA() {
  }
  @MyAnnotationB() // <== отсутствуют аргументы, скобки обязательны
  fun testB() {
  }
  @MyAnnotationC() // <== избыточные скобки
  fun testC() {
  }
