Сообщает об использовании модулей в предложениях с суперклассами. Предложения с суперклассами могут ссылаться только на классы.

Недопустимые предложения с суперклассами можно удалить с помощью быстрого исправления.

Пример:

# 'Kernel' модуль используется как суперкласс
class Example < Kernel
end

После применения быстрого исправления:

class Example
end
