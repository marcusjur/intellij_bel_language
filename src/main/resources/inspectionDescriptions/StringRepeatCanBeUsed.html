Цикл может быть заменен единственным методом String.repeat() (доступен с Java 11).

Пример:


  void append(StringBuilder sb, int count, Object obj) {
    for (int i = 0; i < count; i++) {
      sb.append(obj);
    }
  }

После применения быстрого исправления:


  void append(StringBuilder sb, int count, Object obj) {
    sb.append(String.valueOf(obj).repeat(Math.max(0, count)));
  }

По умолчанию, если нельзя статически доказать, что count неотрицательный, проверка может обернуть count в Math.max(0, count). Это делается для предотвращения возможных семантических изменений, так как String.repeat() не принимает отрицательные числа. При необходимости это поведение можно отключить с помощью опции Добавить Math.max(0, count) во избежание возможных семантических изменений.

Аналогично, если неизвестно, может ли повторяемая строка быть null, её можно обернуть в String.valueOf, чтобы предотвратить возможное исключение NullPointerException.

Эта проверка срабатывает только если уровень языка проекта или модуля установлен на 11 или выше.
