Сообщает о выделении объектов или массивов внутри циклов. Хотя это не обязательно является проблемой, выделение объектов в циклах - хорошее место для поиска утечек памяти и проблем с производительностью.

Проверка обнаруживает следующие структуры:

  • Явное выделение памяти через оператор new
  • Известные методы, возвращающие новые объекты
  • Ссылки на методы с привязкой к экземпляру
  • Лямбды, захватывающие переменные или ссылку на this

Пример:


  // 显式分配
  for (Status status : Status.values()) {
    declarationsMap.put(status, new ArrayList<>());
  }
  // Lambda 捕获变量
  String message = "Engine running.";
  for (Engine engine : engines) {
      if (!isRunning(engine)) {
          logger.warn(() -> {
              return String.format(message);
          });
      }
  }
  // 实例绑定方法引用
  for(Node node : nodes) {
    descriptor = node.getDescription();
    descriptor.ifPresent(dynamicTestExecutor::execute);
  }
