Сообщает о вызовах метода toString(), использующих реализацию по умолчанию из java.lang.Object.

Использование реализации по умолчанию редко является преднамеренным и может указывать на ошибку.

Эта проверка игнорирует вызовы toString() для объектов, имеющих тип java.lang.Object, интерфейс или абстрактный класс.

Пример:


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // предупреждение
          /* ... */
      }
      void foo2(Object obj) {
          String s = obj.toString(); // здесь предупреждения нет
          /* ... */
      }
  }
