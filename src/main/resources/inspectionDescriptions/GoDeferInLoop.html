Обнаружение операторов defer внутри циклов.

Использование defer внутри цикла может привести к утечке ресурсов или непредсказуемому порядку выполнения операторов.

Пример:

func main() {
 for {
  field, err := db.Query("SELECT 1")
  if err != nil {
   // ...
  }
  defer field.Close()
  // ...
 }
}

Вызов defer row.Close() внутри цикла не будет выполнен до завершения функции, а не в конце каждой итерации for цикла. Такая реализация может привести к переполнению стека функции и другим проблемам.
