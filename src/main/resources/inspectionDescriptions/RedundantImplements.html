Интерфейс в списке implements класса или в списке extends интерфейса, который уже реализован суперклассом или расширен суперинтерфейсом. Такое объявление является избыточным и может быть безопасно удалено.

Пример:


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

После применения быстрого исправления:


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

Используйте эту опцию, чтобы исключить Serializable или Externalizable из проверки в списках extends или implements.
