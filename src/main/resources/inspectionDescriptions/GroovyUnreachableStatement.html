Сообщение о недостижимых операторах. Это может произойти, если оператор находится после бесконечного цикла или после операторов return, break или continue.

Пример: 


  void foo (int n) {
      if (n < 1) {
          return
          print('This statement is unreachable')
      }
      while (true){
          print ('Hello, world!')
      }
      print('This statement is unreachable too')
  }
