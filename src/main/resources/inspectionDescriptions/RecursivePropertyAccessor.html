Сообщает о рекурсивных вызовах аксессоров свойств, которые могут завершиться с ошибкой StackOverflowError. Такие вызовы часто путают с доступом к поддерживающему полю.

Пример:


  var counter: Int = 0
      set(value) {
          counter = if (value < 0) 0 else value
      }

После применения быстрого исправления:


  var counter: Int = 0
      set(value) {
          field = if (value < 0) 0 else value
      }
