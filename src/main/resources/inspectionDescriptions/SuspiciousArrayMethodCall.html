Сообщать о вызовах негенерических операций с массивами, таких как Arrays.fill(), где типы аргументов не совпадают. Такие вызовы не выполняют никакой полезной работы и, вероятно, являются ошибочными.

Пример:


  int foo(String[] strings) {
    return Arrays.binarySearch(strings, 1);
  }
