Сообщает о несоответствии имен параметров, указанных в блоках комментариев при вызове методов Java.

Пример:


  // Java
  public class JavaService {
      public void invoke(String command) {}
  }


  // Kotlin
  fun main() {
      JavaService().invoke(/* name = */ "fix")
  }

Быстрое исправление корректирует имя параметра в блоке комментария:


  fun main() {
      JavaService().invoke(/* command = */ "fix")
  }
