Сообщает об атрибуте autowire в элементе <bean> и предлагает использовать явное внедрение свойств Bean, где это возможно.

Пример:


  public class MyComponent {
    public void setOtherBean(OtherBean bean){...}
  }


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean autowire="byType" class="beans.MyComponent"/>
    <!-- "Ненужная автоматическая внедряемая зависимость" -->
    <!-- Рекомендуется "сделать автоматически внедряемую зависимость явной" -->
  </beans>

После применения быстрого исправления:


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean class="beans.MyComponent" id="foo">
      <property name="otherBean" ref="otherBean"/>
    </bean>
  </beans>
