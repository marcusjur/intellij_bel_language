Обнаружено использование одинарных кавычек в строках JSON-массива.

В формате JSON-массива необходимо использовать двойные кавычки (") вместо одинарных ('). В противном случае сборка Docker завершится с ошибкой.

Пример:


  # Все следующие команды завершатся ошибкой
  RUN ['/bin/bash', '-c', 'echo hello']
  ADD ['binaryA.jar', 'binary2.jar', 'destination/']
  COPY ['binaryA.jar', 'binary2.jar', 'destination/']

После применения исправления:


  RUN ["/bin/bash", "-c", "echo hello"]
  ADD ["binaryA.jar", "binary2.jar", "destination/"]
  COPY ["binaryA.jar", "binary2.jar", "destination/"]
