Сообщает о случаях выполнения SELECT из функции или выполнения DBLINK без указания типов в псевдонимах (например, AS t1 (s VARCHAR) ).

Это требование не применяется к скалярным функциям.

Пример (PostgreSQL):

CREATE FUNCTION produce_a_table() RETURNS RECORD AS $$
SELECT 1;
$$ LANGUAGE sql;
SELECT * FROM produce_a_table() AS s (c1 INT);
SELECT * FROM produce_a_table() AS s (c1);
SELECT * FROM DBLINK('dbname=mydb', 'SELECT proname, prosrc FROM pg_proc') AS t1;

AS s (c1 INT) содержит типизованный псевдоним, в то время как AS s (c1) и AS t1 - нет. В этом случае будут подсвечены второй вызов produce_a_table() и вызов DBLINK().
