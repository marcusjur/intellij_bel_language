Обнаружены вызовы wait(), notify() и notifyAll(), которые не находятся внутри соответствующего блока synchronized или синхронизованного метода.

Вызов этих методов без удержания блокировки объекта приведет к исключению IllegalMonitorStateException. Такая конструкция не обязательно является ошибкой, поскольку необходимая блокировка могла быть получена перед вызовом включающего метода, но всё же следует проверить это внимательно.

Пример:


  class Sync {
    private final Object lock = new Object();
    void test() throws InterruptedException {
      synchronized (this) {
        lock.wait(); // 'lock.wait()' не синхронизован по объекту 'lock'
      }
    }
  }
