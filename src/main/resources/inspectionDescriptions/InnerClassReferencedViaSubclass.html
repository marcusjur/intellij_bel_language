Сообщает о доступе к внутренним и вложенным классам, где обращение происходит через подкласс объявляющего класса, а не через сам объявляющий класс.

Java допускает такую квалификацию, но подобный доступ может указывать на неочевидную путаницу между наследованием и переопределением.

Пример:


  class Super {
    static class Inner {}
  }
  class Sub extends Super {
    void test() {
      Sub.Inner s = new Sub.Inner(); // Класс 'Inner' объявлен в классе 'Super', но ссылка идет через класс 'Sub'
    }
  }

После применения быстрого исправления:


  class Super {
    static class Inner {}
  }
  class Sub extends Super {
    void test() {
      Super.Inner s = new Super.Inner();
    }
  }
