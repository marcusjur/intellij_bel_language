Сообщение о расширении, которое инстанцируется платформой IntelliJ, но объявлено как объект Kotlin.

Жизненным циклом расширения управляет платформа IntelliJ. Использование объекта Kotlin для регистрации расширения может привести к созданию ненужных экземпляров расширения и сделать невозможным выгрузку плагина.

Пример

Регистрация расширения:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Реализация расширения:


// Неправильно:
object MyAnnotator : Annotator {
  ...
}
// Правильно:
class MyAnnotator : Annotator {
  ...
}
