Сообщает об операциях инкремента или декремента, вложенных в другие выражения. Такие выражения могут вводить в заблуждение и нарушают общий принцип проектирования, согласно которому каждая конструкция должна выполнять только одну функцию.

Быстрое исправление выделяет операцию инкремента или декремента в отдельный оператор.

Пример:


  int i = 10;
  while (i-- > 0) {
    System.out.println(i);
  }

После применения быстрого исправления:


  int i = 10;
  while (i > 0) {
    i--;
    System.out.println(i);
  }
  i--;
