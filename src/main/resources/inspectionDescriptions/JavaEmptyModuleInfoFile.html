Сообщает о пустом файле module-info.java, указывая на неразрешённые зависимости модуля. Необходимые операторы requires добавляются автоматически путем проверки импортов. Чтобы отключить это предупреждение, вы можете добавить любой комментарий в тело оператора модуля, как показано ниже: 

module module.name {
    // Нет зависимостей
}
Быстрое исправление: Заполнение зависимостей модуля добавляет отсутствующие requires на основе импортов в исходном коде. Новое в версии 2024.1
