Сообщает о синтаксических ошибках в коде Spring Expression Language (SpEL), таких как неразрешённые ссылки и недопустимые позиции.

Пример:


@Controller
public class MyController {
    @Value("#{T(java.lang.Math).unknown()}") // Невозможно разрешить свойство или метод 'unknown'
    public void setValue(String s) {}
}
