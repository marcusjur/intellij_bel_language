Сообщает о недопустимых аргументах, передаваемых методам с параметрами, аннотированными как @PropertyKey.

Эти аргументы должны быть действительными ключами свойств в соответствующем файле свойств. Кроме того, проверка также верифицирует, что аргумент resourceBundle аннотации @PropertyKey указывает на существующий пакет ресурсов.

Используйте быстрое исправление для создания нового свойства или выбора существующего.

Пример:


  @PropertyKey(resourceBundle = "myBundle") String value = "invalid.key";
