Сообщает о переменных, методах и классах, в названиях которых содержится знак доллара ($). Хотя такие имена являются допустимыми в Java, настоятельно не рекомендуется использовать их вне сгенерированного Java-кода.

Пример:


  class SalaryIn${}

Быстрое исправление для переименования доступно только в редакторе.
