Отчёт показывает случаи, когда преобразование строки в версию с интерполяционным префиксом с несколькими знаками доллара может быть полезным.

Интерполяционные префиксы делают строки более понятными, поскольку строки с префиксами могут использовать символы доллара напрямую, без дополнительного экранирования.

Пример:


val jsonSchema: String = """
{
  "${'$'}id": "https://example.com/product.schema.json",
}"""
Быстрое исправление преобразует все безопасно заменяемые экранированные символы доллара в обычные символы доллара: 

val jsonSchema: String = $$"""
{
  "$id": "https://example.com/product.schema.json",
}"""
