Сообщает о следующих возможных проблемах в файле ejb-jar.xml и других XML-дескрипторах, используемых для развертывания EJB на серверах Java EE:
  • Ссылки на неинстанцируемые классы
  • Ссылки на классы, не расширяющие требуемые классы
  • Ссылки на классы с некорректной областью видимости
  • Пустые теги и значения атрибутов
  • Теги и значения атрибутов, не соответствующие требуемой схеме (например, Java-идентификаторы)
  • Теги без обязательных дочерних тегов или атрибутов
  • Теги, определяющие объекты с повторяющимися именами
