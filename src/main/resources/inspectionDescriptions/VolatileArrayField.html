Сообщает о полях массива, объявленных как volatile. Такое объявление может вводить в заблуждение, поскольку доступ к самому массиву следует правилам для volatile полей, но это не относится к доступу к содержимому массива.

Пример:


  class Data {
    private volatile int[] idx = new int[0];
  }

Если требуется такой волатильный доступ к содержимому массива, лучше использовать классы из пакета java.util.concurrent.atomic: 


  class Data {
    private final AtomicIntegerArray idx = new AtomicIntegerArray(new int[0]);
  }
