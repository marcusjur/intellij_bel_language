Сообщает о вызовах метода java.util.Map.put(). Такие вызовы могут быть заменены более короткой и явной формой доступа по ключу.

Пример:


def map = ["foo": "bar"]
map.put("foo", "baz") // map.put("foo", "baz") можно заменить на map["foo"] = "baz"

После применения быстрого исправления:


def map = ["foo": "bar"]
map["foo"] = "baz"
