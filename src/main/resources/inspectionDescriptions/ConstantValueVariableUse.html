Сообщает обо всех использованиях переменных, которые известны как константы.

Это происходит, когда (читаемое) использование переменной окружено операторами if, while или for, и существует условие ==, сравнивающее эту переменную с константой. В таких случаях использование переменной, известной как константа, может быть заменено фактической константой.

Пример:


    private static void foo(double number) {
        if (number == 1.0) {
            f(number);
        }
    }
    private static void f(double number) {}

После применения быстрого исправления:


    private static void foo(double number) {
        if (number == 1.0) {
            f(1.0);
        }
    }
    private static void f(double number) {}
