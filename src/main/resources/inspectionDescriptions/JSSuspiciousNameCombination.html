Сообщает о присваиваниях или вызовах функций, где имена целевых переменных или параметров функции не соответствуют именам присваиваемых им значений.

Пример:


  var x = 0;
  var y = x;
или 

  var x = 0, y = 0;
  var rc = new Rectangle(y, x, 20, 20);
В данном случае проверка предполагает, что x и y перепутаны местами.

Указывает имена, которые не следует использовать вместе. Ошибка возникает, если имя параметра или целевой переменной содержит слово из одной группы, а имя присваиваемой или передаваемой переменной содержит слово из другой группы.
