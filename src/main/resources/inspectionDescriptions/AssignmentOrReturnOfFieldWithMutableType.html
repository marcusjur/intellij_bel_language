Сообщает о возврате или присваивании из параметров метода в массивы или изменяемые типы (такие как Collection, Date, Map, Calendar и другие)

Поскольку эти типы изменяемы, такая структура может привести к непреднамеренным изменениям состояния объекта извне класса. Хотя такая структура может быть полезна для оптимизации производительности, она по своей природе подвержена ошибкам.

Отслеживаются следующие изменяемые типы:

  • java.util.Date
  • java.util.Calendar
  • java.util.Collection
  • java.util.Map
  • com.google.common.collect.Multimap
  • com.google.common.collect.Table

Быстрое исправление добавляет вызов метода .clone() для поля.

Пример:


  class Log {
    String[] messages;
    ...
    String[] getMessages() {
      return messages; // Предупреждение: возврат поля String[] 'messages'
    }
  }

После применения быстрого исправления:


  class Log {
    String[] messages;
    ...
    String[] getMessages() {
      return messages.clone();
    }
  }

Используйте опцию Игнорировать присваивания и возвраты в private методах, чтобы игнорировать присваивания и возвраты значений в методах с модификатором private.
