Сообщает об операторах import, которые импортуют отдельные классы (вместо всего пакета).

Некоторые стандарты кодования запрещают такие операторы import.

Вы можете настроить IntelliJ ИСРA для обнаружения и исправления таких операторов с помощью команды Оптимизовать импорты. Перейдите в Настройки | Редактор | Стиль кода | Java | Импорты и снимите флажок Использовать импорт отдельных классов. Таким образом, эта проверка особенно полезна для автономного анализа кодовой базы, в которую не планируется вносить изменения.
