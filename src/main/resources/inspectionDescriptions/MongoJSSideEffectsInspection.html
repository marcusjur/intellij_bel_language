Сообщает об операторах, которые могут вызвать побочные эффекты, когда источник данных находится в режиме только для чтения.

Подробную информацию о включении режима только для чтения см. в разделе «Включение режима только для чтения для подключения» в документации ИСР.

Быстрое исправление Отключить режим только для чтения выключает режим только для чтения для соответствующего источника данных.

Пример:


db.my_collection.insertOne()
