Обнаружение операторов package, которые не соответствуют структуре каталогов проекта. Кроме того, сообщается о классах без оператора package, если они расположены не непосредственно в корневом каталоге исходного кода.

Хотя язык Java строго не регламентирует это, рекомендуется хранить классы из пакета com.example.myapp непосредственно в каталоге com/example/myapp под корневым каталогом исходного кода. Несоблюдение этого правила может привести к путанице при чтении кода и некорректной работе некоторых инструментов.
