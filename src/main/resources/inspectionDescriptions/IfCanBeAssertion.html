Проверка обнаруживает операторы if, которые только выбрасывают java.lang.Throwable в ветке then без ветки else. Такие операторы можно преобразовать в более компактные операторы assert.

Проверка также распознает вызовы Guava Preconditions.checkNotNull(). Их можно заменить вызовами Objects.requireNonNull(), которые не требуют дополнительной библиотеки.

Пример:


  if (x == 2) throw new RuntimeException("fail");
  if (y == null) throw new AssertionError();
  Preconditions.checkNotNull(z, "z");

После применения быстрого исправления:


  assert x != 2 : "fail";
  Objects.requireNonNull(y);
  Objects.requireNonNull(z, "z");

По умолчанию эта проверка предлагает быстрое исправление в редакторе без подсветки кода.
