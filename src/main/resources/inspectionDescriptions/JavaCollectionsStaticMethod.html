Сообщает о возможности замены статических методов Java Collections на методы стандартной библиотеки Kotlin.

Пример:


  import java.util.Collections
  fun test() {
      val mutableList = mutableListOf(1, 2)
      Collections.fill(mutableList, 3)
  }

Быстрое исправление заменит вызов статического метода Java Collections на соответствующий вызов метода из стандартной библиотеки Kotlin:


  import java.util.Collections
  fun test() {
      val mutableList = mutableListOf(1, 2)
      mutableList.fill(3)
  }
