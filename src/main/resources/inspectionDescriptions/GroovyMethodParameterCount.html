Сообщает о методах с чрезмерным количеством параметров. Большое количество параметров метода может указывать на необходимость рефакторинга. Эта проверка игнорирует методы, унаследованные из библиотечных классов.

Используйте поле Максимальное количество параметров: чтобы указать максимально допустимое количество параметров метода.
