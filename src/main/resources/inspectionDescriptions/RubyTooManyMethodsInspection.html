Обнаружены классы/модули с чрезмерным количеством методов. Обычно это означает, что они пытаются "делать слишком много" и, возможно, требуют рефакторинга на несколько меньших компонентов.

Используйте поле количество методов для указания максимально допустимого числа методов в классе/модуле.

Используйте флажок игнорировать тестовые классы для исключения тестовых классов из проверки.
