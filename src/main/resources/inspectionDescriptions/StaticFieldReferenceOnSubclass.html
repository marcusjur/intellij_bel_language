Сообщает о доступе к статическим полям, который определяется подклассом объявляющего класса, а не самим объявляющим классом.

Java допускает такое определение, но подобный доступ может указывать на неочевидную путаницу между наследованием и переопределением.

Пример:


  class Parent {
    static int foo = 0;
  }
  class Child extends Parent { }
  void bar() {
    System.out.println(Child.foo);
  }

После применения быстрого исправления результат будет выглядеть так:


  class Parent {
    static int foo = 0;
  }
  class Child extends Parent { }
  void bar() {
    System.out.println(Parent.foo);
  }
