Сообщать о вызовах методов в EL, которые не соответствуют ограничению method-signature в TLD файле.

Пример Java класса:



public class ExampleAction {
  public void goodAction() { ... }
  public void badAction(String str) { ... }
}

Пример JSP файла:



<jsp:useBean id="example" scope="request" type="org.example.ExampleAction"/>
<f:view>
  <h:commandButton action="#{example.goodAction}"/> // Правильно: метод принимает ноль аргументов и возвращает `void`
  <h:commandButton action="#{example.badAction}"/> // Ошибка: метод `void badAction(String)` возвращает тип `java.lang.String` вместо `void`
</f:view>
