Сообщает о ненужных или чрезмерно сложных логических выражениях.

Такие выражения включают: комбинации && с true, || с false, сравнения с логическими литералами или отрицания логических литералов. Такие выражения можно упростить.

Примеры: 


  boolean a = !(x && false);
  boolean b = false || x;
  boolean c = x != true;

После применения быстрого исправления:


  boolean a = true;
  boolean b = x;
  boolean c = !x;

Настройка проверки:

Используйте опцию Игнорировать именованные константы при определении бессмысленных выражений, чтобы не учитывать именованные константы при определении бессмысленности выражений.
