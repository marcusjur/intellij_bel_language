Сообщает о ненужных проверках длины массива после итерации по массиву. Когда длина массива равна нулю, итерация пропускается автоматически, поэтому явная проверка длины не требуется.

Пример:


  void f(String[] array) {
    if (array.length != 0) { // ненужная проверка
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

Используйте быстрое исправление, чтобы развернуть или удалить проверку длины:


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
