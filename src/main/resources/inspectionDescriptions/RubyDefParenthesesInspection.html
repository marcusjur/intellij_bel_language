Сообщает об определениях методов с неправильным использованием круглых скобок. В отличие от определений методов без параметров, определения методов с параметрами должны использовать круглые скобки.

Пример:

# 不良做法
def example_method(); end
def example_method_with_parameters param1, param2; end
Круглые скобки можно удалить или добавить с помощью быстрого исправления. После применения быстрого исправления код будет выглядеть так: 
# 优良做法
def example_method; end
def example_method_with_parameters(param1, param2); end

На основе 'RuboCop'
