Оператор return может быть заменен синтаксисом тела-выражения (однострочный код или when).

Рекомендации по использованию синтаксиса тела-выражения приведены в руководстве по стилю.

Быстрое исправление Преобразовать в тело-выражение доступно для автоматического исправления кода.

Пример:


  fun sign(x: Int): Int {
      return when { // <== можно упростить
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

После применения быстрого исправления:


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
