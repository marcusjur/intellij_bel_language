Сообщает о доступе к полям static для типов, небезопасных в многопоточной среде.

Когда доступ к полю static осуществляется из методов экземпляра или несинхронизованных блоков, несколько потоков могут получить доступ к этому полю. Это может привести к непредсказуемым побочным эффектам, таким как исключения и некорректные результаты.

Пример:


  class Sample {
    private static final SimpleDateFormat df = new SimpleDateFormat("yyyy-MM-dd");
    String method() {
      return df.format("");
    }
  }

Вы можете указать, какие типы следует считать небезопасными в многопоточной среде. Проверка выполняется только для полей с этими конкретными типами или инициализованных этими типами, поскольку могут существовать потокобезопасные подклассы этих типов.
