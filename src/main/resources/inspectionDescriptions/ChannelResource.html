Сообщает о незакрытых ресурсах Channel, включая все экземпляры, созданные вызовом getChannel() для файловых или сокетных ресурсов.

По умолчанию проверка предполагает, что ресурс может быть закрыт любым методом, содержащим в названии 'close' или 'cleanup'.

Пример:


  void send(Socket socket) throws IOException {
    SocketChannel channel = socket.getChannel(); //предупреждение
    channel.write(ByteBuffer.wrap("message".getBytes()));
  }

Проверка настраивается следующими параметрами:

  • Разрешено ли открывать ресурсы Channel внутри блока try. Этот стиль менее предпочтителен, так как он сложнее, чем открытие Channel перед блоком try.
  • Может ли ресурс быть закрыт любым вызовом метода, которому этот ресурс передается в качестве аргумента.
