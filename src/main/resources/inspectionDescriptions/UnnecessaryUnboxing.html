Сообщает о ненужной распаковке (unboxing), то есть явном извлечении примитивного значения из обёрнутого объекта.

Для Java 5 и выше распаковка не требуется и может быть безопасно удалена.

Пример:

  • Integer i = Integer.valueOf(42).intValue(); → Integer i = Integer.valueOf(42);
  • int k = Integer.valueOf(42).intValue(); → int k = Integer.valueOf(42);

    (Сообщается только если не выбрана опция Сообщать только о действительно избыточных операциях распаковки)

При использовании опции Сообщать только о действительно избыточных операциях распаковки будут отмечаться только случаи, когда распакованное значение сразу же неявно или явно упаковывается обратно. В этом случае можно удалить весь процесс распаковки-упаковки. Проверка не сообщает о простых явных распаковках.

Эта проверка выполняется только если уровень языка проекта или модуля установлен на 5 или выше.
