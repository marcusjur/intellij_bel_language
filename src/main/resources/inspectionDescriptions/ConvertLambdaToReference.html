Лямбда-выражение может быть заменено ссылкой на функцию.

Замена лямбда-выражения на ссылку на функцию обычно делает код более лаконичным и понятным.

Пример:


  fun Int.isEven() = this % 2 == 0
  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.isEven() }
  }

После применения быстрого исправления:


  fun Int.isEven() = this % 2 == 0
  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::isEven)
  }
