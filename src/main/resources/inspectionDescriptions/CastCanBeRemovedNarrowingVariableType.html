Сообщается о возможности удаления приведения типов, если тип переменной можно сузить до приводимого типа.

Пример:


  Object x = "  string  ";
  System.out.println(((String)x).trim());

В данном случае изменение типа переменной x на String делает приведение типа избыточным. Предлагаемое быстрое исправление обновит тип переменной и удалит все избыточные приведения типов для этой переменной: 


  String x = "  string  ";
  System.out.println(x.trim());

Новое в версии 2018.2
