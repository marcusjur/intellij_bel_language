Android API уровня 30 - это первая версия Android, которая включает версию SQLite с поддержкой логических литералов "TRUE" и "FALSE". Смотрите документацию по типам данных SQLite и версиям Android SQLite.

На более низких уровнях API следует использовать эквивалентные целочисленные литералы 1 (TRUE) и 0 (FALSE).
