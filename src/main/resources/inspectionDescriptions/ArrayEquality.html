Сообщает об использовании операторов == и != для сравнения массивов. В большинстве случаев, когда требуется сравнить содержимое массивов, следует использовать метод java.util.Arrays.equals().

Рекомендуется использовать быстрое исправление для замены == на java.util.Arrays.equals().

Пример:


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

После применения быстрого исправления:


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
