Сообщает о повторяющихся условиях в выражениях && и ||, а также в ветвях оператора if. Такие повторяющиеся условия иногда являются преднамеренными, но чаще всего возникают из-за невнимательности.

Пример:


  boolean result = digit1 != digit2 || digit1 != digit2;

Чтобы игнорировать условия, которые могут иметь побочные эффекты, используйте опцию Игнорировать условия с побочными эффектами. Отключение этой опции может привести к ложным срабатываниям, например когда один и тот же метод возвращает разные значения при последующих вызовах.

Пример:


  native boolean unknownMethod();
  ...
  if (unknownMethod() || unknownMethod()) {
    System.out.println("Got it");
  }

Поскольку unknownMethod() может иметь побочные эффекты (как в примере), предупреждение будет выдано только если опция Игнорировать условия с побочными эффектами отключена.
