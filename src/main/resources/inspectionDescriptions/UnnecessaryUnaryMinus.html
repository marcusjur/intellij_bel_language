Сообщает о ненужных унарных минусах. Такие выражения могут быть трудны для понимания и могут содержать ошибки.

Пример:

void unaryMinus(int i) {
    int x = - -i;
  }

Предлагаются следующие варианты быстрого исправления:

  • Удалить оператор - перед переменной i:

    void unaryMinus(int i) {
    int x = i;
  }

  • Заменить оператор - на префиксный оператор декремента:

    void unaryMinus(int i) {
    int x = --i;
  }

Другой пример:

void unaryMinus(int i) {
    i += - 8;
  }

После применения быстрого исправления:

void unaryMinus(int i) {
    i -= 8;
  }
