Сообщение о возможности упрощения выражения when путем добавления аргумента.

Пример:


  fun test(obj: Any): String {
      return when {
          obj is String -> "string"
          obj is Int -> "int"
          else -> "unknown"
      }
  }

Быстрое исправление добавит аргумент:


  fun test(obj: Any): String {
      return when (obj) {
          is String -> "string"
          is Int -> "int"
          else -> "unknown"
      }
  }
