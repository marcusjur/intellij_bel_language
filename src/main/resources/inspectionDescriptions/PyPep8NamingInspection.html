Сообщает о нарушениях соглашений об именовании PEP8.

Пример:


class mammalia(object):
    extremities = 4
    def feeds(self):
        print("milk")

В этом коде ИСР предлагает переименовать mammalia в Mammalia. После применения быстрого исправления код становится:


class Mammalia(object):
    extremities = 4
    def feeds(self):
        print("milk")
