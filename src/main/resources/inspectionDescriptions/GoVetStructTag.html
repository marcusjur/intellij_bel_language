Сообщает о структурных тегах, которые не соответствуют соглашениям о структурных тегах Go.

Согласно этим соглашениям, строка тегов представляет собой конкатенацию необязательных пар key:"value", разделенных пробелами. Каждый ключ является непустой строкой, состоящей из непробельных символов, за исключением space (U+0020 ' '), quote (U+0022 '"') и colon (U+003A ':'). Каждое значение заключено в кавычки с использованием символа U+0022 '"' и синтаксиса строковых литералов Go.

Кроме того, эта проверка также контролирует, что поля с тегами экспортованы.

Пример корректных тегов:

type Example struct {
  Field int `json:"field" xml:"demo"`
}
