Сообщает о ситуациях, когда каждая ветка содержит общее подвыражение, поэтому конструкция switch может быть перемещена внутрь выражения и операторов switch. Эта операция сокращает код. Во многих случаях целесообразно выделить полученное выражение switch в отдельную переменную или метод.

Пример:


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

После применения быстрого исправления:


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

Эта проверка применяется только к расширенным конструкциям switch, использующим синтаксис со стрелками.
