Сообщает об использовании функции withIndex() вместо переменной индекса в цикле for при итерации по коллекции.

Быстрое исправление "Удалить индекс из цикла 'for'" поможет очистить код.

Пример:


  fun foo(bar: List<String>) {
     for ((index : Int, value: String) in bar.withIndex()) { // <== 'index' не используется
         println(value)
     }
  }

После применения быстрого исправления:


  fun foo(bar: List<String>) {
      for (value: String in bar) { // <== '.withIndex()' и 'index' удалены
          println(value)
      }
  }
