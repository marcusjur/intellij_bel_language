Сообщает об операторах if с веткой else, где условие отрицательное.

Перестановка веток if и else обычно делает такие операторы более понятными.

Это можно исправить, инвертировав текущий оператор if.

Пример:


    void m(Object o1, Object o2) {
        if (o1 != o2) {
            System.out.println(1);
        }
        else {
            System.out.println(2);
        }
    }

После применения быстрого исправления:


    void m(Object o1, Object o2) {
        if (o1 == o2) {
            System.out.println(2);
        } else {
            System.out.println(1);
        }
    }

Используйте опцию Игнорировать сравнения '!= null', чтобы игнорировать сравнения вида != null.

Используйте опцию Игнорировать сравнения '!= 0', чтобы игнорировать сравнения вида != 0.
