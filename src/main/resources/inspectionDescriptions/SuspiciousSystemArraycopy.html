Сообщать о подозрительных вызовах System.arraycopy().

Вызовы считаются подозрительными в следующих случаях:

  • Источник или приемник не являются массивами
  • Типы источника и приемника различаются
  • Длина копируемого блока больше чем src.length - srcPos
  • Длина копируемого блока больше чем dest.length - destPos
  • Диапазоны всегда пересекаются, когда источник и приемник являются одним и тем же массивом

Пример:


  void foo() {
    int[] src = new int[] { 1, 2, 3, 4 };
    System.arraycopy(src, 0, src, 1, 2); // Предупреждение: копирование в тот же массив с пересекающимися диапазонами
  }
