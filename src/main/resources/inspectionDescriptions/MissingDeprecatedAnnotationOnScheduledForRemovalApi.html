Сообщает об объявлениях, помеченных @ApiStatus.ScheduledForRemoval (без @Deprecated).

Пример:


  @ApiStatus.ScheduledForRemoval(inVersion = "2017.3")
  public void myLegacyMethod() { }

После применения быстрого исправления результат будет выглядеть следующим образом:


  @Deprecated
  @ApiStatus.ScheduledForRemoval(inVersion = "2017.3")
  public void myLegacyMethod() { }
