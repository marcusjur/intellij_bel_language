Сообщение о недопустимых объявлениях методов жизненного цикла транзакций с аннотациями @BeforeTransaction и @AfterTransaction в тестовом классе, помеченном аннотацией @Transactional.

Аннотированные методы не должны иметь параметров и возвращаемого типа.

Пример:


  @ContextConfiguration
  @Transactional
  public class AbstractShowcaseTest {
    @BeforeTransaction // метод должен возвращать 'void'
    public boolean setupData() {...}
    @AfterTransaction // неверное количество параметров
    public void disposeData(boolean a) throws Exception {...}
  }
