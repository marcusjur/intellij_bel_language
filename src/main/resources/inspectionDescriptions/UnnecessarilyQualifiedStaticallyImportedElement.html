Обнаружено использование статически импортованного члена с избыточным указанием имени класса.

Такая квалификация класса является излишней и может быть удалена, поскольку статически импортованный член доступен напрямую по его имени.

Пример:


  import static foo.Test.WIDTH;
  class Bar {
    void bar() {
      System.out.println(Test.WIDTH);
    }
  }

После применения быстрого исправления:


  import static foo.Test.WIDTH;
  class Bar {
    void bar() {
      System.out.println(WIDTH);
    }
  }
