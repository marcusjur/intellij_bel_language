Сообщает о сервисах, зарегистрированных в plugin.xml, которые могут быть преобразованы в облегченные сервисы.

Сервисы, которые не планируется переопределять, не нужно регистрировать в файле plugin.xml. Вместо этого можно использовать аннотацию @com.intellij.openapi.components.Service для класса сервиса. Для сервисов уровня проекта укажите @Service(Service.Level.PROJECT).

Требования:

  • IntelliJ Platform 2019.3 или выше.
  • Класс сервиса должен быть final.
  • Не указан serviceInterface.
  • Если сервис уровня приложения является com.intellij.openapi.components.PersistentStateComponent, роуминг должен быть отключен (roamingType = RoamingType.DISABLED).
  • Не указаны следующие атрибуты: os, client, overrides, id, preload, configurationSchemaKey.

Также сообщает о сервисах, зарегистрированных в plugin.xml, классы которых уже имеют аннотацию @Service.

Для получения дополнительной информации см. раздел Сервисы в документации SDK платформы IntelliJ.
