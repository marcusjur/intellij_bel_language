Обнаружен ресурс/источник данных, который не может быть проанализован на основе известных провайдеров. Провайдер должен быть указан в разделе required_providers или должен происходить из пространства имён hashicorp.

Например, следующий код вызовет ошибку: 


terraform {
  required_version = "1.1.3"
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

Terraform попытается импортовать провайдер hashicorp/abbey для ресурса abbey_demo.

Чтобы исправить эту проблему, необходимо указать точные координаты провайдера: 


terraform {
  required_version = "1.1.3"
  required_providers {
    abbey = {
      source  = "abbeylabs/abbey"
      version = "0.2.9"
    }
  }
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

Для получения дополнительной информации см. документацию Terraform.
