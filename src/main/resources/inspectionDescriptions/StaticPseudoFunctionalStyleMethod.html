Сообщает об использовании псевдофункционального кода Guava, когда доступен Java Stream API.

Хотя Guava Iterable API предоставляет функциональность, похожую на Java Streams API, он немного отличается и может не иметь некоторых возможностей. В частности, примитивные специализации потоков, такие как IntStream, работают эффективнее, чем их обобщенные варианты.

Пример:


List<String> transformedIterable = Iterables.transform(someList, someTransformFunction);//предупреждение: псевдофункциональный стиль кода

После применения быстрого исправления:

List<String> transformedIterable = someList.stream().map(someTransformFunction).collect(Collectors.toList());

Примечание: семантика кода может измениться; например Iterable.transform в Guava создает итерируемый объект с отложенным вычислением, а замена производит немедленное вычисление.
