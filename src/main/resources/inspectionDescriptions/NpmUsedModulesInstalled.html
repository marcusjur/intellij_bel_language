Сообщает о модулях в вызовах require() или операторах import, которые не установлены или не указаны в зависимостях package.json.

Рекомендуется установить этот модуль и/или добавить его в package.json.

Для вызовов require() работает только в файлах, относящихся к Node.js Core JavaScript библиотеке.
