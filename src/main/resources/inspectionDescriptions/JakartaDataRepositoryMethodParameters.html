Сообщает о неправильных типах параметров методов репозитория.

Пример:


  public class Person {
    private int id;
    @Column
    private String lastname;
    @Column
    private int age;
  }
  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // должен быть тип 'String'
    List<Person> deleteByAge(String str);   // должен быть тип 'int'
  }
