Сообщать о путях, начинающихся с тильды (~) в операторах LOAD.

Пример (MySQL):

CREATE TABLE table_name (id int);
LOAD DATA LOCAL INFILE '~/Documents/some_file.txt'
INTO TABLE table_name FIELDS TERMINATED BY ',' LINES TERMINATED BY '\n'
IGNORE 1 LINES;

Используйте полный путь к файлу вместо тильды.
