Сообщает о константах, которые определены, но никогда не используются в коде. 
func main() {
 const i = 100
}

В отличие от неиспользуемых переменных и import, этот код скомпилируется. Неиспользуемые константы могут увеличивать кодовую базу и замедлять компиляцию программы. Чтобы удалить неиспользуемые константы, воспользуйтесь быстрым исправлением Удалить константу.
