Обнаружение избыточных операторов import.

Обычные операторы import не требуются, если классы и пакеты, импортованные в исходный файл, не используются. Это также относится к static полям и методам, которые были импортованы, но не используются в исходном файле.

Пример:


  import java.util.ArrayList;
  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }

После применения быстрого исправления: 


  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }
