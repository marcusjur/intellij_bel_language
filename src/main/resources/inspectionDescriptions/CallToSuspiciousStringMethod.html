Сообщает о вызовах следующих методов:
  • equals()
  • equalsIgnoreCase()
  • compareTo()
  • compareToIgnoreCase() и
  • trim()

(выполняемых над объектами String). Для сравнения интернационализованных строк лучше использовать java.text.Collator. String.trim() удаляет только управляющие символы между 0x00 и 0x20. Метод String.strip(), представленный в Java 11, лучше работает с Unicode и может использоваться как альтернатива.
