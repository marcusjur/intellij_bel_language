Сообщает о вызовах методов и yield, где выведенные типы фактических параметров не соответствуют ожидаемым типам формальных параметров. Ожидаемые типы параметров определяются одним из следующих способов:
  • Сигнатуры типов RBS
  • Сигнатуры типов Sorbet
  • Тег YARD @param

Пример:

# @param [Integer] param
def method(param) end
method([1, 2])
