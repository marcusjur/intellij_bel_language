Сообщает о некорректном доступе к свойствам:
  • Установка значения свойства только для чтения
  • Чтение свойства только для записи
  • Удаление неудаляемого свойства

Пример:


class MyClass:
    @property
    def read_only(self): return None
    def __write_only_setter(self, value): pass
    write_only = property(None, __write_only_setter)
a = MyClass()
a.read_only = 10 # нельзя установить свойство
del a.read_only # нельзя удалить свойство
print(a.write_only) # нельзя прочитать свойство
