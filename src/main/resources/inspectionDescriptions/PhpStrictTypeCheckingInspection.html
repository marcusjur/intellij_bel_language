Сообщает о нарушениях правил строгой типизации.

В строгом режиме принимаются только переменные, которые точно соответствуют объявленному типу. Режим строгой проверки типов может быть включен следующими способами:

  • Для отдельного файла, если указана директива declare(strict_types=1). Эта директива должна быть первым выражением в файле, блочный режим явно не допускается.
  • Для всех файлов, если включена опция проверки для всех файлов.

Для получения дополнительной информации см. Strict typing (php.net).
