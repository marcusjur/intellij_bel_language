Сообщает о параметрах метода, имена которых слишком короткие, слишком длинные или не соответствуют указанному шаблону регулярного выражения.

Пример: если проверка включена с настройками по умолчанию (где имя параметра должно начинаться с маленькой буквы), будет выдано предупреждение для void fooBar(int X).

Настройка проверки:

Используйте поля в разделе Параметры, чтобы указать минимальную длину, максимальную длину и ожидаемый шаблон регулярного выражения для имен параметров метода. Укажите 0, чтобы отключить проверку длины имени.

Регулярное выражение должно быть указано в стандартном формате java.util.regex.
