Сообщает о несоответствиях между полями с аннотацией @UiField и атрибутами ui:field в файле ui.xml.

В файле ui.xml должен быть ровно один тег, связанный с таким полем через атрибут ui:field, поле не должно быть private, а его тип должен соответствовать связанному тегу.

При необходимости быстрое исправление удалит модификатор private и установит правильный тип для поля.
