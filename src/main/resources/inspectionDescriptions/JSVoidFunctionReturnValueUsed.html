Сообщает об использовании возвращаемого значения функций, которые ничего не возвращают. Вызов таких функций всегда генерирует значение undefined, такие присваивания могут указывать на ошибку.

Пример:


let a = console.log('foo');

Следующие случаи игнорируются:
  • Внутри оператора return
  • В некоторых бинарных операциях
  • Для переопределённых функций, не помеченных как void
