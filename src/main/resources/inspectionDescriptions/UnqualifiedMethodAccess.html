Сообщает о вызовах нестатических методов без явного указания this для одного и того же экземпляра.

Пример:


  class Foo {
    void bar() {}
    void foo() {
      bar();
    }
  }

После применения быстрого исправления:


  class Foo {
    void bar() {}
    void foo() {
      this.bar();
    }
  }
