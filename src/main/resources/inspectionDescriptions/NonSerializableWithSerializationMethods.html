Сообщает о классах, не реализующих Serializable, но определяющих методы readObject() или writeObject(). Наличие таких методов в данном контексте обычно указывает на ошибку.

Пример:


  public class SampleClass {
    private void readObject(ObjectInputStream str) {}
    private void writeObject(ObjectOutputStream str) {}
  }
