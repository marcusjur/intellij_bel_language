Обнаружены пустые строки в комментариях Javadoc.

Пустые строки в Javadoc могут указывать на намерение разделить текст на отдельные абзацы. Однако инструмент Javadoc и IntelliJ ИСРA игнорируют их при отображении документационных комментариев.

Быстрое исправление предложит заменить пустые строки тегами абзаца (<p>).

Пример:


  class Main {
    /**
     * Не работает.
     *
     * Совершенно не работает
     */
    void foo() {}
  }

После применения быстрого исправления:


  class Main {
    /**
     * Не работает.
     * <p>
     * Совершенно не работает
     */
    void foo() {}
  }
