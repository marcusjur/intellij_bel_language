Сообщать о методах, которые имеют то же имя, что и суперкласс их класса, поскольку такие имена методов могут вызывать путаницу.

Эта проверка не распространяется на интерфейсы или суперклассы, находящиеся глубоко в иерархии.

Пример: 


  class Parent {}
  class Child extends Parent {
     public Parent Parent() {
       return null;
     }
  }

Быстрое исправление для переименования таких методов доступно только в редакторе.
