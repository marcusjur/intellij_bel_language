Сообщает об использовании ненужного модификатора non-sealed для методов, полей или переменных.

Этот модификатор действителен только для классов, интерфейсов и трейтов.

Пример:


  non-sealed boolean foo() {} // Модификатор не нужен
  non-sealed Object bar // Модификатор не нужен
  // Модификатор обязателен, поэтому не подсвечивается
  non-sealed class A {}
