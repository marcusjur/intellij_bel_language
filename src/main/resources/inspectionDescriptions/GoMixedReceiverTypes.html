Сообщает о структурах, которые используют методы со смешанными типами приёмников (значения и указатели). Такое использование не рекомендуется документацией Go.

Для получения дополнительной информации см. раздел Should I define methods on values or pointers? в FAQ по Go.

Пример:

type S struct{
}
func (s *S) fun() {}
func (s S) fun2() {}
