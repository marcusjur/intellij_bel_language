Сообщение предлагает заменить побитовые операции вызовом методов Long.hashCode() или Double.hashCode(). Проверка обнаруживает конструкцию (int)(x ^ (x >>> 32)), где x является переменной типа long или результатом предыдущего вызова Double.doubleToLongBits(). Такая замена сокращает код и улучшает его читаемость.

Пример:


  int result = (int)(var ^ (var >>> 32));

После применения быстрого исправления:


  int result = Long.hashCode(var);

Эта проверка выполняется только если уровень языка проекта или модуля установлен на 8 или выше.
