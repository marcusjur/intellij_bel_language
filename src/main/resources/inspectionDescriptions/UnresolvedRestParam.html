Сообщает о параметрах @PathParam, которые объявлены в сигнатуре метода, но отсутствуют в URL-пути, или наоборот, объявлены в URL-пути, но отсутствуют в сигнатуре метода.

Пример:


  @Get("/path/{myVariable}/")
  public String handler(@PathParam("name_is_not_equal_to_myVariable") String myVariable) { // ошибка
    return "...";
  }
