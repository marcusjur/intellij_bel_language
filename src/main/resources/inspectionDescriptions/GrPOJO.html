Использование аннотации @groovy.transform.stc.POJO без @groovy.transform.CompileStatic.

Аннотация @POJO изменяет процесс компиляции класса Groovy в байт-код. Она не работает без явно включенной статической компиляции (которая активируется с помощью аннотации @CompileStatic).

Пример:


  @POJO // предупреждение @POJO
  class A {}
