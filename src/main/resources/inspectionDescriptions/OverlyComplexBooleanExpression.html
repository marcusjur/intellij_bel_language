Логическое выражение содержит слишком много операндов. Такие выражения могут быть сложными для понимания и подвержены ошибкам.

Пример:


  cond(x1) && cond(x2) ^ cond(x3) && cond(x4);

Настройка проверки:

  • Используйте поле Максимальное количество операндов, чтобы указать максимально допустимое количество операндов в логическом выражении.
  • Используйте опцию Игнорировать чистые конъюнкции и дизъюнкции, чтобы исключить из проверки логические выражения, использующие только один тип логического оператора.
