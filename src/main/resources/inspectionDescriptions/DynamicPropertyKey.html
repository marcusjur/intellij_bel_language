번들 프로퍼티 키로 사용된 문자열이 아닌 리터럴을 보고합니다.

이러한 리터럴은 유지 관리하기 힘듭니다. 이미 해결된 자연어 문자열을 다시 사용하는 것이 좋습니다.

예:


  val key =
    if (???) "property.key.1"
    else "property.key.2"

  Bundle.message(key)

대신 다음을 사용하세요.


  val string =
    if (???) Bundle.message("property.key.1")
    else Bundle.message("property.key.2")

  string
