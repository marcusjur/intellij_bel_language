Сообщает о вызовах методов, использующих java.nio.file.Files для чтения или записи String в виде байтов Такие вызовы могут быть заменены вызовами методов Files.readString() или Files.writeString(), представленных в Java 11.

Пример:


  String s = "example";
  Files.write(Paths.get("out.txt"), s.getBytes(StandardCharsets.UTF_8), StandardOpenOption.WRITE);
  s = new String(Files.readAllBytes(Paths.get("in.txt")), StandardCharsets.ISO_8859_1);

После применения быстрого исправления:


  String s = "example";
  Files.writeString(Paths.get("out.txt"), s, StandardOpenOption.WRITE);
  s = Files.readString(Paths.get("in.txt"), StandardCharsets.ISO_8859_1);
