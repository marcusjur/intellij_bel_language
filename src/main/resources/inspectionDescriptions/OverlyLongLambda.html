Сообщает о лямбда-выражениях, содержащих количество операторов, превышающее указанный максимум.

Длинные лямбда-выражения могут быть сложными для понимания, поэтому лучше выделить операторы в отдельный метод.

Следующие операторы не учитываются:

  • Пустые операторы (точка с запятой)
  • Блочные операторы
  • Операторы инициализации цикла for, то есть int i = ... в выражении for(int i = ...;...)
  • Операторы обновления цикла for, то есть i += 2 в выражении for(int i = ...;...; i += 2)

Используйте поле Ограничение количества операторов без комментариев, чтобы указать максимальное допустимое количество операторов в лямбда-выражении.
