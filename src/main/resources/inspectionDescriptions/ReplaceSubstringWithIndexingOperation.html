Сообщает, что вызов "abc".substring(0, 1) может быть заменен на "abc"[0].

Получение элемента по индексу делает код более лаконичным.

Это быстрое исправление заменит вызов substring на оператор индексации.

Пример:


  fun foo() {
      "abc".substring(0, 1)
  }

После применения быстрого исправления:


  fun foo() {
      "abc"[0]
  }
