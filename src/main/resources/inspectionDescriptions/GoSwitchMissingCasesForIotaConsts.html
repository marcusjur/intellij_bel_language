Сообщать о необработанных значениях в операторе switch, когда идентификатор iota появляется в объявлении const. Для обработки этих значений рекомендуется использовать блоки case или default.

Ключевое слово iota в Go создает последовательность целых чисел в блоке const. Хотя компилятор Go не требует обработки всех значений iota, отсутствие некоторых из них может указывать на наличие ошибки.

Подробную информацию о iota смотрите в разделе Iota на go.dev.

Пример:

const (
  a = iota
  b
  c = -5
)
func main() {
  v := 5
  switch v {
  case a:
    break;
  case b:
    break;
  }
}

ИСР выполнит проверку, поскольку c не обрабатывается оператором switch. Обратите внимание, что c находится в том же блоке const, даже если он не использует iota.

Варианты быстрого исправления: создать блок 'case' для значения и создать блок 'default'. После применения быстрого исправления создать блок 'default':

const (
  a = iota
  b
  c
)
func main() {
  v := 5
  switch v {
  case a:
    break;
  default:
  }
}
