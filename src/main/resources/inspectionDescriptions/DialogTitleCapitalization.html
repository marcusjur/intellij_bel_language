Сообщает о строковых параметрах метода и возвращаемых значениях с аннотацией @Nls, преобразуя параметры в заглавные буквы в соответствии с правилами написания заголовков, принятыми в большинстве руководств по UI платформ.

Пример:


  void setTitle(@NlsContexts.DialogTitle String title) {}
  setTitle("This is sentence capitalization but should be title");

После применения быстрого исправления:


  setTitle("This Is Sentence Capitalization but Should Be Title");
