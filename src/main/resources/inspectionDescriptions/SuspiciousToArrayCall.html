Сообщение о подозрительных вызовах Collection.toArray().

Следующие типы вызовов считаются подозрительными:

  • Тип массива-аргумента отличается от типа массива, к которому приводится результат.
  • Тип массива-аргумента не соответствует параметру типа в объявлении коллекции.

Пример:


void m1(List list) {
  Number[] ns = (Number[]) list.toArray(new String[0]);
}
void m2(List<Number> list) {
  Number[] ns = list.toArray(new String[0]);
}
