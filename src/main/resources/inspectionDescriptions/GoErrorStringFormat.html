Сообщает о проблемах форматования в строках ошибок.

Пример:

err := fmt.Errorf("Cannot read the file!")
log.Printf("Reading %s: %v", file, err)

Согласно Error Strings на github.com/golang, строки ошибок не должны начинаться с заглавной буквы или заканчиваться знаками препинания, поскольку они могут появляться в других контекстах.

Чтобы исправить форматование, примените быстрое исправление Исправить формат строки ошибки. После применения быстрого исправления строка ошибки будет выглядеть так:

err := fmt.Errorf("cannot read the file")
log.Printf("Reading %s: %v", file, err)
