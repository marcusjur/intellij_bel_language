Сообщать о конструкторах классов с аннотацией @Singleton, если они не объявлены как non-strict.

Пример:


  @Singleton
  class Foo{
      Foo(){
      }
  }

Доступны два варианта быстрого исправления: удалить конструктор или объявить @Singleton как non-strict.

После применения быстрого исправления:


  @Singleton
  class Foo{
  }

или:


  @Singleton(strict = false)
  class Foo{
      Foo(){
      }
  }
