Сообщает о выражениях блокировки synchronized, которые ссылаются на нестатические поля без модификатора final. Такие выражения, вероятно, не имеют полезной семантики, поскольку разные потоки могут получать разные блокировки, даже при работе с одним и тем же объектом.

Пример:


  private Object o;
  public void foo() {
    synchronized (o) // синхронизация по нестатическому полю без final
    { }
  }
