Сообщать о локальных классах.

Локальный класс — это именованный вложенный класс, объявленный внутри блока кода. Локальные классы встречаются редко и могут вызывать путаницу. Кроме того, некоторые стандарты кодования не рекомендуют их использование.

Пример:


  class Example {
    void test() {
      class Local { // здесь
      }
      new Local();
    }
  }

После применения быстрого исправления:


  class Example {
    void test() {
      new Local();
    }
    private static class Local { // здесь
    }
  }
