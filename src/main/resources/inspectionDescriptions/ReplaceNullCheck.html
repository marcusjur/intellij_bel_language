Проверки на null могут быть заменены вызовом статических методов из Objects или Stream.

Пример:


  if (message == null) {
    application.messageStorage().save(new EmptyMessage());
  } else {
    application.messageStorage().save(message);
  }

После применения быстрого исправления:


  application.messageStorage()
    .save(Objects.requireNonNullElseGet(message, () -> new EmptyMessage()));

Используйте опцию Не предупреждать, если замена длиннее оригинала, чтобы игнорировать случаи, когда замена превышает длину исходного кода.
