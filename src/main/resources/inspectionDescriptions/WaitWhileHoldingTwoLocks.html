Сообщает о возможных вызовах метода wait(), когда текущий поток удерживает две блокировки.

Вызов wait() освобождает только одну блокировку на целевом объекте, поэтому ожидание при удержании двух блокировок может легко привести к взаимоблокировке.

Пример:


  synchronized (lockA) {
    synchronized (lockB) {
      lockB.wait(); //предупреждение
      //поток A заблокирован здесь, удерживая lockA
    }
  }
  synchronized (lockA) { //поток B не может войти в этот блок и освободить поток A
    lockB.notify();
  }
