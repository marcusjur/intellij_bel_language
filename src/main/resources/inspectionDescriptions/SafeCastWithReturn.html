Обнаружено безопасное приведение типа с return, которое можно заменить на проверку типа с помощью if.

Код можно упростить, используя соответствующую функцию.

Это быстрое исправление заменит безопасное приведение типа на проверку типа с помощью if.

Пример:


  fun test(x: Any) {
      x as? String ?: return
  }

После применения быстрого исправления:


  fun test(x: Any) {
      if (x !is String) return
  }
