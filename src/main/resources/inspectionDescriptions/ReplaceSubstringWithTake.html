Сообщается, что вызов s.substring(0, x) может быть заменен на s.take(x).

Использование take() делает код более лаконичным.

Это быстрое исправление заменит вызов substring на take().

Пример:


  fun foo(s: String) {
      s.substring(0, 10)
  }

После применения быстрого исправления:


  fun foo(s: String) {
      s.take(10)
  }
