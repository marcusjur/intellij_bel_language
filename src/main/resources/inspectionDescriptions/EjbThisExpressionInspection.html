Сообщает о классах EJB с методами, которые возвращают this или используют его в качестве параметра.

Пример:


@Stateless
public class ExampleBean {
  private AnotherBean anotherBean;
  public ExampleBean setAnotherBean(AnotherBean anotherBean) {
    this.anotherBean = anotherBean;
    return this; // Ошибка: возврат `this` здесь недопустим
  }
}
