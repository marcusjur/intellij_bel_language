Сообщает о локальных переменных, имена которых слишком короткие, слишком длинные или не соответствуют указанному шаблону регулярного выражения.

Пример: если проверка включена с настройками по умолчанию, где имена переменных должны начинаться с маленькой буквы, будет отмечен код int X = 42;.

Настройка проверки:

Используйте поля в разделе Параметры для указания минимальной длины, максимальной длины и ожидаемого регулярного выражения для имен локальных переменных. Укажите 0, чтобы не проверять длину имени. Регулярное выражение должно быть указано в стандартном формате java.util.regex.

Используйте флажок, чтобы игнорировать параметры в циклах for и блоках catch.
