Сообщает о неиспользуемых параметрах catch в соответствующем блоке. Проверка не выдаёт предупреждения для параметров catch, названных "ignore" или "ignored".

Пример:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ex) {
      println('Catching the exception')
  }

Параметр ex никогда не используется в блоке catch.

После применения быстрого исправления:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ignored) {
      println('Catching the exception')
  }
