Сообщает о возможности использования автоматического управления ресурсами Java 7 (что менее подвержено ошибкам) вместо конструкции try-finally.

Конструкцию try-finally можно преобразовать в конструкцию try-with-resources с помощью быстрого исправления.

Пример:


  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

После применения быстрого исправления:


  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }
