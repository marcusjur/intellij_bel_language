Сообщает о свойствах, не соответствующих соглашениям об именовании.

Проверяются следующие свойства:

  • Приватные свойства в объектах и вспомогательных объектах

Вы можете указать желаемый шаблон в настройках проверки.

Рекомендуемые соглашения об именовании: должны начинаться с подчеркивания или заглавной буквы, использовать CamelCase.

Пример:


  class Person {
    companion object {
      // Свойство во вспомогательном объекте
      private val NO_NAME = Person()
    }
  }
