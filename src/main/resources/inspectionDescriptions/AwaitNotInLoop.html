Сообщает о вызовах java.util.concurrent.locks.Condition.await(), которые не находятся внутри цикла.

Метод await() и связанные с ним методы обычно используются для приостановки потока до тех пор, пока определённое условие не станет истинным. Поток может быть разбужен по разным причинам, поэтому условие следует проверять после возврата из вызова await(). Это легко сделать с помощью цикла.

Пример:


  void acquire(Condition released) throws InterruptedException {
    released.await();
  }

Правильный код должен выглядеть так:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
