Сообщает о некорректных ссылках на Bean в параметре value аннотации @Lookup.

Пример:


  @Component public class FooBean {...}
  @Component public class OtherBean {...}
  @Component public class MyComponent {
  @Lookup("fooBean")
  public FooBean fooBean() {...}
  @Lookup("fooBean") // Сообщение "Bean должен быть типа 'OtherBean'"
  public OtherBean otherBean() {...}
  @Lookup("unknown") // Сообщение "Невозможно разрешить Bean 'unknown'"
  public OtherBean fooBean() {...}
  }
