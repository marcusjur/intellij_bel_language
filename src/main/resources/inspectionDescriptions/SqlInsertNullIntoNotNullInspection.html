Сообщает о попытках вставки значений NULL в столбцы, которые принимают только значения NOT NULL.

Пример (Microsoft SQL Server):

CREATE TABLE br2 (
id INT NOT NULL,
col1 NVARCHAR (20) NOT NULL,
col2 NVARCHAR (20) NOT NULL,
);
--
INSERT INTO br2 (id, col1, col2)
VALUES (1, NULL, NULL);

Вы не можете вставить значения NULL в столбцы col1 и col2, так как они определены как NOT NULL. При выполнении скрипта в текущем виде вы получите сообщение об ошибке. Чтобы исправить этот код, замените NULL в части VALUES на конкретные значения (например, 42 и 'bird').

INSERT INTO br2 (id, col1, col2)
VALUES (1, 42, 'bird');
