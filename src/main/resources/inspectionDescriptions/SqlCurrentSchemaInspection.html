Отчёт об неисследованных схемах и базах данных в текущей сессии.

Например, это предупреждение может появиться при попытке создать таблицу в неисследованной схеме.

Исследование - это метод проверки источника данных. При выполнении исследования из базы данных извлекается структурная информация для обнаружения различных объектов и их характеристик. Например, это могут быть таблицы, столбцы, функции и другие элементы.
