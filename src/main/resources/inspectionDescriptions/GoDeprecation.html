Сообщает об использовании устаревших элементов.

Пример:

// Устарело: используйте io.SeekStart, io.SeekCurrent и io.SeekEnd.
const (
 SEEK_SET int = 0 // поиск относительно начала файла
 SEEK_CUR int = 1 // поиск относительно текущей позиции
 SEEK_END int = 2 // поиск относительно конца файла
)

Согласно Constants на go.dev, константы SEEK_SET, SEEK_CUR и SEEK_END являются устаревшими.
