Сообщение о неправильно отформатованных именах тестов, бенчмарков и примеров.

Согласно документации пакета testing на go.dev, имена должны следовать специальным соглашениям, чтобы инструменты go могли правильно их обрабатывать.

Пример:

func Testfoo(*testing.T) {} // инструмент 'go' не запустит этот тест

После применения быстрого исправления переименовать в:

func TestFoo(*testing.T) {}
