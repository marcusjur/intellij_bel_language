Сообщает об условиях в коде, которые всегда вычисляются как true или false.

Пример:


func _(file *os.File) {
    if file == nil {
        return
    }
    if file == nil {
    }
}

В приведенном примере второе условие if file == nil избыточно. Оно всегда будет вычисляться как false, поскольку повторяет проверку, которая уже была выполнена в коде.
