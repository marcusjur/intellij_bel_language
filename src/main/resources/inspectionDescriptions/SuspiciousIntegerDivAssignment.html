Сообщает о присваиваниях с делением справа, которое не должно усекаться до целого числа.

Хотя иногда это может быть преднамеренным, такая структура обычно указывает на ошибку.

Пример:


  int x = 18;
  x *= 3/2; // не изменит x из-за целочисленного деления

Этот код следует заменить на: 


  int x = 18;
  x *= 3.0/2;

В настройках проверки можно отключить предупреждения о подозрительных, но потенциально корректных делениях, например когда делимое не может быть вычислено статически. 


  void calc(int d) {
     int x = 18;
     x *= d/2;
  }
