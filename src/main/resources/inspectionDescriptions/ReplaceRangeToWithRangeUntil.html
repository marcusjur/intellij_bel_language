Сообщает о вызовах оператора rangeTo или .. вместо использования ...

Использование соответствующей функции может упростить код.

Быстрое исправление заменит вызов rangeTo или .. на ...

Пример:


  fun foo(a: Int) {
      for (i in 0..a - 1) {
      }
  }

После применения быстрого исправления:


  fun foo(a: Int) {
      for (i in 0..<a) {
      }
  }
