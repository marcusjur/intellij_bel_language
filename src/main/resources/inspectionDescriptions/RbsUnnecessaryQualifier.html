Обнаружены избыточные квалификаторы в объявлении типов. Некоторые квалификаторы являются излишними при наличии оператора use или объявления пространства имён.

Можно использовать быстрое исправление для удаления избыточных квалификаторов.

Пример:

use Enumerator::Generator
class Example
  # 'Enumerator::Generator' можно сократить
  def call: -> Enumerator::Generator
end

После применения быстрого исправления:

use Enumerator::Generator
class Example
  def call: -> Generator
end
