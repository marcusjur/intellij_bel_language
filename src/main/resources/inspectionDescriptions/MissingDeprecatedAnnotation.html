Сообщает об объявлениях модулей, классах, полях или методах, которые имеют тег Javadoc @deprecated, но не имеют аннотации @java.lang.Deprecated.

Пример:


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

После применения быстрого исправления:


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }
