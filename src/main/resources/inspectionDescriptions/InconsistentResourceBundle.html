Отчёт о проблемах в файлах свойств, содержащихся в ресурсном пакете.
  • Сообщать об отсутствующих переводах

    Используйте эту опцию для обнаружения свойств, которые присутствуют в родительском файле свойств, но отсутствуют в унаследованных свойствах (если это не языковой диалект).

    Пример:

    
    # messages.properties
    abc=xxx
    # messages_fr.properties
    # пустой файл
    Пустое свойство abc будет отмечено как непереведенное.


  • Сообщать о несогласованных свойствах

    Используйте эту опцию для обнаружения свойств, которые присутствуют в унаследованном файле свойств, но отсутствуют в родительском (или в одноуровневом, если родительского нет).

    Пример:

    
    # messages.properties
    # пустой файл
    # messages_fr.properties
    abc=xxx
    В данном случае свойство abc переведено только для французского языка и будет отмечено как отсутствующее в (стандартном) файле свойств messages.properties.


  • Сообщать о свойствах, переопределённых с тем же значением

    Используйте эту опцию для обнаружения свойств, которые были дословно скопированы в несколько файлов свойств.


    Пример:

    
    # messages.properties
    abc=xxx
    # messages_fr.properties
    abc=xxx
    Свойство abc будет отмечено как излишне унаследованное в файле messages_fr.properties.


  • Сообщать о свойствах, переопределённых с другими заполнителями

    Используйте эту опцию для проверки согласованности заполнителей в переопределённых свойствах.

    Пример:

    
    # messages.properties
    qwe={0}xxx{1}
    abc={0}yyy{1}
    # messages_fr.properties
    qwe={0}xxx{0}xxx{1}
    abc={0}yyy
    Свойство abc будет отмечено как содержащее заполнители формата сообщения, не соответствующие messages.properties.


  • Сообщать о свойствах, переопределённых с другим окончанием

    Используйте эту опцию для проверки согласованности окончаний в переопределённых свойствах.

    Пример:

    
    # messages.properties
    abc=xxxzzz
    # messages_fr.properties
    abc=xxx;
    Свойство abc будет отмечено как заканчивающееся специальным символом (! / ? / . / : / ;), в то время как родительское значение в messages.properties - нет.
