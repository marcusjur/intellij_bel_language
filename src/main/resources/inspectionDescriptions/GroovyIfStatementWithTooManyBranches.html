Сообщает о наличии слишком большого количества ветвей в операторе if. Такие операторы могут быть сложными для понимания и обычно указывают на недостаточный уровень абстракции в дизайне.

Пример:


if (a) {
  print "foo"
} else if (b) {
  print "bar"
} else if (c) {
  print "baz"
} else if (d) {
  print "Too many branches"
}

Используйте поле максимальное количество ветвей, чтобы указать ожидаемое максимальное количество ветвей.
