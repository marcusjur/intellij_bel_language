Обнаружение повторных объявлений классов, модулей, интерфейсов, констант, глобальных переменных, псевдонимов типов и методов.
  • Повторные объявления вызывают ошибку DuplicatedDeclarationError.
  • Повторные определения методов вызывают ошибку DuplicatedMethodDefinitionError.
  • Повторные определения методов интерфейса вызывают ошибку DuplicatedInterfaceMethodDefinitionError.

Перейти к повторным объявлениям можно с помощью быстрого исправления.

Пример:

# Повторное объявление интерфейса
interface _Example
end
# Повторное объявление интерфейса
interface _Example
end
