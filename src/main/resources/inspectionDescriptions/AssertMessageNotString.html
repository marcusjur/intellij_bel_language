Сообщает о сообщениях assert, которые не являются типом java.lang.String.

Использование строк позволяет предоставить больше информации, что помогает диагностировать причину сбоя или утверждения.

Пример:


  void foo(List<Integer> myList) {
    assert myList.isEmpty() : false;
  }

Опция Выдавать предупреждение только когда сообщение assert имеет тип 'boolean' или 'java.lang.Boolean' позволяет генерировать предупреждения только когда тип сообщения assert является boolean или java.lang.Boolean. Подробные сообщения типа boolean вряд ли предоставят дополнительную информацию о сбое утверждения и, возможно, являются результатом ошибочного ввода : вместо &.
