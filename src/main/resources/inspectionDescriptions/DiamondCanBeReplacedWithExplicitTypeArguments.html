Сообщает об инстанциировании обобщенных классов, где используется символ <> (ромб) вместо параметров типа.

Быстрое исправление заменит <> (ромб) на явные параметры типа.

Пример:

List <String> list = new ArrayList<>()

После применения быстрого исправления:

List <String> list = new ArrayList<String>()

Ромбовидный оператор появился в Java 7. Эта проверка помогает обеспечить обратную совместимость с более ранними версиями Java.
