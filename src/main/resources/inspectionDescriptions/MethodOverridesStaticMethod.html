Сообщает о статических (static) методах, которые имеют такую же сигнатуру, как и статические методы суперкласса. Такие методы могут выглядеть как переопределённые, но на самом деле они скрывают метод суперкласса, поскольку статические методы в Java не могут быть переопределены.

Пример:


  class Parent {
    static void method(){}
  }
  class Example extends Parent {
    static void method(){} //предупреждение
  }
