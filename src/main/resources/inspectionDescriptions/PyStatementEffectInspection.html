Сообщает о выражениях, которые не имеют никакого эффекта.

Пример:


class Car:
    def __init__(self, speed=0):
        self.speed = speed
        self.time # не имеет эффекта
2 + 3 # не имеет эффекта

В этом примере вы можете добавить поле time в класс Car или присвоить результат проблемного выражения переменной.
