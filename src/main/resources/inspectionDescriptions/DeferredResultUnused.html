Сообщает о вызове функции с типом результата Deferred, если возвращаемое значение не используется.

Если возвращаемое значение Deferred не используется, точка вызова не будет ожидать завершения выполнения функции.

Пример:


  fun calcEverythingAsync() = CompletableDeferred(42)
  fun usage() {
      calcEverythingAsync()
  }

Быстрое исправление предложит создать переменную с инициализатором Deferred:


  fun calcEverythingAsync() = CompletableDeferred(42)
  fun usage() {
      val answer = calcEverythingAsync()
  }
