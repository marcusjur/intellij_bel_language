Сообщает о свойствах, явно присвоенных параметрам основного конструктора.

Свойства можно объявлять непосредственно в основном конструкторе, что сокращает объем кода и улучшает его читаемость.

Пример:


  class User(name: String) {
      val name = name
  }

Это быстрое исправление объединяет объявление параметра и свойства в параметре основного конструктора:


  class User(val name: String) {
  }
