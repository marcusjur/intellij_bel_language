Сообщает о методах, которые определённо приведут к бесконечной рекурсии или выбросят исключение. Эта проверка выявляет методы, которые не могут завершиться корректно.

Пример:


// Эта функция приведет к бесконечному углублению рекурсии
def fibonacci(int n) {
  return fibonacci(n-1) + fibonacci(n-2)
}
