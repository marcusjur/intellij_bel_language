Сообщает об отсутствующих проверках условий в запросах, которые могут изменить всю базу данных.

Например, использование оператора JOIN в запросе DELETE без условий ON или WHERE. Без проверки условий в JOIN, оператор DELETE удалит содержимое всей таблицы.

Пример (MySQL):

CREATE TABLE foo (a INT,b INT,c INT);
CREATE TABLE bar (a INT,b INT,c INT);
DELETE table1 FROM foo table1  INNER JOIN bar table2;
