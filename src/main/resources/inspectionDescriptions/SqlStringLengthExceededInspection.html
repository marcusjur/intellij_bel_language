Обнаружены переменные, значения которых превышают заданную длину.

Пример (Microsoft SQL Server):

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(5) = 'abcde12345';
SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;

Переменная myVarExceeded определена как VARCHAR(5), но оба присвоенных значения ('abcde12345' и '123456789') превышают это ограничение. Вы можете либо обрезать указанные значения, либо увеличить определённую длину. Чтобы увеличить длину, используйте быстрое исправление Увеличить длину типа.

После применения быстрого исправления:

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(10) = 'abcde12345';
SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;
