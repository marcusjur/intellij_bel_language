Сообщение о незарегистрированных задачах в таблице планования Celery Beat

Пример:


  #settings.py
  CELERY_BEAT_SCHEDULE = {
      'task_name': {
          'task': 'example_app.tasks.task_name', # path to a task represented by a class or function
          'schedule': 3.0,
      },
  }
  #example_app/tasks.py
  @app.task()
  def taskName():
      pass
  class TaskName(Task):
      pass

Задача не будет считаться зарегистрированной, если класс задачи не наследуется от Task или если функция задачи не имеет необходимого декоратора
