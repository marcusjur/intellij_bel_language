Обнаружена цепочка из двух вызовов, которую можно заменить одним вызовом.

Это поможет избежать избыточного выполнения кода.

Быстрое исправление заменит цепочку вызовов одним вызовом.

Пример:


  fun main() {
      listOf(1, 2, 3).filter { it > 1 }.count()
  }

После применения быстрого исправления:


  fun main() {
      listOf(1, 2, 3).count { it > 1 }
  }
