Сообщает об использовании оператора unless вместе с else. Рекомендуется размещать положительную ветвь условия первой.

Пример:

# 不良做法
a = 1
unless a > 2
  puts 'a is 2 or less'
else
  puts 'a is greater than 2'
end
Оператор unless можно преобразовать в if с помощью быстрого исправления. После применения быстрого исправления код будет выглядеть так: 
# 优良做法
a = 1
if a > 2
  puts 'a is greater than 2'
else
  puts 'a is 2 or less'
end

На основе 'Ruby Style Guide'
