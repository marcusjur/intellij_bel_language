Сообщает о неправильном использовании аннотации @Specializes.

  • Специализованный бин должен расширять другой бин
  • Специализованный метод должен переопределять метод @Produces своего непосредственного суперкласса
  • Аннотация @Named не может использоваться для специализованных классов и специальных классов
  • Два или более класса не могут специализовать один и тот же суперкласс
  • Специализованный метод не может быть static
