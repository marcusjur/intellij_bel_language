Сообщает о присвоенных значениях переменных, которые никогда не используются. Это может произойти в следующих случаях:
  • Переменная никогда не считывается после присваивания
  • Значение перезаписывается другим присваиванием до того, как будет прочитано
  • Инициализатор переменной избыточен (по одной из двух вышеуказанных причин)
  • Переменная никогда не используется.

Пример:


  class C {
    int field;
    void foo() {
      field = unused_value;
    }
    void bar() {
      use(field);
      foo();
    }
  };
