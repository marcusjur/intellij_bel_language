Сообщает о операциях преобразования, вызываемых для значений Mono<Void> в цепочке Reactor.

Mono<Void> завершается без каких-либо значений, поэтому функция преобразования никогда не будет вызвана. Быстрое исправление предлагает заменить map() и flatMap() на then().

Пример:

Mono.when(Mono.just(1), Mono.just(2))
                .map(v -> "New value")
                .doFinally(System.out::println);

После применения быстрого исправления:

Mono.when(Mono.just(1), Mono.just(2))
                .then(Mono.fromCallable(() -> "New value"))
                .doFinally(System.out::println);
