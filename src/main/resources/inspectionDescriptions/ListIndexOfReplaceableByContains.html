Сообщает о выражениях с List.indexOf(), которые могут быть заменены методом List.contains().

Пример:


  boolean hasEmptyString(List<String> list) {
    // Предупреждение: можно упростить
    return list.indexOf("") >= 0;
  }

Предлагаемое быстрое исправление заменяет вызов indexOf на вызов contains:


  boolean hasEmptyString(List<String> list) {
    // Быстрое исправление применено
    return list.contains("");
  }
