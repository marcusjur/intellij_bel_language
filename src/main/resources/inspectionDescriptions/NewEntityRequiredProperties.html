Сообщает о неназначенных свойствах Entity при его создании с помощью функций new или getOrCreate в ChangeScope.

Все non-null свойства Entity должны быть назначены или помечены аннотацией @Many. В противном случае это может привести к исключению во время выполнения.

Пример:


  interface A : LegacyEntity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }
  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

Отсутствующие обязательные свойства можно назначить с помощью быстрого исправления


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }
