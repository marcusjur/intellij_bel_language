Сообщать об устаревших ключах в YAML-файлах.

Проверка на устаревшие элементы выполняется только при наличии JSON-схемы, связанной с соответствующим YAML-файлом.

Обратите внимание, что механизм пометки устаревших элементов еще не определен в спецификации JSON Schema, и эта проверка использует нестандартное расширение deprecationMessage.

Пример объявления устаревшего элемента в схеме:


  {
    "properties": {
      "SomeDeprecatedProperty": {
        "deprecationMessage": "Baz",
        "description": "Foo bar"
      }
    }
  }

Пример с соответствующим предупреждением:


  SomeDeprecatedProperty: some value
