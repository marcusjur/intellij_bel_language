Сообщает об интерфейсах, которые могут быть помечены аннотацией @FunctionalInterface.

Добавление аннотации @FunctionalInterface к интерфейсу указывает, что это функциональный интерфейс, и в него нельзя добавлять дополнительные методы abstract.

Пример:


  interface FileProcessor {
    void execute(File file);
  }

После применения быстрого исправления:


  @FunctionalInterface
  interface FileProcessor {
    void execute(File file);
  }
