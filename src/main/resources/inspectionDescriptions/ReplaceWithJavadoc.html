Сообщает об обычных комментариях к полям, методам или классам, которые могут быть заменены на комментарии Javadoc.

Пример:


  public class Main {
    /*
    * Hello,
    */
    // World!
    void f() {
    }
  }

После применения быстрого исправления:


  public class Main {
    /**
     * Hello,
     * World!
     */
    void f() {
    }
  }
