Сообщает о необходимости обновления зависимости библиотеки kotlinx.coroutines в Gradle для обеспечения совместимости с Kotlin 1.3+.

Пример:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.0.1'
  }

Быстрое исправление изменит версию библиотеки kotlinx.coroutines на версию, совместимую с Kotlin 1.3:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.27.0-eap13'
  }
