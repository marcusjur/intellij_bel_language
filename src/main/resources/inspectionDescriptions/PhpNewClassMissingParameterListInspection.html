Отсутствует список параметров при создании экземпляра класса.

Согласно стандарту PSR-12, при создании нового экземпляра класса всегда необходимо использовать круглые скобки, даже если конструктору не передаются аргументы.

Для получения дополнительной информации см. PSR-12: Extended Coding Style (php-fig.org).
