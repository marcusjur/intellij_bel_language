Сообщает о любых ссылках на внутренние классы, которые излишне уточнены именем включающего класса.

Такие уточнения можно безопасно удалить, что иногда требует добавления импорта для внутреннего класса.

Пример:


  class X {
     X.Y foo;
     class Y{}
  }

После применения быстрого исправления:


  class X {
     Y foo;
     class Y{}
  }

Используйте опцию Игнорировать ссылки, требующие импорта, чтобы пропускать ссылки на внутренние классы, для которых удаление уточнения потребует добавления импорта.
