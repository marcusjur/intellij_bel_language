Сообщает о switch-выражениях, которые не охватывают все возможные результаты сопоставления.

Groovy не требует, чтобы switch-выражения были исчерпывающими. Он работает так, как если бы была добавлена неявная ветка default -> null. Если разработчик забудет добавить необходимую ветку case, это может привести к неожиданному значению null.

Пример:


  enum A { X, Y }
  def foo(A a) {
    def x = switch (a) { // reports switch
      case A.X -> ...
    }
  }
