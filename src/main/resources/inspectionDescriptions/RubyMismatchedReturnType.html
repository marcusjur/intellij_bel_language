Сообщает о методах и блоках, где выведенный тип возвращаемого значения не соответствует ожидаемому типу возвращаемого значения. Ожидаемый тип возвращаемого значения определяется одним из следующих способов:
  • Сигнатура типа RBS
  • Сигнатура типа Sorbet
  • Тег YARD @return

Пример:

# @return [Integer]
def method
  1.0
end
