Сообщает о неразрешённых Spring Bean, на которые ссылаются аннотации Spring Security.

Пример:

  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "unknownUserDetailsService")
      // Если 'unknownUserDetailsService' не определен в модели Spring, будет выведено "Невозможно разрешить Bean 'unknownUserDetailsService'"
    
  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "jpaUserDetailsService")
      // "Bean должен быть типа 'org.springframework.security.core.userdetails.UserDetailsService'"
      // Будет выведено, если Bean "jpaUserDetailsService" не является типом 'UserDetailsService'
