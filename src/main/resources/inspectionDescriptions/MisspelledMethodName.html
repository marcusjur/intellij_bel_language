Сообщает о методах класса, имена которых отличаются только регистром букв. Такие имена могут вызывать путаницу.

Пример: 


  public int hashcode() { // обнаружено, возможно, имелось в виду hashCode?
    return 0;
  }

Быстрое исправление для переименования таких методов доступно только в редакторе.

Используйте опцию Игнорировать переопределение/реализацию методов суперкласса, чтобы исключить методы, которые переопределяют или реализуют методы суперкласса.
