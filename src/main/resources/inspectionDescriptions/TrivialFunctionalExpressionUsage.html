Сообщает о прямых вызовах методов функционального интерфейса в лямбда-выражениях, ссылках на методы или определениях анонимных классов. Такие вызовы методов могут быть заменены реализацией функционального интерфейса.

Пример:


  boolean contains(List<String> names, String name) {
    return ((Predicate<String>)x -> {
      return names.contains(x);
    }).test(name);
  }

После применения быстрого исправления вызов метода становится:


  boolean contains(List<String> names, String name) {
    return names.contains(name);
  }
