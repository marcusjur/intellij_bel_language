Сообщает о попытках удаления столбцов, которые используются в индексных таблицах. Эта проверка доступна в Microsoft SQL Server и Sybase ASE.

Пример (Microsoft SQL Server):

CREATE TABLE test_index
(
col  INT NOT NULL,
col2 INT NOT NULL,
col3 INT NOT NULL UNIQUE,
col4 VARCHAR(200)
);
CREATE UNIQUE INDEX aaaa ON test_index (col, col2);
ALTER TABLE test_index
DROP COLUMN col;

Невозможно удалить столбец col, так как он используется в индексной таблице. Чтобы удалить этот столбец, сначала необходимо удалить индекс aaaa (например, DROP INDEX aaaa).
