Сообщает о любых случаях использования жестко закодованных строковых литералов String.

Жестко закодованные строковые литералы String могут быть ошибкой в контексте интернационализации. Эта проверка не распространяется на пустые строки и строки, содержащие только пробелы. С помощью быстрого исправления строковые литералы можно преобразовать в вызов метода java.util.ResourceBundle.getString().
