Сообщает о вызовах метода java.lang.String.concat().

Если метод вызывается для константы с константным аргументом, такой вызов можно заменить оператором + для улучшения читаемости и производительности.

Пример:


  String foo(String name) {
    return name.concat("foo");
  }

После применения быстрого исправления:


  String foo(String name) {
    return name + "foo";
  }
