Сообщает о присваивании или модификации полей, объявленных в суперклассе, из конструктора подкласса.

Рекомендуется инициализовать поля суперкласса в его собственном конструкторе и делегировать вызов этому конструктору из подкласса. Это также позволит объявить поле как final, если оно не изменяется после конструирования.

Пример:


  class Super {
    int x;
  }
  class Sub extends Super {
    Sub(int _x) {
      // Warning: x is declared in a superclass
      x = _x;
    }
  }

Чтобы избежать этой проблемы, объявите конструктор суперкласса:


  class Super {
    final int x;
    Super(int _x) {
      x = _x;
    }
  }
  class Sub extends Super {
    Sub(int _x) {
      super(_x);
    }
  }
