Сообщает о классах, которые имеют такое же имя, как один из их суперклассов, но при этом имеют разные полные квалифицированные имена. Такие имена классов могут вводить в заблуждение.

Пример:


  package util;
  abstract class Iterable<T> implements java.lang.Iterable<T> {}

Переименовать такой класс можно только с помощью быстрого исправления в редакторе.
