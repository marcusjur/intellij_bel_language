Сообщает о вызовах System.out.println() с исключением в качестве аргумента.

Вывод исключений через операторы print скрывает трассировку стека, что может усложнить исследование проблемы. Рекомендуется использовать логгер.

Также сообщает о вызовах System.out.print(), System.err.println() и System.err.print() с исключением в качестве аргумента. Рекомендуется использовать логгер для записи исключений.

Быстрое исправление поддерживает SLF4J и Log4j 2. Оно заменяет вызов System.out.println() на вызов логгера

Пример:


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

После применения быстрого исправления:


  try {
      foo();
  } catch (Exception e) {
    log.error("e: ", e);
  }

Опция Исправленный метод логования позволяет указать метод, используемый для записи сообщений.
