Сообщает о методе equals() с параметром типа, отличным от Any?, когда в классе отсутствует другой метод equals() с параметром типа Any?.

Пример:


  class Foo {
      fun equals(other: Foo?): Boolean {
          return true
      }
  }

Для исправления этой проблемы создайте метод equals() с параметром типа Any?.
