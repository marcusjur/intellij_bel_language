Сообщает о контрольных операторах, содержащих только одно выражение в блоке кода, и предлагает удалить фигурные скобки из тела оператора.

Пример:


  if (x > 0) {
    System.out.println("x is positive");
  }

После применения быстрого исправления:


  if (x > 0) System.out.println("x is positive");
