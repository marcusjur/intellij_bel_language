Сообщает о ресурсах Java ME javax.microedition.io.Connection, которые не открываются в блоке try и не закрываются в соответствующем блоке finally. Такие ресурсы могут случайно утекать, если исключение возникает до закрытия ресурса.

Пример:


  void example() throws IOException {
    Connection c = Connector.open("foo");
  }
