Сообщает о вызове функции errors.As, когда второй аргумент не является указателем на интерфейс или тип, реализующий error.

Такие вызовы приводят к панике во время выполнения.

Подробную информацию о функции As смотрите в func As на go.dev.

Пример:

_, err := os.Open("non-existing")
var pathError *fs.PathError
if errors.As(err, pathError) { // требуется указатель на *fs.PathError
}

После применения быстрого исправления добавить '&':

_, err := os.Open("non-existing")
var pathError *fs.PathError
if errors.As(err, &pathError) {
}

Эта проверка выполняется только для версий языка 1.13 и выше.
