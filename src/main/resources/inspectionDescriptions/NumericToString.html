Сообщает о вызовах метода toString() для объектов классов, расширяющих Number. Такие вызовы обычно некорректны в интернационализованной среде, вместо них следует использовать форматование с учетом локали.

Пример:


  void print(Double d) {
    System.out.println(d.toString());
  }
Один из возможных способов исправления: 

  void print(Double d) {
    System.out.printf("%f%n", d);
  }
Это будет форматовать число, используя локаль по умолчанию, которая устанавливается при запуске JVM и основывается на настройках системы.
