Сообщает о функциях с именами, которые слишком короткие, слишком длинные или не соответствуют указанному шаблону регулярного выражения. Некоторые стили кодования имеют специальные соглашения об именовании функций.

Используйте параметры проверки, чтобы указать минимальную/максимальную длину имени функции и ожидаемое регулярное выражение. Чтобы игнорировать длину имени, укажите 0.

Для получения дополнительной информации о регулярных выражениях обратитесь к Quick Start guide (regular-expressions.info).
