Сообщает о лямбда-параметрах, которые слишком короткие, слишком длинные или не соответствуют указанному шаблону регулярного выражения.

Пример: если проверка включена с настройками по умолчанию (где имя параметра должно начинаться со строчной буквы), будет отмечен код Function<String, String> id = X -> X;.

Настройка проверки:

Используйте поля в разделе Параметры, чтобы указать минимальную длину, максимальную длину и ожидаемый шаблон регулярного выражения для имен лямбда-параметров. Укажите 0, чтобы отключить проверку длины имени.

Регулярное выражение должно быть указано в стандартном формате java.util.regex.
