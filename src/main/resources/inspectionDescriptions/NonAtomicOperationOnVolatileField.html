Обнаружены неатомарные операции с volatile-полем.

Одним из примеров неатомарной операции является обновление поля с помощью оператора инкремента. Поскольку операция включает чтение и запись, и между ними могут происходить другие изменения, это может привести к повреждению данных. Операцию можно сделать атомарной, обернув её в блок synchronized или используя один из классов пакета java.util.concurrent.atomic.

Пример:


  private volatile int v = 1;
  void foo() {
    v = 2 * v;
  }
