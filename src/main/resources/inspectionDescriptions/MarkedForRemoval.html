Сообщает об использовании устаревших API (классов, полей и методов), помеченных для удаления с помощью @Deprecated(forRemoval=true).

Код, использующий API, помеченные для удаления, может привести к ошибкам времени выполнения в будущих версиях API. Именно поэтому рекомендуемый уровень серьезности для этой проверки — Error.

Если вы хотите использовать такую же подсветку кода, как для обычных устаревших элементов, вы можете изменить уровень серьезности на Предупреждение.

Новое в версии 2017.3
