Сообщает о ресурсах Java ME javax.microedition.rms.RecordStore, которые не открываются в блоке try и не закрываются в соответствующем блоке finally.

Такие ресурсы могут быть случайно утечены, если исключение возникнет до закрытия ресурса.

Эта проверка применима для Java ME и других сред с сильно ограниченными ресурсами. Бездумное применение этой проверки может негативно повлиять на читаемость кода и его архитектуру.

Пример:


  void foo1() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // предупреждение
  }
  void foo2() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // нет предупреждения
      try {
          /* ... */
      } finally {
          rs.closeRecordStore();
      }
  }
