Сообщает о случаях, когда вызовы CASE и COALESCE являются взаимозаменяемыми. Эта проверка предлагает следующие действия: заменить на вызов 'COALESCE' и обратное действие заменить на выражение CASE.

Пример (MySQL):

SELECT
  -- this CASE may be replaced by COALESCE
	CASE
		WHEN C1 IS NOT NULL THEN C1
		ELSE 0
		END
FROM dual;

В этом примере выражение CASE может быть заменено на SELECT COALESCE(C1, 0), что даст тот же результат.

Если вы предпочитаете использовать выражения CASE, выберите опцию Предпочитать выражения CASE вместо функций COALESCE на странице проверки.
