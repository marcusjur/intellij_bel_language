Сообщается о возможности замены конструктора SAM (Single Abstract Method) на лямбда-выражение.

Пример:


  fun main() {
      foo(Runnable { println("Hi!") })
  }
  fun foo(other: Runnable) {}

После применения быстрого исправления:


  fun main() {
      foo( { println("Hi!") })
  }
  fun foo(other: Runnable) {}
