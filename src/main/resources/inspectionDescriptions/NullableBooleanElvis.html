Сообщает о случаях, когда следует использовать проверку на равенство вместо оператора elvis.

Пример:


fun check(a: Boolean? == null) {
    if (a ?: false) throw IllegalStateException()
}

После применения быстрого исправления:


fun check(a: Boolean? == null) {
    if (a == true) throw IllegalStateException()
}
