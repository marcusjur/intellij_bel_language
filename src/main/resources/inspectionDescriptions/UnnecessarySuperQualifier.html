Сообщает о ненужных квалификаторах super в вызовах методов и ссылках на поля.

Квалификатор super не требуется, когда поле или метод суперкласса не скрыт/не переопределен в вызывающем классе.

Пример:


  class Foo {
    void foo() {}
  }
  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }

После применения быстрого исправления:


  class Foo {
    void foo() {}
  }
  class Bar extends Foo {
    void bar() {
      foo();
    }
  }

В настройках проверки можно игнорировать квалификаторы, которые помогают различать доступ к членам суперкласса и одноименным членам внешнего класса.

См. также следующие проверки:

  • Java | Видимость | Доступ к унаследованным полям аналогичен доступу к элементам в окружающем коде
  • Java | Видимость | Вызовы унаследованных методов аналогичны вызовам локальных методов
