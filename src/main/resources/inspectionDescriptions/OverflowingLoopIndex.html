Сообщает о циклах, которые не могут завершиться без переполнения индекса, или о циклах, которые вообще не выполняются. Обычно это происходит из-за ошибки в операции обновления.

Пример:


  void foo(int s) {
    for (int i = s; i > 12; i++) { // здесь должно быть i--
      System.out.println(i);
    }
  }
