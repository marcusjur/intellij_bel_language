Сообщение о методах репозитория Spring Data CRUD, для которых построитель Spring Data Query не может сгенерировать соответствующие запросы.

Пример:


interface PersonRepository extends Repository<Person, Long> {
  List<Person> findByEmailAddressAndLastname(EmailAddress emailAddress, String lastname);
  List<Person> findByEmailUnknownAndLastname(EmailAddress emailAddress, String lastname); // Невозможно разрешить свойство 'Unknown'
  List<Person> findByAndLastname(EmailAddress emailAddress, String lastname); // Отсутствует имя свойства
  List<Person> findDistinctPeopleByLastnameOrFirstname(String lastname, String firstname);
  List<Person> findPeopleDistinctByUnknownOrFirstname(String lastname, String firstname); // Невозможно разрешить свойство 'Unknown'
  List<Person> findByLastnameOrderByFirstnameAsc(String lastname);
  List<Person> findByLastnameOrderByUnknownDesc(String lastname);  // Невозможно разрешить свойство 'Unknown'
}
