Сообщает о выражении типа byte, возвращаемом из метода, реализующего метод InputStream.read().

Это подозрительно, так как InputStream.read() должен возвращать значения в диапазоне от 0 до 255, в то время как выражение типа byte содержит значения от -128 до 127. Быстрое исправление преобразует выражение в беззнаковый byte путем применения битовой маски 0xFF.

Пример:


class MyInputStream extends InputStream {
  int pos = 0;
  byte[] data;
  MyInputStream(byte[] input) {
    data = input;
  }
  @Override
  public int read() {
    if (pos == data.length) {
      return -1;
    }
    return data[pos++]; // Проблема
  }
}

После применения быстрого исправления:


class MyInputStream extends InputStream {
  int pos = 0;
  byte[] data;
  MyInputStream(byte[] input) {
    data = input;
  }
  @Override
  public int read() {
    if (pos == data.length) {
      return -1;
    }
    return data[pos++] & 0xFF;
  }
}
