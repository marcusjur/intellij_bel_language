Обнаружены следующие нарушения спецификации EJB:
  • Недопустимый тип внедряемого свойства
  • Ссылка EJB не соответствует `ejb-jar.xml`

Пример:


@LocalBean
@Stateless
public class ExampleBean {
  private AnotherBean anotherBean;
  @Inject
  public void getAnotherBean() { // Ошибка: внедрение должно выполняться в поле или метод setter
    return anotherBean;
  }
  ...
}
