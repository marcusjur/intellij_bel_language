Сообщает о вызове Throwable.initCause(), когда конструктор исключения также включает параметр Throwable cause.

В этом случае вызов initCause() можно удалить, а его аргумент добавить в вызов конструктора исключения.

Пример:


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing");
    wrapper.initCause(ex); // Ненужный вызов 'Throwable.initCause()'
    throw wrapper;
  }

Аргумент cause можно передать в конструктор с помощью быстрого исправления. После применения быстрого исправления:


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing", ex);
    throw wrapper;
  }
