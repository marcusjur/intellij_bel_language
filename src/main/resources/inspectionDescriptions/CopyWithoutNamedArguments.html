Сообщает о вызовах метода copy() для классов данных без именованных аргументов.

Поскольку все аргументы функции copy() являются необязательными, может быть сложно понять, какие свойства изменяются. Явное указание имен параметров делает код более понятным и устраняет необходимость переходить к объявлению data class.

Пример:


  data class User(val name: String, val age: Int)
  fun copyUser(user: User): User {
      return user.copy("John")
  }

Быстрое исправление добавит имена параметров для всех аргументов copy():


  data class User(val name: String, val age: Int)
  fun copyUser(user: User): User {
      return user.copy(name = "John")
  }
