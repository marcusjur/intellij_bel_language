Сообщает о проблемах регистрации расширений в plugin.xml.

Будут отображены следующие проблемы:

  • Отсутствует объявление language. Если расширение не предназначено для конкретного языка, используйте быстрое исправление для создания явного объявления "для любого языка".
  • Проверки: отсутствующие функции
  • Сервисы: избыточное объявление serviceInterface
  • com.intellij.stubElementTypeHolder без externalIdPrefix, см. индексы заглушек
  • com.intellij.statusBarWidgetFactory без id
  • com.intellij.applicationConfigurable|projectConfigurable без displayName/key

Изменения в версии 2022.3
