Сообщает о методах getter или setter, которые обращаются к полям с именами, не соответствующими ожидаемым. Например, когда getY() возвращает поле x. Обычно это может указывать на ошибку копирования-вставки.

Пример:

class Point {
    private int x;
    private int y;
    public void setX(int x) { // Предупреждение: setter 'setX()' присваивает значение полю 'y'
      this.y = x;
    }
    public int getY() { // Предупреждение: getter 'getY()' возвращает поле 'x'
      return x;
    }
  }
