Сообщает о docstring, которые не соответствуют формату строки документации с тройными кавычками.

Пример:


def calc(self, balance=0):
    'param: balance'
    self.balance = balance

После применения быстрого исправления код станет:


def calc(self, balance=0):
    """param: balance"""
    self.balance = balance
