Обнаружение некорректных обозначений зависимостей.

Gradle поддерживает следующие типы/форматы:
  • Экземпляры Dependency;
  • Значения String или CharSequence, например 'org.gradle:gradle-core:1.0';
  • Отображения (maps), например [group: 'org.gradle', name: 'gradle-core', version: '1.0'];
  • FileCollection, например files('some.jar', 'someOther.jar');
  • Проекты, например project(':some:project:path');
  • ClassPathNotation, например gradleApi();
  • Списки обозначений зависимостей, например ['org.gradle:gradle-core:1.0'];
  • (Gradle 7.0+) Средства доступа к каталогу версий, например libs.groovy.core.
См. документацию Gradle

Пример:


dependencies {
  implementation(1) // reports '1'
}
