Сообщает о методах, помеченных аннотацией @DataProvider, которые не возвращают Object[][] или Iterator<Object>. TestNG выбросит исключение, если возвращается другой тип данных.

Пример:


public class TestNgTest {
  @DataProvider(name = "Languages")
  List<String> getData() {
    return List.of("Java", "Kotlin");
  }
  @Test(dataProvider = "Languages")
  public void testData(String language) {
    System.out.println(language);
  }
}
