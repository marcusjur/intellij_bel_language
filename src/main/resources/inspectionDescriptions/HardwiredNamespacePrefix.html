Сообщает о сравнении функции name() со строкой, содержащей двоеточие (:). Такое использование обычно указывает на жёстко закодованный префикс пространства имён в сравнении. В результате код может перестать работать при использовании другого префикса для того же пространства имён во время выполнения XML.

Пример:


  <xsl:if test="name() = 'xlink:href'">...<xsl:if>
