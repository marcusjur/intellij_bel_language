Сообщает о коде, который использует == или != для сравнения новых созданных объектов вместо вызова equals().

Ссылка на новый созданный объект не может указывать на существующий объект, поэтому результат сравнения всегда будет false. Проверка также выявляет новые объекты, возвращаемые из простых методов.

Пример:


  void test(Object obj) {
    if (new Object() == obj) {...}
  }

После применения быстрого исправления:


  void test(Object obj) {
    if (new Object().equals(obj)) {...}
  }
