Сообщать о классах без конструктора (без аргументов, т.е. без параметров). Конструктор без аргументов необходим в некоторых контекстах. Например, когда нужно создать экземпляр класса с помощью рефлексии.

Пример:


  public class Bean {
    private String name;
    public Bean(String name) {
      this.name = name;
    }
  }

Используйте флажок ниже, чтобы игнорировать классы без явного конструктора. Компилятор предоставляет конструктор по умолчанию без аргументов для таких классов.
