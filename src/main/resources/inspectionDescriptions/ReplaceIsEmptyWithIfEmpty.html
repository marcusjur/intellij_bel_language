Сообщает о вызовах isEmpty, isBlank, isNotEmpty или isNotBlank в операторе if для присвоения значения по умолчанию.

Быстрое исправление заменяет условие if на вызов ifEmpty или ifBlank.

Пример:


  fun test(list: List<Int>): List<Int> {
      return if (list.isEmpty()) {
          println()
          foo()
      } else {
          list
      }
  }

После применения быстрого исправления:


  fun test(list: List<Int>): List<Int> {
      return list.ifEmpty {
          println()
          foo()
      }
  }

Эта проверка выполняется только если версия языка Kotlin для проекта или модуля 1.3 или выше.
