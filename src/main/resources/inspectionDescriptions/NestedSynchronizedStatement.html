Обнаружены вложенные операторы synchronized. Рекомендуется избегать вложенной синхронизации, так как в некоторых случаях это может привести к взаимоблокировке.

Пример:


  synchronized (lockA){
    //поток 1 ожидает lockB
    synchronized (lockB){ //предупреждение
    }
  }
  ...
  synchronized (lockB) {
    //поток 2 ожидает lockA
    synchronized (lockA) { //предупреждение
    }
  }
