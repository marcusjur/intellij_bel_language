Сообщает об ошибках типов в вызовах функций, целевых объектах и возвращаемых значениях. В динамически типизованных языках такое иногда случается.

Типы параметров функции можно указать в docstring или в аннотациях функций Python 3.

Пример:


def foo() -> int:
    return "abc" # должен быть int, получен str
a: str
a = foo() # должен быть str, получен int

Быстрое исправление может изменить проблемный тип:


def foo() -> str:
    return "abc"
a: str
a = foo()
