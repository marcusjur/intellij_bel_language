Обнаружение нераспознанных символов в именованных запросах, нативных запросах и связанных методах API:
  • Неизвестный именованный запрос
  • Неизвестный параметр запроса

Пример:


  @Entity
  @NamedQuery(name = "SelectByConditionQuery", query = "...")
  public class JavaEntity { ... }
  class Main {
    public static void executeQuery() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .createNamedQuery("SelectAllQuery") // Ошибка: неизвестное имя запроса
        ...
    }
  }
