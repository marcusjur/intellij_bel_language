Сообщает о прямых наследниках sealed класса, у которых отсутствует состояние и не переопределен метод equals().

Настоятельно рекомендуется переопределить метод equals() для обеспечения стабильности сравнения или преобразовать class в object для достижения того же эффекта.

Пример:


  sealed class Receiver {
      class Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }

Быстрое исправление преобразует class в object:


  sealed class Receiver {
      object Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }
