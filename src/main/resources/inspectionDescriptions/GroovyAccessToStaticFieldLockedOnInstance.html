Сообщает о доступе к нестатическим полям static через блокировку на this или полях экземпляра this.

Блокировка статических полей на данных экземпляра не предотвращает изменение этого поля другими экземплярами, что может привести к непредвиденным состояниям гонки.

Пример:


  static String s;
  def foo() {
      synchronized (this) {
          System.out.println(s); // Предупреждение
      }
  }
