Сообщает об использовании объектов groovy.lang.GString в качестве ключей отображения. Обычно объекты GString изменяемы и не должны использоваться в качестве ключей. Кроме того, доступ к элементам GString невозможен при использовании объектов java.lang.String с тем же значением.

Пример:


def map = [:]
def key = 'foo'
map << ["${key}": 'bar']
assert map[key] == null // неожиданный результат сравнения 'true'

Изменения в версии 2017.1
