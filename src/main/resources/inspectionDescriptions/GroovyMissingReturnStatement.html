Сообщает об отсутствии оператора return в конце метода с возвращаемым типом, отличным от void. Поток выполнения метода может достигать конца метода.

Пример:


String foo(int a) {
  if (a > 0) {
    return "more than zero"
  }
} // foo(-1) вернет 'null'
int bar(int a) {
  if (a > 0) {
    return a
  }
} // bar(-1) завершится с ошибкой и вызовет исключение времени выполнения
