Сообщает о вызовах метода openSession(), если возвращаемый ресурс org.hibernate.Session не был безопасно закрыт.

По умолчанию проверка предполагает, что ресурс может быть закрыт любым методом, содержащим в названии 'close' или 'cleanup'.

Пример:


  void doHibernateQuery(SessionFactory factory) {
    Session session = factory.openSession(); //предупреждение
    session.createQuery("...");
  }

Проверка настраивается следующими параметрами:

  • Разрешать ли открытие ресурса org.hibernate.Session внутри блока try. Этот стиль менее предпочтителен, так как он более многословен, чем открытие ресурса перед блоком try.
  • Может ли ресурс быть закрыт любым вызовом метода, который принимает ресурс в качестве аргумента.
