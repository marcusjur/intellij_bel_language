Сообщает о методах equals(), которые принимают тип аргумента, отличный от java.lang.Object, если включающий класс не имеет других перегрузок equals() с java.lang.Object в качестве типа аргумента.

Ковариантная версия equals() не переопределяет метод Object.equals(Object). Это может привести к непредвиденному поведению во время выполнения. Например, если класс используется для создания стандартной коллекции, которая ожидает переопределения метода Object.equals(Object).

Пример:


  class Foo {
      public boolean equals(Foo foo) {  // предупреждение
          return false;
      }
  }
  class Bar {
      public boolean equals(Bar bar) {  // здесь нет предупреждения
          return false;
      }
      @Override
      public boolean equals(Object obj) {
          return false;
      }
  }
