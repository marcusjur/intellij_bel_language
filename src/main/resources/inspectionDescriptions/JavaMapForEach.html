Сообщение о возможности замены Java Map на вызов метода forEach в Kotlin.forEach.

Пример:


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }
  fun foo(i: Int, s: String) {}

Быстрое исправление добавит круглые скобки:


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }
  fun foo(i: Int, s: String) {}
