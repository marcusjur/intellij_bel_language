Сообщает об object (включая data object), который реализует java.io.Serializable, но не реализует метод readResolve

Пример:


  import java.io.Serializable
  object Foo : Serializable

Быстрое исправление реализует метод readResolve:


  import java.io.Serializable
  object Foo : Serializable {
      private fun readResolve() = Foo
  }
