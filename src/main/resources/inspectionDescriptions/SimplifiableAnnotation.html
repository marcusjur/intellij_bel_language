Аннотации можно упростить до одноэлементной формы или сокращенной формы записи.

Обнаруженные проблемы:

  • Избыточное указание value= в парах имя-значение аннотации
  • Избыточные фигурные скобки для массивов, содержащих единственное значение
  • Избыточный пробел между символом @ и именем аннотации
  • Избыточный пробел между именем аннотации и списком параметров
  • Избыточные круглые скобки в аннотациях без параметров

Пример:


  @interface Foo { String[] value(); }
  @ Foo({"foo"})
  public String name;

После применения быстрого исправления:


  @interface Foo { String[] value(); }
  @Foo("foo")
  public String name;
