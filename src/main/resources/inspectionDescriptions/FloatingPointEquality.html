Сообщает об использовании операторов == или != при сравнении значений с плавающей точкой.

Значения с плавающей точкой по своей природе неточны, и их сравнение на полное равенство редко является желаемым поведением.

Эта проверка игнорирует сравнения с нулем и литералами бесконечности.

Пример: 


  void m(double d1, double d2) {
    if (d1 == d2) {}
  }
