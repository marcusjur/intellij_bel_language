Сообщает о некорректном использовании аннотаций Spring Web Services.

Пример:


  // 1.  @PayloadRoot on non-endpoint classes…
  public class HolidayEndpoint {
	@PayloadRoot(namespace = NAMESPACE_URI, localPart = "HolidayRequest") // @Endpoint класс поддерживает @PayloadRoot
	public void handleHolidayRequest(@RequestPayload Element holidayRequest) throws Exception {…}
  }
  // 2.  @XPathParam на методе @PayLoad
  @Endpoint
  public class AirlineEndpoint {
	  @Namespace(prefix = "m", uri = MESSAGES_NAMESPACE) @ResponsePayload
	  public GetFlightsResponse getFlights(@XPathParam("//m:sc") String serviceClassString) {…} // Метод @PayLoad поддерживает @XPathParam
  }
