Сообщает о полях, методах или классах, для которых возможно уменьшение области видимости модификатора доступа.

Пример:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // может быть private
  }

После применения быстрого исправления:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

Правила для предложений по изменению модификаторов можно определить в настройках проверки.
