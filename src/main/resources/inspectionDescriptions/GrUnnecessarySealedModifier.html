Сообщает об использовании ненужного модификатора sealed для методов, полей или переменных.

Этот модификатор действителен только для классов, интерфейсов и трейтов.

Пример:


  sealed boolean foo() {} // Модификатор не нужен
  sealed Object bar // Модификатор не нужен
  // Модификатор обязателен, поэтому не подсвечивается
  sealed class A {}
