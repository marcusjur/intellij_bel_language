Сообщает о несоответствиях между именами и количеством параметров в JSDoc-комментариях и фактическими параметрами функции. Рекомендуется обновить параметры в JSDoc-комментариях.

Пример:


/**
 * @param height Height in pixels
 */
function sq(height, width) {} // ширина не документирована

После применения быстрого исправления:


/**
 * @param height Height in pixels
 * @param width
 */
function sq(height, width) {}
