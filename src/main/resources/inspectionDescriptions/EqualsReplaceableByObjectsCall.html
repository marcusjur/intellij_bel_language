Выражение может быть заменено вызовом метода java.util.Objects#equals.

Пример:


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

После применения быстрого исправления:


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

Замена выражений типа a != null && a.equals(b) на Objects.equals(a, b) немного изменяет семантику. Это поведение можно включить или отключить с помощью опции Подсвечивать выражения похожие на 'a != null && a.equals(b)'.
