Сообщать об экспортуемых объявлениях без документирующих комментариев. Согласно разделу Doc Comments на github.com/golang, все экспортуемые имена верхнего уровня должны иметь документирующий комментарий.

Дополнительную информацию о правилах написания комментариев см. в разделе Comment Sentences на github.com/golang.

Чтобы добавить комментарий, воспользуйтесь быстрым исправлением Добавить комментарий.
