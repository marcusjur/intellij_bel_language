Сообщать обо всех методах с переменной арностью (varargs), которые могут быть аннотированы как @SafeVarargs. Аннотация @SafeVarargs подавляет непроверенные предупреждения о создании параметризованного массива в точке вызова.

Пример:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

После применения быстрого исправления:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();
    @SafeVarargs
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Эта аннотация не поддерживается в JVM версии 1.6 и ниже.
