Сообщает о синхронизации выражений this или class. Проверяются блоки synchronized и вызовы методов wait(), notify() или notifyAll().

Синхронизация по выражениям this или class может быть нежелательной по нескольким причинам:

  1. Это делает синхронизацию частью внешнего интерфейса класса, что затрудняет переход на другие механизмы блокировки в будущем,
  2. Сложно отследить, кто именно заблокировал данный объект,
  3. Делает возможными атаки типа "отказ в обслуживании", как преднамеренные, так и случайные при наследовании.

Альтернативным решением является синхронизация по private final объекту блокировки, доступ к которому полностью контролируется.

Пример:


  public void print() {
    synchronized(this) { // Предупреждение: блокировка по 'this' может привести к непредсказуемым побочным эффектам
      System.out.println("synchronized");
    }
  }
