Сообщает об использовании API, недоступного на настроенном уровне языка. Эта проверка работает в трех направлениях:
  • Подсвечивает использование обобщенных классов, когда уровень языка ниже Java 7.
  • Подсвечивает, когда методы по умолчанию не переопределены и уровень языка ниже Java 8.
  • Подсвечивает использование API, когда уровень языка ниже того, который указан в документации с помощью тега @since.

Используйте опцию Запретить использование API для запрета использования API на уровне проекта или пользовательском уровне языка.
