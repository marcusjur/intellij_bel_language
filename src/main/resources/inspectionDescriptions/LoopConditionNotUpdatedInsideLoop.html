Сообщает о любых переменных и параметрах, которые используются в условиях цикла, но не обновляются внутри цикла.

Такие переменные и параметры обычно используются ошибочно, так как их выполнение может привести к бесконечному циклу.

Пример:


  void loopDoesNotLoop(boolean b) {
    while (b) {
      System.out.println();
      break;
    }
  }

Настройка проверки:

Используйте опцию Игнорировать возможные нелокальные изменения, если условие может обновляться косвенно (например, через вызываемые методы или при параллельном обновлении из другого потока).
