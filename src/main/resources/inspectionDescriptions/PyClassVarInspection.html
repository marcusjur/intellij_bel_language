Сообщает о некорректном использовании аннотации ClassVar.

Пример:


from typing import ClassVar
class Cat:
    color: ClassVar[str] = "white"
    weight: int
    def __init__(self, weight: int):
        self.weight = weight
Cat.color = "black"  # OK
my_cat = Cat(5)
my_cat.color = "gray"  # Ошибка: установка значения классовой переменной для экземпляра
