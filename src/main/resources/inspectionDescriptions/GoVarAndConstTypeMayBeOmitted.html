Сообщает о типах переменных и констант, которые можно опустить, так как они могут быть выведены компилятором. Такое избыточное указание типов можно опустить для улучшения читаемости кода.

Пример:

var s string = fmt.Sprintln("hi")

Тип string в объявлении переменной можно опустить. Чтобы удалить этот тип, используйте быстрое исправление Удалить тип. После применения быстрого исправления:

var s = fmt.Sprintln("hi")
