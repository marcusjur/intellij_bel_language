Сообщает об использовании функций и методов Python, помеченных как устаревшие и вызывающих предупреждения DeprecationWarning или PendingDeprecationWarning.

Кроме того, проверка подсвечивает использование декораторов abc.abstractstaticmethod, abc.abstractproperty и abc.abstractclassmethod.

Пример:


class Foo:
    @property
    def bar(self):
        import warnings
        warnings.warn("this is deprecated", DeprecationWarning, 2)
        return 5
foo = Foo()
print(foo.bar)
