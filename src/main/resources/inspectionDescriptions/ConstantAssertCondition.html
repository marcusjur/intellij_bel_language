Сообщает об условиях в операторах assert, которые являются константами. Операторы assert с постоянными условиями либо всегда проваливаются, либо всегда выполняются успешно. Такие операторы могли остаться после рефакторинга и, возможно, не являются преднамеренными.

Пример:


    void foo() {
        assert true;
    }
