Цикл for не содержит ни инициализации, ни обновления компонентов. Рекомендуется заменить этот цикл на более простой оператор while.

Пример:


  for(; exitCondition(); ) {
    process();
  }

После применения быстрого исправления результат будет выглядеть следующим образом:


  while(exitCondition()) {
    process();
  }
