Обнаружен вызов wait() вне цикла.

wait() обычно используется для приостановки потока до тех пор, пока определённое условие не станет истинным. Поток может быть пробужден по разным причинам, поэтому условие следует проверять после возврата из вызова wait(). Это легко сделать с помощью цикла.

Пример:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      if (count >= 10) wait();
      ++count;
    }
  }

Правильный код должен выглядеть так:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      while (count >= 10) wait();
      ++count;
    }
  }
