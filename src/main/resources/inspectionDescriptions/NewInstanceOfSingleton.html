Сообщает о создании нового экземпляра класса с аннотацией @groovy.lang.Singleton. Такая конструкция может привести к исключению времени выполнения невозможность создания экземпляра синглтона.

Пример:


  @Singleton
  class Foo{
  }
  Foo foo = new Foo()

После применения быстрого исправления:


  @Singleton
  class Foo{
  }
  Foo foo = Foo.instance
