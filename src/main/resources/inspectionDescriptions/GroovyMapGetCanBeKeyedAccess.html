Сообщает о вызовах метода java.util.Map.get(). Такие вызовы могут быть заменены более короткой и явной формой доступа по ключу.

Пример:


def map = ["foo": "bar"]
def str = map.get("foo") // map.get("foo") можно заменить на map["foo"]

После применения быстрого исправления:


def map = ["foo": "bar"]
def str = map["foo"]
