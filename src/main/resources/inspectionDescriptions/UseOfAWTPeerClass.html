Сообщает об использовании классов-пиров AWT. Эти классы представляют собой нативные виджеты оконной системы и не являются переносимыми между различными оконными системами.

Пример:


  import java.awt.peer.ButtonPeer;
  abstract class Sample implements ButtonPeer {
    public void foo() {
        Sample sample;
    }
  }
