Сообщает о повторяющихся именах псевдонимов столбцов в списке SELECT.

Пример (Sybase ASE):

CREATE TABLE t1 (a TEXT, b INT, c INT);
SELECT a AS x, b AS x FROM t1;

Псевдоним x используется для столбцов a и b. Эти назначения подсвечиваются как ошибки, поскольку в Sybase ASE нельзя использовать одинаковые псевдонимы для столбцов.
