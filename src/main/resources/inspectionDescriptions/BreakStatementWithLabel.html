Обнаружен оператор break с меткой.

Операторы break с метками усложняют рефакторинг и могут вызывать путаницу.

Пример:


  void handle(List<String> strs) {
    outer:
    for (String s: strs) {
      for (char ch : s.toCharArray()) {
        if ('s' == ch) break outer;
        handleChar(ch);
      }
    }
  }
