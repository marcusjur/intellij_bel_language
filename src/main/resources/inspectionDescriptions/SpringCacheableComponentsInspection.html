Сообщает об ошибках в аннотациях кэширования: @Cacheable, @CacheEvict, @CachePut, @CacheConfig и других.

Пример:


@org.springframework.stereotype.Component
public class MyCacheManager implements CacheManager {...}
public class MyConfiguration {
   @Cacheable(value = "a",
       cacheResolver ="myCacheManager") // Сообщение "Bean должен быть типа 'org.springframework.cache.interceptor.CacheResolver'"
   public String getCache(String isbn) {...}
   @Cacheable(value = "abc",
   private String getAbc() // Сообщение "Аннотации кэширования должны быть определены для public методов"
   {...}
}
