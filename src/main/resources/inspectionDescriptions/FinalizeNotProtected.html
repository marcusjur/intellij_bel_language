Сообщает о любой реализации метода Object.finalize(), объявленного как public.

Согласно соглашению о методе Object.finalize(), этот метод должен вызываться только сборщиком мусора. Объявление этого метода как public может вводить в заблуждение, так как это подразумевает возможность его вызова из другого кода.

С помощью быстрого исправления метод можно сделать protected, чтобы предотвратить его вызов из других классов.

Пример:


  class X {
    public void finalize() {
      /* ... */
    }
  }

После применения быстрого исправления:


  class X {
    protected void finalize() {
      /* ... */
    }
  }
