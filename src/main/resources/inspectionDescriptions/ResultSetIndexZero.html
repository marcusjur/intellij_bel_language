Обнаружена попытка доступа к колонке 0 в java.sql.ResultSet или java.sql.PreparedStatement. Исторически сложилось, что нумерация колонок в java.sql.ResultSet и java.sql.PreparedStatement начинается не с 0, а с 1, и попытка доступа к колонке 0 является распространенной ошибкой при программовании с JDBC.

Пример:


  String getName(ResultSet rs) {
    return rs.getString(0);
  }
