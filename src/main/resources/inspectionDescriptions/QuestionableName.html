Имена, которые являются проблематичными и не описывают должным образом переменные, методы или классы. Такие имена не помогают понять код и, вероятно, были созданы временно, но впоследствии забыты.

Пример:


  int aa = 42;

Быстрое исправление для переименования предлагается только в редакторе.

Используйте эту опцию, чтобы указать имена, которые должны быть отмечены.
