-Xsource:3 사용 시 Scala 3으로 바꿀 수 있는 Scala 2 구문을 보고합니다.

컴파일러 플래그 -Xsource:3 아래에서 다음 Scala 3 구문은 Scala 2.13.6에서 이미 지원됩니다.

  • 와일드카드 타입의 와일드카드 마커로서 _ 대신 사용된 ?
  • for 컴프리헨션의 패턴 바인딩에 사용된 case
  • * import 문에서 _ 대신 사용된 *
  • import 별칭에서 => 대신 사용된 as 소프트 키워드
  • open 및 infix 제어자가 허용됨
  • 새 Scala 3 vararg 스플라이스 구문
  • 복합 타입에서 with 대신 사용된 &
