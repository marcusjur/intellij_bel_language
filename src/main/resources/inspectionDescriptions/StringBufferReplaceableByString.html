Сообщает о случаях использования StringBuffer, StringBuilder или StringJoiner, которые могут быть заменены конкатенацией String.

Использование конкатенации String делает код короче и проще.

Эта проверка срабатывает только в тех случаях, когда предлагаемая замена не приведет к значительному снижению производительности на современных JVM. Во многих случаях конкатенация String может работать даже эффективнее.

Пример:


  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

После применения быстрого исправления:


  String result = "i = " + i + ";";
  return result;
