Сообщает о случаях использования зарезервированных ключевых слов SQL в качестве имен идентификаторов в запросах.

Пример (Microsoft SQL Server):

CREATE TABLE select (identity INT ИСРNTITY NOT NULL, order INT NOT NULL);

Мы используем select, identity и order в качестве имен таблицы и столбцов. Однако они являются зарезервированными ключевыми словами в Microsoft SQL Server. Поэтому для их использования в качестве имен объектов в запросе необходимо заключить эти идентификаторы в кавычки. Чтобы заключить их в кавычки, вы можете использовать быстрое исправление Заключить идентификатор в кавычки.

После применения быстрого исправления:

CREATE TABLE [select] ([identity] INT ИСРNTITY NOT NULL, [order] INT NOT NULL);
