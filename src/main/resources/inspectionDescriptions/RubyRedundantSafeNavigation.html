Сообщает об использовании оператора безопасной навигации (&.) с non-nil получателями, которые можно игнорировать.

Используйте список игнорируемых получателей для указания вызовов методов, которые должны игнорироваться в качестве получателей (например, Foo#call или Foo::call для методов экземпляра или методов-синглтонов соответственно).

Пример:

value = 1
if value&.even?
  # ...
end
