Сообщает о хвостовой рекурсии, то есть о случаях, когда функция вызывает сама себя в качестве последней операции перед возвратом. Хвостовая рекурсия всегда может быть заменена циклом, который работает значительно быстрее. Некоторые движки JavaScript выполняют такую оптимизацию, а другие — нет. Поэтому решения с хвостовой рекурсией могут иметь кардинально разную производительность в разных средах выполнения.
