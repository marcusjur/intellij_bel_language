Сообщает о попытке чтения неинициализованного типизованного свойства. Такая попытка приведет к TypeError.

Типизованные свойства должны быть инициализованы одним из следующих способов:

  • значением по умолчанию
  • в конструкторе
  • через магический метод __get()
  • непосредственно при использовании

Для получения дополнительной информации см. Uninitialized and Unset Properties (php.net).
