Сообщать о неразрешённых символах в следующих JPA аннотациях и методах API:
  • AttributeOverride
  • AssociationOverride
  • MapsId
  • MapKey
  • Методы Criteria API

Пример:



  @Entity
  @AssociationOverrides({
    @AssociationOverride(name = "missingAttribute", joinColumns = { ... }), // Ошибка: неразрешённый атрибут
    @AssociationOverride(name = "existingAttribute", joinColumns = { ... })
  })
  public class JavaEntity {
    @OneToOne
    AnotherEntity existingAttribute;
  }
