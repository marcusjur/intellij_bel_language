Сообщает о попытках выполнения рефлексивных проверок на наличие нерантайм-аннотаций.

Использование Class.isAnnotationPresent() для проверки аннотаций с политикой удержания SOURCE или CLASS (значение по умолчанию) всегда возвращает отрицательный результат. Эту ошибку легко пропустить.

Пример:


  {
    getClass().isAnnotationPresent(SourceAnnotation.class); //всегда false
  }
  @Retention(RetentionPolicy.SOURCE)
  @interface SourceAnnotation {}
