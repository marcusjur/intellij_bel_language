Сообщает об операторе return с возвращаемым значением в методе __init__ класса.

Пример:


class Sum:
    def __init__(self, a, b):
        self.a = a
        self.b = b
        self.sum = a + b
        return self.sum

Конструктор не должен возвращать какие-либо значения. Метод __init__ должен только инициализовать значения членов нового экземпляра объекта.

Быстрое исправление, предлагаемое ИСР, заключается в удалении оператора return.
