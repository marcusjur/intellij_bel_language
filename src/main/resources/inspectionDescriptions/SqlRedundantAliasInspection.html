Отчёт может содержать перекрывающиеся имена столбцов в таблице и избыточные псевдонимы.

Пример (PostgreSQL):

CREATE TABLE foo(a INT, b INT);
SELECT * FROM foo foo(a, b);
SELECT * FROM foo foo(a);
SELECT * FROM foo foo(x);
SELECT * FROM foo foo(x, y);

Первые два псевдонима используют те же имена столбцов, что и в таблице foo. Они считаются избыточными, так как столбцы имеют одинаковые имена.
