Сообщает о полях, которые не используются в методе toString() класса.

Помогает обнаружить поля, добавленные после последнего обновления метода toString(). Быстрое исправление позволяет заново сгенерировать метод toString().

В диалоговом окне Сгенерировать | toString() можно исключить поля из этой проверки. Если включена опция использовать геттеры при генерации кода, проверка также будет искать проблемы в методах-геттерах.

Пример:


  public class Relevant {
    private String name; // Не используется в toString()
    private int index;
    private int length;
    @Override
    public String toString() {
        return "Relevant{" + "index=" + index +
          ", length=" + length + '}';
    }
  }

После применения быстрого исправления:


  public class Relevant {
    private String name;
    private int index;
    private int length;
    @Override
    public String toString() {
        return "Relevant{" + "name='" + name + '\'' +
          ", index=" + index + ", length=" + length + '}';
    }
  }
