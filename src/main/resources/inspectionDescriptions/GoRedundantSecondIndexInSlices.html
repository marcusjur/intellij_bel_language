Сообщает о избыточном втором индексе (верхней границе) в срезах.

Обычно второй индекс является необязательным. Его удаление может улучшить читаемость кода.

Пример:

var a []int
a = a[0:len(a)] // `a[0:len(a)]` эквивалентно `a[0:]`

Вы можете применить быстрое исправление Удалить избыточный индекс. После применения быстрого исправления код будет выглядеть так:

var a []int
a = a[0:]
