Сообщает о любых директивах include в JSP-файле, которые ссылаются на тот же JSP-файл.

Пример JSP-файла с именем main.jsp:


  <jsp:include page="./main.jsp" /> // Ошибка: попытка включить содержащий файл
  ...

Примечание: такое самовключение приведет к бесконечной рекурсии, что делает невозможной компиляцию JSP-файла.
