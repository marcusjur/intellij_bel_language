Обнаружено поле логгера, не объявленное как static и/или final. Обеспечение того, чтобы каждый логгер класса был действительной константой и привязан к этому классу, упрощает задачу предоставления единой реализации логования для приложения.

Доступно быстрое исправление для изменения модификаторов логгера на static final.

Пример:


  public class Significant {
    private Logger LOG = Logger.getLogger(Critical.class);
  }

После применения быстрого исправления:


  public class Significant {
    private static final Logger LOG = Logger.getLogger(Critical.class);
  }

Настройка проверки:

  • В таблице имён классов логгеров можно указать имена классов логгеров. Проверка будет сообщать о полях, которые не являются static и final, и тип которых соответствует одному из указанных имён классов.
