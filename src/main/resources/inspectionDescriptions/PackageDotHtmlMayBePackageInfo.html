Отчёт для записи любых файлов package.html в пакете программного обеспечения.

Начиная с JDK 1.5, рекомендуется использовать файл package-info.java, так как он также может содержать аннотации пакета. Таким образом, package-info.java становится единым хранилищем для аннотаций и документации на уровне пакета.

Пример: package.html


<html>
  <body>
    Documentation example.
  </body>
</html>

После применения быстрого исправления: package-info.java


/**
 * Documentation example.
 */
package com.sample;
