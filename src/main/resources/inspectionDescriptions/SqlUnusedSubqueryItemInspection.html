Показывает неиспользуемые столбцы, псевдонимы и другие элементы подзапросов во внешних запросах.

Пример (PostgreSQL):

CREATE TABLE for_subquery(id INT);
SELECT a, q FROM (SELECT 1 AS a, 10 AS b, 2 + 3 AS q, id
      FROM for_subquery) x;

Мы используем псевдонимы a и q из подзапроса. Однако псевдоним b и столбец id не используются во внешнем SELECT-запросе. Поэтому b и id будут отображаться серым цветом.
