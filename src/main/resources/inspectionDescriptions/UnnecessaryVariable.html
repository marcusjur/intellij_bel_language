Сообщает о локальных переменных, которые используются только в следующем операторе return или в точной копии другой переменной.

Такие переменные можно безопасно встроить для улучшения читаемости кода.

Пример:


  fun sum(a: Int, b: Int): Int {
      val c = a + b
      return c
  }

После применения быстрого исправления:


  fun sum(a: Int, b: Int): Int {
      return a + b
  }

Настройка проверки:

Опция Сообщать о переменных с немедленным возвратом позволяет отслеживать переменные, которые сразу возвращаются. Такие переменные могут улучшить читаемость кода в некоторых случаях, когда им даются описательные имена — именно поэтому эта опция по умолчанию отключена.
