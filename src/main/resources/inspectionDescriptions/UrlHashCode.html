Сообщает о вызовах методов hashCode() и equals() для объектов java.net.URL, а также о добавлении объектов URL в отображения и коллекции.

Методы equals() и hashCode() класса URL могут выполнять DNS-запросы для разрешения имени хоста. Это может привести к значительным задержкам в зависимости от доступности и скорости работы сети и DNS-серверов. Использование java.net.URI вместо java.net.URL позволит избежать DNS-запросов.

Пример:


  boolean urlEquals(URL url1, URL url2) {
      return url1.equals(url2);
  }
