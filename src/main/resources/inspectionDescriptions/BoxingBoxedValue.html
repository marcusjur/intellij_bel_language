Сообщает об упаковке уже упакованных значений.

Это избыточная операция, поскольку любое упакованное значение сначала автоматически распаковывается, а затем снова упаковывается. Такой код может вызвать проблемы с производительностью, если он выполняется внутри циклов.

Пример:


  Integer value = 1;
  method(Integer.valueOf(value));

После применения быстрого исправления:


  Integer value = 1;
  method(value);
