Сообщает о выражениях с ветками then и else, которые достаточно похожи, чтобы выражение можно было переместить внутрь. Эта операция сокращает код.

Пример:


  double g(int a, int b) {
    return a == b ? Math.cos(0) : Math.cos(1);
  }

После применения быстрого исправления:


  double g(int a, int b) {
    return Math.cos(a == b ? 0 : 1);
  }

Новое в версии 2017.2
