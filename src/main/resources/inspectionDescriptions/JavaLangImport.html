Сообщает об операторах import для пакета java.lang.

Классы из java.lang всегда импортуются неявно, поэтому такие операторы import избыточны и могут вводить в заблуждение.

IntelliJ ИСРA может автоматически обнаруживать и исправлять такие операторы с помощью команды Оптимизовать импорты. Эта проверка особенно полезна для автономного анализа кодовой базы, в которую вы не планируете вносить изменения.
