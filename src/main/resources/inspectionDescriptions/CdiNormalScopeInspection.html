Сообщает об управляемых бинах, имеющих публичные поля экземпляра, которые находятся вне стандартной области видимости @Dependent. Согласно спецификации CDI, контекстные области видимости (@RequestScoped, @ApplicationScoped, @SessionScoped и ConversationScoped) не допускаются для управляемых бинов с публичными полями экземпляра.

Пример:


  @ApplicationScoped // Предупреждение: бин с нестандартной областью видимости не должен иметь публичных полей
  class Car {
    public String engine;
  }
