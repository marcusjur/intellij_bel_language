Сообщать об объявлениях переменных, которые перекрываются с объявлениями во внешней области видимости.

Такие ситуации могут вызвать путаницу и привести к неожиданным последствиям, поскольку значение переменной зависит от области видимости.

Пример:

for i := 0; i < len(nums); i++ {
    for i := 0; i < len(nums); i++ {
    }
  }

Переменная i во вложенном цикле скрывает внешнюю переменную. Чтобы устранить сокрытие, рекомендуется переименовать переменную во вложенном цикле.

for i := 0; i < len(nums); i++ {
    for j := 0; j < len(nums); j++ {
    }
  }
