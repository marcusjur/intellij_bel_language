Обнаружены избыточные условия ограничения строк в запросе, такие как FETCH и LIMIT.

Пример (PostgreSQL):

CREATE TABLE foo(a INT);
SELECT * FROM foo WHERE EXISTS(SELECT * FROM foo LIMIT 2);
SELECT * FROM foo WHERE EXISTS(SELECT * FROM foo FETCH FIRST 2 ROWS ONLY);

Для исправления предупреждения можно добавить OFFSET к условию ограничения. Без OFFSET использование LIMIT является избыточным, так как оно не влияет на результат выполнения EXISTS. В случае с OFFSET мы пропускаем первые N строк, что влияет на результат.

SELECT * FROM foo WHERE EXISTS(SELECT * FROM foo OFFSET 1 ROW LIMIT 2);
SELECT * FROM foo WHERE EXISTS(SELECT * FROM foo OFFSET 1 ROW FETCH FIRST 2 ROWS ONLY);
