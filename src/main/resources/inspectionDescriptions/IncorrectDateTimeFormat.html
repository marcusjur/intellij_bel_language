Обнаружение некорректных шаблонов форматования даты и времени.

Сообщает о следующих ошибках:

  • Неподдерживаемые символы шаблона, например "TT"
  • Использование зарезервированных символов, например "#"
  • Неправильное использование заполнителей
  • Несбалансированные квадратные скобки
  • Неверное количество последовательных символов шаблона

Примеры:


  DateTimeFormatter.ofPattern("[][]]"); // Отсутствует левая '[' перед правой ']'
  DateTimeFormatter.ofPattern("TT"); // Недопустимый символ шаблона 'T'
  DateTimeFormatter.ofPattern("{"); // Использование зарезервированного символа '{'
  DateTimeFormatter.ofPattern("MMMMMM"); // Слишком много последовательных символов 'M'
