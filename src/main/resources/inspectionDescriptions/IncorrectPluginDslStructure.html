Обнаружены недопустимые операторы перед блоком 'plugins {}'.

Из-за ограничений DSL плагинов Gradle, перед блоком 'plugins {}' разрешен только ограниченный набор операторов Groovy. Единственные допустимые варианты - это 'buildscript {}', 'pluginManagement {}' и другие 'plugins {}'.
См. документацию Gradle

Пример:


import foo.bar.Baz
plugins {} // 报告 'plugins'
plugins {
  foo() // 报告 'foo'
  id 'java'
}
