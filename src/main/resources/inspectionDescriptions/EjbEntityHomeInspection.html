Сообщает о некорректных определениях следующих элементов, связанных с EJB:
  • Класс не реализует ожидаемые интерфейсы (EJBLocalObject, EJBObject и т.д.)
  • Некорректные модификаторы класса EJB
  • Класс не определяет публичный конструктор без параметров
  • Использование класса вместо интерфейса или интерфейса вместо класса

Пример:


public final class SessionEJB implements javax.ejb.SessionBean {
  private SessionEJB() {} // Ошибка: класс должен определять публичный конструктор без параметров
}
