Сообщает об использовании атрибута #[Override] для методов, у которых нет родительских методов для переопределения.

PHP 8.3 и более поздние версии поддерживают использование атрибута #[Override] для выражения намерений в коде. Для получения дополнительной информации см. PHP RFC: Marking overridden methods (wiki.php.net).
