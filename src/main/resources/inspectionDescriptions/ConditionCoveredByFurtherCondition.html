Сообщает об избыточных условиях, которые полностью перекрываются последующими условиями.

Например, в условии value != -1 && value > 0 первая часть избыточна: если она false, то вторая часть тоже false. Или в условии вида obj != null && obj instanceof String проверка на null избыточна, поскольку оператор instanceof подразумевает не-null значение.

Новое в версии 2018.3
