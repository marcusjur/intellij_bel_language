Сообщает о вызовах java.util.Random.nextDouble(), которые используются для создания положительных целых чисел путем умножения результата вызова на коэффициент и преобразования в целое число.

Для генерации случайных положительных целых чисел в заданном диапазоне метод java.util.Random.nextInt(int) является более простым и эффективным.

Пример:


  int getRandomInt() {
    return (int) ((new Random()).nextDouble() * SIZE);
  }

После применения быстрого исправления:


  int getRandomInt() {
    return (new Random()).nextInt(SIZE);
  }
