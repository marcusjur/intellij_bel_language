Сообщает о выражениях, которые передаются как аргументы параметру @Pattern и возвращаются из методов с аннотацией @Pattern, не соответствующих указанному шаблону.

Пример:


  @Pattern("\\d\\d-\\d\\d\\d") String getId() {
    return "1";
  }

Используя опцию Отмечать выражения, не являющиеся константами времени компиляции, проверка будет сообщать о выражениях с неизвестными значениями и предлагать добавить аннотацию замены (@Subst)
