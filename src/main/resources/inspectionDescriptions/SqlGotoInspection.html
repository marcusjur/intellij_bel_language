Сообщает об использовании оператора GOTO после метки и для выхода из цикла.

Частое использование оператора GOTO обычно не рекомендуется. Подробнее см. GOTO statement in SQL procedures на ibm.com.

Рекомендуется использовать циклы вместо GOTO для возврата к предыдущему оператору.

Рекомендуется использовать другие операторы управления потоком выполнения, такие как RETURN или BREAK, вместо GOTO для выхода из цикла WHILE.

Пример (Oracle):

CREATE PROCEDURE test(n INT) AS
DECLARE
    x INT;
BEGIN
    x := 0;
    GOTO a;
    <<a>> x := 1;
    IF (n = 0) THEN
        GOTO a;
    END IF;
    WHILE TRUE
        LOOP
            GOTO b;
        END LOOP;
    <<b>> x := 3;
END;
