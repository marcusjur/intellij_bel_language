Сообщает о прямом сравнении ошибок с помощью == и предлагает использовать вместо этого errors.Is.

Начиная с Go 1.13, для обёртывания ошибок можно использовать функцию fmt.Errorf с глаголом %w. Следовательно, прямое сравнение ошибок с помощью оператора равенства не будет работать для обёрнутых ошибок. Предпочтительным способом проверки конкретной ошибки является использование функции errors.Is из стандартной библиотеки, так как эта функция обходит цепочку обёрнутых ошибок при проверке.

Для получения подробной информации о errors.Is см. working with Errors in Go 1.13.

Это быстрое исправление заменяет прямое сравнение ошибок на вызов errors.Is.

Пример:


if err == MyError {
  fmt.Println("MyErr")
}

После применения быстрого исправления:


if errors.Is(err, MyError) {
  fmt.Println("MyErr")
}
