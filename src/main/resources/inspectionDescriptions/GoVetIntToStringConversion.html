Сообщает о преобразованиях вида string(x), где x является целым числом, но не является байтом или руной.

Такие преобразования не рекомендуются, поскольку они возвращают UTF-8 представление кодовой точки Unicode x, а не десятичное строковое представление x, как можно было бы ожидать. Кроме того, если x представляет недопустимую кодовую точку, преобразование не может быть отклонено статически.

Для преобразований, где предполагается использование кодовых точек, рассмотрите возможность замены на string(rune(x)). В противном случае, strconv.Itoa и эквивалентные ему функции вернут строковое представление значения должным образом.

Пример:

func main() {
 a := 1
 _ = string(a)
}

После применения быстрого исправления преобразование целого числа в руну:

func main() {
 a := 1
 _ = string(rune(a))
}
