Сообщает о методе clone() без модификатора final.

Метод clone() может использоваться для создания экземпляров объектов без использования конструктора, поэтому возможность переопределения метода clone() может привести к повреждению объекта или даже к уязвимостям безопасности. Этого можно избежать, объявив метод clone() или сам включающий класс как final.

Пример:


  class Main implements Cloneable {
      @Override
      protected Object clone() throws CloneNotSupportedException {
        return super.clone();
      }
  }
