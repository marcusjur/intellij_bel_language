Сообщает о возможности преобразования функций области видимости (let, run, apply, also).

Использование соответствующей функции может упростить код.

Это быстрое исправление позволяет заменять одни функции области видимости на другие.

Пример:


  val x = "".let {
      it.length
  }

После применения быстрого исправления:


  val x = "".run {
      length
  }
