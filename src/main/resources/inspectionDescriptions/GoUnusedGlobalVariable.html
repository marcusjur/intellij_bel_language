Сообщает о глобальных переменных, которые объявлены, но никогда не используются в коде.

Код не скомпилируется, если у вас есть неиспользуемые переменные. Подробнее о неиспользуемых переменных и импортах см. в разделе Unused imports and variables на go.dev. 

func main() {
 a := 422
}

Код в примере не скомпилируется. Поэтому он будет подсвечен как ошибка. Для таких случаев доступны два быстрых исправления: удалить переменную и переименовать в _. Первое исправление удалит переменную, а второе преобразует её в пустой идентификатор.

После применения быстрого исправления переименовать в _:

func main() {
 _ := 422
}
