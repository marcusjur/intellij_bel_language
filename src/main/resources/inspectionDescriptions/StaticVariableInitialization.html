Сообщает о возможно неинициализованных static переменных при инициализации класса.

Пример:


  class Foo {
    public static int bar;
    static { }
  }

Обратите внимание, что эта проверка использует очень консервативный алгоритм анализа потока данных и может ошибочно сообщать о static переменных как о неинициализованных. Переменные, помеченные как инициализованные, всегда будут инициализованы.

Используйте опцию Игнорировать примитивные поля, чтобы игнорировать неинициализованные примитивные поля.
