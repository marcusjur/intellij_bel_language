Сообщает об операторах throw, которые выбрасывают некорректные исключения. Например, исключение может быть некорректным из-за слишком общего типа, такого как java.lang.Exception или java.io.IOException.

Пример:


  void setup(Mode mode) {
      if (mode == null)
          throw new RuntimeException("Problem during setup"); // Предупреждение: выброшено запрещенное исключение 'RuntimeException'
      ...
  }

Используйте список запрещенных исключений, чтобы указать, о каких исключениях следует сообщать.
