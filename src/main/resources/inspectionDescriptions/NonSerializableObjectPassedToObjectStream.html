Сообщение о нессериализуемом объекте (не реализующем Serializable), используемом в качестве аргумента для java.io.ObjectOutputStream.write(). Такие вызовы приведут к исключению во время выполнения.

Эта проверка предполагает, что объекты типов java.util.Collection и java.util.Map являются сериализуемыми (Serializable), если только их типы явно не объявлены как несериализуемые.

Пример:


  public class IWantToSerializeThis {
    public static void main(String[] args) throws IOException {
      try(var stream = new ObjectOutputStream(Files.newOutputStream(Paths.get("output")))) {
        // Предупреждение -- завершится неудачей с исключением NotSerializableException
        stream.writeObject(new IWantToSerializeThis());
      }
    }
  }
