Обнаруживает операции конкатенации строк с пустыми строками. Конкатенация с пустой строкой может использоваться для преобразования объектов не-String или примитивов в String, но использование метода String.valueOf() делает код более понятным.

Рекомендуется использовать быстрое исправление для упрощения конкатенации.

Пример:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

После применения быстрого исправления:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

Опция Сообщать только о случаях, когда пустая строка может быть удалена без дополнительных изменений позволяет игнорировать случаи, когда удаление пустой строки потребует добавления преобразования String.valueOf() для другого операнда.
