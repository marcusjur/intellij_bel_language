Сообщает о вызовах filter… для уже отфильтрованных коллекций в стандартной библиотеке.

Функции стандартной библиотеки, такие как filterNotNull() или filterIsInstance, имеют смысл только при вызове на приемнике с типом, отличным от типа результата. В остальных случаях такие вызовы можно игнорировать, так как результат будет тем же самым.

Удаление избыточного вызова доступно как быстрое исправление для автоматической корректировки кода.

Пример:


  fun test(list: List<String>) {
      val x = list.filterNotNull() // Быстрое исправление упростит до 'list'
      val y = list.filterIsInstance<String>() // Быстрое исправление упростит до 'list'
  }
