Сообщает о строках форматования, которые не соответствуют стандартному синтаксису Java.

По умолчанию проверка рассматривает строки форматования, если константы времени компиляции используются в качестве аргументов соответствующих методов в java.util.Formatter, java.lang.String, java.io.PrintWriter или java.io.PrintStream.

Пример:


  String.format("x = %d, y = %d", 42);

В настройках проверки можно пометить другие классы и методы как связанные с форматованием строк.

В качестве альтернативы вы можете использовать аннотацию org.intellij.lang.annotations.PrintFormat для пометки параметров метода как строк форматования. В этом случае параметры форматования должны следовать сразу после строки форматования и должны быть последними параметрами метода. Пример:


  void myFormatMethod(int mode, @PrintFormat String formatString, Object... args) {...}

Методы, аннотированные таким образом, также будут распознаваться этой проверкой.
