Сообщение о неправильном использовании фреймворка Python Behave BDD:

  • В файле определения шагов отсутствует from behave import *.
  • Директория с определениями шагов не названа steps.

В случае возникновения одной из этих ситуаций, ИСР подсветит соответствующие шаги в файле .feature. Вы можете применить быстрое исправление для создания определения шага или переименования директории, содержащей этот шаг.

Для получения дополнительной информации см. создание определения шага.
