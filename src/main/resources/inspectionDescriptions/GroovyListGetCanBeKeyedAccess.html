Сообщает о вызовах метода java.util.List.get(). Такие вызовы могут быть заменены более короткой и явной формой доступа по индексу.

Пример:


def list = ["foo"]
def str = list.get(0) // list.get(0) можно заменить на list[0]

После применения быстрого исправления:


def list = ["foo"]
def str = list[0]
