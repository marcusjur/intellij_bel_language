Значение атрибута parameters содержит имя параметра метода, которое не определено в аннотациях Micronaut Cache.

Поддерживаются следующие аннотации:

  • io.micronaut.cache.annotation.Cacheable
  • io.micronaut.cache.annotation.CachePut
  • io.micronaut.cache.annotation.CacheInvalidate

Пример:


  @CachePut(parameters = "myParameter") // Ошибка: не удаётся разрешить символ 'myParameter'
  public String cachedMethod(String name_is_not_equal_to_myParameter) {
    return "...";
  }
