Сообщает о вызовах equals(), которые можно заменить на выражения == или != без изменения семантики.

Такие вызовы можно заменить, когда они используются для сравнения final классов, которые не имеют собственной реализации equals() и используют стандартный метод Object.equals(). Такая замена может улучшить производительность.

Вызовы equals() для значений enum проверяются отдельно: Использование 'equals()' для значений Enum.
