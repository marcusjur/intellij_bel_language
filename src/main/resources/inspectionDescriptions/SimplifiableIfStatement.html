Сообщение предлагает заменить оператор if условным выражением с использованием операторов &&, ||, ==, != или ?:.

Результат обычно короче, но не всегда более понятен, поэтому применение этого исправления рекомендуется не во всех случаях.

Пример:


  if (condition) return true; else return foo;

После применения быстрого исправления:


  return condition || foo;

Настройка проверки:

  • Используйте опцию Не предлагать оператор '?:', чтобы отключить предупреждения при предложении использовать условный оператор. В этом случае будут подсвечиваться только предложения с операторами &&, ||, == и !=. Быстрое исправление останется доступным в редакторе.
  • Используйте опцию Игнорировать цепочки операторов 'if', чтобы отключить предупреждения для цепочек if-else. Быстрое исправление останется доступным в редакторе.
