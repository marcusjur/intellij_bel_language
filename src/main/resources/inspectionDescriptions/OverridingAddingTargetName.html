@targetName 어노테이션을 가진 정의 중 @targetName 어노테이션이 없는 정의를 재정의하는 정의를 보고합니다.

@targetName 어노테이션을 참조하세요.

빠른 수정에서는 @targetName 어노테이션을 제거합니다.

예:


  class A:
    def foo: Int = 1

  class B extends A:
    @targetName("overriddenFoo")
    override def foo: Int = 2

빠른 수정 적용 후:


  class A:
    def foo: Int = 1

  class B extends A:
    override def foo: Int = 2
