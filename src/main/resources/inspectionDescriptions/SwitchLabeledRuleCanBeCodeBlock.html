Сообщает о правилах выражения switch или расширенного оператора switch с телом выражения. Они могут быть преобразованы в блоки кода.

Пример:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

После применения быстрого исправления:


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };

Новое в версии 2019.1
