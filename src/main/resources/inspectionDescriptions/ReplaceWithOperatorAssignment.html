Сообщает о модификациях переменных с простым присваиванием (например, y = y + x), которые можно заменить на присваивание с операцией.

Быстрое исправление заменит простое присваивание на присваивание с операцией.

Пример:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list = list + 4
  }

После применения быстрого исправления:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list += 4
  }
