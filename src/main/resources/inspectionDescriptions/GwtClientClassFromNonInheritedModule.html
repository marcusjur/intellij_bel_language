Сообщает об использовании клиентского кода из другого модуля GWT, который не унаследован в текущем модуле.

Клиентский код из одного модуля GWT может использовать код из другого модуля GWT только при наличии соответствующего тега inherits в файле gwt.xml.

Быстрое исправление добавит необходимый тег inherits в файл gwt.xml.
