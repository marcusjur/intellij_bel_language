Сообщать об изменениях исходного кода объектов базы данных.

Проверка запускается при выполнении интроспекции базы данных или объекта. Интроспекция выполняется при открытии исходного кода объекта, выполнении операторов и проведении рефакторинга кода. Кроме того, вы можете запустить интроспекцию, щелкнув правой кнопкой мыши по объекту и выбрав Обновить.

Проверка включает следующие случаи:

  • Исходный код объекта в базе данных был изменен, но код в редакторе не обновлен. Применимо для PostgreSQL, Microsoft SQL Server, Oracle и Sybase ASE.
  • Вы изменили исходный код объекта, выполнили интроспекцию базы данных, но исходный код был изменен кем-то другим.
  • Интроспектор базы данных в ИСР был обновлен, и вам необходимо загрузить новые свойства объектов, которые отсутствовали в предыдущей версии интроспектора.
