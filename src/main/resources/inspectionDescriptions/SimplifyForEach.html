Сообщается о возможности замены на более лаконичный метод или извлечения промежуточных шагов из вызова forEach().

Пример:


  List<String> findNStrings(List<String> list, int n) {
    List<String> other = new ArrayList<>();
    list.forEach(s -> {
      if(s.length() > n) other.add(s);
    });
    return other;
  }

После применения быстрого исправления:


  List<String> findNStrings(List<String> list, int n) {
    List<String> other = list.stream()
      .filter(s -> s.length() > n)
      .collect(Collectors.toList());
    return other;
  }
