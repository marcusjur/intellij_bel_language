Сообщает о выделении массивов с известной нулевой длиной.

Поскольку длина массивов в Java неизменяема, практически всегда можно использовать общий массив нулевой длины вместо того, чтобы многократно выделять новые массивы. Такое совместное использование может обеспечить полезную оптимизацию времени выполнения и занимаемой памяти.

Обратите внимание, что проверка не сообщает о массивах нулевой длины, выделенных для static final полей, поскольку предполагается, что эти массивы используются для реализации совместного использования массивов.
