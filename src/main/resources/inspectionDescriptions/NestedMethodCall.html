Сообщает о вызовах методов, которые используются в качестве параметров для других вызовов методов.

Быстрое исправление вводит переменную, что делает код более простым и удобным для отладки.

Пример:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }
  public void foo() {
      int x = f(y());
  }

После применения быстрого исправления:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }
  public void foo() {
      int y = y();
      int x = f(y);
  }

С помощью параметров проверки можно включать или отключать отчёты для:

  • Вызовов методов в инициализаторах полей
  • Вызовов статических методов
  • Вызовов простых геттеров
