Сообщает о неполных операторах switch для перечислений.

Пример:


  enum AlphaBetaGamma {
    A, B, C;
    void x(AlphaBetaGamma e) {
      switch (e) {
      }
    }
  }

После применения быстрого исправления:


  enum AlphaBetaGamma {
    A, B, C;
    void x(AlphaBetaGamma e) {
      switch (e) {
        case A -> {}
        case B -> {}
        case C -> {}
      }
    }
  }

Используйте опцию Игнорировать switch с ветвью по умолчанию, чтобы пропускать операторы switch с ветвью default.
