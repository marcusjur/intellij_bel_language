Сообщает об использовании тега <code> в комментариях Javadoc. Начиная с Java 5, эти теги могут быть заменены конструкцией {@code ...}. Это позволяет использовать угловые скобки < и > в комментариях вместо HTML-сущностей.

Пример:


  /**
   * @return empty <code>ArrayList&lt;Integer&gt;</code>
   */
  List<Integer> getList(){ ... }

После применения быстрого исправления:


  /**
   * @return empty {@code ArrayList<Integer>}
   */
  List<Integer> getList(){ ... }
