Сообщается о возможности замены на упрощенный вызов библиотечной функции.

Использование соответствующей функции может сделать код проще.

Это быстрое исправление заменяет один вызов функции на другой.

Пример:


  fun test(list: List<Any>) {
      list.filter { it is String }
  }

После применения быстрого исправления:


  fun test(list: List<Any>) {
      list.filterIsInstance<String>()
  }
