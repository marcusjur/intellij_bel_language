Сообщает о конструкторах, которые не начинаются с вызова конструктора "super" или другого конструктора того же класса.

Можно считать, что такие конструкторы неявно начинаются с вызова super(). Некоторые стандарты кодования предпочитают явный вызов super().

Пример:


  class Foo {
    Foo() {}
  }

После применения быстрого исправления:


  class Foo {
    Foo() {
      super();
    }
  }

В настройках проверки можно игнорировать классы, которые напрямую наследуются от Object. Например: 


  class Foo {
    Foo() {} // не сообщается
  }
  class Bar extends Foo {
    Bar() {} // неявный вызов 'super()'
  }
