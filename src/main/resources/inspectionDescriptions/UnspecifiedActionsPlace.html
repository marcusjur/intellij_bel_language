Сообщать о случаях передачи неопределённого параметра place в методы ActionManager.createActionToolbar() и ActionManager.createActionPopupMenu().

Необходимо указывать корректное значение place, чтобы различать использование действий в методах update()/actionPerformed() через AnActionEvent.getPlace().

Примеры: 


// Неправильно:
actionManager.createActionToolbar("", group, false);
actionManager.createActionToolbar("unknown", group, false);
actionManager.createActionPopupMenu(ActionPlaces.UNKNOWN, group);
// Правильно:
actionManager.createActionToolbar("MyPlace", group, false);
actionManager.createActionPopupMenu(ActionPlaces.EDITOR_TOOLBAR, group);
