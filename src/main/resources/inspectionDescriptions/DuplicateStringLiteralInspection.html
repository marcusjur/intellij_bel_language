Отчёт о строковых литералах, которые дублируются в проекте без изменений. Предлагается два варианта быстрого исправления. Первый - создание константы для повторяющейся строки и её использование во всем проекте, второй - отображение всех мест, где встречается конкретный строковый литерал.

Пример:


  class C1 { String CONST1 = "duplicate string"; }
  class C2 { String CONST2 = "duplicate string"; }

Настройка проверки:

  • Используйте поле Минимальная длина строки для установки минимальной длины строки, необходимой для обнаружения дубликатов.
  • Используйте опцию Игнорировать выражения @PropertyKey для исключения строк, передаваемых в качестве аргументов методам с аннотацией org.jetbrains.annotations.PropertyKey.
