Сообщает о внутренних классах в interface.

Некоторые стандарты кодования не рекомендуют использовать такие классы. Эта проверка не распространяется на перечисления (enum) и аннотации.

Используйте опцию игнорировать внутренние интерфейсы, чтобы исключить внутренние интерфейсы из проверки. Например: 


  interface I {
    interface Inner {
    }
  }
