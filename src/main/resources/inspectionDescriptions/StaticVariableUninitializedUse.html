Сообщает о чтении переменных static до их инициализации.

Эта проверка игнорирует проверки на равенство с null.

Пример:


  class Foo {
    public static int bar;
    public static void main(String[] args) {
      System.out.println(bar);
    }
  }

Обратите внимание, что эта проверка использует очень консервативный алгоритм анализа потока данных и может ошибочно сообщать о неинициализованных переменных static. Переменные, о которых сообщается как об инициализованных, всегда будут инициализованы.

Используйте опцию Игнорировать примитивные поля, чтобы игнорировать неинициализованные примитивные поля.
