Сообщать о любых ссылках на классы из пакета по умолчанию в JSP файлах.

Пример:


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // Ошибка
    new com.example.GoodClass(...); // OK
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

Примечание: Такие ссылки не будут разрешены во время компиляции JSP, поскольку скомпилированные классы JSP обычно помещаются в непустой пакет (например, "org.apache.jsp" для Tomcat или "com.ibm._jsp" для WebSphere), а в Java импорт из пакета по умолчанию не разрешен.
