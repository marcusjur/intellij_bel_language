Сообщает о проблеме определения атрибутов экземпляра класса вне метода __init__.

Пример:


    class Book:
    def __init__(self):
        self.author = 'Mark Twain'
    def release(self):
        self.year = '1889'

После применения быстрого исправления код будет выглядеть так: 


    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'
    def release(self):
        pass
