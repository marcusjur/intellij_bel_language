Сообщает о неразрешённых файлах и директориях в атрибуте locations аннотации @ImportResource и соответствующих псевдонимах мета-аннотации @ImportResource.

Пример:


  @Configuration
  @ImportResource(locations = "classpath:META-INF/unknown-context.xml") // Сообщение "Не удаётся разрешить файл 'unknown-context.xml'"
  public class MyConfiguration {...}
