Сообщает об использовании API, помеченных аннотацией как нестабильные. Такие API могут быть изменены или удалены в будущих версиях, что приведет к поломке кода, использующего их.

Аннотации, используемые для пометки нестабильных API, показаны в списке ниже.

По умолчанию проверка игнорирует использование нестабильных API, если их объявление находится в исходном коде того же проекта. В таких случаях использование API может быть обновлено при внесении изменений. Однако, если проект большой, это может быть неудобно, поэтому вы можете отключить опцию Игнорировать API, объявленные в этом проекте, чтобы получать уведомления о нестабильных API, объявленных как в исходном коде проекта, так и в библиотеках.
