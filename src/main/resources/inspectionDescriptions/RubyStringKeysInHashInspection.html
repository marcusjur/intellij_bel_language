Обнаружено использование строк в качестве ключей хеша. Рекомендуется использовать символы вместо строк в качестве ключей хеша.

Пример:

# 不良做法
{ 'foo' => 1, 'bar' => 2 }
Все строковые ключи можно преобразовать в символы с помощью быстрого исправления. После применения быстрого исправления результат будет выглядеть так: 
# 优良做法
{ foo: 1, bar: 2 }

На основе 'RuboCop'
