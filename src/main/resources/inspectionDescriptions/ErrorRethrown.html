Сообщает о блоках try, которые перехватывают java.lang.Error или любой его подкласс без повторного выброса ошибки.

Не сообщает о блоках, перехватывающих java.lang.ThreadDeath.

Пример:


  try {
    executeTests(request);
  }
  catch (OutOfMemoryError ex) { // Предупреждение: ошибка 'ex' не выброшена повторно
    return false;
  }
