Сообщает о ссылках в следующих аннотациях Persistence ORM, которые не могут быть разрешены в настроенном источнике данных:
  • Table
  • Column
  • [Any]To[Any]
  • JoinColumn / JoinTable
  • NamedQuery / NativeQuery
  • Embeddable

Рекомендуется обновить источник данных или назначить другой источник данных.

Пример таблицы базы данных:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
);

Пример JPA-сущности:


  @Entity
  @Table(name = "UNKNOWN_TABLE") // Неразрешённая ссылка
  public class JavaEntity {
    @Column(name = "ID") // Невозможно разрешить столбец в неопределённой таблице
    Integer id;
  }

Примечание: Источник данных может быть настроен через действие `Назначить источник данных` или через соответствующий значок на панели привязки сущности
