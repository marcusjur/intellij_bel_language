Сообщает об операторах if, when и try, которые могут быть преобразованы в выражения путем извлечения операторов return или присваивания.

Пример:


  fun foo(arg: Int): String {
      when (arg) {
          0 -> return "Zero"
          1 -> return "One"
          else -> return "Multiple"
      }
  }
После применения быстрого исправления: 

  fun foo(arg: Int): String {
      return when (arg) {
          0 -> "Zero"
          1 -> "One"
          else -> "Multiple"
      }
  }

Если вы хотите, чтобы эта проверка подсвечивала более сложный код с несколькими операторами в ветвях, снимите флажок "Сообщать только когда каждая ветвь содержит единственный оператор".
