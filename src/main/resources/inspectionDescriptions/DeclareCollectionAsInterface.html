Сообщает об объявлениях переменных типа Collection, где используется конкретный класс коллекции вместо соответствующего интерфейса. Предупреждение не выдается, если ослабление типа переменной приведет к ошибке компиляции.

Пример:


  // Предупреждение: используется конкретный класс коллекции ArrayList.
  int getTotalLength(ArrayList<String> list) {
    return list.stream().mapToInt(String::length).sum();
  }
  // Нет предупреждения, так как метод trimToSize()
  // недоступен в интерфейсе List
  void addData(ArrayList<String> data) {
    data.add("Hello");
    data.add("World");
    data.trimToSize();
  }
Рекомендуется использовать соответствующий интерфейс коллекции (например, Collection, Set или List) с помощью быстрого исправления.
