Сообщает об операторах или выражениях switch, содержащих одинаковый код в разных ветках, и предлагает объединить повторяющиеся ветки.

Пример:


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

После применения быстрого исправления:


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
