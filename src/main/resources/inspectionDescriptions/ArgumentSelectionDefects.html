Сообщает о несоответствиях между именами фактических и формальных параметров. Это помогает обнаружить ошибки программования, например когда вы случайно меняете местами аргументы одного типа. Оригинальная исследовательская статья доступна здесь.

Пример:


  const User* get_User(int64_t company_id, int64_t user_id) {
    return new User{company_id, user_id, "foo"};
  }
  void call_User(int64_t company_id, int64_t user_id) {
    const User* user = get_User(user_id, company_id); // здесь потенциальная проблема
    //...
  }
