Сообщает о методах, у которых количество формальных параметров превышает указанное максимальное значение. Большое количество параметров метода может указывать на необходимость рефакторинга.

Не учитываются методы, имеющие super-методы.

Используйте поле лимит параметров для указания максимально допустимого количества параметров метода.
