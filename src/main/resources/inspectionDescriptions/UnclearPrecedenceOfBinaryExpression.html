Сообщение о бинарных выражениях, содержащих различные операторы без круглых скобок.

Такие выражения могут быть трудночитаемыми из-за различных правил приоритета операторов.

Пример:

  fun foo(b: Boolean?, i: Int?) {
      val x = b ?: i == null // вычисляется как `(b ?: i) == null`
      val y = i ?: 0 + 1 // вычисляется как `i ?: (0 + 1)`
  }
