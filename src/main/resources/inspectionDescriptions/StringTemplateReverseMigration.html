Использование обработчика STR для отчётов о строковых шаблонных выражениях с возможностью быстрого исправления для возврата к обычной конкатенации строк.

Пример:


  String name = "Bob";
  String greeting = STR."Hello, \{name}. You are 29 years old.";

После применения быстрого исправления:


  String name = "Bob";
  String greeting = "Hello, " + name + ". You are 29 years old.";

Изменения в версии 2024.2
