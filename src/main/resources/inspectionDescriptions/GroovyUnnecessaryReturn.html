Сообщает об операторах return в конце конструкторов и методов, возвращающих void. Такие вызовы являются излишними и могут быть безопасно удалены.

Пример: 


  void foo (String s){
      print(s)
      return
  }

После применения быстрого исправления:


  void foo (String s){
      print(s)
  }

Дополнительную информацию см. в аналогичной проверке для Java.
