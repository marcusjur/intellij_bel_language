Обнаруживает числовые литералы с подчеркиваниями и предлагает быстрое исправление для их удаления. Это может быть полезно, если вам нужно понизить уровень языка.

Быстрое исправление удаляет подчеркивания в числовых литералах. Например, 1_000_000 будет преобразовано в 1000000.

Числовые литералы с подчеркиваниями были введены в Java 7. Эта проверка помогает обеспечить обратную совместимость с более ранними версиями Java.
