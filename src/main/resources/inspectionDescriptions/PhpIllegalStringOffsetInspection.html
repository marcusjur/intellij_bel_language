Сообщает об использовании нецелочисленного (integer) смещения в выражениях доступа к строкам, например $str[42].

Начиная с PHP 5.4, смещения в строках должны быть целыми числами или строками, представляющими целые числа. Поскольку использование строк в качестве смещения считается плохой практикой, проверка также сообщает о таких случаях.

Для получения дополнительной информации см. String access and modification by character (php.net).
