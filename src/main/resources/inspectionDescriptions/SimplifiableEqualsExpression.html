Сообщает о сравнении с null при вызове equals() с последующим использованием строкового литерала в качестве аргумента.

Пример:


  if (s != null && s.equals("literal")) {}

После применения быстрого исправления:


  if ("literal".equals(s)) {}

С помощью настроек проверки можно также обнаруживать вызовы equals() с нелитеральными аргументами, когда доказано, что аргумент метода equals() не является null.
