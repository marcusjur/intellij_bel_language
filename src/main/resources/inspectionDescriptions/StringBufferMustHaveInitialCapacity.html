Сообщает о попытках создания экземпляров новых объектов StringBuffer или StringBuilder без указания начальной емкости.

Если начальная емкость не указана, используется емкость по умолчанию, что редко является оптимальным выбором. Отсутствие указания начальной емкости для StringBuffer может привести к проблемам с производительностью, когда требуется перераспределение памяти и её копирование при превышении начальной емкости.

Пример:


  // Емкость не указана
  var sb = new StringBuilder();
