Сообщает об операторах return, возвращающих локальные переменные, значения которых вычисляются в другом месте того же метода.

Быстрое исправление встраивает возвращаемую переменную путем перемещения оператора return в место вычисления значения переменной. Когда возвращаемое значение не может быть встроено в оператор return, быстрое исправление пытается переместить оператор return как можно ближе к месту вычисления возвращаемого значения.

Пример:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      n = i;
      break;
    }
  }
  return n;

После применения быстрого исправления:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      return i;
    }
  }
  return n;
