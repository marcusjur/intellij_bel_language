Сообщает о полях, которые не гарантированно инициализуются после десериализации объекта в методе readObject().

Проверка не распространяется на transient-поля.

Примечание: эта проверка использует очень консервативный алгоритм анализа потока управления и может ошибочно сообщать о неинициализованных полях.

Пример:


class DataObject implements Serializable {
  String s; // s не инициализуется в readObject
  int i;
  private void readObject(ObjectInputStream stream) throws IOException {
    i = stream.readInt();
  }
}
