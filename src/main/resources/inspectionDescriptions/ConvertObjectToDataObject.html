Сообщает об object, который может быть преобразован в data object

data object автоматически генерирует методы toString, equals и hashCode

Эта проверка предлагает преобразовать object в data object в следующих двух случаях:

  • Когда пользовательский метод toString возвращает имя класса
  • Когда object наследуется от sealed class/interface

Пример:


    object Foo {
        override fun toString(): String = "Foo"
    }

После применения быстрого исправления:


    data object Foo

Эта проверка выполняется только если уровень языка Kotlin в проекте или модуле 1.9 или выше
