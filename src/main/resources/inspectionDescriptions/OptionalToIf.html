Сообщает о цепочке вызовов Optional, которую можно заменить последовательностью операторов if.

Пример:


  return Optional.ofNullable(name)
    .map(this::extractInitials)
    .map(initials -> initials.toUpperCase(Locale.ENGLISH))
    .orElseGet(this::getDefault);

После применения быстрого исправления:


  if (name != null) {
    String initials = extractInitials(name);
    if (initials != null) return initials.toUpperCase(Locale.ENGLISH);
  }
  return getDefault();

Эта проверка помогает при понижении версии для обеспечения обратной совместимости со старыми версиями Java.
