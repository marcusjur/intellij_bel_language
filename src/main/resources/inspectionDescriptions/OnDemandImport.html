Сообщать о любых операторах import с импортом всего пакета ('* imports').

Некоторые стандарты кодования запрещают такие операторы import.

Вы можете настроить IntelliJ ИСРA для обнаружения и исправления таких операторов с помощью команды Оптимизовать импорты. Перейдите в Настройки | Редактор | Стиль кода | Java | Импорты, убедитесь, что включена опция Использовать импорт отдельных классов, и укажите значения в полях Количество классов для импорта со звёздочкой (*) и Количество статических имён для импорта со звёздочкой (*). Таким образом, эта проверка особенно полезна для автономного анализа кодовой базы, которую не планируется изменять.
