Сообщение о использовании цикла for. Рекомендуется использовать .each вместо цикла for.

Пример:

# 不良做法
for i in 0..5
  puts "Value is #{i}"
end
Цикл for можно преобразовать в цикл .each с помощью быстрого исправления. После применения быстрого исправления результат будет выглядеть так: 
# 优良做法
(0..5).each do |i|
  puts "Value is #{i}"
end
