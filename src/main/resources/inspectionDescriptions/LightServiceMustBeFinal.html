Сообщает о классах с аннотацией @com.intellij.openapi.components.Service, которые не являются final.

Рекомендуется объявлять конкретные классы как final.

Пример:


  // MyService.kt
  @Service(Service.Level.APP)
  open class MyService

После применения быстрого исправления:


  // MyService.kt
  @Service(Service.Level.APP)
  class MyService

Для абстрактных классов или интерфейсов рекомендуется удалить аннотацию @Service.

Пример:


  // MyService.java
  @Service(Service.Level.APP)
  abstract class MyService {}

После применения быстрого исправления:


  // MyService.java
  abstract class MyService {}

Изменения в версии 2023.2
