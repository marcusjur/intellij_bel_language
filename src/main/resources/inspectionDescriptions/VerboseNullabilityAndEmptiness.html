Проверку можно упростить до единой комбинированной проверки на null и пустоту.

Это быстрое исправление заменит выделенные проверки на комбинированный вызов проверки, например isNullOrEmpty().

Пример:


  fun test(list: List<Int>?) {
      if (list == null || list.isEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }

После применения быстрого исправления:


  fun test(list: List<Int>?) {
      if (list.isNullOrEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }
