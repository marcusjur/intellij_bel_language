Сообщать о следующих неразрешённых атрибутах в аннотациях EntityGraph:
  • Значения в @NamedAttributeNode
  • Именованные подграфы в @NamedAttributeNode
  • Имена в @NamedEntityGraph

Пример:



  @Entity
  @NamedEntityGraph(name = "entity-graph", ...)
  public class JavaEntity {
    public void foo() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .getEntityGraph("unknown-entity-graph"); // Неразрешённая ссылка
    }
  }
