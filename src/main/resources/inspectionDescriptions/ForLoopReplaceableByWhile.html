Инспекция обнаруживает циклы for без инициализации и обновления компонентов и предлагает преобразовать их в циклы while. Это делает код более читаемым.

Пример:


  for(; exitCondition(); ) {
    process();
  }

После применения быстрого исправления:


  while(exitCondition()) {
    process();
  }

Быстрое исправление также доступно для других циклов for, поэтому вы можете заменить любой цикл for на цикл while.

Если вы хотите игнорировать циклы for с несущественными или отсутствующими условиями, используйте опцию игнорировать безусловные 'бесконечные' циклы for.
