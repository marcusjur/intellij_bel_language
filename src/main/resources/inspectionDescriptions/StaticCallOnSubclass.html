Сообщает о вызовах статических методов, которые определены через подкласс объявленного класса, а не через сам объявленный класс.

Java допускает такую квалификацию классов, но подобные вызовы могут указывать на неоднозначность между наследованием и переопределением.

Пример:


  class Parent {
    public static void print(String str) {}
  }
  class Child extends Parent {}
  Child.print("Hello, world!");

После применения быстрого исправления:


  Parent.print("Hello, world!");
