Сообщает о списке экспортуемых переменных или констант, разделенных запятыми.

Пример:

const C1, C3, C2, C44, C9, C11, C6 = 1, 2, 3, 1, 3, 2, 1

В таком объявлении сложно определить значение каждой константы. Чтобы сделать это объявление более читаемым, вы можете применить быстрое исправление Вынести в отдельное объявление. После применения быстрого исправления к каждой константе объявление будет выглядеть так:

const (
 C3  = 2
 C2  = 3
 C44 = 1
 C9  = 3
 C11 = 2
 C6  = 1
 C1  = 1
)
