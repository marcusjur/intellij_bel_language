Сообщать об экспортуемых функциях с неэкспортуемым возвращаемым типом.

При просмотре документации в go doc неэкспортуемые типы могут быть сложны в использовании.

Пример:

type hidden struct{}
func Exported() hidden { // Экспортуемая функция с неэкспортуемым возвращаемым типом `hidden`
 return hidden{}
}

Вы можете применить быстрое исправление Экспортовать, чтобы экспортовать тип. После применения быстрого исправления название типа будет написано с заглавной буквы:

type Hidden struct{}
func Exported() Hidden { // Исправлено
 return Hidden{}
}
