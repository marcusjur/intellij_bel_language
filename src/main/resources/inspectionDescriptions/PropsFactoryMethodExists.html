동적 생성자 호출을 안전하게 factory 메서드 호출로 교체할 수 있는 사례를 보고합니다.

예:


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Props(classOf[Foo])

빠른 수정 적용 후:


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Foo.props()
