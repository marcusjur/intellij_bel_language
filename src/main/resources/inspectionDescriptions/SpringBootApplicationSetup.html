Сообщает о наличии избыточных аннотаций @EnableAutoConfiguration или @ComponentScan при использовании @SpringBootApplication в пакете по умолчанию.

Быстрое исправление удалит избыточные аннотации.

Пример:


@SpringBootApplication
@ComponentScan // Сообщение 'Избыточное объявление: @SpringBootApplication уже включает @ComponentScan'
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

После применения быстрого исправления:


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }

Что нового в версии 2018.2
