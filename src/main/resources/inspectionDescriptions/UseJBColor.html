Отчёт об использовании java.awt.Color.

Это может не учитывать "темные" темы (например, встроенную "Darcula"), что может привести к нежелательным эффектам в интерфейсе.

Быстрое исправление заменит использование на JBColor, который позволяет определять варианты цветов для темной темы.

Пример:


// Неверно:
Color darkGreen = new Color(12, 58, 27);
Color blue = Color.BLUE;
// Верно:
Color darkGreen = new JBColor(12, 58, 27);
Color blue = JBColor.BLUE;
Color green = new JBColor(new Color(12, 58, 27), new Color(27, 112, 39));
