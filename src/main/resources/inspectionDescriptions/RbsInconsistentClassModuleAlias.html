Сообщает о несовместимых объявлениях классов и псевдонимов модулей. Класс не может использоваться как псевдоним модуля, а модуль не может использоваться как псевдоним класса.

Использование класса в качестве псевдонима модуля или модуля в качестве псевдонима класса вызывает ошибку InconsistentClassModuleAliasError.

Доступно быстрое исправление для перехода к конфликтующим объявлениям.

Пример: 


class User
end
# Класс 'User' используется как цель псевдонима модуля
module Customer = User
