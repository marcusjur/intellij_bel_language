Сообщает об избыточных указаниях направления сортировки (ASC и DESC) в предложении ORDER BY.

Пример (MySQL):

CREATE TABLE foo(a INT, b INT, c INT);
SELECT * FROM foo ORDER BY a ASC, b DESC, c ASC;

Ключевое слово ORDER BY по умолчанию сортирует записи по возрастанию. Следовательно, ключевое слово ASC для столбцов a и c является избыточным.
