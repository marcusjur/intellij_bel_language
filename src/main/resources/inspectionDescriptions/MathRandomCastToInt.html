Сообщает о вызовах Math.random() с немедленным приведением к типу int.

Приведение double между 0.0 (включительно) и 1.0 (исключительно) к типу int всегда округляется до нуля. Перед приведением к типу int значение следует умножить на некоторый множитель, чтобы получить значение между нулем (включительно) и множителем (исключительно). Альтернативным решением является использование метода nextInt() класса java.util.Random.

Пример:

  int r = (int)Math.random() * 10;

После применения быстрого исправления:

  int r = (int)(Math.random() * 10);
