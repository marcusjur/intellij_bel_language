Сообщает о потенциально опасных преобразованиях из uintptr в unsafe.Pointer.

Преобразование из uintptr в unsafe.Pointer недопустимо, если значение указателя хранится в переменной типа uintptr в памяти, поскольку такое значение не видно для копирования стека и сборщика мусора.

Пример неправильного использования:

nums := []int8{42, 24}
ptr := unsafe.Pointer(&nums[0])
addr := uintptr(ptr) // адрес сохраняется в локальной переменной
ptr = unsafe.Pointer(addr + uintptr(1))

Пример правильного использования:

nums := []int8{42, 24}
ptr := unsafe.Pointer(&nums[0])
ptr = unsafe.Pointer(uintptr(ptr) + uintptr(1))
