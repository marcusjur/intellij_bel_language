Сообщает об использовании self в качестве квалификатора, когда это не требуется для замыкания.

Рекомендуется удалить избыточное использование self.

Пример: 


  struct URL {
      func loadAsync() {
          print(self.path)
          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

После применения быстрого исправления: 


  struct URL {
      func loadAsync() {
          print(path)
          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
