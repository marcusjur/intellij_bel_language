Сообщает об операторах return в конце конструкторов и методов, возвращающих void. Такие операторы избыточны и могут быть безопасно удалены.

Эта проверка не выполняется в JSP файлах.

Пример:


  void message() {
    System.out.println("Hello World");
    return;
  }

После применения быстрого исправления: 


  void message() {
    System.out.println("Hello World");
  }

Используйте опцию Игнорировать в ветке then оператора if, имеющего ветку else, чтобы игнорировать операторы return в ветке then оператора if, который имеет ветку else.
