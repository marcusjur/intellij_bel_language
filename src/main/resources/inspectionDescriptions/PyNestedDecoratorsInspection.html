Сообщает о проблемах с вложенными декораторами. Эта проверка подсвечивает случаи, когда classmethod или staticmethod применяются после другого декоратора.

Пример:


def innocent(f):
    return f
class A:
    @innocent  # декоратор не получит ожидаемый вызываемый объект
    @classmethod
    def f2(cls):
        pass
    @innocent  # декоратор не получит ожидаемый вызываемый объект
    @staticmethod
    def f1():
        pass

Быстрое исправление, предлагаемое ИСР, заключается в удалении этого декоратора.
