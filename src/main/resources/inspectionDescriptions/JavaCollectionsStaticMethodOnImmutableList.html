Сообщает о вызовах модифицирующих методов Java (таких как fill, reverse, shuffle, sort) для неизменяемых коллекций Kotlin.

Это может привести к исключению UnsupportedOperationException во время выполнения.

Пример:


  import java.util.Collections
  fun test() {
      val immutableList = listOf(1, 2)
      Collections.reverse(immutableList)
  }

Чтобы исправить эту проблему, сделайте список изменяемым.
