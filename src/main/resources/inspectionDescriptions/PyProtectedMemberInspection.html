Сообщает о случаях доступа к защищённым (protected) членам класса за пределами класса, его наследников или модуля.

Пример:


class Foo:
    def _protected_method(self):
        pass
class Bar(Foo):
    def public_method(self):
        self._protected_method()
foo = Foo()
foo._protected_method() # доступ к защищённому методу
