Сообщает об общих частях кода в ветвях оператора if, которые можно вынести.

Эти общие части не зависят от условий и усложняют понимание оператора if.

Пример:


  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

После применения быстрого исправления:


  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();

Функция добавлена в версии 2018.1
