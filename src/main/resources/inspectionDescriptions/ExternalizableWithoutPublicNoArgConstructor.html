Сообщает о классах Externalizable, у которых отсутствует публичный конструктор без параметров.

При восстановлении объекта Externalizable экземпляр создаётся с помощью публичного конструктора без параметров перед вызовом метода readExternal. Если публичный конструктор без параметров недоступен, во время выполнения будет выброшено исключение java.io.InvalidClassException.
