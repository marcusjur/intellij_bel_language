Сообщать о любых ссылках на локальные переменные без Serializable, находящиеся вне Serializable лямбда-выражений, локальных и анонимных классов.

При ссылке на локальную переменную из анонимного класса её значение сохраняется в неявном поле этого класса. То же самое происходит для локальных классов и лямбда-выражений. Если тип переменной не является Serializable, сериализация завершится с ошибкой.

Пример:


  interface A extends Serializable {
      abstract void foo();
  }
  class B {}
  class C {
      void foo() {
          B b = new B();
          A a = new A() {
              @Override
              public void foo() {
                  System.out.println(b); // предупреждение
              }
          };
      }
  }
