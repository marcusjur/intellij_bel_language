Сообщает о сравнениях, где левый и правый операнды представляют одно и то же выражение. Хотя иногда сравнение выражения с самим собой может быть преднамеренным, чаще всего это результат невнимательности.

Пример:


  // возможно left.getLength() == right.getLength() было задумано
  boolean result = left.getLength() == left.getLength();

Чтобы игнорировать сравнения, которые могут иметь побочные эффекты, используйте опцию Игнорировать условия с побочными эффектами. Отключение этой опции может привести к ложным срабатываниям, например когда один и тот же метод возвращает разные значения при последующих вызовах.

Пример:


  native int unknownMethod();
  ...
  if (unknownMethod() > unknownMethod()) {
    System.out.println("Got it");
  }

Поскольку unknownMethod() может иметь побочные эффекты (как в примере), предупреждение будет выдано только если опция Игнорировать условия с побочными эффектами отключена.
