Сообщать о избыточном ключевом слове else в конструкциях if—else и цепочках операторов.

Ключевое слово else является избыточным, когда все предыдущие ветви заканчиваются операторами return, throw, break или continue. В таких случаях операторы из ветви else можно разместить после оператора if, а ключевое слово else можно удалить.

Пример:


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

После применения быстрого исправления:


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

Отключите опцию Сообщать, когда после оператора 'if' нет других операторов, чтобы игнорировать случаи, когда конструкция if—else является последним оператором в блоке кода.
