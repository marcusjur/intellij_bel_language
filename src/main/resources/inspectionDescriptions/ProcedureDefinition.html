프로시저 구문을 포함하는 메서드 정의를 보고합니다.

메서드에 프로시저와 같은 구문을 사용하는 것은 권장되지 않습니다. 이는 일관되지 않고 오류로 이어질 수 있으며 Scala 3에서 제거될 예정입니다.

빠른 수정은 해당 정의를 프로시저 구문에서 함수 구문으로 변환합니다.

예:


  class Test {
    def method() { }
  }

빠른 수정 적용 후:


  class Test {
    def method(): Unit = { }
  }

힌트: 분석 / 이름별로 검사 실행(Ctrl+Alt+Shift+I)을 사용해 이 검사를 전체 프로젝트에 적용할 수 있습니다.
