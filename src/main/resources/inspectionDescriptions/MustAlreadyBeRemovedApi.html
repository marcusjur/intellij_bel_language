Отчёт об объявлениях, помеченных @ApiStatus.ScheduledForRemoval, которые следует удалить из текущей версии библиотеки объявлений.

Он сравнивает версию, запланованную для удаления, с версией, которую можно установить ниже.

Версии указываются в виде строки с разделителями-точками, опционально с суффиксами alpha, beta, snapshot или eap.

Примеры корректных версий: 1.0, 2.3.1, 2018.1, 7.5-snapshot, 3.0-eap.

Сравнение версий интуитивно понятно, например: 1.0 < 2.0, 1.0-eap < 1.0, 2.3-snapshot < 2.3 и т.д. Подробную логику сравнения можно найти в реализации VersionComparatorUtil.
