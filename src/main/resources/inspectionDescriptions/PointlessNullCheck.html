Сообщает о проверках на null перед вызовом методов, которые гарантированно возвращают false при передаче null (например, Class.isInstance).

В таких случаях метод всегда возвращает false, поэтому подобные проверки представляются избыточными.

Пример:


  if (x != null && myClass.isInstance(x)) { ... }

После применения быстрого исправления:


  if (myClass.isInstance(x)) { ... }
