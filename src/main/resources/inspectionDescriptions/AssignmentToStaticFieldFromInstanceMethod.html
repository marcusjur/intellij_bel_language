Сообщает о присваивании или модификации static полей в методах экземпляра класса.

Хотя это и допустимо, такие присваивания сложно реализовать безопасно, и обычно являются результатом случайного объявления поля как static.

Пример:


  class Counter {
    private static int count = 0;
    void increment() {
      // Предупреждение: обновление
      // static поля из метода экземпляра
      count++;
    }
  }
