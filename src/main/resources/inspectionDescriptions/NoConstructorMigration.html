Сообщает о вызовах конструктора на функциональном супертипе, которые вызывают ошибки компиляции начиная с версии 1.9.

Тип несоответствия:

  • Реализация не соответствует опубликованной спецификации или документации

Подробнее: KT-46344: Отсутствует ошибка при вызове суперконструктора на функциональном интерфейсе в списке супертипов

Быстрое исправление удалит вызов конструктора.

Пример:


  abstract class A : () -> Int()

После применения быстрого исправления:


  abstract class A : () -> Int

Эта проверка выполняется только если уровень языка Kotlin для проекта или модуля установлен на 1.7 или выше.
