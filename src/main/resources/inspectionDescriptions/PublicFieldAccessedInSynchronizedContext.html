Сообщает о доступе к нефинальным (final) и неприватным (private) полям в синхронизованном контексте.

Невозможно гарантировать, что доступ к неприватным (private) полям всегда осуществляется синхронизованно. Такой "частично синхронизованный" доступ может случайно привести к несогласованности структуры данных.

Пример:


  class Bar {
    public String field1;
  }
  public Bar myBar;
  synchronized public void sample() {
   myBar.field1 = "bar";
  }
