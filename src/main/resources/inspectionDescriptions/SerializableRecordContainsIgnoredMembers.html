Сообщает о методах или полях сериализации, определённых в классе record. Методы сериализации включают writeObject(), readObject(), readObjectNoData(), writeExternal() и readExternal(), а также поле serialPersistentFields. Эти члены не используются для сериализации или десериализации записей и поэтому являются излишними.

Пример:


  record R1() implements Serializable {
    // Это поле игнорируется при сериализации записи
    @Serial
    private static final ObjectStreamField[] serialPersistentFields = new ObjectStreamField[0];
    // Этот метод игнорируется при сериализации записи
    @Serial
    private void writeObject(ObjectOutputStream out) throws IOException {
    }
  }


  record R2() implements Externalizable {
    // Этот метод игнорируется при сериализации записи
    @Override
    public void writeExternal(ObjectOutput out) throws IOException {
    }
    // Этот метод игнорируется при сериализации записи
    @Override
    public void readExternal(ObjectInput in) throws IOException, ClassNotFoundException {
    }
  }
