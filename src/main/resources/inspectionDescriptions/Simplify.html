Сообщает об операторах, которые можно упростить, например константные условия, идентичные ветки if, избыточные логические выражения и т.д. Используйте следующие флажки для управления параметрами анализа:
  • Упростить "if (true)", "while(false)": предлагает заменить операторы if/while на логические литералы.
  • Упростить "condition == true": предлагает заменить операторы сравнения на логические литералы.
  • Упростить "condition1 ? true : condition2;": предлагает упростить тернарный оператор, где одно из выражений является логическим литералом.
  • Упростить "if (condition1) return true; return condition2;": предлагает упростить оператор if, который возвращает логический литерал при истинном условии и другое условие в противном случае.
