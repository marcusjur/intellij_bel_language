Сообщает об использовании аннотаций @Ignore в JUnit 4 или @Disabled в JUnit 5. Длительное использование этих аннотаций для тестов считается признаком проблемного кода, особенно когда не указана причина.

Пример:


  @Ignore
  public class UrgentTest {
    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

Настройка проверки:

  • Опция Сообщать только об аннотациях без причины позволяет обнаруживать только случаи, когда не указано значение атрибута value аннотации.
