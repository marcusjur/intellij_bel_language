Инспекция предлагает заменить строковые литералы кодировок String (например, "UTF-8") на предопределённую константу StandardCharsets.UTF_8 в методах и конструкторах.

Исправление ускоряет выполнение кода, поскольку устраняет необходимость поиска кодования. Кроме того, может отпасть необходимость в обработке исключения UnsupportedEncodingException. В этом случае блок catch будет автоматически удален.

Пример:


  try {
    byte[] bytes = "str".getBytes("UTF-8");
  } catch (UnsupportedEncodingException e) {
  }

После применения быстрого исправления:


  byte[] bytes = "str".getBytes(StandardCharsets.UTF_8);

Эта проверка доступна в Java 7 и более поздних версиях.
