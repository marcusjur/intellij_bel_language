Сообщает о некорректных шаблонах форматования сообщений или неправильных индексах заполнителей

Сообщает о следующих ошибках:

  • Нераспознанные или отрицательные индексы
  • Незакрытые фигурные скобки
  • Непарные кавычки. В этом случае часть шаблона может остаться неиспользованной
  • Возможно неправильное количество кавычек
  • Некорректные нижние границы во вложенных шаблонах выбора
  • Неправильные индексы заполнителей. В этом случае заполнители не могут быть заменены или аргументы не могут быть использованы

Примеры:


  MessageFormat.format("{wrong}", 1); // неправильный индекс
  MessageFormat.format("{0", 1); // несоответствующие фигурные скобки
  MessageFormat.format("'{0}", 1); // непарные кавычки
  MessageFormat.format("It''''s {0}", 1); // выведет "It''s" вместо "It's"
  MessageFormat.format("{0}", 1, 2); // аргумент с индексом '1' не используется в шаблоне

Используйте таблицу Пользовательские методы MessageFormat для указания вызовов методов, аргументы которых должны проверяться как шаблоны MessageFormat. Таблица содержит пары полных имен классов и регулярных выражений имен методов для сопоставления с именами включаемых классов и вызовов методов. Имена классов также соответствуют подклассам.
