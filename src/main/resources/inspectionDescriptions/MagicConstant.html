Сообщает о выражениях, которые могут быть заменены "магическими" константами.

Пример 1:


  // Использование числового литерала "2", предупреждение:
  Font font = new Font("Arial", 2)

Пример 2:


  // Использование предопределённой константы, правильно:
  Font font = new Font("Arial", Font.ITALIC)

Если возможно, быстрое исправление вставит соответствующую предопределённую константу.

Поведение этой проверки контролируется аннотацией org.intellij.lang.annotations.MagicConstant. Некоторые стандартные методы Java библиотек уже предварительно аннотированы, но вы также можете использовать эту аннотацию в своем коде.
