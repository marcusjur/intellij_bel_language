exists 또는 forall 호출 내 이중 부정을 보고합니다.

빠른 수정에서는 이중 부정을 제거합니다.

예:


  def condition(x: Int): Boolean = ???
  !Seq(1, 2).exists(x => !condition(x))
  !Seq(1, 2).forall(x => !condition(x))

빠른 수정 적용 후:


  def condition(x: Int): Boolean = ???
  Seq(1, 2).forall(x => condition(x))
  Seq(1, 2).exists(x => condition(x))
