Сообщать о неявных преобразованиях примитивных числовых типов, вставляемых компилятором. Кроме того, сообщать о любых преобразованиях примитивных числовых типов, которые будут удалены компилятором.

Пример:

int x = (short)5; // это преобразование будет удалено компилятором javac

После применения быстрого исправления:

int x = 5;
