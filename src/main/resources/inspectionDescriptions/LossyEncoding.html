Сообщение о символах, которые не могут быть отображены из-за текущего кодования документа.

Примеры:

  • При вводе интернациональных символов в документ с кодованием US-ASCII некоторые символы будут потеряны при сохранении.
  • При загрузке файла в кодовании UTF-8 с использованием однобайтового кодования ISO-8859-1 некоторые символы будут отображаться некорректно.

Для решения этой проблемы можно изменить кодование файла, указав её непосредственно в файле, например изменив атрибут encoding= в XML-прологе для XML-файлов, или изменить соответствующие параметры в разделе Настройки | Редактор | Кодования файлов.
