Сообщает о вызовах функций стандартной библиотеки PHP, которые считаются устаревшими начиная с PHP 8.1.

Отслеживаются следующие вызовы:

  • Вызовы функций key(), current(), next(), prev() и reset() с объектами в качестве аргументов.
  • Вызовы mb_check_encoding() без аргументов.
  • Вызовы get_class(), get_parent_class() и get_Called_class() без аргументов.
  • Вызовы IntlCalendar::roll() с булевым значением в качестве аргумента $value.
  • Вызовы ctype_*() с нестроковыми аргументами.

Для получения дополнительной информации см. PHP RFC: Устаревшие функции в PHP 8.1.
