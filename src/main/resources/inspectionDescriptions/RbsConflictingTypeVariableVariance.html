Сообщает о конфликте вариантности типовой переменной (in, out или unchecked) с другим объявлением. Типовые переменные должны совпадать по вариантности между частичными объявлениями одного и того же класса или модуля, даже если они имеют разные имена.

Пример:

# Типовая переменная 'B' класса 'Example' имеет вариантность, конфликтующую с другим объявлением...
class Example[A, in B, out C]
end
# Типовая переменная 'Y' класса 'Example' имеет вариантность, конфликтующую с другим объявлением...
class Example[X, Y, Z]
end
