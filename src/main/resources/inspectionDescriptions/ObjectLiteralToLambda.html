Сообщает об анонимных объектных литералах, реализующих Java-интерфейс с единственным abstract методом, который может быть преобразован в лямбда-выражение.

Пример:


class SomeService {
  val threadPool = Executors.newCachedThreadPool()
  fun foo() {
    threadPool.submit(object : Runnable {
      override fun run() {
        println("hello")
      }
    })
  }
}

После применения быстрого исправления:


  fun foo() {
    threadPool.submit { println("hello") }
  }
