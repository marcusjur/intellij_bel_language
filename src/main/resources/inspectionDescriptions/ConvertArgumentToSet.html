Обнаружение вызовов функций, которые можно ускорить путем преобразования аргументов в Set.

Операции вроде 'minus' или 'intersect' работают эффективнее, когда аргументы являются множествами. Явное преобразование Iterable<T> или Array<T> в Set<T> обычно повышает производительность кода.

Это быстрое исправление добавит явное преобразование к вызову функции.

Пример:


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b).size

После применения быстрого исправления:


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b.toSet()).size
