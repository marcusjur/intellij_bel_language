Сообщает об интерполируемых переменных (экземпляра или глобальных) без фигурных скобок в строковых выражениях.

Пример:

# 不良做法
@name = 'John Doe'
puts "Hi #@name"
Отсутствующие фигурные скобки можно добавить с помощью быстрого исправления. После применения быстрого исправления результат будет выглядеть так: 
# 优良做法
@name = 'John Doe'
puts "Hi #{@name}"
