Сообщает о блоках catch, которые перехватывают некорректные исключения.

Некоторые исключения (например, java.lang.NullPointerException или java.lang.IllegalMonitorStateException) указывают на ошибки программования и, следовательно, практически никогда не должны перехватываться в производственном коде.

Пример:


  try {
    return component.getMousePosition(true) != null;
  } catch (NullPointerException e) { // warning: Prohibited exception 'NullPointerException' caught
    return false;
  }

Используйте список запрещенных исключений, чтобы указать, о каких исключениях следует сообщать.
