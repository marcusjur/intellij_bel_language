Сообщает о методах equals(), которые не проверяют тип своего параметра.

Если метод equals() не проверяет тип своего параметра, это может привести к потенциальным ошибкам при использовании объекта в нетипизованных коллекциях.

Пример:


  class MyClass {
    int x;
    @Override
    public boolean equals(Object obj) {
      // Метод equals должен возвращать false, если obj не является MyClass
      return ((MyClass)obj).x == x;
    }
  }
