Сообщает о любых необязательных точках с запятой, включая те, которые используются между членами класса, внутри блоков операторов или после определений классов.

Хотя эти точки с запятой допустимы в Java, они избыточны и могут быть удалены.

Пример:


  class C {
    ;
    void m() throws Exception {
        try (AutoCloseable r1 = createAutoCloseable();) {
          ;
        }
    }
    ;
  }

После применения быстрого исправления:


  class C {
    void m() throws Exception {
      try (AutoCloseable r1 = createAutoCloseable()) {
      }
    }
  }
