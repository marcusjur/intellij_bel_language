Передан недопустимый метод в атрибут dependsOnMethods аннотации @Test.

Имя метода считается недопустимым, если оно не может быть разрешено как действительный и доступный метод с аннотацией @Test в текущем классе или любом из его родительских классов.

Пример:


  public class SampleTest {
    @Test(dependsOnMethods = "testSpellignError")
    public void testSample() {}
    @Test
    public void testSpellingError(){}
  }
