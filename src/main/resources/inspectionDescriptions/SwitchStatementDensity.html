Сообщает об операторах или выражениях switch с низким соотношением меток switch к исполняемым операторам.

Такие операторы switch могут быть непонятными и, возможно, требуют рефакторинга.

Пример:


    switch (i) { // 1 case, 5 исполняемых операторов -> плотность 20%
        case 1:
            System.out.println("1");
            System.out.println("2");
            System.out.println("3");
            System.out.println("4");
            System.out.println("5");
            break;
    }

Используйте поле Минимальная плотность веток для указания допустимого соотношения меток switch к исполняемым операторам.
