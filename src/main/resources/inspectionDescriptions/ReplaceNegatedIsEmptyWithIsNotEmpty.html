Сообщает о случаях отрицания методов isEmpty() и isNotEmpty() для коллекций и String, а также методов isBlank() и isNotBlank() для String.

Использование соответствующих функций делает код более лаконичным.

Быстрое исправление заменит отрицание вызова на соответствующий вызов из стандартной библиотеки.

Пример:


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // выполнить что-то
      }
  }

После применения быстрого исправления:


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // выполнить что-то
      }
  }
