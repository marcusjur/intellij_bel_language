Некорректное использование аннотации типа.

Пример:


from typing import TypeVar
T0 = TypeVar('T1') # Аргумент 'TypeVar' должен быть 'T0'
def b(p: int) -> int:  # Тип указан и в комментарии, и в аннотации
    # type: (int) -> int
    pass
def c(p1, p2): # Слишком много аргументов в сигнатуре типа
    # type: (int) -> int
    pass

Быстрые исправления предлагают различные действия. Вы можете переименовать, удалить или переместить проблемный элемент. Также можно вручную изменить объявление типа, чтобы предупреждение не отображалось.
