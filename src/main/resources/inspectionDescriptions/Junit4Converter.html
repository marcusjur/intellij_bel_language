Тесты JUnit 3, которые могут быть преобразованы в тестовые классы JUnit 4.

Пример:


  public class MainTestCase extends junit.framework.TestCase {
    public void test() {
      Assert.assertTrue(true);
    }
  }

После применения быстрого исправления:


  public class MainTestCase {
    @org.junit.Test
    public void test() {
      Assert.assertTrue(true);
    }
  }

Эта проверка срабатывает только когда уровень языка проекта или модуля установлен на 5 или выше, и JUnit 4 доступен в classpath.
