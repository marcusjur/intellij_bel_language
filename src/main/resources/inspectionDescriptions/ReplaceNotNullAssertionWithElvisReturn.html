Сообщение о том, что non-null утверждение (!!) может быть заменено на оператор elvis с return (?: return).

Non-null утверждение может привести к неожиданному NPE (NullPointerException). Рекомендуется избегать использования !!.

Это быстрое исправление заменит non-null утверждение на return или return null.

Пример:


  fun test(number: Int?) {
      val x = number!!
  }

После применения быстрого исправления:


  fun test(number: Int?) {
      val x = number ?: return
  }
