Сообщать о non-static внутренних классах, реализующих Serializable, объявленных в классах, не реализующих Serializable.

Такие классы, вероятно, не будут корректно сериализованы из-за неявной ссылки на внешний класс.

Пример:


  class A {
      class Main implements Serializable {
    }
  }

Настройте проверку с помощью следующих параметров:

  • Укажите классы, наследники которых не должны проверяться. Это применимо к классам, которые наследуют Serializable от суперкласса, но не используются для сериализации.
  • Следует ли игнорировать анонимные классы с Serializable.
