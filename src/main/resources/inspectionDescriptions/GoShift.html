Сообщает о выражениях сдвига, которые равны или превышают битовую ширину целого числа.

В таких выражениях все биты левого операнда сдвигаются, что приводит к константному значению. Константное значение указывает на некорректное смещение сдвига или избыточность выражения сдвига.

Пример:


  func shift(i int8) {
    fmt.Println(i << 8) // всегда печатает 0
  }
