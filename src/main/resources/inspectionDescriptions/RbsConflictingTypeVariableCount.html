Сообщает о конфликтах в объявлениях частичных контейнеров из-за несоответствия количества типовых параметров. Ошибка возникает, когда частичные контейнеры объявляются с разным количеством типовых параметров.

Можно исправить количество типовых параметров в объявлении с помощью быстрого исправления или перейти к конфликтующим объявлениям.

Пример:

# 'Example' класс имеет 1 типовой параметр, что конфликтует с другими объявлениями
class Example[A]
end
# 'Example' класс имеет 2 типовых параметра, что конфликтует с другими объявлениями
class Example[Y, Z]
end

После применения быстрого исправления:

class Example[A, B]
end
class Example[Y, Z]
end
