Сообщает о необязательных круглых скобках в типах функций. Функции без параметров не требуют скобок.

Лишние скобки можно удалить с помощью быстрого исправления.

Пример:

class Example
  # Необязательные круглые скобки в типе функции
  def call: () -> untyped
end

После применения быстрого исправления:

class Example
  def call: -> untyped
end
