Сообщает о классах, которые переопределяют метод clone(), но не реализуют интерфейс Cloneable. Это обычно указывает на ошибку программования.

Используйте опцию предупреждать только для 'public' методов clone, чтобы игнорировать методы, которые не являются public.

Для классов, предназначенных для наследования, вы можете не реализовывать интерфейс Cloneable, а вместо этого переопределить clone() и объявить его как protected, оставляя решение о реализации интерфейса Cloneable подклассам.
