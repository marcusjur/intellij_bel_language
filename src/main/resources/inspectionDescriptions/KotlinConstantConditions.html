Сообщает о нетривиальных условиях и значениях, которые статически известны как всегда true, false, null или ноль. Хотя иногда это может быть преднамеренным, обычно это признак логической ошибки в программе. Кроме того, сообщает о недостижимых ветках when и некоторых выражениях, которые статически известны как всегда неуспешные.

Пример:


fun process(x: Int?) {
  val isNull = x == null
  if (!isNull) {
    if (x != null) {} // Условие всегда true
    require(x!! < 0 && x > 10) // Условие всегда false
  } else {
    println(x!!) // Оператор !! всегда будет неуспешным
  }
}
fun process(v: Any) {
  when(v) {
    is CharSequence -> println(v as Int) // Приведение типа всегда будет неуспешным
    is String -> println(v) // Ветка недостижима
  }
}

Снимите флажок "Предупреждать, когда константы хранятся в переменных", чтобы избежать сообщений о переменных с константными значениями, не используемых в условиях.
