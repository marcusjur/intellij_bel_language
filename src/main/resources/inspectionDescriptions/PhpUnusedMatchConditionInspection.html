Сообщает о условиях в выражении match, которые никогда не будут выполнены.

Аналогично поведению PHP Engine, проверка использует строгое сравнение (===) для проверки типов в условиях match с типом аргумента match, независимо от директивы strict_types. Поскольку приведение типов не происходит, некоторые условия могут быть недостижимы из-за несоответствия типов.

Дополнительную информацию смотрите в Match expression (php.net).
