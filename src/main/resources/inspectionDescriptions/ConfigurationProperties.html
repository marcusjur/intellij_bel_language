Сообщает о недопустимых префиксах, определённых в аннотации @ConfigurationProperties:

  • Отсутствующий префикс
  • Пустой префикс
  • Повторяющийся префикс
  • Префикс в нотации, отличной от написания через дефис

Проверка выполняется для классов с аннотацией @ConfigurationProperties, которые не зарегистрированы через @EnableConfigurationProperties, не отмечены как компонент Spring или не сканируются через @ConfigurationPropertiesScan.

Когда версия Spring Boot проекта или модуля установлена на 2.2.0 или выше, эта проверка срабатывает только для классов с аннотацией @ConfigurationProperties, которые не сканируются через @ConfigurationPropertiesScan.
