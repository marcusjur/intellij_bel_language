Сообщает о случаях, когда аннотация @RunWith используется как в родительском, так и в дочернем классах в иерархии JUnit-тестов. Это может привести к непредсказуемому поведению тестов.

Пример:


  @RunWith(Suite.class)
  @SuiteClasses(MySuiteClass.class)
  public abstract Parent {
  }
  @RunWith(Parameterized.class)
  public MyTest {
  }

Новое в версии 2024.1
