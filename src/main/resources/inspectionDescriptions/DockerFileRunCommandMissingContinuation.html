Сообщает об отсутствующих символах продолжения строки в команде RUN.

В командах RUN, использующих форму shell, следует использовать '\'(обратный слеш) для переноса одной инструкции RUN на следующую строку. В противном случае сборка Docker завершится с ошибкой.

Пример:


  # the command below will fail
  RUN /bin/bash -c 'source $HOME/.bashrc;
      echo $HOME'

После применения быстрого исправления:


  RUN /bin/bash -c 'source $HOME/.bashrc; \
      echo $HOME'
