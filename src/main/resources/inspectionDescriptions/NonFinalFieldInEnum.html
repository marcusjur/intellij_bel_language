Сообщать о нефинальных полях в перечислениях (enum). Нефинальные поля вводят глобальное изменяемое состояние, что обычно считается нежелательным.

Пример:


  enum Enum {
    FIRST("first"),
    SECOND("second");
    public String str;
    Enum(String str) {
        this.str = str;
    }
  }

После применения быстрого исправления:


  enum Enum {
    FIRST("first"),
    SECOND("second");
    public final String str;
    Enum(String str) {
        this.str = str;
    }
  }
Используйте опцию "Игнорировать поля, которые не могут быть 'final'" для отображения предупреждений только для полей, которые можно сделать финальными с помощью быстрого исправления.
