Сообщает об опасной инициализации массива (например, $arr[] = value).

Такой подход не рекомендуется, поскольку если $arr уже содержит некоторое значение (например, строку из переменных запроса), это значение сохранится, и [] фактически может представлять оператор доступа к строке. Лучше всегда инициализовать переменные путем прямого присваивания.

Для получения дополнительной информации см. Creating/modifying with square bracket syntax (php.net).
