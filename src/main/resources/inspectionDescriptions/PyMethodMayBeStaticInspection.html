Сообщает о методах, которые не требуют создания экземпляра класса и могут быть объявлены как static.

Пример:


class MyClass(object):
    def my_method(self, x):
        print(x)

При применении быстрого исправления Преобразовать метод в функцию код будет преобразован в:


def my_method(x):
    print(x)
class MyClass(object):
    pass

При выборе быстрого исправления Сделать метод статическим будет добавлен декоратор @staticmethod:


class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
