Сообщает о числовых значениях, возвращаемых методами, которые не соответствуют заявленному диапазону возвращаемых значений. Вы можете использовать несколько аннотаций для объявления диапазона возвращаемых значений метода:
  • org.jetbrains.annotations.Range из пакета аннотаций JetBrains (указывает 'from' и 'to')
  • org.checkerframework.common.value.qual.IntRange из пакета аннотаций Checker Framework (указывает 'from' и 'to')
  • org.checkerframework.checker.index.qual.GTENegativeOne из пакета аннотаций Checker Framework (диапазон '>= -1')
  • org.checkerframework.checker.index.qual.NonNegative из пакета аннотаций Checker Framework (диапазон '>= 0')
  • org.checkerframework.checker.index.qual.Positive из пакета аннотаций Checker Framework (диапазон '> 0')
  • javax.annotation.Nonnegative из пакета аннотаций JSR 305 (диапазон '>= 0')
  • javax.validation.constraints.Min (указывает минимальное значение)
  • javax.validation.constraints.Max (указывает максимальное значение)

Пример:


  @Range(from = 0, to = Integer.MAX_VALUE) int getValue() {
    // Предупреждение: -1 выходит за пределы заявленного диапазона
    return -1;
  }
