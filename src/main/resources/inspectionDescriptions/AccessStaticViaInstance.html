Сообщает об обращении к static методам и полям через экземпляр класса вместо самого класса.

Хотя спецификация Java допускает обращение к статическим членам через переменную экземпляра, это может ввести читателей в заблуждение, заставив их думать, что метод зависит от экземпляра.

Быстрое исправление заменит переменную экземпляра на имя класса.

Пример:


  String s1 = s.valueOf(0);

После применения быстрого исправления:


  String s = String.valueOf(0);
