Сообщать о нераспознанных свойствах в файлах *.properties.

Пример JSP файла:


<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%@ taglib uri="/tags/fmt" prefix="fmt" %>
<fmt:message bundle="bundle.properties" key="unknown.property"/> // Ошибка: такое свойство отсутствует в указанном файле
