Сообщает о методах с чрезмерным количеством точек ветвления.

Точкой ветвления считается одно из следующего:

  • Циклические конструкции
  • Оператор if
  • Тернарный оператор
  • Блок catch
  • Выражения, содержащие один или несколько операторов && или ||
  • Блок switch с нестандартными ветвями

Методы с высокой цикломатической сложностью могут быть запутанными и сложными для тестования.

Используйте поле Предел сложности метода для указания максимально допустимой цикломатической сложности метода.
