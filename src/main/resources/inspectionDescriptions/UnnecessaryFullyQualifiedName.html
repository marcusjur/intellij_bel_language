Сообщает о полных именах классов, которые можно сократить.

Быстрое исправление сокращает полные имена и при необходимости добавляет операторы import.

Пример:


  class ListWrapper {
    java.util.List<Integer> l;
  }

После применения быстрого исправления:


  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

Настройка проверки:

Используйте опцию Игнорировать в операторах модулей Java 9 для игнорирования полных имен в операторах модулей provides и uses Java 9.

В Настройки | Редактор | Стиль кода | Java | Импорты используйте следующие опции для настройки проверки:

  • Используйте опцию Вставлять импорты для внутренних классов, если ссылки на внутренние классы должны быть квалифицированы внешним классом.
  • Используйте опцию Использовать полные имена классов в JavaDoc, чтобы разрешить использование полных имен в Javadoc.
