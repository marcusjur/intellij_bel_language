Сообщает о параметрах типа и аргументах типа подстановочного знака, которые явно объявлены как расширяющие java.lang.Object.

Такое объявление 'extends' является избыточным, поскольку java.lang.Object является родительским типом для всех классов.

Пример:

  class ClassA<T extends Object> {}

Если вам нужно сохранить объявление 'extends Object' из-за аннотаций, отключите опцию Игнорировать при аннотировании java.lang.Object. Это может быть полезно, например, когда вы используете анализатор null-безопасности, и объявление 'extends Object' содержит аннотации @Nullable/@NotNull.

Пример:

  class MyClass<T extends @NotNull Object> {}
