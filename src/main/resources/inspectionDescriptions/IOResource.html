Сообщает о незакрытых I/O ресурсах. Проверка охватывает следующие I/O ресурсы: java.io.InputStream, java.io.OutputStream, java.io.Reader, java.io.Writer, java.util.zip.ZipFile, java.io.Closeable и java.io.RandomAccessFile.

Не сообщается о I/O ресурсах, обёрнутых другими I/O ресурсами, так как обёртывающий ресурс закроет обёрнутый ресурс.

По умолчанию проверка предполагает, что ресурс может быть закрыт любым методом, содержащим в названии 'close' или 'cleanup'.

Пример:


  void save() throws IOException {
    FileWriter writer = new FileWriter("filename.txt"); //предупреждение
    writer.write("sample");
  }

Проверка настраивается следующими параметрами:

  • Список классов I/O ресурсов, которые не требуют закрытия и должны игнорироваться при проверке.
  • Разрешать ли открытие I/O ресурсов внутри блока try. Этот стиль менее предпочтителен, так как он более многословен, чем открытие ресурса перед блоком try.
  • Может ли ресурс быть закрыт любым вызовом метода, которому этот ресурс передаётся в качестве аргумента.
