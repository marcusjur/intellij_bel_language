Обнаружение избыточных ссылок на Companion.

Пример:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.Companion.create()
  }

После применения быстрого исправления:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.create()
  }
