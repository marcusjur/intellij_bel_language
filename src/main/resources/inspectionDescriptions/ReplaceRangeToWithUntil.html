Сообщает об использовании оператора rangeTo или .. вместо until.

Использование соответствующей функции может сделать код более лаконичным.

Быстрое исправление заменит вызов rangeTo или .. на until.

Пример:


  fun foo(a: Int) {
      for (i in 0..a - 1) {
      }
  }

После применения быстрого исправления:


  fun foo(a: Int) {
      for (i in 0 until a) {
      }
  }
