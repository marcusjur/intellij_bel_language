Сообщает о вложенных операторах switch внутри других операторов switch. Такие вложенные операторы switch затрудняют понимание кода и могут привести к непредвиденному поведению.

Пример:


switch (outer) {
  case 1:
    switch (inner) {
      case 1:
        print "inner 1"
        break
      default:
        print "inner default"
    }
    break
  default:
    print "default"
}
