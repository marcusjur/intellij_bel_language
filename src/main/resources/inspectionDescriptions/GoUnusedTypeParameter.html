Сообщать о неиспользуемых параметрах типа. 
func main() {
 printAll(
  42,
  "bird",
 )
}
func printAll[I int, S string](
 i I,
 s string,
) {
 fmt.Println(i)
 fmt.Println(s)
}

Функция printAll имеет два параметра типа I и S, но использует только первый из них. Поэтому S string будет отображаться серым цветом.
