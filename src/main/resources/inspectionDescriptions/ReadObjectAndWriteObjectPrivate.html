Сообщает о классах Serializable, в которых методы readObject или writeObject не объявлены как private. Нет причин для установки видимости этих методов выше, чем private.

Рекомендуется использовать быстрое исправление для установки соответствующего метода как private.

Пример:


  public class Test implements Serializable {
    public void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }

После применения быстрого исправления:


  public class Test implements Serializable {
    private void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }
