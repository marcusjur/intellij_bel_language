Сообщает о стандартных методах setter, которые могут быть заменены аннотацией lombok @Setter.

Пример:


  import java.util.Date;
  public class MyClass {
    /**
     * Дата обработки.
     */
    private Date processDate;
    /**
     * Установить дату.
     *
     * @param The date
     */
    public void setProcessDate(Date param) {
      processDate = param;
    }
  }

После применения быстрого исправления/очистки:


  import lombok.Setter;
  import java.util.Date;
  @Setter
  public class MyClass {
    /**
     * Дата обработки.
     * -- SETTER --
     * Установить дату.
     *
     * @param The date
     */
    private Date processDate;
  }

Отображается только при настроенной библиотеке lombok. Для удаления lombok посетите сайт lombok.
