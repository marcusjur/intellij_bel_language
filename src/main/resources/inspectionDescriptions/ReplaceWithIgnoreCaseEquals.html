Сообщение предлагает заменить сравнение без учета регистра на equals(..., ignoreCase = true).

Использование equals() позволяет сравнивать строки без создания дополнительных строковых объектов с помощью toLowerCase() или toUpperCase().

Это быстрое исправление заменит сравнение без учета регистра, использующее toLowerCase() или toUpperCase(), на equals(..., ignoreCase = true).

Примечание: может изменить семантику для некоторых локалей.

Пример:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

После применения быстрого исправления:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
