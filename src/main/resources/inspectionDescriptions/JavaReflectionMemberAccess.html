Сообщает о рефлексивном доступе к несуществующим или невидимым полям и методам.

Пример:


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getField("hash");
  }

После применения быстрого исправления:


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getDeclaredField("hash");
  }

Для классов с модификатором final всегда точно известно, существует ли поле или метод с указанным именем.

Для классов без модификатора final поля или методы с таким именем могут существовать в подклассах, что может привести к ложным срабатываниям. Используя настройки этой проверки, можно исключить такие ложные срабатывания для любых локаций или конкретных классов.
