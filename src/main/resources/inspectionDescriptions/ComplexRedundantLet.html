Сообщает о избыточных вызовах let на основе аргументов.

Вызов let является избыточным, когда параметр лямбда-выражения используется только как квалификатор в вызываемом выражении.

Если необходимо присвоить имя квалифицирующему выражению, объявите локальную переменную.

Пример:


  fun splitNumbers() {
      "1,2,3".let { it.split(',') }
  }

Быстрое исправление удалит лишний вызов let():


  fun example() {
      "1,2,3".split(',')
  }

Или:


  fun splitNumbers() {
      val numbers = "1,2,3"
      numbers.split(',')
  }
