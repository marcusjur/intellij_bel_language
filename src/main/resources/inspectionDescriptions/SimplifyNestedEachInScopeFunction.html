Сообщает о функции forEach, которая может быть упрощена в функциях области видимости, таких как also или apply.

Быстрое исправление "Преобразовать forEach в onEach" доступно для автоматического исправления кода.

Пример:


  fun test(list: List<Int>) {
      val x = list.also { it.forEach { it + 4 } }.toString()
      val y = list.apply { forEach { println(it) } }
  }

После применения быстрого исправления:


  fun test(list: List<Int>) {
      val x = list.onEach { it + 4 }.toString()
      val y = list.onEach { println(it) }
  }
