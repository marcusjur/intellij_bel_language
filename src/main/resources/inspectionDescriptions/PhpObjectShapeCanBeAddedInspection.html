Сообщает о публичных свойствах объектов указанного типа. Рекомендуется указывать форму возвращаемого объекта с помощью атрибута #[ObjectShape] или через описание формы объекта в комментариях PHPDoc.

Пример:


  $a = new stdClass();
  $a->name = "name";
  $a->age = 2;

После применения быстрого исправления:


  /** @var $a object{name: string, age: int} */
  $a = new stdClass();
  $a->name = "name";
  $a->age = 2;
