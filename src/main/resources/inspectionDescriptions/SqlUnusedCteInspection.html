Сообщает о неиспользуемых обобщенных табличных выражениях (CTE).

Пример (PostgreSQL):

CREATE TABLE foo(a INT);
WITH a AS (SELECT 1 AS x FROM foo)
SELECT 1 + 2 FROM foo;

С помощью WITH мы создаем временный именованный набор результатов с именем a, также известный как обобщенное табличное выражение (CTE). Однако далее в коде мы не используем этот CTE. Неиспользуемый CTE отображается серым цветом.
