Сообщает о классах Serializable, в которых поле serialPersistentFields не объявлено как private static final ObjectStreamField[].

Если поле serialPersistentFields объявлено без этих модификаторов, поведение сериализации будет таким же, как если бы поле вообще не было объявлено.

Пример:


  class List implements Serializable {
    private List next;
    ObjectStreamField[] serialPersistentFields = {new ObjectStreamField("next", List.class)};
  }
