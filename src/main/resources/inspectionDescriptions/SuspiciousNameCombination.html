Сообщает о присваиваниях и вызовах функций, где имена целевых переменных или параметров функций не соответствуют именам присваиваемых им значений.

Пример 1:


  int x = 0;
  int y = x; // x используется как координата y

Пример 2:


  int x = 0, y = 0;
  // x используется как координата y, y используется как координата x
  Rectangle rc = new Rectangle(y, x, 20, 20);

Настройка проверки:

В разделе Группы имен можно указать имена, которые не должны использоваться вместе: ошибка будет выдана, если имя целевого параметра или присваивания содержит слово из одной группы, а имя присваиваемой или передаваемой переменной содержит слово из другой группы.

В разделе Игнорируемые методы можно указать методы, которые не следует проверять, хотя их имена могут вызывать сомнения. Например, параметры метода Integer.compare() названы x и y, но не имеют отношения к координатам.
