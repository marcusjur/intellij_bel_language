Сообщает об анонимных классах, которые могут быть заменены лямбда-выражениями.

Пример:


  new Thread(new Runnable() {
    @Override
    public void run() {
      // Выполнение потока
    }
  });

После применения быстрого исправления:


  new Thread(() -> {
    // Выполнение потока
  });

Обратите внимание, что при преобразовании анонимного класса в неизменяемую лямбду, среда выполнения Java может повторно использовать один и тот же объект лямбды при последующих вызовах. С другой стороны, при использовании анонимного класса каждый раз создаётся отдельный объект. Поэтому в редких случаях применение этого быстрого исправления может привести к изменению семантики, например когда экземпляр анонимного класса используется в качестве ключа HashMap.

Используйте опцию Сообщать о случаях, когда интерфейс не помечен аннотацией @FunctionalInterface, чтобы игнорировать случаи, когда анонимный класс реализует интерфейс без аннотации @FunctionalInterface.
