Сообщает о конкатенации строк String, используемых в качестве аргументов для StringBuffer.append(), StringBuilder.append() или Appendable.append().

Такие вызовы могут быть эффективно преобразованы в цепочку вызовов append для существующего StringBuffer/Builder/Appendable, что позволяет избежать дополнительных затрат на выделение памяти для StringBuffer/Builder. Эта проверка игнорирует конкатенацию строк String, вычисляемую во время компиляции, в этом случае преобразование только ухудшит производительность.

Пример:


  void bar(StringBuilder builder, String name) {
    builder.append("Hello," + name);    //предупреждение
    builder.append("Hello," + "world"); //без предупреждения
  }
