Сообщает о блоках try-finally, содержащих resource.close() в секции finally, которые могут быть преобразованы в вызов resource.use().

Использование use() делает код более читаемым и менее подверженным ошибкам, так как не требует явного вызова close().

Пример:


  fun example() {
      val reader = File("file.txt").bufferedReader()
      try {
          reader.lineSequence().forEach(::print)
      } finally {
          reader.close()
      }
  }

После применения быстрого исправления:


  fun example() {
      File("file.txt").bufferedReader().use { reader ->
          reader.lineSequence().forEach(::print)
      }
  }
