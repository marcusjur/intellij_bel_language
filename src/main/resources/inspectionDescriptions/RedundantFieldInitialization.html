Сообщать о полях с явной инициализацией значениями по умолчанию.

Пример:


  class Foo {
    int foo = 0;
    List bar = null;
  }

После применения быстрого исправления:


  class Foo {
    int foo;
    List bar;
  }

В настройках проверки можно указать, чтобы сообщалось только о явной инициализации значением null, например: 


  class Foo {
    int foo = 0;     // без предупреждения
    List bar = null; // предупреждение об избыточной инициализации поля
  }
