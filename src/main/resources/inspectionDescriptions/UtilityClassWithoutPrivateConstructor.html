Отчёт о служебных классах, не имеющих private конструктора.

Служебные классы объявляют все поля и методы как static. Создание private конструктора в служебном классе предотвращает их случайную инстанциацию.

Используйте опцию Игнорировать при наличии аннотаций, чтобы указать специальные аннотации. Проверка будет игнорировать классы, помеченные одной из этих аннотаций.

Используйте опцию Игнорировать классы, содержащие только main-метод, чтобы исключить из проверки классы, которые содержат только main-метод.
