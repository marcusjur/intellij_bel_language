Сообщает об операторах или выражениях switch для перечислений, где ветка default никогда не будет достигнута, так как все возможные значения уже обработаны в ветках case.

Такие элементы избыточны, особенно в случае выражений switch, поскольку они не будут компилироваться, если ветки case не охватывают все константы перечисления.

Для проверки выражений switch требуется установить уровень языка 14.

Предлагаемое быстрое исправление может удалить ветку default.

Пример:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
      default -> 3;
    };
  }

После применения быстрого исправления:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
    };
  }

Используйте опцию Проверять только выражения switch, чтобы обнаруживать избыточные ветки default только в выражениях switch.
