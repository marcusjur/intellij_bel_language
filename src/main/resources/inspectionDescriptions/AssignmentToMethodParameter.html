Сообщает о присваивании или модификации параметров метода.

Хотя иногда это делается намеренно, такая структура может вызывать путаницу и поэтому запрещена в некоторых Java проектах.

Быстрое исправление добавит объявление новой переменной.

Пример:


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

После применения быстрого исправления:


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

Используйте опцию Игнорировать, если присваивание является преобразованием исходного параметра, чтобы игнорировать присваивания, которые изменяют значение параметра на основе его предыдущего значения.
