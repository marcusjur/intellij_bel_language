Сообщает о методах, принимающих переменное количество параметров (также известных как vararg-методы).

Пример:


  enum EnumConstants {
  A(null), B, C;
  EnumConstants(String... ss) {}
}

С помощью быстрого исправления параметр переменной длины можно заменить на эквивалентный параметр массива. Соответствующие аргументы в вызовах метода будут обернуты в выражение инициализации массива. После применения быстрого исправления:


  enum EnumConstants {
  A(null), B(new String[]{}), C(new String[]{});
  EnumConstants(String[] ss) {}
}

Varargs-методы появились в Java 5. Эта проверка помогает обеспечить обратную совместимость с более ранними версиями Java.
