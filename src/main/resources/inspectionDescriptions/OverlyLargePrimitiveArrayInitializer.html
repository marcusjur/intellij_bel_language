Сообщает о выражениях инициализации массивов примитивных типов, содержащих слишком много элементов. Такие инициализаторы могут привести к чрезмерно большим файлам классов, поскольку для инициализации каждого элемента массива должен быть сгенерирован код. В средах с ограниченной памятью или пропускной способностью может быть эффективнее загружать большие массивы примитивов из файлов ресурсов.

Эта проверка актуальна для Java ME и других сред с сильно ограниченными ресурсами. Бездумное применение этой проверки может негативно повлиять на читаемость кода и архитектуру.

Используйте эту опцию, чтобы указать максимально допустимое количество элементов в инициализаторах массивов примитивных типов.
