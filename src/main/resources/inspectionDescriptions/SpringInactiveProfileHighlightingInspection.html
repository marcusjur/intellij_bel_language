Отображение неиспользуемых профилей в контексте Spring XML приложения.

Пример:


  <!-- Для этого контекста приложения активен профиль 'DEV' -->
  <beans profile="DEV">
    <bean class="MyFactory" factory-method="createBean"/>
  </beans>
  <!--  Содержимое этого тега подсвечивается как 'unused' -->
  <beans profile="TESTS">
    <bean class="MyTestFactory" factory-method="createBean"/>
  </beans>
  <!--  Конец секции 'unused' -->
  </beans>
