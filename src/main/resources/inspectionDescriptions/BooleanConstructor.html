Сообщает о создании объектов Boolean.

Создание новых объектов Boolean обычно не требуется и может привести к проблемам с производительностью при частом использовании. Кроме того, конструктор Boolean считается устаревшим начиная с Java 9 и может быть удален или стать недоступным в будущих версиях Java.

Пример:


  Boolean b1 = new Boolean(true);
  Boolean b2 = new Boolean(str);

После быстрого исправления:


  Boolean b1 = Boolean.TRUE;
  Boolean b2 = Boolean.valueOf(str);
