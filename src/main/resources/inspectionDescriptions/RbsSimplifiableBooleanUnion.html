Сообщает о возможности замены использования true | false на тип bool. Ключевое слово bool является сокращенной формой, которая улучшает читаемость кода.

Любая комбинация true | false может быть преобразована в bool с помощью быстрого исправления.

Пример:

# 'true | false' можно упростить
type example = true | false

После применения быстрого исправления:

type example = bool
