Сообщает о выражениях, затронутых автоматической упаковкой (автоматическое преобразование примитивных значений в объекты). Рекомендуется использовать примитивы вместо объектов. Это может существенно влиять на производительность.

Пример:

Integer x = 42;

Быстрое исправление сделает преобразование явным:

Integer x = Integer.valueOf(42);

Автоматическая упаковка появилась в Java 5. Эта проверка помогает обеспечить обратную совместимость с более ранними версиями Java.
