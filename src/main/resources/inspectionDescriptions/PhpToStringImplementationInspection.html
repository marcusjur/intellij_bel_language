Сообщает о попытках преобразования в string объектов, не реализующих метод __toString.

До PHP 7.4 это приводило к фатальной ошибке уровня E_RECOVERABLE_ERROR. В PHP 7.4 выбрасывается фатальная неустранимая ошибка ERROR. Начиная с PHP 8.0, возвращаемое значение следует стандартной семантике типов PHP и приводится к строке, если это возможно и отключен строгий режим типизации.

Если включена опция проверять наличие __toString для всех типов выражений, проверка будет выполняться для всех возможных типов выражения, и если хотя бы один из типов не содержит реализации метода __toString, будет выдано сообщение.

Подробнее см. __toString (php.net).
