Сообщает о возможном разыменовании nil. Разыменование nil может привести к NoMethodError.

Используя список игнорируемых получателей, можно указать вызовы методов, которые должны игнорироваться в качестве получателей (например, Foo#call или Foo::call для методов экземпляра или статических методов соответственно).

Пример:

def foo
  rand(2) == 1 ? nil : 1
end
a = foo
a.nomethod # Вызов метода может вызвать NoMethodError
