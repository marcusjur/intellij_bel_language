Сообщать о методах с определёнными именами в следующих случаях:
  • Имя метода совпадает с именем одного из общеизвестных методов интерфейса из стандартной библиотеки
  • Сигнатура не соответствует сигнатуре соответствующего метода интерфейса

Такой метод может указывать на то, что тип получателя предназначен для реализации интерфейса из стандартной библиотеки, но не может этого сделать из-за ошибки в сигнатуре метода.

Пример:

type MyReader []byte
func (r MyReader) ReadByte(data []byte) (byte, error) {
}

Такое использование подозрительно, так как похоже на попытку реализовать io.ByteReader, но с неправильной сигнатурой. Правильная реализация должна выглядеть так:

type MyReader []byte
func (r MyReader) ReadByte() (byte, error) {
}
