Сообщает о текстовых блоках, которые могут быть заменены обычными строковыми литералами. Эта проверка помогает при необходимости обеспечить обратную совместимость со старыми версиями Java.

Пример:


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }
                           hello();
                           """);

После применения быстрого исправления:


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

Новое в версии 2019.3
