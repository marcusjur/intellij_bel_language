Сообщает о попытках синхронизации экземпляров value-based классов, которые начиная с Java 16 вызывают предупреждения во время компиляции и исключения во время выполнения.

Например, java.lang.Double аннотирован с помощью jdk.internal.ValueBased, поэтому следующий код вызовет предупреждение при компиляции: 


Double d = 20.0;
synchronized (d) { ... } // предупреждение javac

Новое в версии 2021.1
