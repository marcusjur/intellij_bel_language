Сообщать о следующих недостатках в Javadoc-комментариях:
  • Отсутствие Javadoc там, где он требуется
  • Отсутствие необходимых тегов
  • Недействительные или неполные теги
  • Отсутствующее или неполное описание в Javadoc

Пример:


  class Main {
  }

После применения быстрого исправления:


  /**
   *
   */
  class Main {
  }

Пример:


  String id(String value) {
    return value;
  }

После применения быстрого исправления:


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
Используйте вкладки и соответствующие параметры проверки для определения области видимости, требующей Javadoc, а также необходимых тегов Javadoc для классов, методов, полей и внутренних классов.
