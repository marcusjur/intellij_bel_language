Сообщает о создании экземпляра Throwable или его подкласса, когда созданный Throwable фактически не выбрасывается. Кроме того, эта проверка сообщает о вызовах методов, возвращающих экземпляр Throwable или его подкласса, когда результат вызова не выбрасывается.

Вызовы методов, аннотированных с помощью @CanIgnoreReturnValue из Error Prone или AssertJ, не отмечаются.

Пример:


    void check(String s) {
        if (s == null) {
            new NullPointerException("s");
        }
        // ...
    }
