Блок catch, который обрабатывает более общие исключения, чем те, которые выбрасываются в соответствующем блоке try.

Пример:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  } catch (Exception ex) { // warning: 'catch' of 'Exception' is too broad, masking exceptions 'RuntimeException'
    return defaultFilePath;
  }

После применения быстрого исправления:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  } catch (RuntimeException ex) {
    return defaultFilePath;
  }

Настройка проверки:

  • Используйте опцию Предупреждать только для RuntimeException, Exception, Error или Throwable, чтобы проверка выдавала предупреждения только для наиболее общих исключений.
  • Используйте опцию Игнорировать исключения, которые маскируют другие исключения, но сами выбрасываются, чтобы игнорировать любые исключения, которые хотя и маскируют другие исключения, но могут быть выброшены сами, и поэтому технически не являются слишком общими.
