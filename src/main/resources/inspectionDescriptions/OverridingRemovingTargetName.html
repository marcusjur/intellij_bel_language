@targetName 어노테이션이 없는 정의 중 @targetName을 가진 정의를 재정의하는 정의를 보고합니다.

@targetName 어노테이션을 참조하세요.

빠른 수정에서는 재정의된 정의의 외부 이름을 가진 @targetName 어노테이션을 추가합니다.

예:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    override def *(i: Int): A = ???

빠른 수정 적용 후:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???
