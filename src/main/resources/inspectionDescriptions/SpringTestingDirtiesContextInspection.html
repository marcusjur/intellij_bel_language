Сообщение о неправильной конфигурации параметра 'mode' в аннотации @DirtiesContext.

Пример:


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // Использование methodMode на уровне тестового класса не имеет смысла. Для управления на уровне класса используйте classMode.
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // hierarchyMode следует использовать, когда контекст настроен как часть иерархии через @ContextHierarchy
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // Использование classMode на уровне тестового метода не имеет смысла. Для управления на уровне метода используйте methodMode.
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // hierarchyMode следует использовать, когда контекст настроен как часть иерархии через @ContextHierarchy
    public void testFoo() {...}
  }
