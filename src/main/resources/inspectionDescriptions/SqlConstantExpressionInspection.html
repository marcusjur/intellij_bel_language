Сообщает об условиях и выражениях, которые всегда возвращают true, false или null.

Пример (MySQL):

CREATE TABLE t1 (a TEXT, b INT, c BOOLEAN);
SELECT a FROM t1 WHERE 'Cat' = 'Cat';
SELECT a FROM t1 WHERE 'Cat' = null;

'Cat' = 'Cat' всегда возвращает true, будет отмечено.

'Cat' = null всегда возвращает null, будет отмечено.
