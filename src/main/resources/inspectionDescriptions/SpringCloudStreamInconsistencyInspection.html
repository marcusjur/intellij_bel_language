Сообщает о несоответствиях в методах обработки потоков.

Методы обработки входных и выходных потоков, аннотированные с помощью @StreamListener или @StreamEmitter, имеют различные ограничения. Нарушение этих ограничений приводит к ошибкам во время выполнения.

Например:

  • Каналы, определённые в атрибуте value аннотации @StreamListener, никогда не могут комбинироваться с параметрами метода, аннотированными @Input или @Output
  • Входной канал должен быть указан для методов, аннотированных @StreamListener
  • Выходной канал должен быть указан для методов, возвращающих значение
  • Выходной канал не может быть указан для методов, не возвращающих значение
  • Атрибут condition аннотации @StreamListener не может быть установлен для методов, возвращающих значение
