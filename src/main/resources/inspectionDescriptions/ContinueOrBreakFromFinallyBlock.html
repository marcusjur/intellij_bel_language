Обнаружены операторы break или continue в блоке finally.

Хотя иногда это может быть преднамеренным, такие операторы могут вводить в заблуждение, маскировать выброшенные исключения и усложнять отладку.

Пример:


    while (true) {
        try {
          throwingMethod();
        } finally {
            continue;
        }
    }
