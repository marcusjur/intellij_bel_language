Сообщать о вызовах функций с типовыми аргументами, которые могут быть автоматически выведены. Такие типовые аргументы избыточны и могут быть безопасно опущены.

Используйте быстрое исправление "Удалить явные типовые аргументы" для очистки кода.

Пример:


  // 在这里可以推断 'String' 类型
  fun foo(): MutableList<String> = mutableListOf<String>()
  // 在这里无法推断 'String'，类型实参为必选项。
  fun bar() = mutableListOf<String>()

После применения быстрого исправления:


  fun foo(): MutableList<String> = mutableListOf() <== обновлено
  fun bar() = mutableListOf<String>()
