Отчёт о проблемах стиля кодования, обнаруженных PHP Mess Detector.

Для работы этой проверки требуется корректно установленный PHP Mess Detector и правильная настройка в Настройки | PHP | Инструменты качества | Mess Detector.

Для получения дополнительной информации об установке PHP Mess Detector см. PHP Mess Detector installation (phpmd.org).

Вы можете использовать предопределённые наборы правил или добавить собственные правила, указав действительный файл ruleset.xml. Подробнее см. rules index (phpmd.org) и How to create a custom rule set (phpmd.org).
