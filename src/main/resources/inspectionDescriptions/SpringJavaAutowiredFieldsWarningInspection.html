Сообщает о полях в Spring-компонентах, которые внедряются или автоматически подключаются.

Это быстрое исправление предлагает внедрение зависимостей на основе конструктора для обязательных полей в Bean-компонентах и добавляет проверки утверждений.

Пример:


class MyComponent {
  @Inject MyCollaborator collaborator; // внедряемое поле
  public void myBusinessMethod() {
    collaborator.doSomething(); // выбрасывает NullPointerException
  }
}

После применения быстрого исправления:


class MyComponent {
  private final MyCollaborator collaborator;
  @Inject
  public MyComponent(MyCollaborator collaborator) {
    Assert.notNull(collaborator, "MyCollaborator must not be null!");
    this.collaborator = collaborator;
  }
  public void myBusinessMethod() {
    collaborator.doSomething(); // теперь этот вызов безопасен
  }
}
