Сообщает о недопустимых ссылках в аннотации Spring Shell org.springframework.shell.standard.ShellMethodAvailability.

Пример:


@ShellComponent
public class MyCommands {
    @ShellMethod("sum")
    @ShellMethodAvailability("sumAvailability") // значение аннотации будет подсвечено как нераспознанный символ
    public int sum(int a, int b) {
        return a + b;
    }
    @ShellMethodAvailability("sumMethod") // значение аннотации будет подсвечено как нераспознанный символ
    public Availability isSumAvailable() {
        return Availability.available();
    }
}
