Сообщает о избыточных вызовах requireNotNull или checkNotNull для выражений, которые не могут быть null.

Пример:


  fun foo(i: Int) {
      requireNotNull(i) //Переменная 'i' никогда не может быть null, поэтому вызов 'requireNotNull' избыточен.
      ...
  }

После применения быстрого исправления:


  fun foo(i: Int) {
      ...
  }
