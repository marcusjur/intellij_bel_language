Сообщает о проблемах, связанных с отсутствием настроенного интерпретатора Python или его недействительностью для проекта. Без правильно настроенного интерпретатора невозможно выполнять Python-скрипты и использовать некоторые функции анализа Python-кода.

ИСР предоставляет быстрый доступ к настройкам интерпретатора.
