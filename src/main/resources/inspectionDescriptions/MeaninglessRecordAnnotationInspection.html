Обнаружены аннотации, которые применены к компонентам record-класса, но не имеют эффекта.

Это может произойти в двух случаях:

  • Аннотация имеет цель METHOD, но для неё явно определён метод доступа.
  • Аннотация имеет цель PARAMETER, но явно определён канонический конструктор.

Пример:


@Target(ElementType.METHOD)
@interface A { }
// Эта аннотация не появится в байт-коде,
// так как она должна была распространиться на метод доступа, но он определён явно
record R(@A int x) {
  public int x() { return x; }
}
