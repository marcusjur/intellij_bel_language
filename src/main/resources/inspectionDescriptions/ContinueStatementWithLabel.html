Обнаружен оператор continue с меткой.

Использование оператора continue с меткой усложняет рефакторинг и может вызвать путаницу.

Пример:


  void handle(List<String> strs) {
    outer:
    for (String s: strs) {
      for (char ch : s.toCharArray()) {
        if ('s' == ch) continue outer;
        handleChar(ch);
      }
    }
  }
