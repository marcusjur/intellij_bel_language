Сообщает о коде, который использует цикл для копирования содержимого массива в коллекцию.

Более короткий и потенциально более быстрый способ (в зависимости от реализации коллекции) — использовать Collection.addAll(Arrays.asList()) или Collections.addAll().

Сообщается только о циклах, которые не содержат дополнительных операторов внутри.

Пример:


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

После применения быстрого исправления:


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
