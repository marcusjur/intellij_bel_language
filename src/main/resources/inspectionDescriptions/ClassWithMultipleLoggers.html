Класс объявляет несколько регистраторов. Важным шагом в обеспечении единообразного ведения журнала для приложения является наличие выделенного регистратора для каждого класса.

Например:


  public class Critical {
    protected static final Logger LOG = Logger.getLogger(Critical.class);
    protected static final Logger myLogger = Logger.getLogger(getClass());
  }
