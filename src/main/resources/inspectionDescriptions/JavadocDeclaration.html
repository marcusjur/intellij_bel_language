Сообщать о следующих проблемах в комментариях и тегах Javadoc:
  • Недопустимое имя тега
  • Неполное описание тега
  • Повторяющиеся теги
  • Отсутствует описание Javadoc

Пример:


  /**
   * Недопустимое имя тега
   * @poram param description
   */
  public void sample(int param){
  }

Пример:


  /**
   * Ссылка на себя {@link #sample(int)}
   */
  public void sample(int param){
  }

Это быстрое исправление добавит неизвестные теги Javadoc в список пользовательских дополнительных тегов.

Используйте текстовое поле ниже для определения дополнительных тегов Javadoc.

Используйте первый флажок, чтобы игнорировать повторяющиеся теги 'throws'.

Используйте второй флажок, чтобы игнорировать отсутствие или неполноту первого предложения в описании.

Используйте третий флажок, чтобы игнорировать ссылки на себя.
