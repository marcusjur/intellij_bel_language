Сообщает о реализациях Iterator.next(), которые не выбрасывают исключение java.util.NoSuchElementException.

Такие реализации нарушают контракт java.util.iterator и могут привести к труднообнаруживаемым ошибкам при нестандартном использовании итератора.

Пример:


  class Numbers implements Iterator<Integer> {
    @Override
    public Integer next() { //warning
        if (hasNext()) {
            return generateNext();
        } else {
            return null; //вместо этого нужно выбросить NoSuchElementException
        }
    }
    ...
  }
