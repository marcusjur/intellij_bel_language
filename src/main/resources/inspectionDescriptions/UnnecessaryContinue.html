Сообщает о последних достижимых операторах continue в цикле. Такие операторы continue являются избыточными и могут быть безопасно удалены.

Пример:


  for (String element: elements) {
    System.out.println();
    continue;
  }

После применения быстрого исправления:


  for (String element: elements) {
    System.out.println();
  }

Эта проверка не анализует JSP файлы.

Используйте опцию Игнорировать в ветке then оператора if с веткой else, чтобы игнорировать операторы continue, находящиеся в ветке then полного оператора if-else.

Пример:


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
