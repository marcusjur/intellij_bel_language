Сообщает о методах, объявленных в одном классе, имеющих одинаковые имена и одинаковое количество параметров. Такая перегрузка может быть очень запутанной, так как может быть неясно, какая именно перегрузка вызывается.

Пример:


  class Main {
    public static void execute(Runnable r) {}
    public static <T> void execute(RunnableFuture<T> c) {}
  }

Используйте эту опцию, чтобы игнорировать перегруженные методы, у которых типы параметров абсолютно несовместимы.
