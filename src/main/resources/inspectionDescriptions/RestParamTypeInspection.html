지원되지 않는 타입의 리소스 메서드 매개변수(@PathParam, @QueryParam 등의 어노테이션이 추가됨)를 보고합니다.

어노테이션이 추가된 매개변수, 필드 또는 프로퍼티의 타입은 다음 요구 조건 중 하나를 만족해야 합니다.

  1. 기본 타입이어야 함
  2. 하나의 String 인수를 허용하는 생성자가 있어야 함
  3. 하나의 String 인수를 허용하고, 이름이 valueOf() 또는 fromString()인 static 메서드(예: Integer.valueOf(String))가 있음
  4. 문자열을 특정 타입으로 변환할 수 있는 ParamConverter 인스턴스를 반환하는 ParamConverterProvider JAX-RS 확장 SPI의 구현이 등록되어 있음
  5. T가 위의 2, 3 또는 4를 만족하는 List<T>, Set<T> 또는 SortedSet<T>이어야 함 결과 컬렉션은 읽기 전용입니다.
