Сообщает о вызовах native методов внутри synchronized блоков или методов.

Если возможно, рекомендуется выносить вызовы native методов за пределы синхронизованного контекста, поскольку такие вызовы приводят к затратному переключению контекста и могут вызвать проблемы с производительностью.

Пример:


  native void nativeMethod();
  void example(){
    synchronized (lock){
      nativeMethod();//предупреждение
    }
  }
