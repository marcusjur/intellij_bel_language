Сообщает об использовании выражения instanceof для проверки типа параметра в блоке catch.

Для проверки типа параметра catch лучше использовать отдельные блоки catch, чем применять instanceof.

Пример:


  void foo(Runnable runnable) {
    try {
        runnable.run();
    } catch (Throwable throwable) {
        if (throwable instanceof NoClassDefFoundError) { // Предупреждение: использование 'instanceof' для параметра 'throwable' в блоке 'catch'
            System.out.println("Class not found!");
        }
    }
  }
