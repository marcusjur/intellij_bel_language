Рекомендуется добавить атрибут #[Override] к методам, которые переопределяют методы родительского класса или реализуемых интерфейсов с тем же именем.

Атрибут #[Override] доступен только начиная с PHP 8.3. Для получения дополнительной информации см. PHP RFC: Marking overridden methods (wiki.php.net).
