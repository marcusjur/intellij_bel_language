Сообщение о приведении типов, которому предшествует проверка instanceof для другого типа.

Хотя это может быть преднамеренным, такая структура, вероятно, является ошибкой и приведет к исключению java.lang.ClassCastException во время выполнения.

Пример:


  class Main {
    int whenCharSequenceCastToNumber(Object o){
      if (o instanceof CharSequence) {
        return ((Number) o).intValue();
      }
      return 0;
    }
    int earlyReturnWhenNotCharSequence(Object o){
      if (!(o instanceof CharSequence)) return 0;
      return ((Number)o).intValue();
    }
  }
