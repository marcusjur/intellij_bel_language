Сообщает о возможных проблемах с EL, таких как неразрешённые ссылки и недопустимые позиции EL.

Пример:


Рассмотрим функцию `exampleFun(String)`, определённую в текущем пространстве имён. 

<jsp:root xmlns:example="urn:jsptld:exampleNamespace.tld">
  ${ example:exampleFun("1") + example:unknownFun("2") } // Ошибка: невозможно разрешить функцию 'unknownFun'
  ${ example:exampleFun("1", "2") } // Ошибка: количество формальных и фактических параметров не совпадает
</jsp:root>
