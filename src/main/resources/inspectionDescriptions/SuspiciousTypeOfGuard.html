Сообщает о ненадежных проверках типов с использованием typeof или instanceof. В следующих двух случаях проверка типов typeof x может быть ненадежной:
  • typeof x не соответствует указанному значению (например, когда x имеет тип 'string | boolean', а typeof x === 'number')
  • typeof x всегда соответствует указанному значению (например, когда x имеет тип 'string', а typeof x === 'string')

В следующих двух случаях проверка типов x instanceof A может быть ненадежной:
  • тип x не связан с A
  • тип x является A или подтипом A
