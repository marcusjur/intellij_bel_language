Сообщать о объявлениях свойств, которые могут быть объединены с последующим присваиванием.

Пример:


  val x: String
  x = System.getProperty("")

Быстрое исправление объединит объявление с присваиванием:


  val x = System.getProperty("")

Настройка проверки:

Вы можете отключить опцию Сообщать о свойствах с комплексной инициализацией, чтобы пропускать свойства со сложной инициализацией. Это включает два случая:

  1. Инициализатор свойства сложный (многострочное или составное выражение/выражение с управляющим потоком)
  2. Свойство сначала инициализуется, а затем сразу используется в последующем коде (например, для вызова других методов инициализации)
