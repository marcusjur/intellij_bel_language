Сообщает об использовании функции asDynamic на приёмниках динамического типа.

Функция asDynamic не влияет на выражения динамического типа.

Использование функции asDynamic на приёмниках динамического типа может привести к проблемам во время выполнения, поскольку asDynamic будет выполняться в среде JavaScript, и такие функции могут быть недоступны во время выполнения. Рекомендуется использовать эту функцию на обычных типах Kotlin.

Удаление вызова "asDynamic" доступно как быстрое исправление для автоматической корректировки кода.

Пример:


  fun wrongUsage(d: Dynamic) {
     d.asDynamic().foo() // <== избыточно, быстрое исправление упростит выражение вызова до "d.foo()"
  }
