Сообщает о необязательных преобразованиях типов.

Пример:

var s = string("hello")

Значение "hello" уже имеет тип string, поэтому дополнительное преобразование в string избыточно. Чтобы удалить преобразование, воспользуйтесь быстрым исправлением Удалить преобразование.

После применения быстрого исправления:

var s = "hello"

Иногда преобразование выражений с плавающей точкой в тип с плавающей точкой может быть намеренным (см. эту проблему). В таких случаях ИСР предупреждает о потенциально избыточном преобразовании.
