Сообщается, что вспомогательный конструктор может быть заменен более лаконичным основным конструктором.

Пример:


  class User {
      val name: String
      constructor(name: String) {
          this.name = name
      }
  }

Быстрое исправление автоматически преобразует код:


  class User(val name: String) {
  }
