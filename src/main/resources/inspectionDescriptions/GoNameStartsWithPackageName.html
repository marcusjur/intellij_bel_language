Сообщает об экспортуемых именах, которые начинаются с имени пакета. Эта проверка не выполняется для имен в пакете main.

Пример:

package myPackage
func MyPackageGetIP() {
}

Имя MyPackageGetIP будет подсвечено, так как оно начинается с имени пакета.

Согласно Package Names на github.com/golang, все ссылки на имена в пакете будут дополнены именем пакета, поэтому его можно опустить в идентификаторе. Например, если вы находитесь в пакете foo, вам не нужен тип FooFile, так как клиенты будут писать его как foo.FooFile. Вместо этого следует назвать тип просто File, и клиенты будут использовать его как foo.File.
