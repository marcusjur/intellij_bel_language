Сообщать об использовании замыкающих запятых в объектных литералах.

Это предупреждение отображается только если версия JavaScript установлена на ECMAScript 5.1.

Хотя спецификация разрешает использование замыкающих запятых в объектных литералах, некоторые браузеры могут выдавать ошибку при их использовании.

Вы можете настроить параметры форматования замыкающих запятых в разделе Стиль кода | JavaScript или TypeScript | Пунктуация.
