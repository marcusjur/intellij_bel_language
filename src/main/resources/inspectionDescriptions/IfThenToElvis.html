Выражение if-then можно свернуть в elvis-выражение (?:).

Пример:


  fun maybeFoo(): String? = "foo"
  var foo = maybeFoo()
  val bar = if (foo == null) "hello" else foo

Это быстрое исправление преобразует выражение if-then в elvis-выражение (?:):


  fun maybeFoo(): String? = "foo"
  var foo = maybeFoo()
  val bar = foo ?: "hello"
