Сообщает о шаблонном instanceof и предлагает преобразовать его в обычный instanceof с приведением типа.

С помощью быстрого исправления эта проверка может перенести шаблонный instanceof в кодовую базу, использующую более ранние версии Java.

Обратите внимание, что при использовании сложных выражений перед instanceof результат может не быть полностью эквивалентным исходному шаблонному instanceof. В этом случае выражение будет вычислено повторно.

Пример:


  if (object instanceof String txt && txt.length() == 1) {
      System.out.println(txt);
  } else {
      return;
  }
  System.out.println(txt);

После применения быстрого исправления:


  if (object instanceof String && ((String) object).length() ==1) {
      String txt = (String) object;
      System.out.println(txt);
  } else {
      return;
  }
  String txt = (String) object;
  System.out.println(txt);
