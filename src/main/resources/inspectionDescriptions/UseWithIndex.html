Обнаружен цикл for с переменной индекса, увеличивающейся вручную.

Цикл for с переменной индекса, увеличивающейся вручную, можно упростить с помощью функции withIndex().

Быстрое исправление Использовать withIndex() вместо ручного увеличения индекса доступно для автоматического исправления кода.

Пример:


  fun foo(list: List<String>): Int? {
      var index = 0
      for (s in list) { <== can be simplified
          val x = s.length * index
          index++
          if (x > 0) return x
      }
      return null
  }

После применения быстрого исправления:


  fun foo(list: List<String>): Int? {
      for ((index, s) in list.withIndex()) {
          val x = s.length * index
          if (x > 0) return x
      }
      return null
  }
