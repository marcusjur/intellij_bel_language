Сообщает о константных значениях, находящихся на неправильной стороне операции сравнения.

Некоторые соглашения о кодовании определяют, на какой стороне сравнения должна находиться константа – слева или справа.

Пример:


  boolean compare(int x) {
    return 1 > x; // Константа '1' находится на левой стороне сравнения
  }

После применения быстрого исправления:


  boolean compare(int x) {
    return x < 1;
  }

В настройках проверки можно выбрать предпочтительную сторону для констант в сравнении, а также указать, следует ли выдавать предупреждение, когда литерал null находится на неправильной стороне.
