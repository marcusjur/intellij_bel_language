Сообщает о вызовах String.equals() с аргументом типа CharSequence.

Метод String.equals() может вернуть true только для аргументов типа String. Для сравнения содержимого String с аргументом типа CharSequence, не являющимся String, используйте метод contentEquals().

Пример:


  boolean equals(String s, CharSequence ch) {
    return s.equals(ch);
  }

После применения быстрого исправления:


  boolean equals(String s, CharSequence ch) {
    return s.contentEquals(ch);
  }
