Сообщает об операторах, которые могут изменить базу данных во время работы в режиме только для чтения.

Чтобы включить режим только для чтения для подключения, щёлкните правой кнопкой мыши на источнике данных в окне инструмента База данных (Вид | Окна инструментов | База данных) и выберите Свойства. В диалоговом окне Источники данных и драйверы перейдите на вкладку Параметры и установите флажок Только для чтения.

Пример (MySQL):

CREATE TABLE foo(a INT);
INSERT INTO foo VALUES (1);

Операторы CREATE TABLE и INSERT INTO будут подсвечены в режиме подключения только для чтения, так как они вносят изменения в базу данных.
