Сообщает о несоответствии типов свойств в JPA-аннотациях.

Пример:



  @Entity
  public class JavaEntity {
    @OneToOne
    Map<Integer, AnotherEntity> incorrectRelationship; // Ошибка: тип связи 'One To One' должен быть Entity, а не Map
    @ManyToMany
    Map<Integer, AnotherEntity> correctRelationship;
  }
