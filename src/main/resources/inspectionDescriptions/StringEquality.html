Обнаружено сравнение строк с использованием операторов == или !=.

Эти операторы проверяют равенство ссылок, а не содержимого строк. В большинстве случаев для сравнения строк следует использовать метод equals(), который выполняет посимвольное сравнение, когда строки являются разными объектами.

Пример:


  void foo(String s, String t) {
    final boolean b = t == s;
  }

Если известно, что t не является null, можно безопасно применить "небезопасное" быстрое исправление, что приведет к следующему результату:


  void foo(String s, String t) {
    final boolean b = t.equals(s);
  }
