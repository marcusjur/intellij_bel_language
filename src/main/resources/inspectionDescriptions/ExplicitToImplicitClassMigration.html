Сообщает об обычных классах, которые могут быть преобразованы в неявные объявления.

Пример:


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

После применения быстрого исправления:


  public static void main(String[] args) {
      System.out.println("Hello, world!");
  }
