Обнаружено объявление метода equal() с одним параметром. Обычно это опечатка, вероятно, подразумевался метод equals().

Рекомендуется переименовать метод в equals с помощью быстрого исправления.

Пример:


  class Main {
    public boolean equal(Object obj) {
      return true;
    }
  }

После применения быстрого исправления:


  class Main {
    public boolean equals(Object obj) {
      return true;
    }
  }
