Обнаружены следующие ошибки в EJB QL запросе:
  • Несоответствие типов выражений
  • Некорректные параметры
  • Пустые или константные условия
  • Неразрешённые символы

Примеры запросов:


SELECT OBJECT(e) FROM Employee e WHERE SQRT(ABS(e.firstName - ?1)) > 50000; // Ошибка: несоответствие типов, требуется числовой тип
SELECT e.firstName FROM Employee e WHERE SUBSTRING(e.firstName, ?1, ?2) = ?4; // Ошибка: определено только 3 фактических параметра запроса
