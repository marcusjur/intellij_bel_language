Сообщает о переменных, объявленных как java.lang.String, к которым происходят повторные присоединения. Такие переменные могут быть более эффективно объявлены как java.lang.StringBuffer или java.lang.StringBuilder.

Пример:


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

Такой цикл можно заменить на:


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

Или даже на:


  String s = String.join(" ", names);

Используйте эту опцию, чтобы проверка сообщала только о переменных, к которым происходит присоединение в цикле.
