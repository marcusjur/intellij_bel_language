Сообщение об инстанцировании класса java.lang.ClassLoader.

Хотя обычно это безопасно, любое создание экземпляра ClassLoader должно быть тщательно проверено при проведении аудита безопасности.

Пример:

    Class<?> loadExtraClass(String name) throws Exception {
      try(URLClassLoader loader =
          new URLClassLoader(new URL[]{new URL("extraClasses/")})) {
        return loader.loadClass(name);
      }
    }
