Внутренний класс может быть объявлен как static.

static внутренний класс не сохраняет неявную ссылку на экземпляр внешнего класса. При использовании Java 17 или более ранних версий это предотвращает распространенные утечки памяти и уменьшает объем памяти, используемой каждым экземпляром класса.

Пример:


  public class Outer {
    class Inner { // не static
      public void foo() {
        bar("x");
      }
      private void bar(String string) {}
    }
  }

После применения быстрого исправления:


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }
      private void bar(String string) {}
    }
  }
