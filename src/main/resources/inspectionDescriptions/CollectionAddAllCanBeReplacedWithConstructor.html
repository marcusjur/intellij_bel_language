Сообщать о вызовах Collection.addAll() и Map.putAll(), которые следуют сразу после создания коллекции с помощью конструктора без параметров.

Такая структура может быть заменена одним вызовом параметризованного конструктора, что упрощает код. Кроме того, для некоторых коллекций такая замена может быть более эффективной.

Пример:


  Set<String> set = new HashSet<>();
  set.addAll(Arrays.asList("alpha", "beta", "gamma"));

После применения быстрого исправления:


  Set<String> set = new HashSet<>(Arrays.asList("alpha", "beta", "gamma"));

По умолчанию поддерживаются коллекции JDK. Дополнительные классы можно указать на панели Проверяемые классы.
