Сообщать о столбцах в предложении USING, которые отсутствуют в обеих таблицах.

Пример (MySQL):

CREATE TABLE t1 (i INT, j INT);
CREATE TABLE t2 (k INT, l INT);
SELECT * FROM t1 JOIN t2 USING (j);

В предложении USING имена столбцов должны существовать в обеих таблицах, запрос SELECT автоматически объединяет эти таблицы по указанным именам столбцов. Поскольку у нас нет столбца j в таблице t2, мы можем переписать запрос с использованием ON. Предложение ON позволяет объединять таблицы, где имена столбцов не совпадают.

SELECT * FROM t1 JOIN t2 ON t1.j = t2.l;
