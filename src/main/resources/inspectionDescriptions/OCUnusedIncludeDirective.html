Сообщать о неиспользуемых или ненужных директивах include, которые можно безопасно удалить.
Обнаруживать полностью неиспользуемые директивы include
Директива include считается неиспользуемой, если объявления из нее никогда не используются в единице трансляции.
Обнаруживать директивы include без прямого использования
Директива include считается неиспользуемой, если объявления из нее не используются напрямую в файле. Наиболее эффективно работает со стратегией "Включать только то, что используется".
Обнаруживать избыточные директивы include
Определяет минимальный набор директив include, необходимых для компиляции файла. Все остальные директивы #include считаются неиспользуемыми.
