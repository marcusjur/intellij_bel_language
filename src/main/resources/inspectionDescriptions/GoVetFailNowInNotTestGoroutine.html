Сообщает о вызовах testing.T.FailNow и подобных методов в goroutine тестового файла.

Методы, подобные FailNow, вызывают runtime.Goexit и останавливают вызывающую goroutine, а не тест. Поэтому их следует вызывать только из goroutine, выполняющей тест или бенчмарк.

Подробную информацию о FailNow смотрите в func (*T) FailNow на go.dev.

Пример:

func TestFoo(t *testing.T) {
  go func() {
    t.Fatal("oops") // завершает goroutine, а не TestFoo
  }()
}

После применения быстрого исправления заменить на 'Error' и 'return':

func TestFoo(t *testing.T) {
  go func() {
    t.Error("oops")
    return
  }()
}
