Сообщается о возможности замены приведения типа на существующую локальную переменную или переменную шаблона с тем же значением.

Пример:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(((String) obj).trim());
  }

После применения быстрого исправления:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(s.trim());
  }

Новое в версии 2022.3
