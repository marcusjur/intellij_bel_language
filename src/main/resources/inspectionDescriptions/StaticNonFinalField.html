Сообщает о полях, не отмеченных как final static.

С помощью быстрого исправления можно добавить модификатор final к полям static, не отмеченным как final.

Эта проверка не анализует изменяемость полей. Например, добавление модификатора final к полю, значение которого где-то устанавливается, приведет к ошибке компиляции.

Используйте опцию Проверять только 'public' поля, чтобы проверка выполнялась только для полей с модификатором public.
