Обнаружение отсутствующих ключей в CONSTANCE_CONFIG_FIELDSETS (используется для настройки констант в Django Constance).

Пример:


CONSTANCE_CONFIG = {
    'SITE_NAME': ('My Title', 'Website title'),
    'SITE_DESCRIPTION': ('', 'Website description'),
}
CONSTANCE_CONFIG_FIELDSETS = {
    'General Options': ('SITE_NAME'),
}
