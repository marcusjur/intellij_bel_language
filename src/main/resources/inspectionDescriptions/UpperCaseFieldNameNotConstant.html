Сообщает о полях, написанных заглавными буквами, которые не являются static или final.

Такие поля могут вызвать путаницу, так как нарушают общепринятые соглашения об именовании, и часто используются неправильно.

Пример: 


  public static int THE_ANSWER = 42; // Предупреждение: отсутствует модификатор final

Быстрое исправление для переименования таких полей доступно только в редакторе.
