Сообщать о ненужных операциях приведения типов.

Пример:


  static Object toObject(String s) {
    return (Object) s;
  }

Используйте флажок ниже, чтобы игнорировать уточняющие приведения типов, например приведение типа в вызовах коллекций, которые должны быть типа Object: 


  static void removeFromList(List<String> l, Object o) {
    l.remove((String)o);
  }
