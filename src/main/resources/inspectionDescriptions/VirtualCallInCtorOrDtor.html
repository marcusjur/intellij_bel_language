Сообщает о вызовах виртуальных функций-членов из конструкторов или деструкторов. Поскольку конструирование начинается с базового класса и движется к производному, ресурсы производного класса еще не инициализованы. Деструкция выполняется в обратном порядке, поэтому вызов виртуальных функций может привести к доступу к уже разрушенным ресурсам.

Пример:


  class Test {
  public:
      void virtual setUp();
      Test() {
          setUp();
      }
  };
