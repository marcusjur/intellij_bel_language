Сообщает о вызове Supplier lambda в Optional.orElseThrow(), где исключение выбрасывается явно вместо возврата.

Пример:


optional.orElseThrow(() -> {
    throw new RuntimeException();
});

После применения быстрого исправления:


optional.orElseThrow(() -> new RuntimeException());

Новое в версии 2023.1
