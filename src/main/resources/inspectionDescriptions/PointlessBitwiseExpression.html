Обнаружение бессмысленных побитовых выражений.

К таким выражениям относятся применение оператора & к максимальному значению для данного типа, применение оператора or к нулю, а также сдвиг на ноль. Подобные выражения могут появиться в результате незавершенного автоматического рефакторинга и вряд ли отражают изначальное намерение.

Пример:


  // Предупреждение: операция бессмысленна, можно заменить на `flags`
  // 0xFFFF_FFFF является максимальным значением для целого числа, и оба литерала
  // рассматриваются как 32-битные целочисленные литералы.
  int bits = flags & 0xFFFF_FFFF;
  // Предупреждение: операция бессмысленна, можно заменить на `bits`
  // Операция OR с 0 всегда возвращает другой операнд.
  int or = bits | 0x0;
  // Предупреждение: операция бессмысленна, результат всегда 0
  int xor = or ^ or;
