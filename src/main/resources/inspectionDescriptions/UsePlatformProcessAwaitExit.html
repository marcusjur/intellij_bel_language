Сообщение об использовании Process.waitFor() и Process.onExit() в контексте корутин. Использование этих методов в контексте корутин запрещено.

Вместо этого используйте com.intellij.util.io.awaitExit().

Пример:


suspend fun doSomething(process: Process) {
  val exitCode = process.waitFor() // Неверно
  // ...
}
suspend fun doSomething(process: Process) {
  val exitCode = process.awaitExit() // Верно
  // ...
}
