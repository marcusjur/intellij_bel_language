Обнаружен избыточный сеттер свойства.

Сеттер считается избыточным в одном из следующих случаев:

  1. Сеттер не имеет тела. Видимость аксессора не изменена, объявление не является external и отсутствуют аннотации. 
    
  var myPropWithRedundantSetter: Int = 0
      set // избыточный
  var myPropA: Int = 0
      private set // допустимо - видимость свойства изменена на private
  var myPropB: Int = 0
      external set // допустимо - не реализовано в Kotlin (внешняя реализация)
  var myPropC: Int = 0
      @Inject set // допустимо - аксессор имеет аннотацию
  2. Тело сеттера представляет собой блок с единственным выражением, которое присваивает параметр теневому полю. 
    
  var prop: Int = 0
      set(value) { // избыточный
          field = value
      }
