Сообщает о вызовах методов java.util.Collection или java.util.Map, где аргументом является сама коллекция или отображение. Такая ситуация может возникнуть в коде с сырыми типами из-за копирования и вставки.

Пример:


  ArrayList list = new ArrayList<>();
  list.add(list); // здесь будет предупреждение
  return list.hashCode(); // выбрасывает StackOverflowError
