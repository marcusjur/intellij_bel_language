Сообщает о недопустимых вызовах protected/private методов.

Protected методы можно вызывать с использованием неявного или явного получателя правильного типа.

Private методы можно вызывать только через неявного получателя (за исключением методов-сеттеров self, например self.foo=).

Protected и private методы можно вызывать только в контексте обработчика метода.
