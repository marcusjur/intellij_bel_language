Сообщает о избыточных вызовах функции with, которые не обращаются к содержимому приёмника.

Пример:


  class MyClass {
      fun f(): String = ""
  }
  fun testRedundant() {
      with(c) { // <== 'with' избыточен, так как 'c' не используется
          println("1")
      }
  }
  fun testOk() {
      val c = MyClass()
      with(c) { // <== Корректно, так как 'f()' является допустимым 'c.f()'
          println(f())
      }
  }
