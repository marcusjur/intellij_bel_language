Сообщает об использовании помеченного return в последнем выражении лямбда-функции.

Такой return избыточен и может быть безопасно удален.

Пример:


fun foo() {
  listOf(1,2,3).find {
    return@find true
  }
}

После применения быстрого исправления:


fun foo() {
  listOf(1,2,3).find {
    true
  }
}
