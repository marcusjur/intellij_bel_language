Обнаружена ошибка в объявлении конечной точки WebSocket.

Согласно спецификации JSR 356, конечная точка WebSocket должна иметь только один метод @OnMessage для каждого типа сообщений: text, message и pong

Класс конечной точки с аннотацией javax.websocket.server.ServerEndpoint может иметь только один метод с одной из следующих аннотаций жизненного цикла:

  • @OnOpen
  • @OnClose
  • @OnError

Класс конечной точки, расширяющий javax.websocket.Endpoint, должен реализовывать соответствующие методы жизненного цикла вместо использования аннотаций.
