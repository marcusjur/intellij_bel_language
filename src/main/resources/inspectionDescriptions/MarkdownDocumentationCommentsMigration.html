Отчёт может быть преобразован из Javadoc-комментариев в документационные комментарии в формате Markdown.

Пример:


  /**
   * For blubbering of the florz.
   *
   * Similar to {@link com.deeps.Querp querping} class.
   */
  class FlorzBlubber {
  }

После применения быстрого исправления:


  /// For blubbering of the florz.
  /// Similar to [querping][com.deeps.Querp] class.
  class FlorzBlubber {
  }
