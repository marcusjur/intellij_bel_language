Сообщает об использовании операторов return, throw, break, continue и yield внутри блока finally. Это приводит к преждевременному завершению блока finally вместо его нормального выполнения. Любые исключения, выброшенные из блоков try и catch того же оператора try-catch, будут подавлены.

Пример:


  void x() {
    try {
      throw new RuntimeException();
    } finally {
      // Если bar() вернет true, RuntimeException будет подавлено
      if (bar()) return;
    }
  }
