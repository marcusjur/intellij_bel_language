Сообщать о случаях использования унарного оператора +. Унарный плюс обычно является избыточной операцией, и его присутствие может указывать на ошибку в коде. Например, при использовании с оператором инкремента (как в +++) или оператором присваивания (как в =+).

Пример:


  void unaryPlus(int i) {
    int x = + +i;
  }

Предлагаются следующие варианты быстрого исправления:

  • Удалить оператор + перед переменной i:

    
  void unaryPlus(int i) {
    int x = i;
  }

  • Заменить оператор + на префиксный оператор инкремента:

    
  void unaryPlus(int i) {
    int x = ++i;
  }

Используйте флажок ниже, чтобы включить проверку унарного плюса в сочетании с бинарными или другими унарными выражениями. Это означает, что проверка не будет выявлять случаи использования унарного плюса в инициализаторах массивов или в качестве аргументов методов.
