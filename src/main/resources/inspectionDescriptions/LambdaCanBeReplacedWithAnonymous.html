Сообщается, что лямбда-выражение может быть преобразовано в анонимный класс.

Расширение лямбда-выражения до анонимного класса может быть полезно, если необходимо реализовать дополнительные методы в анонимном классе.

Пример:


  s -> System.out.println(s)

После применения быстрого исправления:

new Consumer<String>() {
  @Override
  public void accept(String s) {
    System.out.println(s);
  }
}
