Сообщать о ссылках на статические методы и поля через экземпляр класса, а не через сам класс.

Хотя спецификация PHP позволяет обращаться к статическим членам через переменные экземпляра, это может привести к путанице, так как читатели могут ошибочно предположить, что метод зависит от экземпляра.

Это быстрое исправление заменит переменную экземпляра на имя класса.

Пример:


  class StaticClass {
    public static function someStaticFunction(): int
    {
        return 3;
    }
  }
  $var = new StaticClass
  $var::someStaticFunction();

После применения быстрого исправления:


  StaticClass::someStaticFunction()
