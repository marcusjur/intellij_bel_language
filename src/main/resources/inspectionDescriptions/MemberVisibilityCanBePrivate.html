Отчёт может быть объявлен как private в соответствии с принципом инкапсуляции.

Пример:


class Service(val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}

После применения быстрого исправления (учитывая, что url не используется за пределами класса Service):


class Service(private val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}
