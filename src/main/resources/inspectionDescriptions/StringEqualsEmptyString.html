Сообщает о случаях, когда метод equals() используется для сравнения String с пустой строкой. В таких случаях лучше использовать .isEmpty(), так как это точнее отражает то, что вы проверяете.

Пример:


  void checkString(String s){
    if ("".equals(s)) throw new IllegalArgumentException();
  }

После применения быстрого исправления:


  void checkString(String s){
    if (s != null && s.isEmpty()) throw new IllegalArgumentException();
  }

"".equals(str) возвращает false, когда str равен null. Для безопасности быстрое исправление этой проверки добавляет явную проверку на null, когда аргумент метода equals() может быть null. Используйте эту опцию, чтобы проверка игнорировала такие случаи.
