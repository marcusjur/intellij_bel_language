Сообщает об использовании встроенных комментариев, которые содержат только имена параметров перед аргументами в вызовах функций и, таким образом, служат подсказками. В PHP 8.0 и выше вместо этого можно использовать именованные аргументы.

Для получения дополнительной информации см. Именованные аргументы (php.net).
