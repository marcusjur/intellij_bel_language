Обнаружена недопустимая ссылка с Entity на SharedEntity.

Невозможно ссылаться на Entity из SharedEntity. Ссылка с Entity на SharedEntity допустима только если свойство является nullable или имеет аннотацию @CascadeDeleteBy. В противном случае это может привести к исключению во время выполнения.

Пример:


  interface A : LegacyEntity {
    var x: SharedEntity
    var y: SharedEntity
  }

Можно использовать быстрое исправление для добавления аннотации или преобразования типа в nullable


  interface A : LegacyEntity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
