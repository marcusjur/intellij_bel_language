Сообщает об использовании java.awt.Insets и JBUI.insetsXyz(), которое можно упростить.

Экземпляры Insets не учитывают DPI и могут вызвать проблемы с компоновкой UI.

Быстрое исправление выполнит замену или упростит выражение с помощью JBUI.insets().

Пример:


// Неверно:
Insets insets1 = new Insets(1, 2, 3, 4);
Insets insets2 = new Insets(1, 2, 1, 2);
Insets insets3 = new Insets(1, 0, 0, 0);
// Верно:
Insets insets1 = JBUI.insets(1, 2, 3, 4);
Insets insets2 = JBUI.insets(1, 2);
Insets insets3 = JBUI.insetsTop(1);
