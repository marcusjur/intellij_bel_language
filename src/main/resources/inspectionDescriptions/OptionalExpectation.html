Сообщает о необязательных ожидаемых аннотациях, для которых отсутствуют фактические аннотации в некоторых платформенных модулях.

Пример:

// 通用代码
@OptionalExpectation
expect annotation class JvmName(val name: String)
@JvmName(name = "JvmFoo")
fun foo() { }
// jvm 代码
actual annotation class JvmName(val name: String)

Эта проверка также сообщает о случаях, когда actual annotation class JvmName опущен для не-JVM платформ (например, Native).
