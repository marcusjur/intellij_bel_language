Сообщать об устаревшем использовании неявно допускающих null параметров.

Начиная с PHP 8.4, использование null в качестве значения параметра по умолчанию без явного объявления null в типе считается устаревшим.

Для получения дополнительной информации см. Устаревание неявно допускающих null типов параметров.
