컬렉션 및 Options의 빈 값을 검사하는 다양한 방식을 보고하고 .isEmpty, .nonEmpty 또는 .isDefined.로 바꿀 것을 제안합니다.

예:


  !Seq(1).isEmpty
  Seq(1).size == 0
  Seq(1).size != 0
  !(Seq(1).length > 0)
  Seq(1).exists(_ => true)
  import scala.Function.const
  Seq(1).exists(const(true))
  !Option(0).isEmpty
  Option(0) == None
  Option(0) != None

빠른 수정 적용 후:


  Seq(1).nonEmpty
  Seq(1).isEmpty
  Seq(1).nonEmpty
  Seq(1).isEmpty
  Seq(1).nonEmpty
  import scala.Function.const
  Seq(1).nonEmpty
  Option(0).isDefined
  Option(0).isEmpty
  Option(0).isDefined
