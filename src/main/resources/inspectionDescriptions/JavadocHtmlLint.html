Сообщает о тех же проблемах с HTML в комментариях Javadoc, которые отмечает DocLint начиная с Java 8.

Эта проверка обнаруживает следующие проблемы:

  • Самозакрывающиеся, незакрытые, неизвестные, неправильно размещенные или пустые теги
  • Неизвестные или некорректные атрибуты
  • Неправильно размещенный текст

Пример:

 * Некорректный или пустой атрибут: <a href=""></a>
 * Самозакрывающийся тег: <br/>
 * …
 */
