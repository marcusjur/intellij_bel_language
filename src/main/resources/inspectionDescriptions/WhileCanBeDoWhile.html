Цикл while может быть более эффективно переписан как цикл do-while. Существует цикл while, в котором код перед циклом идентичен коду в теле цикла. Замена на цикл do-while позволит устранить повторяющийся код. Для циклов while без такого повторяющегося кода быстрое исправление также доступно в редакторе, но без подсветки.

Пример:


  foo();
  while (x) {
      foo();
  }

Может быть заменено на:


  do {
    foo();
  } while (x);
