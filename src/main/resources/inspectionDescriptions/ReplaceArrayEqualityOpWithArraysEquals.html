Сообщает об использовании операторов == или != для массивов, где следует использовать contentEquals().

Операторы == и != сравнивают ссылки на массивы, а не их содержимое.

Пример:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // сравнение по ссылке
  }

После применения быстрого исправления:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
