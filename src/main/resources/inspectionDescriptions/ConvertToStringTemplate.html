Сообщает о конкатенации строк, которые могут быть преобразованы в строковые шаблоны.

Рекомендуется использовать строковые шаблоны, так как это делает код более читаемым.

Пример:


  fun example() {
      val capitals = mapOf("France" to "Paris", "Spain" to "Madrid")
      for ((country, capital) in capitals) {
          print(capital + " is a capital of " + country)
      }
  }

После применения быстрого исправления:


  fun example() {
      val capitals = mapOf("France" to "Paris", "Spain" to "Madrid")
      for ((country, capital) in capitals) {
          print("$capital is a capital of $country")
      }
  }
