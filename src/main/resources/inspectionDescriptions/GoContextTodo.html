Сообщение об использовании context.TODO().

Согласно документации на pkg.go.dev, context.TODO следует использовать, когда неясно, какой Context использовать, или когда Context еще недоступен. Context может быть недоступен, если окружающие функции еще не расширены для принятия параметра Context.

Обратите внимание, что это временный заполнитель, который в будущем необходимо заменить на более подходящий контекст (например, context.Background()).
