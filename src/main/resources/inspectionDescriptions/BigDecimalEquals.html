Сообщает о вызовах метода equals() при сравнении двух чисел типа java.math.BigDecimal. Это обычно является ошибкой, так как два числа java.math.BigDecimal считаются равными только при совпадении как значения, так и масштаба.

Пример:


  if (new BigDecimal("2.0").equals(
    new BigDecimal("2.00"))) {} // false

После применения быстрого исправления:


  if (new BigDecimal("2.0").compareTo(
    new BigDecimal("2.00")) == 0) {} // true
