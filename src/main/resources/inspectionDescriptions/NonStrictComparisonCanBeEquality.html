Анализ потока данных показывает, что условие неравенства выполняется только для одного числового значения операнда. Такое условие может быть заменено на проверку равенства для улучшения читаемости кода.

Пример:


  if (x >= 10) {
    ...
    if (x <= 10) { // can be replaced with 'x == 10'
    }
  }
