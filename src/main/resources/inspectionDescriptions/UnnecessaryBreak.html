Сообщать о любых ненужных операторах break.

Оператор break считается ненужным, если после его удаления не выполняются никакие другие операторы.

Пример:


  switch (e) {
    case A -> {
        System.out.println("A");
        break; // Сообщение о ненужном операторе 'break'
    }
    default -> {
        System.out.println("Default");
        break; // Сообщение о ненужном операторе 'break'
    }
  }
