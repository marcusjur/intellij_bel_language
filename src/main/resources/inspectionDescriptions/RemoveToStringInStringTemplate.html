Сообщает о вызовах toString(), которые можно безопасно удалить из строковых шаблонов.

Пример:

  fun foo(a: Int, b: Int) = a + b
  fun test(): String {
      return "Foo: ${foo(0, 4).toString()}" // 'toString()' избыточен
  }

После применения быстрого исправления:

  fun foo(a: Int, b: Int) = a + b
  fun test(): String {
      return "Foo: ${foo(0, 4)}"
  }
