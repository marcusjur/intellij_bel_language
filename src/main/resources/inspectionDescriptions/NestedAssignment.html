Обнаружение операций присваивания, вложенных в другие выражения.

Такие выражения могут вводить в заблуждение и нарушают общий принцип проектирования, согласно которому каждая конструкция должна выполнять только одну функцию.

Пример:


  String userName;
  // Предупреждение: используется результат присваивания 'userName'
  String message = "Hello " + (userName = "Alice") + "!"
  System.out.println(message);
  System.out.println("Goodbye " + userName);
