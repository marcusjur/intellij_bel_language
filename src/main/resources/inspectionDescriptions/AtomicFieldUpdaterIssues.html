Сообщает о проблемах с полями AtomicLongFieldUpdater, AtomicIntegerFieldUpdater или AtomicReferenceFieldUpdater (пакет java.util.concurrent.atomic).

Сообщаются те же проблемы, которые могут возникнуть во время выполнения с атомарными обновлениями полей: не найдено указанное поле, указанное поле недоступно, неверный тип указанного поля и т.д.

Примеры:

  • 
  class A {
    private static volatile int value = 0;
    private static final AtomicIntegerFieldUpdater updater =
    AtomicIntegerFieldUpdater.newUpdater((A.class), "value"); // Предупреждение: поле 'value' имеет модификатор 'static'
  }
  
  • 
  class B {
    private static final AtomicIntegerFieldUpdater updater =
      AtomicIntegerFieldUpdater.newUpdater(B.class, "value"); // Предупреждение: поле 'value' не найдено в классе 'B'
  }
