Сообщает об отсутствующих и пустых docstring.

Пример отсутствующего docstring


def demo(a):
    c = a ** 2

Пример пустого docstring


def demo(a):
    """
    """
    c = a ** 2

После применения быстрого исправления код становится:


def demo(a):
    """
    :param a:
    """
    c = a ** 2

Вам необходимо предоставить подробную информацию о параметрах в сгенерированном шаблоне.
