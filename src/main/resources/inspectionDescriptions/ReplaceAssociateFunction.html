Сообщает о вызовах associate() и associateTo(), которые могут быть заменены на associateBy() или associateWith().

Обе функции принимают функцию-преобразователь, которая применяется к элементам заданной последовательности или коллекции (выступающей в качестве приемника). Затем эта функция пар используется для построения результирующего Map.

Учитывая, что преобразователь ссылается на it, вызовы associate[To]() могут быть заменены на более производительные associateBy() или associateWith().

Пример:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L
  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // Заменяемый 'associate()'
      listOf(1).associate { it to getValue(it) } // Заменяемый 'associate()'
  }

После применения быстрого исправления:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L
  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }
