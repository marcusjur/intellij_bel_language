Сообщает о вызовах Object.notify() или Object.notifyAll(), которым не предшествовали какие-либо обнаруживаемые изменения состояния.

Object.notify() и Object.notifyAll() обычно используются для уведомления других потоков об изменении состояния. Такое изменение состояния должно происходить в том же синхронизованном контексте, что и вызов Object.notify() или Object.notifyAll(), и предшествовать этому вызову. Хотя отсутствие такого изменения состояния не обязательно является ошибкой, это определённо заслуживает проверки.

Пример:


  synchronized (this) {
    notify();
  }
  // Нет изменения состояния
  synchronized (this) {
    notify(); // Уведомление может быть избыточным
  }
