Сообщает о небезопасных операциях с использованием this во время конструирования объекта, включая:
  • Доступ к не final полям во время инициализации класса: из конструктора или инициализатора свойств
  • Вызов не final методов во время инициализации класса
  • Использование this в качестве аргумента функции в конструкторе не final класса

Если другие классы наследуются от данного класса, они могут быть не полностью инициализованы в момент выполнения небезопасной операции.

Пример:


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }
  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }
  fun testIt() {
      println(Derived(42).code) // Ожидается: 42, Фактически: 0
  }
