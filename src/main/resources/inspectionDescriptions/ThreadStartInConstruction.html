Сообщает о вызовах метода start() для java.lang.Thread или любого его подкласса во время конструирования объекта.

Хотя такая структура иногда может быть полезной, её следует избегать из-за проблем с наследованием. Подклассы класса, который запускает поток во время конструирования объекта, не смогут выполнить свою инициализацию до запуска потока.

Проверка не выполняется, если класс, запускающий поток, объявлен как final.

Пример:


  class MyThread extends Thread {
    MyThread() {
      start();
    }
  }
