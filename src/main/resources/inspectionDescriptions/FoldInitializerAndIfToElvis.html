Сообщает о выражениях if, которые проверяют переменную на null сразу после инициализации, и которые можно преобразовать в оператор elvis в инициализаторе.

Пример:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo
      if (i == null) {
          return bar
      }
      return i
  }

Быстрое исправление преобразует выражение if с инициализатором в выражение elvis:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo ?: return bar
      return i
  }
