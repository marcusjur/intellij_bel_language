Сообщает о вызовах метода equals() для перечислений (enum).

Такие вызовы можно заменить на сравнение по идентичности (==), поскольку константы перечисления равны только тогда, когда они идентичны.

Вызов можно изменить на сравнение с помощью быстрого исправления.

Пример:


  boolean foo(MyEnum value) {
    return value.equals(MyEnum.FOO);
  }

После применения быстрого исправления:


  boolean foo(MyEnum value) {
    return value == MyEnum.FOO;
  }
