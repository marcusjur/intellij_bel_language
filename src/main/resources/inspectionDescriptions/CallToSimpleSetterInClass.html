Сообщает о вызовах простых сеттеров свойств внутри класса, которому эти свойства принадлежат.

Простой сеттер свойства определяется как метод, который только присваивает значение параметра полю без каких-либо дополнительных вычислений. Такие простые вызовы сеттеров можно безопасно встраивать. Некоторые стандарты кодования также рекомендуют избегать использования простых сеттеров для поддержания чистоты кода.

Пример:


  class Foo {
  private int index;
    public Foo(int idx) {
       setIndex(idx);
    }
    public void setIndex(int idx) {
       index = idx;
    }
  }

После применения быстрого исправления:


  class Foo {
    private int index;
    public Foo(int idx) {
      index = idx;
    }
    public void setIndex(int idx) {
      index = idx;
    }
  }

Проверка может быть настроена следующими параметрами:

  • Сообщать только о вызовах сеттеров для this, игнорируя вызовы для объектов того же типа, переданных в качестве параметров.
  • Игнорировать не-private сеттеры.
