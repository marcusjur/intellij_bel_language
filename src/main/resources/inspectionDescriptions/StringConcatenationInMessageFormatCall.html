Обнаружена конкатенация нестатических строк, используемых в качестве аргумента для вызова MessageFormat.format().

Хотя иногда это может быть преднамеренным, обычно это является неправильным использованием метода форматования и может даже вызвать исключение, если конкатенированные строки содержат специальные символы, такие как {.

Кроме того, иногда это может быть результатом ошибочного использования + вместо , при передаче аргументов форматования.

Пример:


  String formatGreeting(String userName, int balance) {
    return MessageFormat.format("Hello, " + userName + "! Your balance is {0}.", balance);
  }

В данном случае userName интерпретируется как часть строки форматования, что может привести к IllegalArgumentException (например, если userName содержит "{"). Этот вызов следует заменить на MessageFormat.format("Hello, {0}! Your balance is {1}.", userName, balance).
