Сообщает о параметрах или локальных переменных, к которым можно добавить модификатор final в их объявлении.

Пример:


  ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

После применения быстрого исправления:


  final ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Параметры проверки позволяют указать, следует ли сообщать о параметрах или локальных переменных.
