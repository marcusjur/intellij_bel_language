Обнаружено некорректное выражение @Profile:

  • Spring-профиль не может быть пустым
  • В выражении Spring-профиля операторы '&' и '|' нельзя использовать совместно без круглых скобок

Примеры:


  // Правильная конфигурация
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}
  // Пустой профиль
  @Configuration
  @Profile() // Ошибка "Выражение профиля должно содержать текст"
  public class MyConfiguration {...}
  // Смешанные операторы без скобок
  @Configuration
  @Profile("a & b | c") // Ошибка "Некорректный формат выражения профиля"
  public class MyConfiguration {...}
