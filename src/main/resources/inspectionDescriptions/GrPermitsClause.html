Сообщает о разрешённых классах, которые не расширяют sealed базовый класс.

Groovy не требует, чтобы все разрешённые классы были доступны во время компиляции и компилировались вместе с базовым классом. Компилятор не предупреждает пользователя при обработке разрешённых подклассов без расширения, что противоречит природе sealed классов.

Пример:


  class A permits B {} // сообщает о B
  class B {}
