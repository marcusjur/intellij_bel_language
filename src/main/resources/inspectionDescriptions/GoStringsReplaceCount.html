Обнаружен вызов strings.Replace с количеством замен 0 вместо -1.

Аргумент количества замен не должен быть равен 0, так как это не выполнит никаких замен и сделает вызов функции избыточным. Используйте -1 вместо этого.

Пример:

a := strings.Replace("SSS", "S", "H", 0) // ничего не заменяется
b := strings.Replace("SSS", "S", "H", -1) // заменяет все вхождения S на H
