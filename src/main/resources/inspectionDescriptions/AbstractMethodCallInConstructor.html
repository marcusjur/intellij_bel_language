Сообщает о вызовах abstract методов текущего класса во время конструирования объекта.

Вызов происходит во время конструирования объекта, если метод находится в:

  • конструкторе
  • нестатическом инициализаторе экземпляра
  • инициализаторе нестатического поля
  • методе clone()
  • методе readObject()
  • методе readObjectNoData()

Такие вызовы могут привести к труднообнаруживаемым ошибкам, поскольку инициализация объекта может произойти после вызова метода.

Пример:


  abstract class Parent {
    abstract void abstractMethod();
  }
  class Child extends Parent {
    Child() {
      abstractMethod();
    }
  }

Эта проверка имеет общую функциональность со следующими проверками:

  • Вызов переопределяемых методов во время конструирования объекта
  • Вызов переопределённых методов во время конструирования объекта

Во избежание дублирования предупреждений можно включить только одну проверку одновременно.
