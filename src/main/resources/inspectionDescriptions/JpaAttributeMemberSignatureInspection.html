Обнаружены следующие ошибки спецификации JPA:
  • Несколько аннотаций настроено для одного метода или поля
  • Недопустимые аннотации используются в Embeddable объекте (JPA 1.0)
  • Метаданные персистентных свойств не указаны на поле или геттере свойства
  • Отсутствует сеттер или геттер для доступа к свойству

Пример:


  @Entity
  @Table(name = "TABLE")
  public class JavaEntity {
    private Integer id; // Ошибка: должны присутствовать и getter, и setter
    public Integer getId() { ... }
    private String data;
    public String getData() { ... }
    @Basic // Ошибка: аннотация персистентности должна быть на поле или методе getter
    public void setData(String data) { ... }
  }
