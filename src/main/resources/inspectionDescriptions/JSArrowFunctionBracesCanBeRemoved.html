Сообщает о стрелочных функциях, тело которых состоит только из фигурных скобок и ровно одного оператора. Рекомендуется преобразовать в краткий синтаксис без фигурных скобок. 

    let incrementer = (x) => {return x + 1};

После применения быстрого исправления код будет выглядеть следующим образом:


    let incrementer = (x) => x + 1;
