Сообщает о циклах, которые можно заменить вызовами Stream API с использованием лямбда-выражений.

Такая замена преобразует императивный стиль в функциональный, делая код более компактным.

Пример:


  boolean check(List<String> data) {
    for (String e : data) {
      String trimmed = e.trim();
      if (!trimmed.startsWith("xyz")) {
        return false;
      }
    }
    return true;
  }

После применения быстрого исправления:


  boolean check(List<String> data) {
    return data.stream().map(String::trim).allMatch(trimmed -> trimmed.startsWith("xyz"));
  }
