Сообщает о пустых основных конструкторах, когда основной конструктор неявно доступен.

Основной конструктор избыточен и может быть безопасно опущен, когда он не имеет аннотаций или модификаторов видимости. Используйте быстрое исправление "Удалить пустой основной конструктор" для очистки кода.

Пример:


  class MyClassA constructor() //  избыточно, можно заменить на 'class MyClassA'
  annotation class MyAnnotation
  class MyClassB @MyAnnotation constructor() //  обязателен из-за аннотации
  class MyClassC private constructor() // обязателен из-за модификатора видимости
