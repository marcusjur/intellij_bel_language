Сообщает об операторах try с несколькими ресурсами, которые могут быть автоматически разделены на несколько отдельных операторов try-with-resources.

Такое преобразование полезно для дальнейшего рефакторинга (например, для извлечения вложенных операторов try в отдельные методы).

Пример:


  try (FileInputStream in = new FileInputStream("in.txt");
     FileOutputStream out = new FileOutputStream("out.txt")) {
     /*чтение и запись*/
  }

После применения быстрого исправления:


  try (FileInputStream in = new FileInputStream("in.txt")) {
    try (FileOutputStream out = new FileOutputStream("out.txt")) {
      /*чтение и запись*/
    }
  }
