Сообщает об объявлениях массивов в стиле C, где квадратные скобки размещаются после имени переменной или после списка параметров метода. Большинство стилей кодования предпочитают объявления массивов в стиле Java, где квадратные скобки размещаются после имени типа.

Пример:


  public String process(String value[])[] {
    return value;
  }

После применения быстрого исправления:


  public String[] process(String[] value) {
    return value;
  }

Настройка проверки:

Используйте опцию Игнорировать объявления в стиле C в переменных, чтобы сообщать только об объявлениях массивов в стиле C для возвращаемых типов методов.
