Сообщается о методе, который вызывает другой метод. Быстрое исправление предложит ввести локальную переменную.

Пример:


  class X {
    int foo(File f) {
      return f.getName().length();
    }
  }

После применения быстрого исправления:


  class X {
    int foo(File f) {
      final String name = f.getName();
      return name.length();
    }
  }

Используйте параметры проверки для включения/выключения предупреждений для:

  • Цепочки вызовов методов в инициализаторах полей, например private final int i = new Random().nextInt();

  • Цепочки вызовов методов, работающих с одним и тем же типом, например new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();
