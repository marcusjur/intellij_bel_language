Сообщает о пакетах с именами, которые слишком короткие, слишком длинные или не соответствуют указанному шаблону регулярного выражения.

Пример:


  package io;

С помощью опций можно указать минимальную и максимальную длину имени пакета, а также регулярное выражение для проверки допустимых имен пакетов (регулярное выражение использует стандартный формат java.util.regex).
