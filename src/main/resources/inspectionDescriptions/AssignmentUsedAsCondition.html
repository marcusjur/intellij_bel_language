Сообщение об использовании оператора присваивания в качестве условия в операторах if, while, for или do, либо в условном выражении.

Хотя иногда это делается намеренно, такое использование может вводить в заблуждение и указывать на опечатку (например, когда вместо = следует использовать ==).

Быстрое исправление заменит = на ==.

Пример:


  void update(String str, boolean empty) {
    // Предупреждение: 'empty' присваивается значение,
    // а не сравнивается с str.isEmpty()
    if (empty = str.isEmpty()) {
      ...
    }
  }

После применения быстрого исправления:


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
