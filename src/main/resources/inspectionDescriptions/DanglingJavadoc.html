Обнаружены Javadoc-комментарии, не принадлежащие никакому классу, методу или полю. Инструмент Javadoc игнорирует такие "висячие" комментарии и не включает их в сгенерированную HTML-документацию.

Пример:


  class A {
    /**
     * Dangling comment
     */
    /**
     * Method javadoc
     */
     public void m(){}
  }

Такие комментарии можно полностью удалить или преобразовать в блочные комментарии с помощью быстрого исправления. После применения быстрого исправления:


  class A {
    /*
      Dangling comment
    */
    /**
     * Method javadoc
     */
    public void m(){}
  }

Используйте опцию Игнорировать комментарии заголовка файла в формате JavaDoc, чтобы исключить комментарии в начале Java-файла. Обычно это сообщения об авторских правах.
