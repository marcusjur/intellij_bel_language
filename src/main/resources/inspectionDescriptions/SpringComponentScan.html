Сообщает о нераспознанных пакетах в аннотации @ComponentScan и соответствующих псевдонимах мета-аннотации @ComponentScan.

Пример:


  @ComponentScan(basePackages = {
     "com.my.company",
     "com.unknown" // Сообщение "Не удаётся разрешить пакет 'unknown'"
  }) +
  @Configuration +
  public class MyConfiguration {}
