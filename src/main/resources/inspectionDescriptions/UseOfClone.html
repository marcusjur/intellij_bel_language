Сообщает о реализации метода clone(), его вызовах и использовании интерфейса java.lang.Cloneable.

Некоторые стандарты кодования запрещают использование clone() и рекомендуют вместо этого использовать конструктор копирования или static фабричный метод.

Эта проверка игнорирует вызовы clone() для массивов, поскольку это правильный и компактный способ копирования массивов.

Пример:


  class Copy implements Cloneable /*предупреждение*/ {
    public Copy clone() /*предупреждение*/ {
      try {
        return (Copy) super.clone(); // предупреждение
      } catch (CloneNotSupportedException e) {
        throw new AssertionError();
      }
    }
  }
