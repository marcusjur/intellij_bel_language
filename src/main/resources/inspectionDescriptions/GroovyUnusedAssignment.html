Сообщает о случаях, когда переменной присваивается значение, которое затем никогда не используется.

Если переменная не используется, рекомендуется удалить её для сокращения кода и избежания избыточных присваиваний.

Будут отмечены следующие случаи:

  • Значение переменной после присваивания никогда не считывается
  • Значение всегда перезаписывается другим присваиванием до того, как будет прочитано
  • Инициализатор переменной является избыточным (по одной из двух вышеуказанных причин)

Для получения дополнительной информации см. аналогичную проверку в Java.
