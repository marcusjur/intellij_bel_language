Сообщает о возможной утечке ссылки this во время инициализации объекта. Утечка происходит, когда this используется как аргумент метода или присваивается в конструкторе или инициализаторе. Такие утечки могут привести к труднообнаруживаемым ошибкам, так как объект становится доступным в контексте, где его инициализация не гарантирована.

Пример:


  class Foo {
    {
      System.out.println(this);
    }
  }
