Сообщать о вложенных классах с модификатором protected.

Пример:


  public class Outer {
    protected static class Nested {} // Предупреждение
    protected class Inner {}         // Предупреждение
    protected enum Mode {}           // Предупреждение зависит от настроек
    protected interface I {}         // Предупреждение зависит от настроек
  }

Настройка проверки:

  • Используйте опцию Игнорировать внутренние enum с модификатором 'protected', чтобы игнорировать protected внутренние перечисления.
  • Используйте опцию Игнорировать внутренние интерфейсы с модификатором 'protected', чтобы игнорировать protected внутренние интерфейсы.
