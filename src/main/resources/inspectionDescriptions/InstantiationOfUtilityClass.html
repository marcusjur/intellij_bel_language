Сообщает об инстанцировании служебных классов с использованием ключевого слова new.

В служебных классах все поля и методы являются static. Создание экземпляров таких классов, вероятно, не требуется и указывает на ошибку.

Пример:


  class MyUtils {
    public static double cube(double x) {
      return x * x * x;
    }
  }
  class Main {
    public static void main(String[] args) {
      // Инстанцирование служебного класса
      MyUtils utils = new MyUtils();
    }
  }

Чтобы предотвратить инстанцирование служебных классов, рекомендуется использовать private конструктор.
