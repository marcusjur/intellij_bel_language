Сообщает о сравнениях 0 с вызовами .size() или .length(), которые могут быть заменены вызовом .isEmpty().

Пример:


  boolean emptyList = list.size() == 0;

После применения быстрого исправления:


  boolean emptyList = list.isEmpty();

Используя список игнорируемых классов, можно добавить классы, для которых не следует заменять сравнения .size() или .length().

Опция игнорировать выражения, которые будут заменены на '!isEmpty()' позволяет исключить любые выражения, которые были бы заменены на !isEmpty().
