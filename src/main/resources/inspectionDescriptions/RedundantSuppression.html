Сообщает об элементах, которые можно безопасно удалить, поскольку проверки, на которые они влияют, больше не применимы в данном контексте:
  • аннотации @SuppressWarning, или
  • однострочные комментарии // noinspection, или
  • комментарии JavaDoc /** noinspection */

Пример:


public class C {
 // Символ уже является private,
 // но аннотация всё ещё присутствует
  @SuppressWarnings({"WeakerAccess"})
 private boolean CONST = true;
 void f() {
    CONST = false;
  }
}
