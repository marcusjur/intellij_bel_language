Сообщает о неразрешённых каналах в аннотациях @StreamListener, @SendTo, @Output и @Input.

Использование недопустимого имени привязки приведет к ошибке 'Bean not found' во время выполнения.

Пример:


  @EnableBinding(Sink.class)
  public class LogSource {
    @StreamListener(Sink.class)
	  public void log1() {
    }
    @StreamListener("invalid")  // Сообщение 'канал не найден'
	  public void log2() {
    }
  }
