Сообщает о вызовах простых геттеров свойств внутри класса, которому принадлежат эти свойства.

Простые геттеры определяются как методы, которые только возвращают значение поля без дополнительных вычислений. Такие простые вызовы геттеров можно безопасно встраивать с помощью быстрого исправления. Некоторые стандарты кодования также рекомендуют избегать использования простых геттеров для поддержания чистоты кода.

Пример:


  public class Salient {
    private String name;
    public String getName() {
      return name;
    }
    @Override
    public String toString() {
      return getName();
    }
  }

После применения быстрого исправления:


  public class Salient {
    private String name;
    public String getName() {
      return name;
    }
    @Override
    public String toString() {
      return name;
    }
  }

Проверка может быть настроена следующими параметрами:

  • Сообщать только о вызовах геттеров для this, игнорируя вызовы для объектов того же типа, переданных как параметры.
  • Игнорировать не-private геттеры.
