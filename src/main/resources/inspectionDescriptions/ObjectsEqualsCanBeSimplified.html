Сообщает о вызовах Objects.equals(a, b), где первый аргумент статически известен как ненулевой.

Если оба аргумента являются примитивами, такой вызов можно безопасно заменить на a.equals(b) или a == b.

Пример:


  String defaultName = "default";
  boolean isDefault = Objects.equals(defaultName, name);

После применения быстрого исправления:


  String defaultName = "default";
  boolean isDefault = defaultName.equals(name);

Новое в версии 2018.3
