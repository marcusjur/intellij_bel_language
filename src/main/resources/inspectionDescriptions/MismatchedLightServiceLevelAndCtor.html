Сообщает о несоответствии между уровнем облегченного сервиса и его конструктором.

Будут отмечены следующие проблемы:

  • Класс облегченного сервиса с конструктором, имеющим два параметра типа com.intellij.openapi.project.Project и kotlinx.coroutines.CoroutineScope или один параметр типа Project, не аннотирован как сервис проектного уровня.
  • Класс облегченного сервиса, аннотированный как сервис уровня приложения, не имеет конструктора без параметров или конструктора с одним параметром типа CoroutineScope.

Пример (Kotlin):


@Service                              // Рекомендуется указать параметр 'Service.Level.PROJECT' в аннотации '@Service'
class MyService(val project: Project) // Рекомендуется удалить параметр из конструктора 'MyService'

После применения быстрого исправления с указанием параметра 'Service.Level.PROJECT' в аннотации '@Service':


@Service(Service.Level.PROJECT)
class MyService(val project: Project)

После применения быстрого исправления с удалением параметра из конструктора 'MyService':


@Service
class MyService()
