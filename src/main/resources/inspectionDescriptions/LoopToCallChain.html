Сообщает о циклах for, которые могут быть заменены последовательностью операций стандартной библиотеки (таких как map, filter и др.).

Пример:


fun foo(list: List<String>): List<Int> {
  val result = ArrayList<Int>()
  for (s in list) {
     if (s.length > 0)
       result.add(s.hashCode())
     }
  return result
}

После применения быстрого исправления:


fun foo(list: List<String>): List<Int> {
  val result = list
    .filter { it.length > 0 }
    .map { it.hashCode() }
  return result
}
