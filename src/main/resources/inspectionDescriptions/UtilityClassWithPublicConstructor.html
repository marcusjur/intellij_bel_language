Сообщает о служебных классах, содержащих public конструктор.

Служебные классы объявляют все поля и методы как static. Создание public конструктора в таком классе вводит в заблуждение и может привести к непреднамеренному созданию экземпляров класса.

Пример:


  public final class UtilityClass {
    public UtilityClass(){
    }
    public static void foo() {}
  }

После применения быстрого исправления:


  public final class UtilityClass {
    private UtilityClass(){
    }
    public static void foo() {}
  }
