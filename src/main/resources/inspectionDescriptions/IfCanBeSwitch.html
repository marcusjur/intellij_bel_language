Сообщает об операторах if, которые могут быть заменены на оператор switch.

Результат замены обычно короче и понятнее.

Пример:


  void test(String str) {
    if (str.equals("1")) {
      System.out.println(1);
    } else if (str.equals("2")) {
      System.out.println(2);
    } else if (str.equals("3")) {
      System.out.println(3);
    } else {
      System.out.println(4);
    }
  }

После применения быстрого исправления:


  void test(String str) {
    switch (str) {
      case "1" -> System.out.println(1);
      case "2" -> System.out.println(2);
      case "3" -> System.out.println(3);
      default -> System.out.println(4);
    }
  }

Эта проверка выполняется только если уровень языка проекта или модуля 7 или выше.

Используйте поле Минимальное количество веток условия 'if' для указания минимального количества условных веток if, при котором будет выполняться проверка оператора if. Обратите внимание, что завершающая ветка else (без if) не учитывается.

Используйте опцию Включить предложения switch для чисел, чтобы включить предложения об использовании оператора switch для примитивных и упакованных числовых типов и символов.

Используйте опцию Включить предложения switch для перечислений, чтобы включить предложения об использовании оператора switch для констант enum.

Используйте опцию Включить предложения только для null-safe выражений, чтобы предлагать только те операторы switch, которые не могут вызвать NullPointerException.
