Сообщение об операторе if, который может быть заменен на оператор case.

Используйте поле минимальное количество веток для указания минимального количества веток when в получаемом операторе case.

Пример:

fruit = gets
if fruit == 'apple' # Simplifiable 'if' statement
  puts 'This is apple!'
elsif fruit == 'peach'
  puts 'This is peach!'
elsif fruit == 'orange'
  puts 'This is orange!'
elsif fruit == 'banana'
  puts 'This is banana!'
end
