Сообщать об использовании замыкающих запятых в литералах массивов.

Это предупреждение выводится только если версия JavaScript установлена на ECMAScript 5.1.

Хотя спецификация разрешает использование замыкающих запятых в массивах, некоторые браузеры могут выдавать ошибки при их использовании.

Вы можете настроить параметры форматования замыкающих запятых в разделе Стиль кода | JavaScript или TypeScript | Пунктуация.
