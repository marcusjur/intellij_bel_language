Сообщает об использовании переменных final static во время инициализации класса.

В таких случаях семантика кода может зависеть от порядка создания классов. Кроме того, это может привести к использованию переменных до их инициализации и обычно вызывает сложные и запутанные ошибки.

Пример:


  class Foo {
    public static int bar = 0;
    static {
      System.out.println(bar);
    }
  }
