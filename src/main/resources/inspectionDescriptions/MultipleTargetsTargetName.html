여러 대상에 @targetName 어노테이션이 사용되면 보고합니다.

빠른 수정에서는 @targetName 어노테이션을 제거합니다.

예:


  // 1
  val list: List[Int] = (1 to 5).toList
  @targetName("head_and_tail")
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    @targetName("b_and_c")
    case B, C
    case D

빠른 수정 적용 후:


  // 1
  val list: List[Int] = (1 to 5).toList
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    case B, C
    case D
