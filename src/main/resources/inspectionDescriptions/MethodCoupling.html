Метод ссылается на слишком большое количество методов других классов. Методы с высокой степенью связанности могут быть очень хрупкими и должны быть разделены на более мелкие методы.

Каждый используемый класс учитывается только один раз, независимо от количества обращений к нему.

Настройка проверки:

  • Используйте поле Ограничение связанности метода, чтобы указать максимально допустимую степень связанности для метода.
  • Используйте опцию Включать связи с системными классами Java, чтобы учитывать ссылки на классы из пакетов java или javax.
  • Используйте опцию Включать связи с библиотечными классами, чтобы учитывать ссылки на классы сторонних библиотек.
