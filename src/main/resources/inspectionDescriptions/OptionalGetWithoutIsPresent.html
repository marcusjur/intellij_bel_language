Сообщает о вызове метода get() у объекта Optional без предварительной проверки наличия значения.

Вызов Optional.get() на пустом экземпляре Optional приведет к выбросу исключения.

Пример:


  void x(List<Integer> list) {
    final Optional<Integer> optional =
      list.stream().filter(x -> x > 10).findFirst();
    final Integer result = optional.get(); // проблема здесь
  }
