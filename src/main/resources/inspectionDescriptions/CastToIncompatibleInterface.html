Сообщает о выражениях приведения типов, где целевой тип приведения является классом/интерфейсом, который не расширяет/не реализует исходный тип, и не имеет подклассов, которые это делают.

Такая структура, вероятно, ошибочна и приведет к исключению java.lang.ClassCastException во время выполнения.

Пример:


  interface A {}
  interface Z {}
  static class C {}
  void x(C c) {
    if (c instanceof Z) {
      A a = ((A)c); // Приведение к несовместимому интерфейсу 'A'
    }
  }
