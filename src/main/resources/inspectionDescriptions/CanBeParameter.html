Сообщает о параметрах основного конструктора с модификаторами val или var, которые могут быть удалены.

Свойства класса, объявленные в конструкторе, увеличивают потребление памяти. Если значение параметра используется только в конструкторе, его можно опустить.

Обратите внимание, что ссылающиеся объекты могли быть ранее удалены сборщиком мусора.

Пример:


  class Task(val name: String) {
      init {
          print("Task created: $name")
      }
  }

Быстрое исправление удалит лишние ключевые слова val или var:


  class Task(name: String) {
      init {
          print("Task created: $name")
      }
  }
