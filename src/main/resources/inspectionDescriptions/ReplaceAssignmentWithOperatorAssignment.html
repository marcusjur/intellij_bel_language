Сообщается о возможности замены присваивания на оператор присваивания с операцией.

Код с использованием оператора присваивания короче и потенциально более понятен.

Пример:

  x = x + 3;
  x = x / 3;

После применения быстрого исправления:

  x += 3;
  x /= 3;

Опция Игнорировать условные операторы позволяет исключить операторы && и ||. Замена условных операторов на операторы присваивания может изменить семантику выражения, превратив ленивые вычисления в энергичные.

Опция Игнорировать неоднозначные операторы позволяет исключить менее распространённые операторы ^ и %.
