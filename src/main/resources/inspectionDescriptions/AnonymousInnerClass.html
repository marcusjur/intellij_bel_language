Обнаружение анонимных классов.

Замена анонимных классов на внутренние классы может улучшить читаемость и сопровождаемость кода. Некоторые стандарты кодования не рекомендуют использовать анонимные классы.

Пример:


  class Example {
    public static void main(String[] args) {
      new Thread() {
        public void run() {
          work()
        }
        private void work() {}
      }.start();
    }
  }

После применения быстрого исправления:


  class Example {
    public static void main(String[] args) {
      new MyThread().start();
    }
    private static class MyThread extends Thread {
      public void run() {
        work();
      }
      private void work() {}
    }
  }
