Сообщает об использовании констант абсолютного выравнивания в AWT и Swing. Интернационализованные приложения должны использовать относительное выравнивание, так как оно учитывает настройки направления компонентов для локали.

Пример:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hello World");
  panel.add(label, BorderLayout.NORTH);

После применения быстрого исправления:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hello World");
  panel.add(label, BorderLayout.PAGE_START);
