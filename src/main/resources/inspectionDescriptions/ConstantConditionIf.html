Сообщает об условных выражениях if с константными литералами true или false, которые можно упростить.

Хотя иногда это делается намеренно, такая структура может вводить в заблуждение и обычно является результатом опечаток или предыдущего рефакторинга.

Пример:


  fun example() {
      if (true) {
          throw IllegalStateException("Unexpected state")
      }
  }

Быстрое исправление удалит условие if:


  fun example() {
      throw IllegalStateException("Unexpected state")
  }
