Сообщает о избыточном создании объекта File в случаях, когда можно использовать только строковый путь (String) в следующих конструкторах: FileInputStream, FileOutputStream, FileReader, FileWriter, PrintStream, PrintWriter, Formatter.

Пример:


InputStream is = new FileInputStream(new File("in.txt"));

После применения быстрого исправления:


InputStream is = new FileInputStream("in.txt");

Что нового в версии 2020.3
