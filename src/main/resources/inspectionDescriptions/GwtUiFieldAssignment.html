Сообщает о присваивании значений полям с аннотацией @UiField.

Такие поля по умолчанию инициализуются с помощью @UiField и не должны изменяться пользовательским кодом. Если необходимо инициализовать такие поля в коде, следует установить атрибут provided в аннотации @UiField в значение true.

Быстрое исправление устанавливает атрибут provided в аннотации @UiField в значение true.
