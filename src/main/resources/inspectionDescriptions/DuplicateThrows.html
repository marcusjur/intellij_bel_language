Обнаружение повторяющихся исключений в списке throws метода.

Пример:


  void f() throws Exception, Exception {}

После применения быстрого исправления:


  void f() throws Exception {}

Используйте опцию игнорировать исключения, являющиеся подклассами других исключений, чтобы пропускать исключения, которые наследуются от других исключений в списке.
