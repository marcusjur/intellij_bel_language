Сообщает об итерации по keySet() экземпляра java.util.Map, где итерируемые ключи используются для получения значений из отображения.

Такая итерация может быть более эффективной при замене на итерацию по entrySet() или values() (если ключи фактически не используются).

Аналогично, если значения извлекаются в лямбда-выражении, keySet().forEach(key -> ...) можно заменить на forEach((key, value) -> ...).

Пример:


  for (Object key : map.keySet()) {
   Object val = map.get(key);
  }

После применения быстрого исправления:


  for (Object val : map.values()) {}
