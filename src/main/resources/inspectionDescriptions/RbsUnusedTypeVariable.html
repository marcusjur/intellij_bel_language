Сообщает о неиспользуемых контейнерах, методах и псевдонимах типов в проекте.

Проверка переменных типа контейнера выполняется только в том случае, если контейнер имеет ровно одно объявление.

Неиспользуемые переменные типа можно удалить с помощью быстрого исправления.

Пример:

# Переменная типа 'A' класса 'Example' не используется
class Example[A]
end

После применения быстрого исправления:

class Example
end
