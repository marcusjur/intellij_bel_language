Сообщает об использовании аргументов без имен при вызове процедур. По умолчанию эта проверка отключена.

Подробную информацию о различиях между именованными и неименованными параметрами см. в разделе Binding Parameters by Name (Named Parameters) на docs.microsoft.com.

Пример (Microsoft SQL Server):

CREATE FUNCTION foo(n INT, m INT) RETURNS INT AS
BEGIN
    RETURN n + m;
END;
CREATE PROCEDURE test AS
BEGIN
    foo n = 1, m = 2;
--- Следующий вызов без имен параметров будет подсвечен
    foo 1, 2;
END;

Параметры 1, 2 в вызове foo 1, 2; подсвечиваются, так как у них отсутствуют имена.
