Сообщает о классах, которые ссылаются на свои подклассы в статических инициализаторах или статических полях.

Такие ссылки могут вызвать взаимоблокировку (deadlock) на уровне JVM в многопоточной среде, когда один поток пытается загрузить суперкласс, а другой поток одновременно пытается загрузить подкласс.

Пример:


  class Parent {
    static final Child field = new Child();
  }
  class Child extends Parent { }
