Классы, названия которых начинаются с имени их пакета без учета регистра символов.

Хотя такие названия иногда могут быть оправданы, они обычно являются результатом неудачной схемы именования и могут быть избыточными и нежелательными.

Пример: 


  package byteCode;
  class ByteCodeAnalyzer {}

Переименовать такие классы можно только с помощью быстрого исправления в редакторе.
