Сообщает об отсутствующих и повторяющихся ID в локальных элементах TMS, объявленных в Markdown.

Элементы TMS с отсутствующими или недействительными ID не могут быть связаны с кодом (модульными тестами).

Быстрое исправление создаст уникальный ID.

Выдает предупреждение при отсутствии ID тестового случая и ошибку при неуникальном ID. Выдает ошибку при неуникальных номерах ID тестового набора/прогона.
