Сообщает об использовании System.out или System.err.

Такие операторы обычно используются для временной отладки и должны быть удалены из производственного кода или заменены более надежным инструментом логования.

Предлагаемое быстрое исправление поддерживает SLF4J и Log4j 2. Оно заменяет вызовы System.out и System.err на вызовы логгера

Пример:


  public static void test(Object o) {
    System.out.println("Test: "+ o);
  }

После применения быстрого исправления:


  private static final Logger log = LoggerFactory.getLogger(Test.class);
  public static void test(Object o) {
      log.info("Test: {}", o);;
  }

Опция метод логования для 'System.err' позволяет указать метод для записи сообщений из вызовов 'System.err'. Опция метод логования для 'System.out' позволяет указать метод для записи сообщений из вызовов 'System.out'.
