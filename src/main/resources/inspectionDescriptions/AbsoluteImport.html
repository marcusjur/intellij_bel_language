기본 패키지(설정 | 언어 및 프레임워크 | Scala | 기본 패키지)에 상대적일 수 있는 정규화된 import 문을 보고합니다.

설정 | 에디터 | 코드 스타일 | Scala | Import 문에서 기본 패키지 제외 옵션을 활성화한 경우에만 적용됩니다.

예:


  package org.example.application
  package ui

  import org.example.application.model.Data

빠른 수정 적용 후:


  package org.example.application
  package ui

  import model.Data
