Отчёт о неиспользуемых зависимостях.

Для каждой зависимости в текущем файле go.mod эта проверка ищет операторы import, начинающиеся с имени зависимости, во всех Go-файлах. Если такой import не найден, зависимость считается неиспользуемой. Эта проверка эквивалентна команде go mod why.

Проверка обрабатывает директивы require и replace. Директивы require с комментарием // indirect также обрабатываются.

Проверка может неточно определить, используется ли зависимость с заменой или косвенным требованием в следующих случаях:

  • Включен режим vendoring
  • Некоторые зависимости не загружены
