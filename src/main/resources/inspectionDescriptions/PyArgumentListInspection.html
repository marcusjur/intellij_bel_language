Сообщает о различиях между объявленными формальными параметрами и фактическими аргументами, а также о некорректных аргументах (например, повторяющихся именованных аргументах) и неправильном порядке аргументов.

Пример:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1
bar = Foo()
bar.__call__() # отсутствуют параметры
bar(5, "#") # некорректные аргументы

Правильный фрагмент кода показан ниже:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1
bar = Foo()
bar.__call__(5)
bar(5, p2="#")
