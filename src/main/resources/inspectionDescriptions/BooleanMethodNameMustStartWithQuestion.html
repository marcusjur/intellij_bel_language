Сообщает о булевых методах, имена которых не начинаются с вопросительного слова.

Эта проверка игнорирует булевы методы, которые переопределяют методы библиотеки.

Пример:

boolean empty(List<String> list) {
  return list.isEmpty();
}

Быстрое исправление для переименования таких методов доступно только в редакторе.

Настройка проверки:

  • Используйте список префиксов имен булевых методов, чтобы указать допустимые вопросительные слова, с которых могут начинаться имена булевых методов.
  • Используйте опцию игнорировать методы с возвращаемым типом 'java.lang.Boolean', чтобы игнорировать методы, возвращающие java.lang.Boolean.
  • Используйте опцию игнорировать булевы методы в @interface, чтобы игнорировать булевы методы в аннотациях (@interface).
  • Используйте опцию игнорировать методы, переопределяющие/реализующие методы суперкласса, чтобы игнорировать методы, имеющие методы суперкласса.
