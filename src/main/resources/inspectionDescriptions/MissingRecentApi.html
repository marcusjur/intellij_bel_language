Обнаружено использование API платформы IntelliJ, который был введен в версии ниже той, что указана в атрибуте @since-build тега файла plugin.xml.

Использование такого API может привести к несовместимости плагина со старыми версиями ИСР.

Чтобы избежать возможных проблем при запуске плагина в старых версиях ИСР, повысьте значение since-build или удалите использование этого API.

Дополнительную информацию смотрите в разделе Диапазоны номеров сборок документации SDK платформы IntelliJ.

Настройка проверки:

Если атрибуты @since/until-build тега <idea-version> не указаны в plugin.xml, явно установите значения Since/Until.
