Сообщает о избыточных круглых скобках в вызовах функций, где единственным параметром является лямбда-выражение вне скобок.

Используйте быструю фиксацию "Удалить ненужные скобки из вызова функции с лямбда-выражением" для очистки кода.

Пример:


  fun foo() {
      listOf(1).forEach() {  }
  }

После применения быстрой фиксации:


  fun foo() {
      listOf(1).forEach {  }
  }
