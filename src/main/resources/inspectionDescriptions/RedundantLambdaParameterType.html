Сообщает о избыточных типах параметров лямбда-выражения, которые можно вывести из контекста.

Пример:


  Map<String, Integer> map = ...
  map.forEach((String s, Integer i) -> log.info(s + "=" + i));

Быстрое исправление удалит эти типы параметров из лямбда-выражения.


  Map<String, Integer> map = ...
  map.forEach((s, i) -> log.info(s + "=" + i));
