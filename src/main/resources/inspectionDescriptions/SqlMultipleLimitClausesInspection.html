Сообщает об использовании нескольких ограничивающих предложений в одном запросе.

Пример (Microsoft SQL Server):

create table foo(a int);
select top 1 * from foo order by a offset 10 rows fetch next 20 rows only;

Предложение SELECT TOP используется для указания того, что должна быть возвращена только 1 запись. Предложение FETCH определяет количество строк, которые нужно вернуть после обработки предложения OFFSET. Однако, поскольку у нас уже есть ограничивающее предложение SELECT TOP, предложение FETCH может быть избыточным.
