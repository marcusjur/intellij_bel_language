Сообщает о private свойствах с методами доступа getter и/или setter, которые могут быть заменены на public свойства с использованием свойств-перехватчиков get и/или set.

Свойства-перехватчики get и set поддерживаются в PHP 8.4 и выше. Дополнительную информацию смотрите в PHP RFC: Свойства-перехватчики.
