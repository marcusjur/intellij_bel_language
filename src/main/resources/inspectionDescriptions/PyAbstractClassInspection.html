Сообщает о случаях, когда не все абстрактные свойства или методы определены в дочернем классе.

Пример:


from abc import abstractmethod, ABC
class Figure(ABC):
    @abstractmethod
    def do_figure(self):
        pass
class Triangle(Figure):
    def do_triangle(self):
        pass

После применения быстрого исправления ИСР реализует абстрактный метод для класса Triangle:


from abc import abstractmethod, ABC
class Figure(ABC):
    @abstractmethod
    def do_figure(self):
        pass
class Triangle(Figure):
    def do_figure(self):
        pass
    def do_triangle(self):
        pass
