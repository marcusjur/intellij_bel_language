Сообщает о блоках synchronized, использующих блокировку на объектах, инициализованных литералами.

Строковые литералы интернируются, а литералы Character, Boolean и Number могут выделяться из кэша. Следовательно, другие части системы (использующие объекты, инициализованные теми же литералами) фактически могут содержать ссылки на абсолютно те же объекты. Это может привести к неожиданным ситуациям взаимной блокировки, если предполагается, что объект блокировки является private.

Пример:


  class Main {
    final String mutex = "Mutex";
    void method() {
      synchronized (mutex) {
      }
    }
  }

Опция предупреждать обо всех возможных литералах позволяет получать уведомления о любой синхронизации объектов String, Character, Boolean и Number.
