Сообщает о вызовах метода java.util.Set.removeAll() с аргументом типа java.util.List.

Такие вызовы могут быть медленными, когда размер аргумента больше или равен размеру множества, и множество является подклассом java.util.AbstractSet. В этом случае для каждого элемента множества вызывается List.contains(), что приводит к линейному поиску.

Пример:


  public void check(String... ss) {
    // Возможная сложность O(n^2)
    mySet.removeAll(List.of(ss));
  }

После применения быстрого исправления:


  public void check(String... ss) {
    // Сложность O(n)
    List.of(ss).forEach(mySet::remove);
  }
