Сообщение об использовании свойства innerHTML для доступа к DOM-узлам через JavaScript в текстовом виде. При использовании явных вызовов DOM (таких как getElementByID() и createElement()), большинство применений innerHTML работают более эффективно. Кроме того, innerHTML не работает с XML DOM, включая DOM для XHTML (при просмотре в формате XML). Это может привести к трудно диагностируемым ошибкам.
