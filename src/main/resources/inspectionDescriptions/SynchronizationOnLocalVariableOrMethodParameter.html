Сообщает о синхронизации локальных переменных или параметров.

При использовании такой синхронизации сложно гарантировать корректность операций. Такой код можно улучшить, например, используя классы-обертки synchronized для контроля доступа или выполняя синхронизацию на уровне полей.

Пример:


  void bar() {
    final Object lock = new Object();
    synchronized (lock) { }
  }
