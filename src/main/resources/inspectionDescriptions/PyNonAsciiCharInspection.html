Сообщение о файлах Python 2, содержащих символы не ASCII без объявления кодования в начале файла.

Пример:


class A(object):
# №5
    def __init__(self):
        pass

В этом примере ИСР сообщает о наличии не ASCII символов в комментарии и отсутствии объявления кодования. Примените предлагаемое быстрое исправление для добавления отсутствующего объявления кодования:


# coding=utf-8
class A(object)
# №5
    def __init__(self):
        pass
