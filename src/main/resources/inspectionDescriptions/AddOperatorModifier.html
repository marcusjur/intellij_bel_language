Сообщает о функции, которая соответствует одному из соглашений об операторах, но не имеет ключевого слова operator.

Добавив модификатор operator, вы позволите пользователям функции писать идиоматический код на Kotlin.

Пример:


  class Complex(val real: Double, val imaginary: Double) {
      fun plus(other: Complex) =
          Complex(real + other.real, imaginary + other.imaginary)
  }
  fun usage(a: Complex, b: Complex) {
      a.plus(b)
  }

Быстрое исправление добавит ключевое слово модификатора operator:


  class Complex(val real: Double, val imaginary: Double) {
      operator fun plus(other: Complex) =
          Complex(real + other.real, imaginary + other.imaginary)
  }
  fun usage(a: Complex, b: Complex) {
      a + b
  }
