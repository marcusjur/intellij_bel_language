Сообщение об использовании Hooks.onOperatorDebug() при вызове ReactorDebugAgent.init().

Если ReactorDebugAgent находится в classpath и параметр spring.reactor.debug-agent.enabled установлен в значение true (значение по умолчанию), ReactorDebugAgent.init() вызывается автоматически при запуске. Одновременное использование Hooks.onOperatorDebug() и ReactorDebugAgent.init() приводит к генерации Reactor двойных отладочных stack-фреймов. Кроме того, Hooks.onOperatorDebug() создает дополнительную нагрузку на производительность.

Это быстрое исправление удаляет вызов Hooks.onOperatorDebug().

Пример:


  public void hook() {
    Hooks.onOperatorDebug(); // Сообщение 'Вызов Hooks.onOperatorDebug() при инициализации ReactorDebugAgent'
  }

После применения быстрого исправления:


  public void hook() {
  }

Эта проверка срабатывает только если версия Spring Boot в проекте или модуле 2.2.0 или выше.
