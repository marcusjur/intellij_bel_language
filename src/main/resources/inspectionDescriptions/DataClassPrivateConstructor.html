Сообщение о private основном конструкторе в классе данных.

Класс data имеет фабричный метод copy(), который можно использовать как конструктор. Для обеспечения должной функциональности конструктор не должен быть помечен как private.

Пример:


  data class User private constructor(val name: String)

Быстрое исправление изменит модификатор видимости конструктора на public:


  data class User(val name: String)
