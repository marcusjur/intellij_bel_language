Сообщает об использовании устаревших типов, которые могут исчезнуть в будущих версиях СУБД.

Отслеживаемые типы:

  • LONG в Oracle (см. страницу Deprecated and Desupported Features на docs.oracle.com).
  • TEXT, NTEXT и IMAGE в Microsoft SQL Server (см. страницу Deprecated Database Engine Features in SQL Server 2016 на docs.microsoft.com).

Пример (Oracle):

CREATE TABLE ot.foo(
a NUMBER GENERATED BY DEFAULT AS ИСРNTITY,
b LONG NOT NULL
);
