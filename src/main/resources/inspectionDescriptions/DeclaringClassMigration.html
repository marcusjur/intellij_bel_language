Сообщение о вызовах свойства 'declaringClass' для Enum, которые приводят к ошибкам компиляции начиная с версии 1.9.

'Enum.getDeclaringClass' - это "скрытая" функция Java, обычно недоступная через разбор. Однако она доступна через синтетическое свойство, что является ошибкой фронтенда.

Подробнее: KT-49653 Объявить устаревшим и удалить синтетическое свойство Enum.declaringClass

Быстрое исправление заменит вызов на 'declaringJavaClass'.

Пример:


  fun <E: Enum<E>> foo(values: Array<E>) {
    EnumSet.noneOf(values.first().declaringClass)
  }

После применения быстрого исправления:


  fun <E: Enum<E>> foo(values: Array<E>) {
    EnumSet.noneOf(values.first().declaringJavaClass)
  }

Эта проверка выполняется только если уровень языка Kotlin для проекта или модуля установлен на 1.7 или выше.
