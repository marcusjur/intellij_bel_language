Обнаружение некорректных объявлений команд Spring Shell:

  • Отсутствует описание команды
  • Команда находится вне org.springframework.shell.standard.@ShellComponent

Пример:


// Все методы будут отмечены из-за отсутствия @ShellComponent
class MathCommands {
  @ShellMethod("")
  public int sum(int a, int b) { // Метод будет отмечен из-за пустого описания
    return a + b;
  }
  @ShellMethod
  public int mul(int a, int b) { // Метод будет отмечен из-за отсутствия описания
    return a * b;
  }
}
