Обнаружен вызов метода notify() или notifyAll() для объекта java.util.concurrent.locks.Condition.

Это может быть ошибкой программования, где следовало использовать метод signal() или signalAll(), иначе может возникнуть исключение IllegalMonitorStateException.

Пример:


  class C {
    final Lock l = new ReentrantLock();
    final Condition c = l.newCondition();
    void release() {
      l.lock();
      try {
        c.notifyAll(); // здесь следовало использовать 'signalAll()'
      } finally {
        l.unlock();
      }
    }
  }
