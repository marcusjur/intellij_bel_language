Обнаружено выражение, которое, вероятно, некорректно определяет равенство массивов или вычисляет их хэш-код.

Сообщается о следующих вызовах методов:

  • Использование Object.equals() для любых массивов
  • Использование Arrays.equals() для многомерных массивов
  • Использование Arrays.hashCode() для многомерных массивов
