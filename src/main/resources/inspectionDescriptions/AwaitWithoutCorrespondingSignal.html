Сообщает о вызовах Condition.await(), для которых не найдены соответствующие вызовы Condition.signal() или Condition.signalAll().

Вызов Condition.await() в потоке без соответствующего Condition.signal() может привести к блокировке потока до тех пор, пока он не будет прерван или не произойдет "спонтанное пробуждение".

Эта проверка отслеживает только вызовы для полей текущего класса.

Пример:


  class Queue {
    private final Condition isEmpty = ...;
    void add(Object elem) {
      // ...
      // isEmpty.signal();
      // ...
    }
    void remove(Object elem) throws InterruptedException {
      // ...
      isEmpty.await(); // Вызов 'await()' без соответствующего вызова 'signal()'/'signalAll()'
      // ...
    }
  }
