Сообщает о присваивании или модификации параметра цикла внутри тела цикла for.

Хотя иногда это делается намеренно, такая структура может вызывать путаницу и обычно является результатом опечатки или использования неправильной переменной.

Быстрое исправление добавит объявление новой переменной.

Пример:


  for (String s : list) {
    // Предупреждение: s изменяется внутри цикла
    s = s.trim();
    System.out.println("String: " + s);
  }

После применения быстрого исправления:


  for (String s : list) {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  }

Присваивания в базовом цикле for без оператора update не отмечаются. В таких случаях присваивание, вероятно, является намеренным и не может быть легко перемещено в секцию обновления цикла for.

Пример:


  for (int i = 0; i < list.size(); ) {
    if (element.equals(list.get(i))) {
      list.remove(i);
    } else {
      // Модификация параметра цикла for не отмечается
      // так как отсутствует оператор update
      i++;
    }
  }

Используйте опцию Проверять параметры расширенного цикла 'for', чтобы указать, следует ли также сообщать о модификации параметров расширенного цикла for.
