Сообщает о свойстве lateinit var, которое переопределяет другое свойство lateinit var.

Один экземпляр подкласса будет иметь два поля, при этом поле из суперкласса останется неиспользованным.

Пример:


    open class BaseClass {
      open lateinit var name: String
    }
    class RealClass : BaseClass() {
      override lateinit var name: String
    }
