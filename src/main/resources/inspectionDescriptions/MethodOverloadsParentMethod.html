Сообщает об экземплярных методах, которые имеют то же имя и количество параметров, что и метод суперкласса, но как минимум один параметр имеет несовместимый тип.

В этом случае метод подкласса будет перегружать, а не переопределять метод суперкласса. Это может привести к потенциальным ошибкам, если такое поведение не предусмотрено.

Пример:


  public class Foo {
    void foo(int x) {}
  }
  public class Bar extends Foo {
    void foo(Number x) {} // В случае, если переопределение было намеренным,
                          // метод 'foo()' перегружает совместимый метод суперкласса
  }

Используйте эту опцию, чтобы определить, должна ли проверка также сообщать о случаях несовместимости типов параметров.
