Сообщает о параметрах типа, объявленных в нижнем регистре.

В официальной документации Go примеры используют параметры типа в верхнем регистре. Эта проверка следует такому правилу написания параметров типа в верхнем регистре.

func PrintSlice[t any](s []t) {
    for _, v := range s{
        print(v)
    }
}

Параметр типа t объявлен в нижнем регистре, поэтому будет отмечен.
