Сообщает о конструкциях case без блока else. Блок else должен всегда присутствовать, чтобы определить поведение по умолчанию, когда совпадений не найдено.

Пример:

# 不良做法
case status
when :active
  perform_action
end
Блок else можно добавить с помощью быстрого исправления. После применения быстрого исправления код будет выглядеть так: 
# 优良做法
case status
when :active
  perform_action
else
  # 此处键入代码
end
