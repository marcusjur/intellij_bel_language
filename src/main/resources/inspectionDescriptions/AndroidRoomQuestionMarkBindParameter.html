Сообщает об использовании неименованных параметров запроса (например, WHERE id = ?) в SQL-запросах Room. Анализатор запросов Room не поддерживает такой подход и требует использования именованных параметров (например, WHERE id = :id), которые соответствуют аргументам метода, помеченного аннотацией @Query.
