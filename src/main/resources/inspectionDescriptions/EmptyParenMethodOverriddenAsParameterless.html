빈 매개변수 절을 가진 메서드를 재정의하는 매개변수 절이 없는 메서드를 보고합니다.

규칙에 따르면 메서드에 부수 효과가 있는 경우 괄호를 사용해야 합니다. 리스코프의 치환 원칙에 따라 재정의된 메서드에 매개변수 절이 있고 부수 효과가 있음을 나타낼 경우, 재정의하는 메서드 역시 부수 효과가 있는 메서드로서 선언해야 합니다.

빠른 수정에서는 빈 매개변수 절을 추가합니다.

예:


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x = 2
  }

빠른 수정 적용 후:


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x() = 2
  }
