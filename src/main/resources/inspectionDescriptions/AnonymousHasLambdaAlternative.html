Сообщение относится к анонимным классам, которые можно преобразовать в вызов конструктора или фабричного метода с аргументом в виде лямбда-выражения.

Проверка обнаруживает следующие категории:

  • Анонимные классы, расширяющие ThreadLocal с методом initialValue() (можно заменить на ThreadLocal.withInitial)
  • Анонимные классы, расширяющие Thread с методом run() (можно заменить на new Thread(Runnable))

Пример:


  new Thread() {
    @Override
    public void run() {
      System.out.println("Hello from thread!");
    }
  }.start();

После применения быстрого исправления:


  new Thread(() -> {
    System.out.println("Hello from thread!");
  }).start();
