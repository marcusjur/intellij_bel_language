Сообщает о полях, помеченных аннотацией non-null, которые не инициализованы в конструкторе.

Пример:

public class MyClass {
  private @NotNull String value;
  public void setValue(@NotNull String value) {
    this.value = value;
  }
  public @NotNull String getValue() {
    return value;
  }
}

Такие поля могут нарушать ограничение non-null. В приведенном примере параметр метода setValue помечен как non-null, но если сеттер не будет вызван, метод getValue может вернуть null.
