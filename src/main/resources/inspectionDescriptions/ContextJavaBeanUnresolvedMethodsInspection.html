Сообщает о нераспознанных ссылках на методы в параметрах initMethod и destroyMethod аннотации @Bean.

Пример:

Если класс MyBean не содержит метод 'public void doInit(){...}', будет выведено сообщение "Невозможно разрешить метод" для выражения 'doInit' 

  public class MyBean {...}
  @Component
  public class MyComponent {
     @Bean(initMethod="doInit" )
     public MyBean myBean() {...}
  }

В этом примере, если MyBean не определяет метод doInit(), проверка сообщит о нераспознанной ссылке на метод.
