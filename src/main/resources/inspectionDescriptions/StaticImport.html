Сообщать о наличии операторов import static.

JVM версии Java 1.4 и более ранних версий не поддерживает такие операторы import.

Настройка проверки:

  • Используйте таблицу ниже, чтобы указать классы, которые будут игнорироваться при проверке операторов import static.
  • Используйте флажок Игнорировать импорт отдельных полей static, чтобы игнорировать операторы import static для отдельных полей.
  • Используйте флажок Игнорировать импорт отдельных методов static, чтобы игнорировать операторы import static для отдельных методов.
