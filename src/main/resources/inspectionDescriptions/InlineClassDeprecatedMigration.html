Сообщает об инлайн-классах, которые считаются устаревшими в Kotlin 1.5 и выше и вызывают предупреждения при компиляции. См. Что нового в Kotlin 1.5.0

Пример:


  inline class Password(val s: String)

После применения быстрого исправления:


  @JvmInline
  value class Password(val s: String)

Проверка выполняется для языкового уровня Kotlin начиная с версии 1.5.
