Сообщать о избыточных объявлениях переопределения.

Переопределение может быть опущено, если оно не изменяет семантику унаследованной сигнатуры, например при изменении только видимости.

Пример:


  open class Foo {
      open fun singleExpression() {
      }
  }
  class Bar : Foo() {
      override fun singleExpression() = super.singleExpression()
  }

После применения быстрого исправления:


  class Bar : Foo() {
  }
