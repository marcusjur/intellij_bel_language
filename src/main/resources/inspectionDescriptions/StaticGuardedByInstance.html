Сообщает об аннотации @GuardedBy на статических полях или методах, где защитный элемент является нестатическим полем или this.

Защита статических элементов нестатическими элементами может привести к чрезмерной конкурентности, когда несколько потоков блокируются в контексте разных объектов для одновременного доступа к защищенному полю.

Пример:


    private ReadWriteLock lock = new ReentrantReadWriteLock();
    @GuardedBy("lock")
    public static void bar() {
        // ...
    }

Поддерживаемые аннотации @GuardedBy:

  • net.jcip.annotations.GuardedBy
  • javax.annotation.concurrent.GuardedBy
  • org.apache.http.annotation.GuardedBy
  • com.android.annotations.concurrency.GuardedBy
  • androidx.annotation.GuardedBy
  • com.google.errorprone.annotations.concurrent.GuardedBy
