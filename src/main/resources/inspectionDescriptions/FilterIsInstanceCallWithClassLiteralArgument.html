Сообщает о вызовах функции стандартной библиотеки Kotlin filterIsInstance с аргументами-литералами классов. Рекомендуется использовать версию этой функции с параметром reified type для избежания синтаксиса ::class.java.

Примечание: проверка не распространяется на литералы обобщенных классов, так как синтаксис Class<*, *> в списке аргументов типа может быть нежелательным.

Пример:


  fun foo(list: List<*>) {
      list.filterIsInstance(Int::class.java)
  }

После применения быстрого исправления:


  fun foo(list: List<*>) {
      list.filterIsInstance<Int>()
  }
