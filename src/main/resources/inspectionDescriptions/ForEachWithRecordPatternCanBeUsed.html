Сообщает об объявлениях локальных переменных и обращениях к методам доступа, которые могут быть заменены переменными шаблона в расширенном операторе 'for' (что обычно делает код более компактным).

Пример:


record Record(Integer x, String y) {
}
public static void test(List<Record> records) {
    for (Record record : records) {
        System.out.println(record.y());
        Integer x = record.x;
        System.out.println(x);
    }
}

Может быть заменено на:


record Record(Integer x, String y) {
}
public static void test(List<Record> records) {
    for (Record(Integer x, String y) : records) {
        System.out.println(y);
        System.out.println(x);
    }
}
  • Используйте опцию Предел глубины вложенности, чтобы указать максимальное количество вложенных шаблонов деструктурирования для отчёта
  • Используйте опцию Максимальное количество компонентов записи для деструктурирования, чтобы указать максимальное количество компонентов, которые может содержать запись для использования в шаблоне деструктурирования
  • Используйте опцию Максимальное количество неиспользуемых компонентов записи, чтобы указать максимальное количество неиспользуемых компонентов в операторе for
