Обнаружены некорректные свойства каналов в аннотациях конечных точек: @Gateway, @ServiceActivator, @Filter и другие.

Пример:


  @Component
  public class MyComponent {}
     @Gateway(requestChannel = "requestChannel", replyChannel = "replyChannel")
     public void gateway() {...}
     @Gateway(
        requestChannel ="simpleBean",   // Bean должен быть одного из следующих типов: org.springframework.integration.MessageChannel,org.springframework.messaging.MessageChannel
        replyChannel = "unknownChannel")   // Канал не найден
     public void errors() {...}
     @Bean
     public MyBean simpleBean() {...}
  }
