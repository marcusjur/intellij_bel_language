Сообщать о неразрешённых ссылках в коде MongoDB и JavaScript.

Пример:

db
use foo
  -- a reference to a non-existing collection
db.non_existing_collection
db['non_existing_collection']
db['non_existing_collection'].find().hasNext()

Коллекция non_existing_collection не существует в базе данных и будет отмечена.
