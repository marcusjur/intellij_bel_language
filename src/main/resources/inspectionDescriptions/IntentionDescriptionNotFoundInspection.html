Сообщение об отсутствии файла описания HTML, файла before.template или файла after.template для намерения. Они отображаются в Настройки | Редактор | Намерения.

Быстрое исправление Создать файл описания создаст шаблонный файл описания HTML.

If no before/after template files are required, add <skipBeforeAfter>true</skipBeforeAfter> to the intention registration in plugin.xml.

See Intentions in IntelliJ Platform Plugin SDK docs for more details.
