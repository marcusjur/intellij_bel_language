Эта проверка подсвечивает случаи в вашем тестовом коде, где возможно происходит попытка переопределения нестандартных команд библиотеки тестования JavaScript Cypress.


Переопределение нестандартных команд может привести к непредсказуемому поведению в тестах и нарушить целостность тестового набора. В рамках правильного использования API Cypress рекомендуется переопределять только встроенные команды.
