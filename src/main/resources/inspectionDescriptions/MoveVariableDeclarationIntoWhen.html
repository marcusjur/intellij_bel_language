Объявление переменной может быть перемещено в выражение when.

Пример:


fun someCalc(x: Int) = x * 42
fun foo(x: Int): Int {
  val a = someCalc(x)
  return when (a) {
    1 -> a
    2 -> 2 * a
    else -> 24
  }
}

После применения быстрого исправления:


fun foo(x: Int): Int {
  return when (val a = someCalc(x)) {
    1 -> a
    2 -> 2 * a
    else -> 24
  }
}
