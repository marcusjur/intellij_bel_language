Сообщает о массивах, используемых в конкатенации String или передаваемых в качестве параметров в методы java.io.PrintStream, такие как System.out.println().

Обычно следует использовать содержимое массива вместо самого объекта массива.

Пример:


  void print(Object[] objects) {
    System.out.println(objects);
  }

После применения быстрого исправления:


  void print(Object[] objects) {
    System.out.println(Arrays.toString(objects));
  }
