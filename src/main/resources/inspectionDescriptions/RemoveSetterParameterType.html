Сообщает о явно указанных типах параметров в сеттерах свойств.

Тип параметра сеттера всегда соответствует типу свойства, поэтому его явное указание не требуется. Быстрое исправление "Удалить явное указание типа" позволяет соответствующим образом изменить код.

Пример:


  fun process(x: Int) {}
  var x: Int = 0
      set(value: Int) = process(value) // <== спецификация 'Int' может быть безопасно опущена

После применения быстрого исправления:


  fun process(x: Int) {}
  var x: Int = 0
      set(value) = process(value)
