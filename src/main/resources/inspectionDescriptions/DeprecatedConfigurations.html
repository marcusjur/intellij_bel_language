Обнаружение использования устаревших методов конфигурации.

Методы конфигурации могут быть признаны устаревшими по одной из двух причин:
  • API Gradle развивается, и следует избегать использования устаревших методов;
  • Плагин предоставляет улучшенную версию устаревшего метода конфигурации.
Скорее всего, для устаревших методов существуют альтернативные решения.

Пример:


plugins {
  id 'java'
}
dependencies {
  archive 'org.gradle.api:gradle:1.0' // reports 'archive'
}
