Сообщает о любых вызовах super(A, B), которые не соответствуют следующим требованиям:

  • B является экземпляром A
  • B является подклассом A

Пример:


class Figure:
    def color(self):
        pass
class Rectangle(Figure):
    def color(self):
        pass
class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square не является подклассом Rectangle

Для исправления вы можете сделать Square экземпляром класса Rectangle.
