Сообщает о включениях ссылочных классов (с помощью include, prepend и extend). Включения могут ссылаться только на модули или интерфейсы.

Некорректные включения можно удалить с помощью быстрого исправления.

Пример:

class Example
  # Использование класса 'Object' в вызове 'include'
  include Object
end

После применения быстрого исправления:

class Example
end
