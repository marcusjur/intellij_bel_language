Делегирование интерфейса свойству var

Делегирование использует только начальное значение свойства, последующие присваивания не влияют на него.

Пример:


  class Example(var text: CharSequence): CharSequence by text

Это быстрое исправление заменит свойство на неизменяемое:


  class Example(val text: CharSequence): CharSequence by text

Если вам по какой-то причине необходима изменяемость, можно использовать альтернативный подход:


  class Example(text: CharSequence): CharSequence by text {
      var text = text
  }
