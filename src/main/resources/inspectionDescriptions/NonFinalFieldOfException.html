Сообщает о полях, не объявленных как final в подклассах java.lang.Exception.

Данные в объектах исключений не должны изменяться, так как это может привести к потере контекста ошибки, необходимого для последующей отладки и логования.

Пример:


  public class EditorException extends Exception {
    private String message; // Предупреждение: нефинальное поле 'message' в классе исключения
  }
