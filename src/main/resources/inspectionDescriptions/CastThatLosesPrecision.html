Сообщать о преобразованиях между примитивными числовыми типами, которые могут привести к потере точности.

Такие преобразования не обязательно являются проблемой, но могут привести к трудноотслеживаемым ошибкам, если потеря точности происходит непреднамеренно.

Пример:


  int a = 420;
  byte b = (byte) a;

Используйте опцию Игнорировать преобразования из int в char, чтобы игнорировать преобразования из int в char. Такой тип преобразования часто используется при реализации операций ввода-вывода, так как метод read() класса java.io.Reader возвращает int.

Используйте опцию Игнорировать преобразования из int 128-255 в byte, чтобы игнорировать преобразования констант (128-255) из int в byte. Такие значения переполняются в отрицательные числа, которые все еще помещаются в один байт.
