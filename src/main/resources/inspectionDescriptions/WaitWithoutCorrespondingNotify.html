Сообщает о вызовах Object.wait(), для которых не найдены соответствующие вызовы Object.notify() или Object.notifyAll().

Эта проверка сообщает только о вызовах, содержащих квалификаторы, которые ссылаются на поля текущего класса.

Пример:


  public class Foo {
    public Object foo = new Object();
    void bar() throws InterruptedException {
      this.foo.wait();
    }
  }
