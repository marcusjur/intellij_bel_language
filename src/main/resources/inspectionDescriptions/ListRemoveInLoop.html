Сообщает о вызовах List.remove(index) в цикле, которые могут быть заменены на List.subList().clear().

Такая замена более эффективна для большинства реализаций List при удалении нескольких элементов.

Пример:


  void removeRange(List<String> list, int from, int to) {
    for (int i = from; i < to; i++) {
        list.remove(from);
    }
  }

После применения быстрого исправления:


  void removeRange(List<String> list, int from, int to) {
    if (to > from) {
      list.subList(from, to).clear();
    }
  }

Быстрое исправление автоматически добавляет проверку диапазона для предотвращения возможного IndexOutOfBoundsException, когда минимальное значение больше максимального. Если такая ситуация невозможна в вашем коде, эту проверку можно удалить.
