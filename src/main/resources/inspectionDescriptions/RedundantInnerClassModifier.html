Модификатор inner помечается как избыточный, если класс не ссылается на члены своего внешнего класса.

Пример:


  class Foo {
      inner class InnerClass { // Избыточный модификатор `inner`
          fun hello() {
              println("Hi!")
          }
      }
  }
  class List {
      val objects = Array<Any>(42) { Any() }
      inner class Iterator { // Неизбыточный модификатор `inner`
          fun next(): Any {
              return objects[0]
          }
      }
  }

После применения быстрого исправления:


  class Foo {
      class InnerClass { // Избыточный модификатор `inner`
          fun hello() {
              println("Hi!")
          }
      }
  }
  class List {
      val objects = Array<Any>(42) { Any() }
      inner class Iterator { // Неизбыточный модификатор `inner`
          fun next(): Any {
              return objects[0]
          }
      }
  }
