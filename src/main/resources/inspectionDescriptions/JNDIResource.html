Сообщает о незакрытых ресурсах JNDI. Проверка отслеживает ресурсы JNDI, включая javax.naming.InitialContext и javax.naming.NamingEnumeration.

По умолчанию проверка предполагает, что ресурс может быть закрыт любым методом, содержащим в названии 'close' или 'cleanup'.

Пример:


  Object findObject(Properties properties, String name) throws NamingException {
    Context context = new InitialContext(properties); //контекст не закрыт
    return context.lookup(name);
  }

Настройка проверки с помощью следующих параметров:

  • Разрешать ли открытие ресурсов JNDI внутри блока try. Этот стиль менее предпочтителен, так как он более многословен, чем открытие ресурса перед блоком try.
  • Может ли ресурс быть закрыт любым вызовом метода, которому этот ресурс передается в качестве аргумента.
