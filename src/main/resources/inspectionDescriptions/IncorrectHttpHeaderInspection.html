Сообщает о неизвестных HTTP-заголовках, которые не совпадают с общеизвестными публичными заголовками. Быстрое исправление предлагает добавить этот заголовок в список пользовательских заголовков при включенной опции Использовать пользовательские HTTP-заголовки. HTTP-заголовки из списка пользовательских заголовков не будут вызывать срабатывание проверки.
