Столбцы с одинаковыми именами, принадлежащие разным таблицам.

Пример (MySQL):

CREATE TABLE foo(id INT PRIMARY KEY);
CREATE TABLE bar(id INT PRIMARY KEY);
SELECT foo.id, bar.id FROM foo, bar WHERE id > 0;

Столбец id присутствует в таблицах foo и bar. Необходимо уточнить имя столбца, чтобы запрос работал корректно.

SELECT foo.id, bar.id FROM foo, bar WHERE foo.id > 0;
