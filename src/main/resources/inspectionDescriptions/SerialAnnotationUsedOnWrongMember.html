Сообщает о методах и полях в классах Serializable и Externalizable, которые некорректно аннотированы с помощью аннотации java.io.Serial.

Пример:


class Test implements Serializable {
  @Serial // поле с аннотацией не является частью механизма сериализации, так как оно не final
  private static long serialVersionUID = 7874493593505141603L;
  @Serial // метод с аннотацией не является частью механизма сериализации, так как он не private
  void writeObject(ObjectOutputStream out) throws IOException {
  }
}


class Test implements Externalizable {
  @Serial // метод с аннотацией не является частью механизма сериализации, так как находится внутри класса Externalizable
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

Информацию обо всех допустимых случаях см. в Javadoc для класса java.io.Serial.
