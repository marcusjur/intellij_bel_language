Сообщает о неиспользуемых унарных операторах числового типа в выражениях.

При использовании унарных операторов без фигурных скобок они могут прервать предыдущее выражение. Таким образом, математические выражения, охватывающие несколько строк, могут вводить в заблуждение.

Пример:


  fun main() {
      val result = 1 + 2 * 3
                  + 3              // <== Обратите внимание, '+ 3' не относится к переменной 'result' и не используется
      println("Result = $result")  // Результат будет '7', а не ожидаемые '10'
  }
