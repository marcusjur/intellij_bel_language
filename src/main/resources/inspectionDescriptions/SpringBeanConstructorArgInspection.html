Сообщает об использовании внедрения зависимостей на основе конструктора в Spring <bean>.

Проверка срабатывает, когда не удаётся найти подходящий конструктор или фабричный метод для <bean> с использованием настроенных тегов <constructor-arg> и определённой стратегии autowire.

Пример:


    public class MyComponent {
    // 构造函数
    public MyComponent(MyBean bean) {}
    // 工厂方法
    public static MyComponent getInstance(String name, int port) {
    }


    <beans>
      <bean class="beans.MyComponent" id="foo"> <!-- Сообщение "Не найден подходящий конструктор в классе 'MyComponent'" -->
          <constructor-arg ref="myBean"/>
         <constructor-arg value="123"/>
      </bean>
      <bean class="beans.MyComponent" factory-method="getInstance"> <!-- Сообщение "Не найден подходящий фабричный метод в классе" -->
         <constructor-arg value="123"/>
      </bean>
    </beans>
