Сообщает о повторяющихся ключевых параметрах в методах, блоках и типах proc. Распознается только первое объявление каждого ключевого параметра; все последующие объявления игнорируются.

Пример:

class Example
  # Повторяющийся ключевой параметр
  def call: (keyword: untyped, keyword: untyped) -> untyped
end
