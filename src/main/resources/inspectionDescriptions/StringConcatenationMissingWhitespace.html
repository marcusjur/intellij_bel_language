Сообщать о конкатенации строк без пробелов, где левая часть заканчивается буквой или цифрой Unicode, а правая часть начинается с буквы или цифры Unicode.

Пример:


  String sql = "SELECT column" +
               "FROM table";

Используйте опцию игнорировать конкатенацию со строковыми переменными, чтобы получать уведомления только когда обе части конкатенации являются литералами.
