Сообщение может быть свёрнуто в выражение безопасного доступа (?.) из выражения if-then.

Пример:


  fun bar(x: String) = ""
  fun foo(a: String?) {
     if (a != null) bar(a) else null
  }

Это быстрое исправление преобразует выражение if-then в выражение безопасного доступа (?.):


  fun bar(x: String) = ""
  fun foo(a: String?) {
     a?.let { bar(it) }
  }
