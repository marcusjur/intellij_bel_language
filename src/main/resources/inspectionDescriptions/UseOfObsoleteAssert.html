Сообщает о любых вызовах методов класса junit.framework.Assert. Этот класс устарел, и такие вызовы можно заменить вызовами методов класса org.junit.Assert.

Например:


  import org.junit.*;
  public class NecessaryTest {
    @Test
    public void testIt() {
      junit.framework.Assert.assertEquals("expected", "actual");
    }
  }

После применения быстрого исправления результат будет выглядеть так:


  import org.junit;
  public class NecessaryTest {
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }
