Сообщает об использовании фигурных скобок для обрамления простых идентификаторов в строковых шаблонах.

Используйте быстрое исправление "Удалить фигурные скобки" для удаления избыточных фигурных скобок.

Пример:


  fun redundant() {
     val x = 4
     val y = "${x}" // <== избыточно
  }
  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== допустимо
  }

После применения быстрого исправления:


  fun redundant() {
     val x = 4
     val y = "$x"
  }
  fun correctUsage() {
      val x = "x" <== обновлено
      val y = "${x.length}"
  }
