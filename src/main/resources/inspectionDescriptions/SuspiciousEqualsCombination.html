Сообщает об использовании операторов == и === для сравнения одной и той же переменной в одном выражении.

Из-за схожести операторов == и ===, они могут быть случайно перепутаны без явного предупреждения. Необходимо внимательно проверять, что используется именно ==, а не ===

Пример:


  if (type === FIELD || type == METHOD || type == ANNOTATION_METHOD || // Обратите внимание на неправильное использование "=="
      type === LAMBDA_EXPRESSION) return
