Сообщает о стандартных геттер-методах, которые могут быть заменены аннотацией lombok @Getter.

Пример:


  import java.util.Date;
  public class MyClass {
    /**
     * Дата обработки.
     */
    private Date processDate;
    /**
     * Возвращает дату.
     *
     * @return The date
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

После применения быстрого исправления/очистки:


  import lombok.Getter;
  import java.util.Date;
  @Getter
  public class MyClass {
    /**
     * Дата обработки.
     * -- GETTER --
     * Возвращает дату.
     *
     * @return The date
     */
    private Date processDate;
  }

Сообщение появляется только при настроенной библиотеке lombok. Для удаления lombok посетите сайт lombok.
