Сообщает о вызовах параметризованных методов с явными аргументами типа, которые можно опустить (поскольку компилятор может однозначно вывести эти типы аргументов).

Использование избыточных аргументов типа является ненужным и снижает читаемость кода.

Пример:


  List<String> list = Arrays.<String>asList("Hello", "World");

Избыточные аргументы типа можно удалить с помощью быстрого исправления:


  List<String> list = Arrays.asList("Hello", "World");
