Сообщает о лишних точках с запятой (;), которые можно безопасно удалить.

Kotlin не требует точку с запятой в конце каждого выражения или оператора. Рекомендуется использовать это быстрое исправление для удаления лишних точек с запятой.

Пример:


  val myMap = mapOf("one" to 1, "two" to 2);
  myMap.forEach { (key, value) ->  print("$key -> $value")};

После применения быстрого исправления:


  val myMap = mapOf("one" to 1, "two" to 2)
  myMap.forEach { (key, value) ->  print("$key -> $value")}

Точка с запятой необходима в двух случаях:

  1. Для разделения нескольких выражений в одной строке: 
    
  map.forEach { val (key, value) = it; println("$key -> $value") }
  2. В классах enum, где определяются свойства или функции, требуется точка с запятой после списка констант: 
    
  enum class Mode {
      SILENT, VERBOSE;
      fun isSilent(): Boolean = this == SILENT
  }
