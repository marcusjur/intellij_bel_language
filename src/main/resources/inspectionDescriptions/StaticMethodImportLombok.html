Сообщение об использовании статического импорта для методов, генерируемых Lombok.

Это приведет к ошибке при компиляции с помощью javac. Причина в том, что статические методы анализуются до запуска процессора аннотаций Lombok. В этот момент javac использует исходный код вместо кода, который должен быть сгенерирован процессором аннотаций, и, естественно, сообщает об отсутствующих символах (сгенерированных методах).
