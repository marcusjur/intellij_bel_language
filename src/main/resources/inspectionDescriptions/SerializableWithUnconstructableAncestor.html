Сообщает о ближайшем несериализуемом предке класса Serializable, у которого отсутствует конструктор без параметров. Такой класс не может быть десериализован, что приведет к ошибке и вызовет исключение InvalidClassException.

Пример:


  class Ancestor {
    private String name;
    Ancestor(String name) {
      this.name = name;
    }
  }
  //Для этого класса будет выдано предупреждение, так как суперкласс
//не сериализуем и его конструктор принимает параметры
  class Descendant extends Ancestor implements Serializable {
    Descendant() {
      super("Bob");
    }
  }
