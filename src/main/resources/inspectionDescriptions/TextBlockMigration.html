Сообщение о возможности упрощения конкатенации String путем замены на текстовый блок.

Требования:

  • \n встречается два или более раз.
  • Текстовый блок не конкатенирован.

Опция сообщать об отдельных строковых литералах позволяет подсвечивать отдельные литералы, содержащие символы переноса строки. Быстрое исправление доступно даже при отключенной опции.

Пример:


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

После применения быстрого исправления:


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;

Новое в версии 2019.3
