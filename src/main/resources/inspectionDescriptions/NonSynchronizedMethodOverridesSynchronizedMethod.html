Сообщает о методах без модификатора synchronized, которые переопределяют методы с synchronized.

Если метод суперкласса объявлен как synchronized, переопределяющий метод не будет автоматически синхронизованным. Это может привести к непредвиденным состояниям гонки при использовании подкласса.

Пример:


  class Super {
    synchronized void process() {}
  }
  class Sub extends Super {
    // Метод 'process()' без synchronized переопределяет synchronized метод
    void process() {}
  }
