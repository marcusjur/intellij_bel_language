Сообщает о упакованных свойствах Range.start и Range.endInclusive.

Эти свойства можно заменить на неупакованные свойства first и last во избежание избыточных вызовов.

Быстрое исправление заменит свойства start и endInclusive на соответствующие first и last.

Пример:


  fun foo(range: CharRange) {
      val lastElement = range.endInclusive
  }

После применения быстрого исправления:


  fun foo(range: CharRange) {
      val lastElement = range.last
  }
