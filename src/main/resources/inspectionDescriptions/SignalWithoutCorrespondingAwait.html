Сообщает о вызовах Condition.signal() или Condition.signalAll(), для которых не найден соответствующий вызов Condition.await().

Эта проверка отслеживает только вызовы, целью которых являются поля текущего класса.

Пример:


  class Queue {
    private final Condition isEmpty = ...;
    void add(Object elem) {
      // ...
      isEmpty.signal(); // Предупреждение: вызов 'signal()' без соответствующего 'await()'
      // ...
    }
    void remove(Object elem) throws InterruptedException {
      // ...
      //      isEmpty.await();
      // ...
    }
  }
