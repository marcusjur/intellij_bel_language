Оператор if, который можно упростить до единственного присваивания, оператора return или assert.

Пример:


  if (foo()) {
    return true;
  } else {
    return false;
  }

После применения быстрого исправления:


  return foo();

Настройка проверки:

Чтобы скрыть предупреждения для цепочки операторов if, используйте опцию Игнорировать цепочки операторов 'if'.

Например, в следующем коде предупреждение будет скрыто, но быстрое исправление останется доступным:


  if (condition1) return true;
  if (condition2) return false;
  return true;

Обратите внимание, что замена if (isTrue()) assert false; на assert isTrue(); может изменить семантику программы при отключенных проверках assert, если условие имеет побочные эффекты. Чтобы скрыть предупреждения для операторов if, содержащих в теле только оператор assert, используйте опцию Игнорировать операторы 'if' с обычным 'assert'.
