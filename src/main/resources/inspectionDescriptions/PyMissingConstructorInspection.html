В классе отсутствует вызов конструктора super.

Пример:


class Fruit:
    def __init__(self):
        pass
class Pear(Fruit):
    def __init__(self):
        pass

Класс Pear должен содержать вызов super в методе __init__.

После применения быстрого исправления код будет выглядеть так:


class Fruit:
    def __init__(self):
        pass
class Pear(Fruit):
    def __init__(self):
        super().__init__()
