Сообщает о распространенной ошибке в TypeScript, когда поле класса объявляется как параметр конструктора, а затем происходит попытка присвоить значение этому параметру.
В этом случае значение не присваивается соответствующему полю, а только изменяет локальный параметр. 

class Foo {
  constructor(private p: number) {
    p = 1; //должно быть this.p = 1;
  }
}
