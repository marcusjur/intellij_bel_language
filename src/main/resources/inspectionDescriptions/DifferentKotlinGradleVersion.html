Текущий плагин ИСР не может корректно поддерживать данную версию плагина Gradle.

Это может привести к несоответствиям между ИСР и Gradle-сборкой в отображении ошибок или поведении кода.

Пример:


  dependencies {
    classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:0.0.1"
  }

Для исправления этой проблемы измените версию плагина Kotlin Gradle, чтобы она соответствовала версии Kotlin, встроенной в плагин ИСР.
