Обнаружен public конструктор в abstract классе.

Конструкторы abstract классов могут вызываться только из конструкторов их подклассов, поэтому объявление их как public может вводить в заблуждение.

Это быстрое исправление изменит модификатор доступа конструктора на protected.

Пример:


  public abstract class Foo {
    public Foo () { // Предупреждение: модификатор 'public'
        /* ... */
    }
  }

После применения быстрого исправления:


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

Настройка проверки:

Используйте опцию игнорировать не public классы, чтобы пропускать public конструкторы в непубличных классах.
