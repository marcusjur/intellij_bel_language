Обнаружены неотсортированные импорты.

Все программы на Go должны быть отформатованы одинаковым образом, согласно правилам, установленным инструментом gofmt. Эти правила требуют сортировки импортов.

Пример неправильной сортировки:

import (
 "net"
 "errors"
 "fmt"
)

Вы можете применить быстрое исправление Отсортировать импорты, чтобы исправить проблему сортировки. После применения быстрого исправления сортировка будет выглядеть так:

import (
 "errors"
 "fmt"
 "net"
)
