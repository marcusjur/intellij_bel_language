Сообщает об улучшенных операторах и выражениях switch. Предлагает заменить их на обычные операторы switch.

Пример:


  boolean even = switch (condition) {
    case 1, 3, 5, 7, 9 -> false;
    default -> true;
  };

После применения быстрого исправления:


  boolean even;
  switch (condition) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 9:
      even = false;
      break;
    default:
      even = true;
      break;
}

Улучшенный switch появился в Java 14. Эта проверка помогает обеспечить обратную совместимость с более ранними версиями Java.
