Сообщает о случаях импорта нескольких трейтов через единственный оператор use.

Согласно PSR-12, каждый отдельный трейт, импортуемый в класс, должен располагаться на отдельной строке и иметь собственный оператор импорта use.

Для получения дополнительной информации см. PSR-12: Extended Coding Style (php-fig.org).
