Сообщает о случаях разыменования указателя, который может быть nil.

Пример:


type Person struct{ Name string }
func _(person *Person) {
	if person != nil || person.Name != "Peter" {
	}
}

В приведенном примере person является указателем на структуру Person, но имеет значение nil, поскольку ему не было присвоено значение. Попытка разыменовать person для доступа к полю Name без проверки person на nil приведет к ошибке времени выполнения.
