Сообщает о компонентах, не имеющих статического текста и не связанных с какими-либо метками с помощью setLabelFor.

Компоненты без статического текста включают поля редактирования и комбинированные списки. Такие компоненты невозможно активировать с помощью клавиатурных сокращений. Быстрое исправление для этой проверки позволяет автоматически связать соседние метки с проблемными компонентами.
