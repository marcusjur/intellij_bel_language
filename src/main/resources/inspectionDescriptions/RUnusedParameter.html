Сообщает о неиспользуемых параметрах функции в теле функции.

Пример:


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((temp - 32) * 5 / 9)
}

Измените выделенный фрагмент кода, чтобы использовать или удалить этот параметр.

Возможное решение:


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((f_temp - 32) * 5 / 9)
}
