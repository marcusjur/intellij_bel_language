Сообщает об упрощаемых логических выражениях.

Пример:


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

После применения быстрого исправления:


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

Пример:


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

После применения быстрого исправления:


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }
