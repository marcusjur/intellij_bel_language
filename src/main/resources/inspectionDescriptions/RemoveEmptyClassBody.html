Сообщает об объявлениях классов и объектов с пустым телом.

Используйте быстрое исправление "Удалить избыточное пустое тело класса" для очистки кода.

Пример:


  class EmptyA() {} // <== пустое тело
  class EmptyB {
      companion object {} // <== пустое тело
  }
  fun emptyC() {
     object {} // <== анонимный объект, допустимо (не отмечается)
  }

После применения быстрого исправления:


  class EmptyA()
  class EmptyB {
      companion object
  }
  fun emptyC() {
     object {}
  }
