Сообщает о классах, объявленных как final. Не сообщает о final-классах, которые расширяют sealed классы или интерфейсы. Такие классы не могут быть унаследованы и могут указывать на недостатки в объектно-ориентированном проектировании. Некоторые стандарты кодования не рекомендуют использовать классы с модификатором final.

Пример:


  public final class Main {
  }

После применения быстрого исправления:


  public class Main {
  }
