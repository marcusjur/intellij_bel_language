Обнаружено сравнение с NULL, которое может быть заменено операторами IS NULL или IS NOT NULL.

Пример (Microsoft SQL Server):

CREATE TABLE foo ( id int );
SELECT * FROM foo WHERE NULL = NULL;
SELECT * FROM foo WHERE NULL != NULL;

NULL = NULL может быть заменено на IS NULL, а сравнение NULL != NULL может быть заменено на IS NOT NULL. Для выполнения этой замены вы можете использовать быстрое исправление Использовать оператор IS NULL или Использовать оператор IS NOT NULL.

SELECT * FROM foo WHERE NULL IS NULL;
SELECT * FROM foo WHERE NULL IS NOT NULL;
