Сообщает об использовании несокращенных форм логических операторов 'and' и 'or' (&, |, &= и |=). Хотя несокращенные версии иногда могут быть полезны, в большинстве случаев следует использовать сокращенные формы (&& и ||), так как непреднамеренное использование несокращенных форм может привести к труднообнаруживаемым ошибкам.

Рекомендуется использовать сокращенные версии с помощью быстрого исправления.

Пример:


  void foo(boolean x, boolean y, boolean z) {
    if (x | y) { x |= z; }
  }

После применения быстрого исправления:


  void foo(boolean x, boolean y) {
    if (x || y) { x = x || z; }
  }
