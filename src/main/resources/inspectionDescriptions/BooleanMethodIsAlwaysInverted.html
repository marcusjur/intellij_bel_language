Сообщает о методах, возвращающих тип boolean, результат которых всегда инвертируется при вызове.

Доступно быстрое исправление для инвертирования и опционального переименования такого метода. По соображениям производительности не все проблемные методы подсвечиваются в редакторе.

Пример:


  class C {
    boolean alwaysTrue() {
      return true;
    }
    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

После применения быстрого исправления:


  class C {
    boolean alwaysFalse() {
      return false;
    }
    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
