Сообщает о случаях переназначения первого параметра (такого как self или cls) внутри метода. ИСР отмечает это как ошибку, поскольку в большинстве случаев такое переназначение не имеет смысла.

Пример:


class Account:
    def calc(self, balance):
        if balance == 0:
            self = balance
        return self

Для исправления следует проверить и модифицировать алгоритм, чтобы убедиться в необходимости такого переназначения. Если всё действительно правильно, можно вызвать операцию намерения для этого кода и выбрать игнорирование предупреждения.
