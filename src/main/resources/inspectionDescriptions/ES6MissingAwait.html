Сообщает о вызовах async функций без ожидаемого префикса await внутри async функций. Такие вызовы возвращают Promise, и поток выполнения продолжается немедленно.

Пример:


async function bar() { /* ... */ }
async function foo() {
    bar(); // плохо
}

После применения быстрого исправления будет добавлен префикс await:


async function bar() { /* ... */ }
async function foo() {
    await bar(); // хорошо
}

Если установлен флажок "Сообщать о промисах в операторах return", также рекомендуется добавлять await в операторы return.
Хотя это обычно не требуется, есть два существенных преимущества.
  • При использовании блока try-catch вы не забудете добавить await.
  • Явное использование await помогает среде выполнения V8 предоставлять асинхронные трассировки стека.
