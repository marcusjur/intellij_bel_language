Сообщает о случаях, когда выражение с концептуальным ограничением присваивается переменной, объявленной как auto, и предлагает добавить ограничение типа.

Пример:


  template<class T>
  concept Foo = true;
  Foo auto func() { return true; }
  void bar() {
      auto f = func(); //предупреждение: здесь требуется добавить ограничение 'Foo'
  }
