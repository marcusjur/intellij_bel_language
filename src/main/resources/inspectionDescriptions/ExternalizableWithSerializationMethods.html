Сообщает о классах Externalizable, которые определяют методы readObject() или writeObject(). Эти методы не вызываются для сериализации объектов Externalizable.

Пример:


  abstract class Crucial implements Externalizable {
    int value;
    private void readObject(ObjectInputStream in) {
      value = in.readInt();
    }
  }
