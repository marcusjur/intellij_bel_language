Сообщает о тегах Javadoc @see, {@link} и {@linkplain}, которые ссылаются на методы с комментариями, super-методы методов с комментариями или классы, содержащие комментарии.

Такие ссылки являются избыточными и могут быть безопасно удалены с помощью быстрого исправления этой проверки. Если тег является единственным содержимым, быстрое исправление удалит весь комментарий Javadoc.

Пример:


  class Example {
    /**
     * @see Example#method
     */
    public void method() { }
  }

После применения быстрого исправления:


class Example {
  public void method() { }
}

Используйте флажок ниже, чтобы игнорировать встроенные ссылки ({@link} и {@linkplain}) на super-методы. Хотя инструмент Javadoc автоматически добавляет ссылки на все super-методы, иногда встроенные ссылки на super-методы могут быть необходимы в тексте комментария Javadoc.
