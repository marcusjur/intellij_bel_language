Результат целочисленного деления используется как число с плавающей точкой. Такое деление обычно указывает на ошибку и может привести к неожиданным результатам из-за усечения при целочисленном делении.

Пример:


  float x = 3.0F + 3 * 2 / 5;

После применения быстрого исправления:


  float x = 3.0F + ((float) (3 * 2)) /5;
