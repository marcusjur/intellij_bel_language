Отчёт включает только комментарии Javadoc с тегом {@inheritDoc}. Поскольку Javadoc автоматически копирует комментарии суперкласса при их отсутствии, комментарии, содержащие только {@inheritDoc}, не добавляют никакой информации.

Кроме того, отчёт показывает использование {@inheritDoc} в недопустимых местах (например, в полях).

Рекомендуется удалить ненужные комментарии Javadoc.

Пример:


  class Example implements Comparable<Example> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }

После применения быстрого исправления:


  class Example implements Comparable<Example> {
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }
