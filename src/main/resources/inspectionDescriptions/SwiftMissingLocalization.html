Сообщает об использовании локализованных строк, которые не были добавлены в файлы строк.

По умолчанию эта проверка срабатывает, если локализационный ключ не определен ни в одном из доступных файлов строк. Используйте флажок ниже, чтобы получать предупреждения только в случае, если ключ не определен во всех языках.

Рекомендуется добавить отсутствующие пары ключ-значение в файлы строк.

Пример: 


  let presentation = "Do It"

После применения быстрого исправления: 


  let presentation = NSLocalizedString("Do It", comment: ""))
