Сообщает о циклах while, do-while и for, которые используют константу true в качестве единственного условия. При этом такие циклы всё ещё могут быть прерваны с помощью операторов if, содержащих выход из цикла.

Такие операторы if должны быть либо первыми, либо единственными операторами в циклах while или for, а также последними или единственными операторами в цикле do-while.

Цикл можно упростить, удалив оператор if и сделав его условие явным условием цикла.
