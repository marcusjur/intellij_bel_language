Сообщает о безусловном вызове wait() в синхронизованном контексте.

Метод wait() обычно используется для блокировки потока до тех пор, пока некоторое условие не станет истинным. Безусловный вызов wait() обычно указывает на то, что условие проверяется до получения блокировки. В этом случае возможна гонка данных, когда условие становится истинным между проверкой и получением блокировки.

Хотя обнаруженная структура не обязательно является некорректной, она определённо заслуживает проверки.

Пример:


  class Bar {
      void foo() throws InterruptedException {
          synchronized (this) {
              wait(); // Предупреждение
          }
      }
  }
