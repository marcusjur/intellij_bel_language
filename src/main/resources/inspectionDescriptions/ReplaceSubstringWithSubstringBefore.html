Сообщается, что вызов s.substring(0, s.indexOf(x)) может быть заменен на s.substringBefore(x).

Использование substringBefore() делает код более лаконичным.

Это быстрое исправление заменит вызов substring на substringBefore.

Пример:


  fun foo(s: String) {
      s.substring(0, s.indexOf('x'))
  }

После применения быстрого исправления:


  fun foo(s: String) {
      s.substringBefore('x')
  }
