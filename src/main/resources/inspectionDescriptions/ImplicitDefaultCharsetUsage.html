Сообщает о методах и вызовах конструкторов, которые неявно используют кодование по умолчанию для платформы. Такие вызовы могут давать разные результаты на системах с различными кодованиеми по умолчанию и могут привести к непредсказуемому поведению.

Пример:

void foo(byte[] bytes) {
  String s = new String(bytes);
}

Если доступна соответствующая перегруженная версия метода, можно применить быстрое исправление, указав явно кодование UTF-8. После применения быстрого исправления:

void foo(byte[] bytes) {
  String s = new String(bytes, StandardCharsets.UTF_8);
}
