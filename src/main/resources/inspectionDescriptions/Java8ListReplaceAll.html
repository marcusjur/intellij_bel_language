Цикл может быть свёрнут в единственный вызов List.replaceAll().

Пример:


  for (int i = 0; i < strings.size(); i++) {
    String str = strings.get(i).toLowerCase();
    strings.set(i, str);
  }

После применения быстрого исправления:


  strings.replaceAll(String::toLowerCase);
