Обнаружение множественных последовательных пробелов в регулярных выражениях. Поскольку пробелы по умолчанию невидимы, сложно определить их необходимое количество. Замена последовательных пробелов на один пробел с квантификатором делает регулярное выражение более понятным.

Пример:


  (     )

После применения быстрого исправления:


  ( {5})
