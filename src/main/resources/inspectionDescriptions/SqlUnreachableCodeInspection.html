Обнаружение недостижимых операторов в SQL-процедурах.

Пример (Microsoft SQL Server):

CREATE FUNCTION foo() RETURNS INT AS
BEGIN
    THROW;
    RETURN 1;
END;

В Microsoft SQL Server оператор THROW вызывает исключение и передает управление в блок CATCH конструкции TRY...CATCH. Следовательно, часть RETURN 1; никогда не будет выполнена.
