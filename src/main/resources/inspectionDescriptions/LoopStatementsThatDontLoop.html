Сообщает о любых случаях использования операторов for, while и do, тело которых выполняется не более одного раза. Обычно это указывает на ошибку.

Используйте опцию Игнорировать расширенный for-цикл, чтобы исключить foreach-циклы. Они иногда используются как компактный способ выполнения операции только с первым элементом итерируемого объекта.

Пример:


  for (String s : stringIterable) {
    doSomethingOnFirstString(s);
    break;
  }
