Обнаруживает ненужные экранированные символы в строковых литералах String и символьных литералах char (опционально).

Не сообщает об экранированных символах табуляции \t, поскольку символы табуляции невидимы.

Пример:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

После применения быстрого исправления:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
  """;

Новое в версии 2019.3
