Сообщает о вызовах методов Assert.assertEquals(), Assert.assertTrue() и подобных, которые можно преобразовать в декларативный стиль Hamcrest с использованием Assert.assertThat().

Например:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      Assert.assertTrue(c.contains(s));
      Assert.assertEquals(c, s);
      Assert.assertNotNull(c);
      Assert.assertNull(c);
      Assert.assertFalse(c.contains(s));
    }
  }

Для выполнения преобразования доступно быстрое исправление:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      assertThat(c, hasItem(o));
      assertThat(o, is(c));
      assertThat(c, notNullValue());
      assertThat(c, nullValue());
      assertThat(c, not(hasItem(o)));
    }
  }

Для работы этой проверки требуется наличие библиотеки Hamcrest в classpath.

Опция статического импорта методов сопоставления позволяет указать, нужно ли использовать статический импорт для методов сопоставления Hamcrest.
