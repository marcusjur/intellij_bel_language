Обнаружено некорректное определение typing.NamedTuple.

Пример:


import typing
class FullName(typing.NamedTuple):
    first: str
    last: str = ""
    middle: str

Для исправления переместите поля со значениями по умолчанию после полей без значений по умолчанию:


import typing
class FullName(typing.NamedTuple):
    first: str
    middle: str
    last: str = ""
