Сообщает об использовании составного присваивания (+=) для неизменяемой Collection.

Операция составного присваивания (+=) для неизменяемой Collection создает временную новую коллекцию, что может повлиять на производительность.

Изменить тип на изменяемый доступно как быстрое исправление для автоматической корректировки кода.

Пример:


  fun test() {
      var list = listOf(0)
      list += 42 // здесь создаётся новый список, эквивалентно list = list + 42
  }

После применения быстрого исправления:


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
