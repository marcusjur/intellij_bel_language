Сообщает об операторах или выражениях switch, где ветка default расположена перед другими case-ветками. Такая конструкция может вызвать ненужную путаницу. При возможности, ветку default можно переместить в конец с помощью быстрого исправления.

Пример:


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

После применения быстрого исправления:


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
