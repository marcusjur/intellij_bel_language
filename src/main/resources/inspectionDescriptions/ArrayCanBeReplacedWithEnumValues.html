Массив констант перечисления может быть заменен вызовом EnumType.values().

Обычно при обновлении такого перечисления необходимо также обновлять массив. Однако если вместо этого использовать EnumType.values(), изменения не потребуются.

Пример:


  enum States {
     NOT_RUN, IN_PROGRESS, FINISHED;
  }
  handleStates(new States[] {NOT_RUN, IN_PROGRESS, FINISHED});

После применения быстрого исправления:


  handleStates(States.values());

Новое в версии 2019.1
