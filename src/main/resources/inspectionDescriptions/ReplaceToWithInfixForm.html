Вызов функции может быть заменен инфиксной формой функции to.

Использование инфиксной формы делает код более лаконичным.

Это быстрое исправление заменит to на инфиксную форму.

Пример:


  fun foo(a: Int, b: Int) {
      val pair = a.to(b)
  }

После применения быстрого исправления:


  fun foo(a: Int, b: Int) {
      val pair = a to b
  }
