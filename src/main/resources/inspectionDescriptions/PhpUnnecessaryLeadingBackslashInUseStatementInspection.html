Обнаружен оператор use с лишним начальным символом \.

Для имен пространств имён (полностью определённое имя пространства имён содержит разделитель пространств имён, как в Foo\Bar, в отличие от имени глобального пространства имён без разделителя, как в FooBar), использование обратной косой черты в начале не требуется и не рекомендуется, поскольку импортуемые имена должны быть полностью определёнными и не обрабатываются относительно текущего пространства имён.

Для получения дополнительной информации см. Using namespaces: Aliasing/Importing (php.net).
