Обнаружен жестко закодованный текст в файле plugin.xml.

Использование текста, определённого в ресурсных пакетах, позволяет поддерживать несколько языков в ИСР. See Internationalization in IntelliJ Platform Plugin SDK docs for more details.

Проверяются следующие элементы:

  • <action>、<group>
  • <separator, <override-text>
  • Известные точки расширения с альтернативами bundle/key
