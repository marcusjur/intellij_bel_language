Обнаружение необязательных пустых переменных в цикле range.

При итерации по срезу с помощью цикла range для каждой итерации возвращаются два значения. Первое значение — это индекс, а второе — копия элемента по этому индексу. Если второе значение не требуется, его можно пропустить, не используя пустой идентификатор.

Пример:

for a, _ = range v {} // `for a, _ =` эквивалентно `for a =`

Чтобы удалить пустой идентификатор, вы можете использовать быстрое исправление Удалить пустой аргумент. После применения быстрого исправления код будет выглядеть так:

for a = range v {}
