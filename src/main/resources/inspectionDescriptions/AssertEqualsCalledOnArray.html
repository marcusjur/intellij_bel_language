Обнаружен вызов JUnit метода assertEquals() с аргументами массива. Такой метод сравнивает ссылки на массивы, а не их содержимое. Следует использовать метод assertArrayEquals() для проверки содержимого массивов.

Пример:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

После применения быстрого исправления:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
