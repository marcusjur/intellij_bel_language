Отчёт об устаревших операторах приведения типов:
  • Оператор приведения типа (unset) считается устаревшим с PHP 7.2.
  • Оператор приведения типа (real) считается устаревшим с PHP 7.4.

Для получения дополнительной информации см. Deprecated features in PHP 7.2.x (php.net) и Deprecations for PHP 7.4 (php.net).
