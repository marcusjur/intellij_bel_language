Сообщает о коде с запахом привязанности к функциям. Предупреждение выдается, когда метод вызывает методы другого класса три или более раз. Вызовы методов библиотечных классов, родительских классов и включающих классов не учитываются в этой проверке. Привязанность к функциям обычно указывает на то, что функциональность находится в неправильном классе.

Пример:


  class JobManager {
    // Предупреждение: этот метод вызывает
    // три метода класса Job
    // Лучше переместить эту цепочку вызовов
    // в сам класс Job.
    void performJob(Job job) {
      job.beforeStart();
      job.process();
      job.afterProcessing();
    }
  }
