Сообщает об операторах assert и утверждениях тестового фреймворка, которые блокируются окружающими блоками catch. Такие утверждения никогда не завершатся неудачей, поскольку выброшенная ошибка AssertionError будет перехвачена и проигнорирована без уведомления.

Пример 1:


  void javaAssertion() {
    try {
      ...
      assert 1 == 2;
    } catch (AssertionError e) {
      // Это утверждение игнорируется без уведомления
    }
  }

Пример 2:


  @Test
  void testWithAssertJ() {
    try {
      ...
      assertThat(1).as("test").isEqualTo(2);
    } catch (AssertionError e) {
      // Это утверждение игнорируется без уведомления
    }
  }

Пример 3:


  @Test
  void testWithJunit() {
    try {
      ...
      assertEquals(1, 2);
    } catch (AssertionError e) {
      // Это утверждение игнорируется без уведомления
    }
  }

Что нового в версии 2020.3
