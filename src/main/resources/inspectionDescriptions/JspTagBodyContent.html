Сообщает о несоответствии содержимого тела JSP-тега спецификации TLD.

Если указано пустое содержимое тела, тело тега должно быть пустым.

Если указано содержимое тела без скриптов, тело тега не должно содержать JSP scriptlet.

Для типа пустого содержимого быстрое исправление удалит тело тега, а если указан тип содержимого без скриптов, то будет удален scriptlet.

Пример:


<s:empty>body not allowed here</s:empty>
<s:scriptless>say: <% System.out.println("hey"); %></s:scriptless>

После применения быстрого исправления:


<s:empty/>
<s:scriptless>say: </s:scriptless>
