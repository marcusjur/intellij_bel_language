Сообщает о неявно объявленных классах и предлагает заменить их обычными классами.

Пример (в файле Sample.java):


  public static void main() {
      System.out.println("Hello, world!");
  }

После применения быстрого исправления:


public class Sample {
    public static void main() {
        System.out.println("Hello, world!");
    }
}

Эта проверка помогает обеспечить обратную совместимость со старыми версиями Java.
