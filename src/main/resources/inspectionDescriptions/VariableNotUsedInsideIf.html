Сообщает о переменных, которые проверяются на null в условиях оператора if или в условных выражениях, но не используются внутри этого оператора if.

Обычно это указывает на ненужную проверку или на то, что переменная ошибочно не используется внутри оператора if.

Пример:


  void test(Integer i) {
    if (i != null) { // Здесь 'i' не используется внутри оператора 'if'
      System.out.println();
    }
  }
