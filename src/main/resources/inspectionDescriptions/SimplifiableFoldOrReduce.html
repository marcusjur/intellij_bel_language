sum, product, max 또는 min으로 바꿀 수 있는 fold 및 reduce 메서드를 보고합니다.

예:


  List(1, 2, 3).foldLeft(1){(x,y) => x * y}
  List(1, 2, 3).fold(0)(_ + _)
  List(1, 2, 3).reduce(_ + _)
  List(1, 2, 3).reduceLeft(_ min _)
  List(1, 2, 3).reduce((x, y) => math.max(x, y))

빠른 수정 적용 후:


  List(1, 2, 3).product
  List(1, 2, 3).sum
  List(1, 2, 3).sum
  List(1, 2, 3).min
  List(1, 2, 3).max
