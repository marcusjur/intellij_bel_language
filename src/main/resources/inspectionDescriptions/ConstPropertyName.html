Сообщает о именах свойств const, которые не соответствуют рекомендуемым соглашениям об именовании.

Последовательное именование делает код более читаемым и понятным. Согласно официальному руководству по стилю Kotlin, свойства const должны использовать имена в верхнем регистре с подчеркиваниями.

Пример:


  const val Planck: Double = 6.62607015E-34

Быстрое исправление переименует свойство:


  const val PLANCK: Double = 6.62607015E-34
