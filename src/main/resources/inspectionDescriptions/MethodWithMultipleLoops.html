Сообщает о методах, содержащих несколько циклов.

Пример:

Следующий метод будет отмечен, так как содержит два цикла:


  void methodWithTwoLoops(int n1, int n2) {
    for (int i = 0; i < n1; i++) {
      System.out.println(i);
    }
    int j = 0;
    while (j < n2) {
      System.out.println(j);
      j++;
    }
  }

Следующий метод также будет отмечен, так как содержит вложенные циклы:


  void methodWithNestedLoop(int n1, int n2) {
    for (int i = 0; i < n1; i++) {
      for (int j = 0; j < n2; j++) {
        System.out.println(i + j);
      }
    }
  }
