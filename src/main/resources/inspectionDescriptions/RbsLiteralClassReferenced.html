Сообщает об использовании классов NilClass, TrueClass и FalseClass в сигнатурах типов. Использование этих классов является излишне многословным.

С помощью быстрого исправления можно заменить использование NilClass, TrueClass или FalseClass на их соответствующие литералы.

Пример:

# Используется 'NilClass' вместо 'nil', …
type example = NilClass | TrueClass | FalseClass

После применения быстрого исправления:

type example = nil | true | false
