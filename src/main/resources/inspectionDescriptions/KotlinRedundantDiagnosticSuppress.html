Сообщает об аннотациях @Suppress, которые можно безопасно удалить, поскольку подавляемые ими предупреждения компилятора больше не актуальны в данном контексте.

Пример:


  fun doSmth(@Suppress("UNUSED_PARAMETER") used: Int) {
    println(used)
  }

После применения быстрого исправления:


  fun doSmth(used: Int) {
    println(used)
  }
