Отчёт об избыточных операциях с объектами 'java.time':
  • Создание объектов даты/времени из пакета JDK java.time, когда можно использовать более простой вызов метода или избежать создания.
  • Вызовы методов java.time с аргументами java.time.temporal.ChronoField и java.time.temporal.ChronoUnit, которые можно заменить более специфичными методами.
  • Сравнения с java.time использующие compareTo(), которые можно заменить вызовами isAfter(), isBefore() или isEqual().

Примеры:

- До: 

LocalDateTime now = LocalDateTime.now();
return LocalDateTime.from(now);

После применения быстрого исправления:


LocalDateTime now = LocalDateTime.now();
return now;
- До: 

LocalTime localTime = LocalTime.now();
int minute = localTime.get(ChronoField.MINUTE_OF_HOUR);

После применения быстрого исправления:


LocalTime localTime = LocalTime.now();
int minute = localTime.getMinute();
- До: 

  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.compareTo(date2) > 0;

После применения быстрого исправления:


  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.isAfter(date2);
