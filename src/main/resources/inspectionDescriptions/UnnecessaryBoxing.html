Сообщает о явной упаковке (boxing), то есть об обёртывании примитивных значений в объекты.

Для Java 5 и более поздних версий явная ручная упаковка не требуется и может быть безопасно удалена.

Пример:

  • Integer i = new Integer(1); → Integer i = Integer.valueOf(1);
  • Integer i = Integer.valueOf(1); → int i = 1;

Опция Сообщать только о действительно избыточных операциях упаковки позволяет выявлять только те случаи, когда упакованное значение сразу же явно или неявно распаковывается. В этом случае весь процесс упаковки-распаковки может быть удален. Проверка не выявляет простые случаи явной упаковки.

Эта проверка выполняется только если уровень языка проекта или модуля установлен на 5 или выше.
