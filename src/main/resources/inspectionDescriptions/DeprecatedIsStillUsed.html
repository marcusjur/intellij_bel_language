Сообщает об использовании устаревших классов, методов и полей в коде.

Пример:


  class MyCode {
  @Deprecated
  void oldMethod() {}// Предупреждение: "Устаревший элемент всё ещё используется"
  void newMethod() {
    oldMethod(); // Забытое использование
  }
  }

Использование внутри устаревших элементов игнорируется.

Примечание: По соображениям производительности проверка выполняется только для non-private элементов, которые редко встречаются в проекте.
