Сообщает о параметрах метода, которые всегда имеют одно и то же константное значение.

Пример:


  static void printPoint(int x, int y) { // x всегда равен 0
    System.out.println(x + ", " + y);
  }
  public static void main(String[] args) {
    printPoint(0, 1);
    printPoint(0, 2);
  }

Быстрое исправление встраивает константное значение. Это может упростить реализацию метода.

Используйте опцию Игнорировать, если быстрое исправление недоступно для отключения проверки в следующих случаях:

  • Параметр модифицируется внутри метода
  • Переданное значение параметра является ссылкой на недоступное поле (только для Java)
  • Параметр является vararg (только для Java)

Используйте опцию Максимальная видимость метода для контроля максимальной видимости методов, о которых следует сообщать.

Используйте поле Минимальное количество использований метода для отчёта о параметрах чтобы указать минимальное количество вызовов метода с одинаковыми значениями параметров.
