Сообщает об отсутствующих комментариях и тегах Javadoc.

Пример:


  /**
   * Сообщает об отсутствующем "@param" (если настроено).
   */
  public void sample(int param){
  }

Быстрое исправление добавит отсутствующие теги или отсутствующие комментарии Javadoc:


  /**
   * Сообщает об отсутствующем "@param" (если настроено).
   * @param param
   */
  public void sample(int param){
  }

Проверку можно настроить так, чтобы игнорировать устаревшие элементы или простые методы доступа, такие как getField() или setField().

Вы также можете настроить обязательные теги и минимальный уровень видимости для определённых элементов кода, таких как методы, поля, классы, пакеты, модули и другие.
