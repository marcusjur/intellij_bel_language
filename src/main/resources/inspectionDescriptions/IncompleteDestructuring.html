Сообщает о неполном деструктурирующем объявлении.

Пример:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name) = person

Быстрое исправление дополнит деструктурирующее объявление новыми переменными:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name, age) = person
