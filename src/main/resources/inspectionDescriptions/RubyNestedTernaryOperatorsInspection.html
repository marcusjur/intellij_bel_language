Обнаружены вложенные тернарные операторы. Вложенные тернарные операторы могут быть трудны для чтения, поэтому обычно лучше использовать как минимум один оператор if.

Пример:

min = x < y ? (x < z ? x : z) : (y < z ? y : z)

После применения быстрого исправления результат будет выглядеть так: 
min = if x < y
  x < z ? x : z
else
  y < z ? y : z
end

Основано на 'Ruby Style Guide'
