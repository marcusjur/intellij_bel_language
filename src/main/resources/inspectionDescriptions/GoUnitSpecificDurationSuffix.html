Сообщает о суффиксах единиц измерения в именах констант и переменных типа time.Duration.

Эта проверка происходит из go lint. Список суффиксов, обозначающих единицы времени, можно найти в репозитории golang на github.com.

time.Duration представляет значение в наносекундах, поэтому добавление суффиксов единиц времени может сделать значение переменной неоднозначным или указывать на неправильное использование API time.Duration.

Пример:

var timeoutSeconds = 5 * time.Second
