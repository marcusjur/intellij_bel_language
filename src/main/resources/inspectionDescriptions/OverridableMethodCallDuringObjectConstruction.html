Сообщает о вызовах переопределяемых методов текущего класса во время конструирования объекта.

Вызов метода происходит во время конструирования объекта, если метод находится в:

  • конструкторе
  • нестатическом инициализаторе экземпляра
  • нестатическом инициализаторе поля
  • методе clone()
  • методе readObject()
  • методе readObjectNoData()

    • Метод считается переопределяемым, если он не объявлен как final, static или private. Методы уровня пакета (package-local) считаются безопасными, даже если они переопределяемы. Такие вызовы могут привести к труднообнаруживаемым ошибкам, так как инициализация объекта может произойти после вызова метода.

    Пример:

    
  class Parent {
    void someMethod() { }
  }
  class Child extends Parent {
    Child() {
      someMethod();
    }
  }

    Эта проверка имеет общую функциональность со следующими проверками:

    • Вызов абстрактных методов во время конструирования объекта
    • Вызов переопределённых методов во время конструирования объекта

    Во избежание дублирования предупреждений можно включить только одну проверку одновременно.
