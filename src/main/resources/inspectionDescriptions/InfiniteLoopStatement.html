Сообщает о циклах for, while или do, которые могут завершиться только путем выброса исключения. Хотя такие конструкции могут быть корректными, они часто являются результатом ошибок программования.

Пример:


  for (;;) {
  }

Используйте опцию Игнорировать при размещении в Thread.run, чтобы исключить бесконечные циклы в методе Thread.run. Это может быть полезно для потоков-демонов.

Пример:


  new Thread(() -> {
    while (true) {
    }
  }).start();
