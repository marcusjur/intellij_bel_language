Сообщать о свойствах, не соответствующих соглашениям об именовании.

Проверяются следующие свойства:

  • Свойства верхнего уровня
  • Свойства в объектах и вспомогательных объектах

Вы можете указать желаемый шаблон в настройках проверки.

Рекомендуемые соглашения об именовании: должны начинаться с заглавной буквы, использовать camelCase и не содержать подчеркиваний.

Пример:


  // Свойство верхнего уровня
  val USER_NAME_FIELD = "UserName"
  // Свойство верхнего уровня, содержащее ссылку на синглтон-объект
  val PersonComparator: Comparator<Person> = /*...*/
  class Person {
    companion object {
      // Свойство во вспомогательном объекте
      val NO_NAME = Person()
    }
  }
