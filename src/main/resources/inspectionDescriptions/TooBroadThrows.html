Объявление в throws более общего исключения, чем фактически выбрасываемое методом.

Пример:


  public void createFile() throws Exception { // Предупреждение: 'throws Exception' слишком общий и скрывает 'IOException'
    File file = new File("pathToFile");
    file.createNewFile();
  }

После применения быстрого исправления:


  public void createFile() throws IOException {
    File file = new File("pathToFile");
    file.createNewFile();
  }

Настройка проверки:

  • Используйте поле Максимальное количество скрытых исключений для предупреждения, чтобы игнорировать исключения, которые скрывают больше других исключений, чем указанное количество.
  • Используйте опцию Предупреждать только о RuntimeException, Exception, Error или Throwable, чтобы проверка выдавала предупреждения только для самых распространенных исключений.
  • Используйте опцию Игнорировать исключения, объявленные в методах, переопределяющих библиотечные методы, чтобы игнорировать слишком общие объявления throws в методах, которые переопределяют библиотечные методы.
  • Используйте опцию Игнорировать исключения, которые скрывают другие исключения, но сами выбрасываются, чтобы игнорировать любые исключения, которые скрывают другие исключения, но всё ещё могут быть выброшены из тела метода и поэтому технически не являются слишком общими.
