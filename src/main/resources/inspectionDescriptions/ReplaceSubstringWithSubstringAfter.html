Сообщается, что вызов s.substring(s.indexOf(x)) может быть заменен на s.substringAfter(x).

Использование s.substringAfter(x) делает код более лаконичным.

Это быстрое исправление заменит вызов substring на substringAfter.

Пример:


  fun foo(s: String) {
      s.substring(s.indexOf('x'))
  }

После применения быстрого исправления:


  fun foo(s: String) {
      s.substringAfter('x')
  }
