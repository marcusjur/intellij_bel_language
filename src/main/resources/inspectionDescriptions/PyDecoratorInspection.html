Сообщает об использовании декораторов @classmethod или @staticmethod в методах вне класса.

Пример:


class State(object):
    @classmethod
    def my_state(cls, name):
        cls.name = name
@classmethod
def change_state(self):
    pass

Метод change_state не должен использовать декоратор @classmethod, или его следует переместить в объявление класса state.

При применении операции удаления декоратора код станет:


class State(object):
    @classmethod
    def my_state(cls, name):
        cls.name = name
def change_state(self):
    pass
