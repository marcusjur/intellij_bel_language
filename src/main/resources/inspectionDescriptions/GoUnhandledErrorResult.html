Сообщает о вызовах функций и методов, которые не обрабатывают результат типа error.

API таких функций подразумевает, что выполнение может завершиться неудачно и вернуть ошибку. Вызовы без обработки результата ошибки могут указывать на неправильное использование API.

Пример:


os.Remove("non-existing") // Ошибка игнорируется

После применения быстрого исправления Обработать ошибку:

err := os.Remove("non-existing") // Ошибка обрабатывается
if err != nil {
  return err
}
