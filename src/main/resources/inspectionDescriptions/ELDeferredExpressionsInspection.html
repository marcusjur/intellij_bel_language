Сообщает о динамических выражениях, где требуются отложенные выражения, или об отложенных выражениях, где требуются динамические выражения.

Пример:



<jsp:useBean id="exampleBean" scope="request" type="java.util.Date"/>
<h:outputText id="${exampleBean.day}" value="#{exampleBean.day}" /> // OK
<h:outputText id="#{exampleBean.day}" value="${exampleBean.day}" /> // Ошибка: атрибут `id` требует динамического выражения, `value` - отложенного
