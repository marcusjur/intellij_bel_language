Вызов конструктора Pair может быть заменен на инфиксный вызов функции to().

Явный вызов конструктора может создавать избыточность, особенно при многократном использовании. Замена вызова конструктора на to() делает код более читаемым и простым в обслуживании.

Пример:


  val countries = mapOf(
      Pair("France", "Paris"),
      Pair("Spain", "Madrid"),
      Pair("Germany", "Berlin")
  )

После применения быстрого исправления:


  val countries = mapOf(
      "France" to "Paris",
      "Spain" to "Madrid",
      "Germany" to "Berlin"
  )
