Сообщает о конфликтах между свойствами-расширениями и синтетическими свойствами, автоматически сгенерированными из методов Java get или set.

Такие свойства следует удалить или переименовать, чтобы избежать поломки кода при будущих изменениях в компиляторе.

Быстрое исправление удалит свойство-расширение.

Пример:


  val File.name: String
      get() = getName()

Быстрое исправление добавит аннотацию @Deprecated:


  @Deprecated("Is replaced with automatic synthetic extension", ReplaceWith("name"), level = DeprecationLevel.HIDDEN)
  val File.name: String
      get() = getName()
