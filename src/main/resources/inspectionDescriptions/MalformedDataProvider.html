Сообщает о ссылках на несуществующие или недоступные методы поставщика данных.

Пример:


public class InstanceDataProviderFromForeignClass {
  // Метод data() не существует в классе A
  @Test(dataProvider = "data", dataProviderClass = A.class)
  public void test() {
  }
}
class A { }

После применения быстрого исправления:


//Требуемый метод data() создан в классе A
class A {
  @DataProvider
  public Object[][] data() {
    return new Object[][]{};
  }
}
