Сообщает о незакрытых сокетных ресурсах. Эта проверка отслеживает сокетные ресурсы, включая java.net.Socket, java.net.DatagramSocket и java.net.ServerSocket.

По умолчанию проверка предполагает, что ресурс может быть закрыт любым методом, содержащим в своем названии 'close' или 'cleanup'.

Пример:


  byte[] getMessage(ServerSocket socket) throws IOException {
    Socket client = socket.accept(); //сокет не закрыт
    return client.getInputStream().readAllBytes();
  }

Проверка настраивается следующими параметрами:

  • Разрешено ли открывать сокеты внутри блока try. Такой стиль менее предпочтителен, так как он более многословен, чем открытие ресурса перед блоком try.
  • Может ли ресурс быть закрыт любым вызовом метода, которому этот ресурс передается в качестве аргумента.
