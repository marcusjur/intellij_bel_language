Сообщает о вызовах функции StringBuilder.append(CharArray, offset, len), которые должны быть заменены на вызовы функции StringBuilder.appendRange(CharArray, startIndex, endIndex) на платформе JVM.

Функция append ведет себя по-разному на платформах JVM, JS и Native, поэтому рекомендуется использовать функцию appendRange.

Пример:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          append(charArray, offset, len)
      }
  }

После применения быстрого исправления:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          appendRange(charArray, offset, offset + len)
      }
  }
