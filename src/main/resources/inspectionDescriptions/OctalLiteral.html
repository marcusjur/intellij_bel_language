Сообщает о восьмеричных целочисленных литералах. Некоторые стандарты кодования запрещают использование восьмеричных литералов, так как их легко перепутать с десятичными литералами.

Пример:


  int i = 015;
  int j = 0_777;

Для этой проверки доступны два различных быстрых исправления. После применения быстрого исправления Преобразовать восьмеричный литерал в десятичный код станет:


  int i = 13;
  int j = 511;

После применения быстрого исправления Удалить ведущий ноль для преобразования в десятичный код станет:


  int i = 15;
  int j = 777;
