Сообщает о non-public методах, на которые ссылается атрибут factory-method в элементе <bean>.

Пример:


public class MyFactory {
  private static FooBean createBean() {...}
  }
}


<beans>
  <bean class="MyFactory" factory-method="createBean"/>
  // Сообщение "Фабричный метод 'createBean' должен быть public"
</beans>
