java.time.temporal.ChronoField 및 java.time.temporal.ChronoUnit을 인수로 취하는 java.time 메서드 호출을 더 구체적인 메서드 호출로 바꿀 수 있는 경우 보고합니다.

예:


LocalTime localTime = LocalTime.now();
int minute = localTime.get(ChronoField.MINUTE_OF_HOUR);

빠른 수정 적용 후:


LocalTime localTime = LocalTime.now();
int minute = localTime.getMinute();
