Сообщает о переменных экземпляра, которые могут быть не инициализованы при создании объекта.

Пример:


  class Foo {
    public int bar;
    static { }
  }

Обратите внимание, что эта проверка использует очень консервативный алгоритм анализа потока данных и может ошибочно сообщать о неинициализованных переменных экземпляра. Переменные, помеченные как инициализованные, всегда будут инициализованы.

Используйте опцию Игнорировать примитивные поля, чтобы игнорировать неинициализованные примитивные поля.
