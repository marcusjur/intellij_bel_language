Сообщает о вложенных классах, объявленных без модификатора доступа (также известных как package-private).

Пример:


  public class Outer {
    static class Nested {}  // предупреждение
    class Inner {}          // предупреждение
    enum Mode {}            // предупреждение зависит от настроек
    interface I {}          // предупреждение зависит от настроек
  }

Настройка проверки:

  • Используйте опцию Игнорировать package-visible внутренние перечисления, чтобы игнорировать внутренние перечисления с уровнем доступа package-private.
  • Используйте опцию Игнорировать package-visible внутренние интерфейсы, чтобы игнорировать внутренние интерфейсы с уровнем доступа package-private.
