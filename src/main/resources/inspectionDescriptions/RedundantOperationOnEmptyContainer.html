Обнаружение избыточных операций с пустыми коллекциями, отображениями или массивами.

Итерация по пустым коллекциям, удаление элементов, сортировка и некоторые другие операции не имеют эффекта и могут быть удалены. Более того, они могут указывать на наличие ошибки.

Пример:


  if (numbers.isEmpty()){
    //ошибка из-за отсутствия отрицания
    int max = numbers.stream().max(Comparator.naturalOrder()).get();
    ...
  }
