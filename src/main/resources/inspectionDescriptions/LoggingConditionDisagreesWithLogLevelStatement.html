Сообщает об условиях is log enabled for в операторах if, где уровень проверки логования не соответствует уровню вызова логования.

Например: 


  if (LOG.isTraceEnabled()) {
    // Логирование на уровне debug, но проверка на уровне trace
    LOG.debug("some log message");
  }

Эта проверка поддерживает фреймворки логования java.util.logging, Log4j, Log4j2, Apache Commons Logging и SLF4J.
