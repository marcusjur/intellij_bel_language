Вызов arrayOf может быть заменен литералом массива [...].

Пример:

  annotation class MyAnnotation(val strings: Array<String>)
  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // Заменяемый 'arrayOf()'
  class MyClass

После применения быстрого исправления:

  annotation class MyAnnotation(val strings: Array<String>)
  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass
