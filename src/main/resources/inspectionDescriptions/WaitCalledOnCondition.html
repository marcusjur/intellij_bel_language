Сообщает о вызове wait() на объекте java.util.concurrent.locks.Condition. Это может быть ошибкой программования, вместо этого следует использовать один из вариантов метода await().

Пример:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

Правильный код должен выглядеть так:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
