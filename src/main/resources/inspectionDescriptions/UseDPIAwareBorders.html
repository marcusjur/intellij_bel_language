Сообщает об использовании javax.swing.border.EmptyBorder и JBUI.Borders.emptyXyz(), которое можно упростить.

Экземпляры EmptyBorder не учитывают DPI и могут вызвать проблемы с компоновкой интерфейса.

Быстрое исправление выполнит замену или упростит выражение с использованием JBUI.Borders.empty().

Пример:


// Неверно:
Border border1 = new EmptyBorder(1, 2, 3, 4);
Border border2 = new EmptyBorder(1, 2, 1, 2);
Border border3 = new EmptyBorder(1, 0, 0, 0);
// Верно:
Border border1 = JBUI.Borders.empty(1, 2, 3, 4);
Border border2 = JBUI.Borders.empty(1, 2);
Border border3 = JBUI.Borders.emptyTop(1);
