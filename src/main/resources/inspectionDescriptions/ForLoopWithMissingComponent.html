Сообщает о циклах for с отсутствующей инициализацией, условием или блоком обновления. Некоторые стили кодования запрещают такие циклы.

Пример:


  for (int i = 0;;i++) {
    //тело цикла
  }

Используйте опцию Игнорировать итерации по коллекциям, чтобы исключить циклы, использующие итераторы. Это стандартный способ итерации по коллекции в цикле for без блока обновления.
