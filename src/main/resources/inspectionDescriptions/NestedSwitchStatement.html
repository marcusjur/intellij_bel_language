Обнаружены вложенные операторы или выражения switch.

Вложенные операторы switch могут сделать код слишком сложным для понимания. Рекомендуется выделить их в отдельный метод.

Пример:


  int res = switch (i) {
            case 0 -> 0;
            default -> switch (i) {
                case 100 -> 0;
                default -> i;
            };
        };
