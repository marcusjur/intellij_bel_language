Сообщает о возможности замены вызовов Java-методов на эквивалентные функции Kotlin, например System.out.println().

Такая замена позволяет избавиться от зависимостей Java и сделать код более идиоматичным для Kotlin.

Это быстрое исправление заменяет вызовы методов Java на эквивалентные вызовы Kotlin.

Пример:


  import java.util.Arrays
  fun main() {
      val a = Arrays.asList(1, 3, null)
  }

После применения быстрого исправления:


  fun main() {
      val a = listOf(1, 3, null)
  }
