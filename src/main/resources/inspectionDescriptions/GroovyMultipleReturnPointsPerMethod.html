Обнаруживает методы с чрезмерным количеством точек возврата. Методы с множественными точками возврата могут быть сложными для понимания и рефакторинга.

Пример:


  int foo(int a) {
    if (a > 0) {
      return a
    }
    if (a < 0) return -a
    return 0
  }

Используйте указанное поле для установки максимально допустимого количества точек возврата в методе.
