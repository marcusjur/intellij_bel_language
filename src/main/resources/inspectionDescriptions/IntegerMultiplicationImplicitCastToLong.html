Сообщать о неявных преобразованиях в long при целочисленном умножении и операциях сдвига влево.

Пример:


  void f(int i) {
    long val = 65536 * i;
  }

После применения быстрого исправления код станет:


  void x(int i) {
    long val = 65536L * i;
  }

Пример:


  void f(int i) {
    long value = i << 24;
  }

После применения быстрого исправления код станет:


  void f(int i) {
    long value = (long) i << 24;
  }

Такие операции умножения часто указывают на ошибку, так как может произойти непреднамеренное переполнение со усечением результата. Преобразование целочисленного литерала int в литерал long (65536L) исправляет эту проблему.
