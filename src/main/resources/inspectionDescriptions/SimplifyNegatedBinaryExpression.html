Сообщает о возможности упрощения отрицания булевых выражений.

Быстрое исправление упрощает булево выражение.

Пример:


    fun test(n: Int) {
        !(0 == 1)
    }

После применения быстрого исправления:


    fun test(n: Int) {
        0 != 1
    }

Обратите внимание, что эта операция может изменить семантику кода при работе со значениями IEEE-754 NaN:


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

После применения быстрого исправления:


    fun main() {
        println(Double.NaN < 0) // false
    }
