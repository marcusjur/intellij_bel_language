Сообщает о методах, содержащих три или более отрицания. Такие методы могут быть сложными для понимания.

Пример:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

Без отрицаний метод становится более понятным:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

Настройка проверки:

  • Используйте опцию Игнорировать отрицания в методе 'equals()', чтобы отключить проверку в методах equals().
  • Используйте опцию Игнорировать отрицания в операторах 'assert', чтобы отключить проверку в операторах assert.
