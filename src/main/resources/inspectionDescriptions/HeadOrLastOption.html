headOption 또는 lastOption이 에뮬레이션된 경우를 보고합니다.

예:


  var seq: Seq[Int]
  if (seq.nonEmpty) Some(seq.head) else None
  seq.lift(0)

  if (seq.size == 0) None else seq.lastOption
  seq.lift(seq.size - 1)

빠른 수정 적용 후:


  var seq: Seq[Int]
  seq.headOption
  seq.headOption

  seq.lastOption
  seq.lastOption
