Сообщает о случаях, когда CASE и IF являются взаимозаменяемыми.

Пример (MySQL):

SELECT CASE
WHEN C1 IS NULL THEN 1
ELSE 0
END
FROM dual;

Вы можете сократить код, заменив структуру CASE на IF. Для этого примените быстрое действие Заменить на вызов 'IF'. Код будет преобразован следующим образом:

SELECT IF(C1 IS NULL, 1, 0)
FROM dual;

Чтобы преобразовать IF обратно в CASE, щёлкните на IF и примените быстрое действие Заменить на выражение CASE.
