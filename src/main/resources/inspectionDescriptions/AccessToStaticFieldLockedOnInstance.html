Сообщает о доступе к нестатическим static полям при синхронизации на this или полях экземпляра this.

Блокировка static полей на данных экземпляра не предотвращает изменение этого поля другими экземплярами, что может привести к непредвиденным условиям гонки.

Пример:


  static String test;
  public void foo() {
      synchronized (this) {
          System.out.println(test); // предупреждение
      }
  }

С помощью быстрого исправления можно игнорировать static поля определённого типа. Игнорируемыми типами можно управлять в настройках проверки.

В настройках проверки можно указать, какие классы следует игнорировать для static полей.
