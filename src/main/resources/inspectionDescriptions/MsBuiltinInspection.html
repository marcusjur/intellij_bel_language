Сообщать об усечении строковых аргументов в функции ISNULL.

Синтаксис ISNULL: ISNULL(check_expression, replacement_value).

Согласно документации ISNULL на docs.microsoft.com, если replacement_value длиннее, чем check_expression, значение replacement_value будет усечено.

Пример (Microsoft SQL Server):

DECLARE @name1 VARCHAR(2) = NULL;
DECLARE @name2 VARCHAR(10) = 'Example';
DECLARE @name3 VARCHAR(2) = 'Hi';
  -- `@name2` имеет тип VARCHAR(10) и будет усечен
SELECT ISNULL(@name1, @name2);
  -- `@name3` имеет тип VARCHAR(2) (как и `@name1`) и не будет усечен
SELECT ISNULL(@name1, @name3);
