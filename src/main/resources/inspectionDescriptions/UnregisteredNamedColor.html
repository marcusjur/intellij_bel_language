Сообщает об использовании JBColor.namedColor() с незарегистрированными ключами.

Все ключи должны быть зарегистрированы с соответствующими метаданными в *.themeMetadata.json.

Быстрое исправление предоставляет навигацию по всем известным файлам-кандидатам *.themeMetadata.json для добавления необходимых записей.

Дополнительную информацию смотрите в разделе Публикация метаданных темы в документации SDK.
