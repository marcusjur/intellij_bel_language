Проверка ошибок в литералах даты и времени. Эта проверка доступна в MySQL, Oracle, Db2 и H2.

Пример (MySQL):

SELECT TIME '10 -12:13:14' FROM dual;
SELECT TIME ' 12 : 13 : 14 ' FROM dual;
SELECT TIME '12 13 14' FROM dual;
SELECT TIME '12-13-14' FROM dual;
SELECT TIME '12.13.14' FROM dual;
SELECT TIME '12:13:' FROM dual;
SELECT TIME '12:13' FROM dual;
SELECT TIME '12:' FROM dual;

В этом примере даты не соответствуют стандартам литералов даты и времени MySQL. Поэтому они будут подсвечены. Для получения подробной информации о литералах даты и времени в MySQL см. Date and Time Literals на dev.mysql.com.

Следующие литералы даты и времени являются допустимыми для MySQL.

SELECT TIME '12:13:14' FROM dual;
SELECT TIME '12:13:14.555' FROM dual;
SELECT TIME '12:13:14.' FROM dual;
SELECT TIME '-12:13:14' FROM dual;
SELECT TIME '10 12:13:14' FROM dual;
SELECT TIME '-10 12:13:14' FROM dual;
