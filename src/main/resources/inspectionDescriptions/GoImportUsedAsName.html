Сообщает об объявлениях переменных, аргументов или функций, которые перекрываются с импортованными идентификаторами.

Хотя такие объявления являются синтаксически корректными, они могут сделать экспортуемые идентификаторы пакета недоступными после объявления или создать путаницу при чтении кода.

Пример:

import "fmt"
import _ "fmt"
import iio "io"
func _() {
 fmt.Println("demo")
 demo := true
 _, _ = iio.EOF, demo
}
func demo() (int, int) {
 return 1, 2
}
func _() {
 _, _ = iio.EOF, demo
 fmt := "demo"
 iio := 1
 _, _ = iio, fmt
 a, _ := demo()
 _ = a
}

Имена переменных fmt и iio конфликтуют с именами импортованных пакетов. Во избежание путаницы в коде рекомендуется переименовать эти переменные.
