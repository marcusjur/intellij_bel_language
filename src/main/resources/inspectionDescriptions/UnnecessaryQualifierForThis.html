Обнаружение необязательных квалификаторов this или super.

Использование квалификаторов this или super для устранения неоднозначности в ссылках кода может стать излишним после автоматического рефакторинга и должно быть удалено для улучшения читаемости.

Пример:


  class Foo {
    void foo() {}
  }
  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

После применения быстрого исправления:


  class Foo {
    void foo() {}
  }
  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
