비교 .length 또는 .size를 보고하고 .sizeIs로 바꿀 것을 제안합니다.

예:


  var seq: Seq[Int]
  seq.length == n
  seq.length != n
  seq.length < n
  seq.length > n

  seq.size == n
  seq.size != n
  seq.size < n
  seq.size > n

빠른 수정 적용 후:


  var seq: Seq[Int]
  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n

  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n
