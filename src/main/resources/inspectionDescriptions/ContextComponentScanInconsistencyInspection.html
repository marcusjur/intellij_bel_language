Обнаружено несоответствие в определении <filter> внутри контекста XML-приложения <component-scan>.

Пример:


  public class NonAnnotationTypeClass {}


  <beans>
    <ctx:component-scan base-package="com.my.company">
      <ctx:exclude-filter expression="com.my.company.anno.NonAnnotationTypeClass" type="annotation"/>
    </ctx:component-scan>
  </beans>

В этом примере NonAnnotationTypeClass должен быть типом аннотации.
