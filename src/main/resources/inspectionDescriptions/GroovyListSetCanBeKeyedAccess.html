Сообщает о вызовах метода java.util.List.set(). Такие вызовы могут быть заменены более короткой и явной формой доступа по индексу.

Пример:


def list = ["foo"]
list.set(0, "bar") // list.set(0, "bar") можно заменить на list[0] = "bar"

После применения быстрого исправления:


def list = ["foo"]
list[0] = "bar"
