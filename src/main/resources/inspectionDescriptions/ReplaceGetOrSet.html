Сообщает о явных вызовах функций get или set, которые могут быть заменены оператором индексации [].

Kotlin позволяет создавать пользовательские реализации предопределённого набора операторов для типов. Чтобы перегрузить оператор, необходимо отметить соответствующую функцию модификатором operator: 


  operator fun get(index: Int) {}
  operator fun set(index: Int, value: Int) {}
Эти функции соответствуют оператору индексации.

Пример:

  class Test {
      operator fun get(i: Int): Int = 0
  }
  fun test() {
      Test().get(0) // Заменяемый 'get()'
  }

После применения быстрого исправления:

  class Test {
      operator fun get(i: Int): Int = 0
  }
  fun test() {
      Test()[0]
  }
