Обнаружено поле массива с модификаторами public static.

Такие поля обычно используются для хранения массивов констант. Тем не менее, они представляют угрозу безопасности, поскольку их содержимое может быть изменено, даже если поле объявлено как final.

Пример:


  public static String[] allowedPasswords = {"foo", "bar"};
