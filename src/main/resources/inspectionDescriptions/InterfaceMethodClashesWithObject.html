Сообщает о методах интерфейса, которые конфликтуют с protected методами clone() и finalize() класса java.lang.Object.

В интерфейсе эти методы могут быть объявлены с типами возврата, несовместимыми с методами java.lang.Object. Классы, реализующие такой интерфейс, не могут быть скомпилированы. Хотя создание лямбда-выражений на основе такого интерфейса все еще возможно, это не рекомендуется делать.

Пример:


  // // Предупреждение: этот интерфейс не может быть реализован никаким классом,
  // только лямбда-выражением или ссылкой на метод
  interface MyInterface {
    double clone();
  }
