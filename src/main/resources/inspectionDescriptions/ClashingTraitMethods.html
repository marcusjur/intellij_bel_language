Сообщает о классах, реализующих два или более интерфейса (содержащих методы с одинаковой сигнатурой).

Вызов таких методов может привести к неожиданным результатам.

Быстрое исправление добавит явное переопределение метода.

Пример:


  trait T1 {
      def foo() {}
  }
  trait T2 {
      def foo() {}
  }
  class X implements T1, T2 {}
  // Будет вызван T2.foo()
  new X().foo()

После применения быстрого исправления:


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }
