Обнаружены члены с модификатором final и protected в классе.

Поскольку final класс не может быть унаследован, использование модификатора protected для методов может вводить в заблуждение. Рекомендуется объявлять такие члены как private или с пакетной видимостью.

Пример:

record Bar(int a, int b) {
  protected int sum() { 
     return a + b;
  }
}

После применения быстрого исправления:

record Bar(int a, int b) {
  int sum() { 
     return a + b;
  }
}

Как показано в примере, класс может быть помечен как final явно или неявно.
