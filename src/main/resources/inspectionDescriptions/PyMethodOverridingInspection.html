Сообщает о несоответствии сигнатур переопределённых методов.

Пример:


class Book:
    def add_title(self):
        pass
class Novel(Book):
    def add_title(self, text):
        pass

Параметры метода add_title в классе Novel не соответствуют сигнатуре метода, определённой в классе Book. ИСР предлагает исправить это с помощью рефакторинга "Изменить сигнатуру".
