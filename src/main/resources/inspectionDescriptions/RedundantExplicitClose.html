Сообщает о ненужных вызовах close() в конце блока try-with-resources и предлагает их удалить.

Пример:


  try(MyAutoCloseable ac = new MyAutoCloseable()) {
    foo();
    ac.close();
  }

После применения быстрого исправления:


  try(MyAutoCloseable ac = new MyAutoCloseable()) {
    foo();
  }

Новое в версии 2018.1
