Сообщать о методах с модификатором final в классах с модификатором final.

Поскольку final класс нельзя наследовать, пометка метода как final является избыточной и может вводить в заблуждение.

Пример:

record Bar(int a, int b) {
  public final int sum() { 
     return a + b;
  }
}

После применения быстрого исправления:

record Bar(int a, int b) {
  public int sum() { 
     return a + b;
  }
}

Как показано в этом примере, класс может быть помечен как final явно или неявно.
