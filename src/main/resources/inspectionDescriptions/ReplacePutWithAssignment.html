Вызов функции map.put может быть заменен оператором индексации ([]).

Использование синтаксического сахара делает код более лаконичным.

Это быстрое исправление заменит вызов put на присваивание.

Пример:


  fun foo(map: MutableMap<Int, String>) {
      map.put(42, "foo")
  }

После применения быстрого исправления:


  fun foo(map: MutableMap<Int, String>) {
      map[42] = "foo"
  }
