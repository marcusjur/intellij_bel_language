Проверка размера Collections/Array/String должна быть заменена на isNotEmpty().

Использование isNotEmpty() делает код более лаконичным.

Это быстрое исправление заменит проверку размера на isNotEmpty().

Пример:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size > 0
  }

После применения быстрого исправления:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isNotEmpty()
  }
