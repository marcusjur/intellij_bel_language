Сообщение о возможности упрощения цепочки вызовов с помощью оператора безопасной навигации (&.). Цепочка вызовов — это распространенный паттерн для проверки nil в каждом компоненте вызова.

Пример:

# Плохая практика
if user && user.profile && user.profile.thumbnails
end
Выражение можно упростить с помощью &., используя быстрое исправление. После применения быстрого исправления результат будет выглядеть так: 
# Хорошая практика
if user&.profile&.thumbnails
end
