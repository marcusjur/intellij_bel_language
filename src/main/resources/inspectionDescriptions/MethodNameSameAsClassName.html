Сообщает о методе с тем же именем, что и его класс. Хотя Java допускает такое именование, по соглашению оно зарезервировано для определения конструкторов. Использование такого имени для метода может быть ошибкой или плохой практикой.

Пример:


  class MyClass {
    int val;
    // Метод MyClass имеет то же имя, что и содержащий его класс.
    // Возможно, 'void' был добавлен по ошибке.
    void MyClass(int val) {
      this.val = val;
    }
  }

При необходимости метод можно преобразовать в конструктор с помощью быстрого исправления:


  class MyClass {
    int val;
    MyClass(int val) {
      this.val = val;
    }
  }

Метод также можно переименовать с помощью другого быстрого исправления.
