Сообщает о конструкторах с ненулевыми параметрами, ссылающимися на самих себя.

Такие конструкторы никогда не позволяют создать экземпляр класса.

Быстрое исправление преобразует тип параметра в допускающий null.

Пример:


  class SelfRef(val ref: SelfRef)

После применения быстрого исправления:


  class SelfRef(val ref: SelfRef?)
