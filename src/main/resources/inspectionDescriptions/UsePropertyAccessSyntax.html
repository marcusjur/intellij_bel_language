Сообщает о возможности замены вызовов Java-методов get и set на синтетические свойства Kotlin.

Быстрое исправление с использованием синтаксиса доступа к свойствам доступно для автоматического исправления кода.

Пример:


  // Java:
  public class JavaClassWithGetter {
      private final String expr = "result";
      // ...
      public String getExpr() {
          return expr;
      }
  }


  // Kotlin:
  fun test(j: JavaClassWithGetter) {
      // ...
      j.getExpr() // <== Быстрое исправление упростит выражение до 'j.expr'
  }
