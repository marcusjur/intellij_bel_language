Сообщается, что вызов s.substring(0, s.length - x) может быть заменен на s.dropLast(x).

Использование соответствующей функции делает код более лаконичным.

Это быстрое исправление заменит вызов substring на dropLast.

Пример:


  fun foo(s: String) {
      s.substring(0, s.length - 5)
  }

После применения быстрого исправления:


  fun foo(s: String) {
      s.dropLast(5)
  }
