Сообщает о небезопасных последовательностях, созданных с помощью двоеточия (:), и предлагает заменить их безопасными вызовами.

Пример:


    a <- 1:length(c)
    b <- 1:nrow(data.frame())

После применения быстрого исправления код будет выглядеть так: 


    a <- seq_along(c)
    b <- seq_len(nrow(data.frame()))
