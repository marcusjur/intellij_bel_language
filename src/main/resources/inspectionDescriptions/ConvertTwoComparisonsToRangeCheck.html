Сообщает о двух последовательных сравнениях, которые можно преобразовать в проверку диапазона.

Проверка диапазона упрощает код за счет устранения повторения условия проверки.

Пример:


  fun checkMonth(month: Int): Boolean {
      return month >= 1 && month <= 12
  }

Быстрое исправление заменит проверку на основе сравнений на проверку диапазона:


  fun checkMonth(month: Int): Boolean {
      return month in 1..12
  }
