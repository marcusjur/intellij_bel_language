Сообщает о методах тестования TestNG, имена которых слишком короткие, слишком длинные или не соответствуют указанному шаблону регулярного выражения.

Когда эта проверка включена, проверка соглашения об именовании методов экземпляра автоматически игнорирует тестовые методы TestNG.

Используйте поля для указания минимальной длины, максимальной длины и ожидаемого регулярного выражения для имен тестовых методов TestNG. Укажите 0, чтобы отключить проверку длины имени. Регулярное выражение указывается в стандартном формате java.util.regex.
