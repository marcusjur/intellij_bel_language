Сообщает о вызовах методов из классов junit.framework.Assert, org.junit.Assert или org.junit.Assume в тестах JUnit 5.

Хотя эти тесты работают корректно, переход на org.junit.jupiter.api.Assertions/org.junit.jupiter.api.Assumptions поможет избежать зависимостей от устаревших версий JUnit.

Пример:


  import org.junit.Assert;
  import org.junit.jupiter.api.Test;
  public class MyTest {
    @Test
    public void simpleTest() {
      Assert.assertEquals(4, 2 + 2);
    }
  }

После применения быстрого исправления:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;
  public class MyTest {
    @Test
    public void simpleTest() {
      Assertions.assertEquals(4, 2 + 2);
    }
  }
