Сообщение об ошибках конфигурации в XML-определении <flow> для Spring Webflow.

  • Неразрешённые ссылки на Bean
  • Отсутствующие обязательные теги или атрибуты
  • Неверные типы атрибутов
  • Несоответствие атрибутов типа enum
  • Неверные типы для ссылок на Bean

Пример:


  <flow
    start-state="unknown-state" <!-- Cannot resolve symbol 'unknown-state' ->
    parent="unknownParentFlow,another-flow" <!-- Не удаётся найти поток 'unknownParentFlow' -->>
    <input name="" // Value must not be empty
         type="foo.com.UnknownType" <!--Cannot resolve class 'UnknownType'-->
         value="42"/>
    <exception-handler bean="dummyBean"; <!--Bean должен быть типа 'org.springframework.webflow.engine.FlowExecutionExceptionHandler' -->
    <view-state id="validFlowButInvalidState_forParent"
           parent="another-flow#unknownParentState"/> <!-- Не удаётся разрешить родительское состояние 'unknownParentState' -->
    <bean-import resource="foo.xml" /> <!-- Не удаётся найти файл 'foo.xml' -->
    <bean-import resource="highlighting-20-wrong-flow.xml"/> <!-- Недействительный файл spring.xml -->
  </flow>
