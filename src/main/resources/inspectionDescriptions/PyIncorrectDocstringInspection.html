Сообщает о несоответствующих параметрах в docstring. Например, b подсвечивается, поскольку в функции add нет такого параметра.


    def add(a, c):
    """
    @param a:
    @param b:
    @return:
    """
    pass

Если в docstring не упоминается ни один из параметров, проверка не будет предупреждать об их отсутствии:


def mult(a, c):
    """
    @return:
    """
    pass
