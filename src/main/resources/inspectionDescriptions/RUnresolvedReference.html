Сообщать о необъявленных символах и ссылках на функции.

Пример:


my_fahrenheit_to_celsius(40)

Функция my_fahrenheit_to_celsius не определена, поэтому ИСР показывает предупреждение. Необходимо добавить отсутствующее объявление.

Возможное решение: 


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((f_temp - 32) * 5 / 9)
}
my_fahrenheit_to_celsius(40)
