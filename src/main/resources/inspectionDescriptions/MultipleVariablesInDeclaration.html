Сообщает о множественном объявлении переменных в одном выражении и предлагает создать отдельные объявления для каждой переменной.

Некоторые стандарты кодования запрещают такие объявления.

Пример:


  int x = 1, y = 2;

После применения быстрого исправления:


  int x = 1;
  int y = 2;

Настройка проверки:

  • Используйте опцию Игнорировать объявления в циклах 'for', чтобы пропускать множественные объявления переменных в инициализации циклов 'for', например: 
    
  for (int i = 0, max = list.size(); i > max; i++) {}
  • Используйте опцию Предупреждать только о разных размерностях массивов в одном объявлении, чтобы получать предупреждения только при объявлении переменных с разными размерностями массивов в одном выражении, например: 
    
  String s = "", array[];
