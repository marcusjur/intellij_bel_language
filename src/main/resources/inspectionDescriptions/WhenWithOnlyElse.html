Сообщает о выражении when, которое имеет только одну ветку else и может быть упрощено.

Упростить выражение доступно как быстрое исправление для автоматического исправления кода.

Пример:


  fun redundant() {
      val x = when { // <== Избыточно, быстрое исправление упростит выражение when до "val x = 1"
          else -> 1
      }
  }
