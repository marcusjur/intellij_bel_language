Сообщает о зарезервированных словах JavaScript, используемых в качестве имен. Спецификация JavaScript зарезервировала множество слов, которые в настоящее время не используются как ключевые слова. Если будущие версии JavaScript начнут использовать эти слова как ключевые, использование их в качестве идентификаторов может нарушить работу кода.
