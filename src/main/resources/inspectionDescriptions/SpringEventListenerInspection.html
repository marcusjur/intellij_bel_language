Сообщение о некорректном методе с аннотацией @EventListener.

Пример:


  @Configuration @ComponentScan
  open class Config
  data class MyEvent(val string: String)
  @Component
  class LogComponent {
    @EventListener // Метод с аннотацией @EventListener должен быть public
    private fun logCommands(commandName: MyEvent) {}
    @EventListener // Метод должен иметь не более одного параметра
    fun processCommand(commandName: MyEvent, text: String) {}
  }
