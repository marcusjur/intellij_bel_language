Сообщает об использовании расширенного оператора for.

Пример:


  for (int x: Arrays.asList(1, 2, 3)) {
    System.out.println(x);
  }

После применения быстрого исправления:


  for (Iterator<Integer> iterator = Arrays.asList(1, 2, 3).iterator(); iterator.hasNext(); ) {
    final int x = iterator.next();
    System.out.println(x);
  }

Расширенный оператор for появился в Java 5. Эта проверка помогает обеспечить обратную совместимость с более ранними версиями Java.
