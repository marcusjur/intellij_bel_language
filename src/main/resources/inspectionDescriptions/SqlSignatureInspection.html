Сообщать о проблемах с сигнатурами встроенных функций.

Проверка будет выявлять ошибки в количестве аргументов, недопустимые ключевые слова, неверные типы данных и другие проблемы.

Пример (MySQL):

CREATE TABLE foo (a INT, b INT, c INT)
SELECT IFNULL() FROM foo; -- ошибка
SELECT IFNULL(a) FROM foo; -- ошибка
SELECT IFNULL(a, b) FROM foo; -- верно
SELECT IFNULL(a, b, c) FROM foo; -- ошибка

В MySQL функция IFNULL() принимает строго два аргумента. Поэтому только запрос SELECT IFNULL(a, b) FROM foo; является правильным.
