Сообщать об ошибках определения управляемых Bean-классов в следующих случаях:
  • Класс не является final
  • Класс является abstract
  • Bean является нестатическим внутренним классом

Пример:



  @ManagedBean
  public final class JavaManagedBean { // Ошибка: управляемый Bean не должен быть final
    ...
  }
