Сообщать о полях внутреннего класса, которые имеют такое же имя, как поля окружающего класса. Такое именование может привести к случайному использованию поля внутреннего класса в ситуациях, когда предполагалось использование одноименного поля окружающего класса.

Рекомендуется переименовать поле внутреннего класса с помощью быстрого исправления.

Пример:


  class Outer {
    private String name;
    class Inner {
      private String name;
    }
  }

Используйте эту опцию, чтобы выбрать, должна ли проверка сообщать обо всех конфликтах имен или только о конфликтах с полями, видимыми во внутреннем классе.
