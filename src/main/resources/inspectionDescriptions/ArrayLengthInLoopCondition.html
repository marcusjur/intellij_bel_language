Сообщает об обращениях к свойству .length массива в условиях циклических операторов. В средах с сильно ограниченными ресурсами такие вызовы могут негативно влиять на производительность.

Эта проверка применима для Java ME и других сред с сильно ограниченными ресурсами. Необдуманное применение этой проверки может отрицательно повлиять на читаемость и структуру кода.

Пример:


  void foo(Object[] x) {
    for (int i = 0; i < x.length; i++) { /**/ }
  }
