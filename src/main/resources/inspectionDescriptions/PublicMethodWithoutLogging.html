Сообщает о любых публичных методах, которые не содержат операторов логования. Эта проверка не распространяется на простые геттеры и сеттеры.

Например:


  public class Crucial {
    private static final Logger LOG = LoggerFactory.getLogger(Crucial.class);
    public void doImportantStuff() {
      // Для этого метода будет выдано предупреждение
    }
    public void doOtherStuff() {
      LOG.info("do other stuff");
    }
  }

Используйте таблицу ниже для указания имен классов логгера. Проверка будет сообщать о публичных методах, которые не используют методы экземпляров указанных классов.
