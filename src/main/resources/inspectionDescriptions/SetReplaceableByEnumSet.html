Сообщает об инициализации объекта java.util.Set для перечислений. Такой объект Set может быть заменен на объект java.util.EnumSet.

По сравнению с другими коллекциями, реализация EnumSet более эффективна, так как использует битовый вектор в качестве базовой структуры данных. Это быстрое исправление позволяет заменить инициализатор на вызов EnumSet.noneOf(). Быстрое исправление недоступно, когда тип переменной является подклассом Set.

Пример:


  enum MyEnum { FOO, BAR; }
  Set<MyEnum> enums = new HashSet<MyEnum>();

После применения быстрого исправления:


  enum MyEnum { FOO, BAR; }
  Set<MyEnum> enums = EnumSet.noneOf(MyEnum.class);
