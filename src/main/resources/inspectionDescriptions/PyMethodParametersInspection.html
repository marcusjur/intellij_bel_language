Сообщает об отсутствии первого параметра метода, обычно называемого self.

Пример:


class Movie:
   def show():
       pass

После применения быстрого исправления код станет:


class Movie:
   def show(self):
       pass

Проверка также сообщает о проблемах именования в методах класса.

Пример:


class Movie:
    @classmethod
    def show(abc):
        pass

Поскольку первый параметр метода класса должен быть cls, ИСР предлагает быстрое исправление для его переименования.
