Сообщает о избыточных параметрах метода, которые можно заменить локальными переменными.

Если все локальные использования параметра следуют непосредственно за присваиванием значения этому параметру, то параметр можно удалить и заменить его использование локальной переменной. Такие параметры бессмысленны, поскольку переданные им значения перезаписываются. Эта проблема обычно является результатом рефакторинга.

Пример:


  void test(int p) {
    p = 1;
    System.out.print(p);
  }

После применения быстрого исправления:


  void test() {
    int p = 1;
    System.out.print(p);
  }
