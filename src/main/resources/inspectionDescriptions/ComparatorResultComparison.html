Сообщает о сравнении результата вызова Comparator.compare() или Comparable.compareTo() с ненулевой константой. По соглашению, эти методы могут возвращать любое целое число (не только -1, 0 или 1), поэтому сравнение с конкретными числами считается плохой практикой. Некоторые широко используемые методы сравнения (например, String.compareTo()) фактически возвращают значения за пределами диапазона [-1..1], и такие сравнения могут привести к некорректному поведению программы.

Пример:


  void validate(String s1, String s2) {
    // Некорректное сравнение с 1
    if (s1.compareTo(s2) == 1) {
      throw new IllegalArgumentException("Incorrect order");
    }
  }

После применения быстрого исправления:


  void validate(String s1, String s2) {
    if (s1.compareTo(s2) > 0) {
      throw new IllegalArgumentException("Incorrect order");
    }
  }
