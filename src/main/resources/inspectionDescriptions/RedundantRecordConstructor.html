Сообщает о избыточных конструкторах, объявленных в Java-записях.

Пример 1:


  record Point(int x, int y) {
    public Point {} // можно удалить
  }
  record Point(int x, int y) {
    public Point(int x, int y) { // можно удалить
      this.x = x;
      this.y = y;
    }
  }

Быстрое исправление удалит избыточный конструктор.

Пример 2:


   // можно преобразовать в компактный конструктор
  record Range(int from, int to) {
    public Range(int from, int to) {
      if (from > to) throw new IllegalArgumentException();
      this.from = from;
      this.to = to;
    }
  }

Быстрое исправление преобразует этот код в компактный конструктор.
