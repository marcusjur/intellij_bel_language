Вызов String.format может быть заменен строковым шаблоном.

Использование строковых шаблонов делает код более читаемым.

Это быстрое исправление заменит вызов на строковый шаблон.

Пример:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

После применения быстрого исправления:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
