Сообщение о проблеме при создании объекта класса VIRTUAL.

Пример:


setClass('MyClass', slots = c('aa', 'bb'), contains = "VIRTUAL")
new('MyClass')

Наиболее распространенное использование класса VIRTUAL — это объединение классов, то есть виртуальный класс, определяемый при вызове setClassUnion() вместо setClass().
