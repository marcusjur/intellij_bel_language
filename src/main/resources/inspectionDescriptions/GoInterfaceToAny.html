Сообщает об использовании пустого интерфейса в качестве типа или ограничения типа.

Пустой интерфейс представляет собой множество всех типов. В Go 1.18 для interface{} был введен более явный псевдоним any, который эквивалентен interface{} во всех аспектах.

Проверка срабатывает, если пустой интерфейс используется как тип или ограничение типа. Проверка не срабатывает для псевдонимов пустого интерфейса и интерфейсов, которые только встраивают другие пустые интерфейсы.

Рекомендуется использовать более явный псевдоним any.
