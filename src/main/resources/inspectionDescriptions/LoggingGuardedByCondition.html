Обнаружение вызовов логования, окруженных защитными условиями. Эта проверка может использоваться для настройки пользовательского стиля кода.

Пример:


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

После применения быстрого исправления:


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

Эта проверка поддерживает фреймворки логования Log4j2 и SLF4J (за исключением построителей).
