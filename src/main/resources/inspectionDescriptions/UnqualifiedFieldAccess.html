Сообщает о доступе к полям, не имеющим квалификатора this или других квалификаторов.

Некоторые стили кодования требуют, чтобы весь доступ к полям был квалифицирован во избежание путаницы с локальными переменными или параметрами.

Пример:


  class Foo {
    int foo;
    void bar() {
      foo += 1;
    }
  }

После применения быстрого исправления:


  class Foo {
    int foo;
    void bar() {
      this.foo += 1;
    }
  }
