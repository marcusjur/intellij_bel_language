Сообщает о декларации непроверяемых исключений (java.lang.RuntimeException или одного из его подклассов) в секции throws метода.

Объявление непроверяемых исключений не является обязательным и может быть удалено или перемещено в тег Javadoc @throws.

Пример:


  public class InvalidDataException extends RuntimeException {}
  class TextEditor {
    void readSettings() throws InvalidDataException {} // Предупреждение: непроверяемое исключение 'InvalidDataException' объявлено в секции 'throws'
  }
