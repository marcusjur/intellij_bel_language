Сообщает о проверяемых исключениях, которые указаны как ожидаемые в TestNG-методе, но никогда не выбрасываются в теле метода.

Пример:


  @Test(expectedExceptions = Exception.class) // Предупреждение: ожидаемое исключение 'Exception' не выбрасывается
  public void testEngineIsRunning() {
    assertTrue(engine.isRunning());
  }
