Сообщает о модификаторах видимости, которые соответствуют модификаторам по умолчанию для элементов (для большинства элементов это final, а для членов с override это open).

Пример:


  final class Foo
  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }

После применения быстрого исправления:


  class Foo
  open class Bar : Comparable<Bar> {
      override fun compareTo(other: Bar): Int = 0
  }
