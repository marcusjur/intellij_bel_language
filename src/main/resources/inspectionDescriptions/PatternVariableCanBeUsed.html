Сообщение предлагает заменить объявление локальной переменной на переменную шаблона, которая обычно более компактна.

Пример:


  if (obj instanceof String) {
    String str = (String) obj;
    System.out.println(str);
  }

Можно заменить на:


  if (obj instanceof String str) {
    System.out.println(str);
  }

Новое в версии 2020.1
