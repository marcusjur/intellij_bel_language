Сообщать о неразрешённых ссылках в комментариях Javadoc.

В следующем примере параметр someParam будет подсвечен, так как он отсутствует:


class A {
    /**
     * @param someParam description
     **/
    void foo() {
    }
}

Отключите опцию Сообщать о недоступных символах, чтобы игнорировать пометки об отсутствующих параметрах методов, классах, полях и методах.
