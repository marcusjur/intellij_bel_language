Сообщение о сравнении двух массивов с помощью метода equals().

Вызов метода equals() для массивов сравнивает идентичность, что эквивалентно использованию оператора ==. Используйте Arrays.equals() для сравнения содержимого двух массивов или Arrays.deepEquals() для сравнения содержимого многомерных массивов.

Пример:


  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

После применения быстрого исправления:


  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }
