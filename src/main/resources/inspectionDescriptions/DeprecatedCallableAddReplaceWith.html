Сообщает о функциях и свойствах, помеченных как устаревшие (deprecated), в аннотации kotlin.deprecated которых отсутствует аргумент kotlin.ReplaceWith. ИСР предложит добавить его на основе их реализации.

Kotlin предоставляет аргумент ReplaceWith для автоматической замены устаревших объявлений. Рекомендуется использовать этот аргумент для исправления проблем с устаревшим кодом.

Пример:


  @Deprecated("Use refined() instead.")
  fun deprecated() = refined()
  fun refined() = 42

Быстрое исправление добавит аргумент ReplaceWith():


  @Deprecated("Use refined() instead.", ReplaceWith("refined()"))
  fun deprecated() = refined()
  fun refined() = 42
