Сообщает об использовании тернарного оператора и предлагает преобразовать его в конструкцию if/else.

Некоторые стандарты кодования запрещают использование тернарного оператора.

Пример:


  Object result = (condition) ? foo() : bar();

После применения быстрого исправления:


  Object result;
  if (condition) {
    comp = foo();
  }
  else {
    comp = bar();
  }

Настройка проверки:

Используйте опцию Игнорировать для простых присваиваний и возвратов, чтобы игнорировать простые присваивания и возвраты, разрешая следующие конструкции:


  String s = (foo == null) ? "" : foo.toString();

Используйте опцию Игнорировать в местах, где невозможно использовать if, чтобы игнорировать условные выражения в контекстах, где автоматическая замена на if невозможна (например, когда условное выражение используется в вызове конструктора super()).
