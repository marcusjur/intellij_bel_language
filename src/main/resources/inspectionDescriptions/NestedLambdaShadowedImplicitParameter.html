Обнаружены вложенные лямбда-выражения со скрытыми неявными параметрами.

Пример:


fun foo(listOfLists: List<List<String>>) {
  listOfLists.forEach {
    it.forEach {
      println(it)
    }
  }
}

После применения быстрого исправления:


fun foo(listOfLists: List<List<String>>) {
  listOfLists.forEach {
    it.forEach { it1 ->
      println(it1)
    }
  }
}
