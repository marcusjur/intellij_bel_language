Сообщает об использовании предложения ORDER BY в представлениях, встроенных функциях, производных таблицах, подзапросах и общих табличных выражениях без использования TOP, OFFSET или FOR XML.

Подробную информацию об использовании ORDER BY см. в разделе SELECT - предложение ORDER BY (Transact-SQL) на docs.microsoft.com.

Пример (Microsoft SQL Server):

CREATE TABLE foo (a INT NOT NULL, b  INT NOT NULL);
SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89
ORDER BY b) ALIAS;

ORDER BY в подзапросе будет подсвечен как ошибка. Вы можете добавить TOP, OFFSET или FOR XML в подзапрос. Или используйте быстрое исправление Удалить элемент, чтобы удалить часть ORDER BY.

После применения быстрого исправления:

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89) ALIAS;
