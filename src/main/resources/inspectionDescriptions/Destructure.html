Сообщает о объявлениях, которые могут быть деструктурированы.

Пример:


  data class My(val first: String, val second: Int, val third: Boolean)
  fun foo(list: List<My>) {
      list.forEach { my ->
          println(my.second)
          println(my.third)
      }
  }

Быстрое исправление деструктурирует объявление и вводит новые переменные с именами из соответствующего класса:


  data class My(val first: String, val second: Int, val third: Boolean)
  fun foo(list: List<My>) {
      list.forEach { (_, second, third) ->
          println(second)
          println(third)
      }
  }
