Рекомендуется выделить фрагмент кода в отдельный метод для улучшения читаемости кода. Эта проверка использует несколько эвристик для выбора подходящих кандидатов на выделение, включая следующие критерии.
  • Выделяемый фрагмент не содержит нелокальных потоков управления
  • Выделяемый фрагмент имеет ровно одну выходную переменную
  • Отсутствуют похожие выходные переменные внутри и снаружи выделяемого фрагмента
  • Выделяемый фрагмент имеет небольшое количество входных параметров (по умолчанию не более трех; настраивается в параметрах проверки)
  • Выделяемый фрагмент не меньше заданной длины (по умолчанию 500 символов), но не превышает 60% от размера содержащего его метода
