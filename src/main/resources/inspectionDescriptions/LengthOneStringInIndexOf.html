Сообщает об использовании односимвольных строк в качестве аргументов при вызовах String.indexOf() и String.lastIndexOf().

Рекомендуется использовать быструю подстановку для замены таких строковых литералов на эквивалентные символьные литералы для повышения производительности.

Пример:


  return s.indexOf("x");

После применения быстрой подстановки:


  return s.indexOf('x');
