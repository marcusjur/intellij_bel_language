Сообщает о модифицируемых public static полях коллекций.

Несмотря на то, что такие поля часто используются для хранения наборов констант, они представляют угрозу безопасности, поскольку их содержимое может быть изменено даже если поле объявлено как final.

Пример:


  public static final List<String> EVENTS = new ArrayList<>();

В таблице раздела Параметры можно указать методы, возвращающие неизменяемые коллекции. Поля коллекций public static, инициализованные с помощью этих методов, не будут отмечаться.
