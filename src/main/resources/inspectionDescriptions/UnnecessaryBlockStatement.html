Сообщает о семантически избыточных блоках кода, которые могут быть заменены их содержимым.

Блоки кода, которые являются телом операторов if, do, while или for, не будут отмечены этой проверкой.

Пример:


  void foo() {
    { // Избыточно
      int result = call();
      analyze(result);
    } // Избыточно
  }

Настройка проверки:

Используйте опцию Игнорировать ветви оператора 'switch', чтобы исключить из проверки блоки кода, используемые в качестве ветвей оператора switch.
