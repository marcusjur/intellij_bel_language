Сообщает о статических полях типа Collection. Хотя это не обязательно является проблемой, статические коллекции часто приводят к утечкам памяти и поэтому запрещены некоторыми стандартами кодования.

Пример:


  public class Example {
    static List<String> list = new ArrayList<>();
  }

Настройка проверки:

  • Используйте опцию Игнорировать слабые статические коллекции или отображения, чтобы игнорировать поля типа java.util.WeakHashMap.
