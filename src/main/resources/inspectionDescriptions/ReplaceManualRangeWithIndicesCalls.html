Сообщение предлагает заменить на Collection.indices или операторы until и rangeTo при итерации по коллекции в цикле for.

Использование синтаксического сахара может сделать код проще.

Это быстрое исправление заменит ручное определение диапазона на соответствующую конструкцию.

Пример:


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

После применения быстрого исправления:


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
