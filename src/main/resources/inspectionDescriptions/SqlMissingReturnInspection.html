Сообщает о функциях без оператора RETURN.

Пример (Oracle):

CREATE FUNCTION foo RETURN int AS
BEGIN
END;

Функция foo должна возвращать целочисленное значение, но тело функции ничего не возвращает. Чтобы исправить ошибку, добавьте оператор RETURN (например, RETURN 1;).

CREATE FUNCTION foo RETURN int AS
BEGIN
 RETURN 1;
END;
