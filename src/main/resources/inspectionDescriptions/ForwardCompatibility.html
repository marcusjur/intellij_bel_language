Обнаружение структур Java-кода, которые могут не компилироваться в будущих версиях Java.

Будут обнаружены следующие проблемы:

  • Использование assert, enum или _ в качестве идентификаторов
  • Использование ограниченных идентификаторов var, yield или record в качестве имен типов
  • Неквалифицированные вызовы метода yield
  • Модификаторы оператора requires java.base внутри module-info.java
  • Избыточные точки с запятой между операторами import

Пример:


  // Этот ранее допустимый класс не может быть скомпилирован в Java 14,
  // так как 'yield' стал ограниченным идентификатором.
  public class yield {}

Своевременное устранение этих проблем упростит переход на будущие версии Java.
