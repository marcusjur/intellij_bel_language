Сообщает о вызове метода getClass() для экземпляра java.lang.Class.

Обычно это является ошибкой, так как результат всегда эквивалентен Class.class. Если это ошибка, лучше удалить вызов getClass() и использовать квалификатор напрямую. Если такое поведение намеренное, лучше явно написать Class.class, чтобы избежать путаницы.

Пример:


  void test(Class<?> clazz) {
    String name = clazz.getClass().getName();
  }

После применения одного из возможных быстрых исправлений:


  void test(Class<?> clazz) {
    String name = clazz.getName();
  }
