Сообщает о явно указанных типах локальных переменных, которые очевидны и избыточны, например val f: Foo = Foo().

Пример:


  class Point(val x: Int, val y: Int)
  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

После применения быстрого исправления:


  class Point(val x: Int, val y: Int)
  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
