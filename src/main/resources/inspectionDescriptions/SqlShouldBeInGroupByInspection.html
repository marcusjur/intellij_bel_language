Обнаружены столбцы, которые отсутствуют в предложении GROUP BY или в вызове агрегатной функции.

Пример (Microsoft SQL Server):

CREATE TABLE t1 (a INT, b INT);
SELECT a, b FROM t1 GROUP BY a;

При выполнении запроса SELECT возникнет ошибка, так как Microsoft SQL Server требует, чтобы столбец b был включен в GROUP BY или использовался внутри агрегатной функции. Следующие два примера исправляют эту ошибку.

SELECT a, b FROM t1 GROUP BY a, b;
SELECT a, max(b) max_b FROM t1 GROUP BY a;
