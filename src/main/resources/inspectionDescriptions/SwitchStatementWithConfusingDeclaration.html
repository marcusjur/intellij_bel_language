Сообщает о локальных переменных, которые объявлены в одной ветке оператора switch и используются в другой. Такие объявления могут быть особенно непонятными.

Пример:


    switch(i) {
      case 2:
          int x = 0;
          break;
      case 3:
          x = 3;
          System.out.println(x);
          break;
    }
