Сообщать о вложенных классах с модификатором public.

Пример:


  public class Outer {
    public static class Nested {} // Предупреждение
    public class Inner {}         // Предупреждение
    public enum Mode {}           // Предупреждение (зависит от настроек)
    public interface I {}         // Предупреждение (зависит от настроек)
  }

Настройка проверки:

  • Используйте опцию Игнорировать 'public' внутренние перечисления, чтобы игнорировать внутренние public enum.
  • Используйте опцию Игнорировать 'public' внутренние интерфейсы, чтобы игнорировать внутренние public интерфейсы.
