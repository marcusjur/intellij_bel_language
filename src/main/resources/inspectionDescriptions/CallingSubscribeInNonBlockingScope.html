Сообщать о вызовах subscribe() в "reactive" методах.

Методы, возвращающие тип Publisher (включая Flux и Mono), не должны напрямую вызывать метод subscribe(), так как это может нарушить цепочку реактивных вызовов.

Вместо использования subscribe() рекомендуется использовать операторы композиции, такие как flatMap(), zip(), then() и другие.

Пример:


  Flux<String> stringFlux(){
    Flux<String> flux = Flux.just("abc");
    flux.subscribe(); // <- Блокирующий вызов 'subscribe' в неблокирующем контексте
    return flux;
  }
