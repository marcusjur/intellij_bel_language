Сообщает о вызовах методов параметризованных коллекций, где фактический тип аргумента не соответствует типу элементов коллекции.

Пример:


  List<Integer> list = getListOfElements();
  list.remove(""); //remove будет подсвечен

В настройках проверки можно отключить предупреждения для потенциально корректного кода, как показано ниже: 


  public boolean accept(Map<Integer, String> map, Object key) {
    return map.containsKey(key);
  }
