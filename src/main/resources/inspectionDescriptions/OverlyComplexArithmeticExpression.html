Сообщать об арифметических выражениях с чрезмерным количеством операндов. Такие выражения могут быть трудны для понимания и могут содержать ошибки.

Параметры, ссылки на поля и другие основные выражения считаются за один операнд.

Пример: 

int calc(int a, int b) {
        return a + a + a + b + b + b + b; // Эта строка содержит 7 операндов и будет отмечена.
    }
