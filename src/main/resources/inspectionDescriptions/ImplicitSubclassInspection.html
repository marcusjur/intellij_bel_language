Сообщает о случаях, когда код во время выполнения препятствует созданию подклассов классов определённым фреймворком (например, Spring или Hibernate).

Типичные примеры, когда создание подклассов необходимо, но невозможно:

  • Классы с модификатором final, помеченные специфичными для фреймворка аннотациями (например, Spring @Configuration)
  • Методы с модификаторами final, static или private, помеченные специфичными для фреймворка аннотациями (например, Spring @Transactional)
  • Методы, помеченные специфичными для фреймворка аннотациями в классах с модификатором final

Список отображаемых случаев зависит от используемого фреймворка.
