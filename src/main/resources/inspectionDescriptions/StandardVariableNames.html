Сообщать о переменных, имеющих "стандартные" имена, не соответствующие их типам. Такие имена могут вводить в заблуждение. Для определённых типов существуют следующие стандартные имена:
  • i, j, k, m, n - int
  • f - float
  • d - double
  • b - byte
  • c, ch - char
  • l - long
  • s, str - String

Быстрое исправление с предложением переименования доступно только в редакторе.

Используйте эту опцию, чтобы игнорировать имена параметров, совпадающие с именами параметров в прямом родительском методе.
