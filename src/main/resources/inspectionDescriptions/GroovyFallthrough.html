Сообщать о fallthrough в операторах switch. Хотя иногда это может быть полезно, обычно это происходит непреднамеренно и может привести к неожиданным ошибкам.

Пример:


switch(n) {
  case 1:
    print 1
  case 2: // "case 1" проваливается в "case 2". Операторы, начинающиеся с "case 2", будут выполнены сразу после выполнения "case 1".
    print 2
    break
  default:
    print "Default"
}
