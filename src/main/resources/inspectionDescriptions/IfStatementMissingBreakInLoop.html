Сообщает о циклах, содержащих оператор if и заканчивающихся оператором break, без изменения семантики. Это помогает избежать избыточных итераций цикла.

Пример:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

После применения быстрого исправления:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
