Сообщает о вызовах методов суперкласса из анонимных классов, внутренних классов или локальных классов, если в окружающем коде класса существуют методы с такой же сигнатурой. В таких случаях может показаться, что вызывается метод из окружающего кода, хотя на самом деле вызывается метод из суперкласса.

Для большей ясности кода рекомендуется добавить явный квалификатор super к вызову метода.

Пример:


  class Parent {
    void ambiguous(){}
  }
  class Example {
    void ambiguous(){}
    class Inner extends Parent {
      void example(){
        ambiguous(); //предупреждение
      }
    }
  }

После применения быстрого исправления:


  class Parent {
    void ambiguous(){}
  }
  class Example {
    void ambiguous(){}
    class Inner extends Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
