Методы, возвращающие this.

Хотя такой возврат и является допустимым, он редко необходим и обычно указывает на то, что метод предназначен для использования как часть цепочки вызовов методов (например, buffer.append("foo").append("bar").append("baz")). Многие стандарты кодования не рекомендуют такие цепочки.

Пример:


  public Builder append(String str) {
    // [...]
    return this;
  }
