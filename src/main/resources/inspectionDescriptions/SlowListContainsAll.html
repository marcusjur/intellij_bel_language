Сообщение о вызове containsAll() для java.util.List.

Временная сложность этого метода составляет O(n·m), где n - количество элементов в списке, для которого вызывается метод, а m - количество элементов в коллекции, передаваемой в качестве параметра. Для больших списков эта операция может быть весьма затратной.

Быстрое исправление обернет список в new java.util.HashSet<>(), поскольку время, необходимое для создания java.util.HashSet из java.util.List и выполнения containsAll() на java.util.HashSet, составляет O(n+m).

Пример:

  public boolean check(List<String> list, Collection<String> collection) {
    // Сложность O(n·m)
    return list.containsAll(collection);
  }

После применения быстрого исправления:

  public boolean check(List<String> list, Collection<String> collection) {
    // Сложность O(n+m)
    return new HashSet<>(list).containsAll(collection);
  }
