Сообщает о классах, которые содержат ссылки на один из своих подклассов. Такие ссылки могут вызывать путаницу и нарушают несколько принципов объектно-ориентированного проектирования.

Пример:


  class Entity {
    // Предупреждение: этот класс ссылается на свой подкласс
    void compare(SimpleEntity entity) {
      ...
    }
  }
  class SimpleEntity extends Entity {
    ...
  }
