Рекомендуется заменить проверку size == 0 для Collections/Array/String на использование метода isEmpty().

Использование isEmpty() делает код более лаконичным.

Это быстрое исправление заменит проверку размера на вызов isEmpty().

Пример:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size == 0
  }

После применения быстрого исправления:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isEmpty()
  }
