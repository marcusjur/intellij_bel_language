Сообщает о случаях в обычных Python-файлах, когда перегрузки размещены после реализации или их сигнатуры несовместимы с реализацией.

Пример:


from typing import overload
@overload
def foo(p1, p2): # сигнатура перегрузки несовместима с реализацией
    pass
@overload
def foo(p1): # сигнатура перегрузки несовместима с реализацией
    pass
def foo(p1, p2, p3):
    print(p1, p2, p3)
