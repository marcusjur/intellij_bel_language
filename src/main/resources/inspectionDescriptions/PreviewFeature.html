Сообщает об использовании API предварительных функций, то есть модулей, пакетов, классов, интерфейсов, методов, конструкторов или констант перечислений в пространствах имен java.* или javax.*, помеченных аннотацией @PreviewFeature.

Предварительная функция - это новая возможность языка Java, виртуальной машины Java или Java SE API, которая полностью специфицирована и реализована, но не является постоянной. Концепция предварительных функций определена в JEP 12.

Если код зависит от предварительного API, он может перестать компилироваться в будущих версиях JDK, если эта функция будет изменена или удалена.

Эта проверка выполняется только в том случае, если уровень языка проекта или модуля установлен на Preview.
