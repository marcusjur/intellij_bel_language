Сообщает о классах Serializable, в которых поле serialVersionUID не объявлено как private static final long.

Пример:


  class SampleClass implements Serializable {
    private long serialVersionUID = 1; // Поле класса Serializable не объявлено как 'private static final long'
    public SampleClass() {
        System.out.println(serialVersionUID);
    }
  }
