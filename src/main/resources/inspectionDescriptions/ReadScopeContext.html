Обнаружен некорректный вызов ReadScope из приостановленного или операционного контекста.

Функция ReadScope не может быть вызвана из корутины или операции. Это может привести к исключению во время выполнения.

Пример:


  fun ReadScope.kernel(/*...*/) { /*...*/ }
  fun ReadScope.foo() {
      readEditorAction { ac ->
          kernel() // Некорректно
      }
  }
