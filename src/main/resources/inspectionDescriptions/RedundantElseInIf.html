Обнаружение избыточного if с оператором return внутри if

Пример:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      else { // этот else избыточен, код в фигурных скобках можно сдвинуть влево
          someCode()
      }
  }

После применения быстрого исправления:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      someCode()
  }
