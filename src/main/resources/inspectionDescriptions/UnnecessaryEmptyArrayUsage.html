Обнаружено выделение массива нулевой длины в классе, который содержит константу с типом элементов этого массива. Поскольку массивы нулевой длины неизменяемы, можно повторно использовать один и тот же экземпляр массива для экономии памяти.

Пример:


  class Item {
    // Повторно используемая публичная константа массива нулевой длины
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // Ненужное создание массива нулевой длины
      return new Item[0];
    }
  }

После применения быстрого исправления:


  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }
