Сообщает о методах, в которых количество операторов превышает указанное максимальное значение.

Методы с большим количеством операторов могут быть сложными для понимания и указывать на необходимость рефакторинга.

Следующие операторы не учитываются:

  • Пустые операторы (точка с запятой)
  • Блочные операторы
  • Операторы инициализации в цикле for, то есть int i = ... в выражении for(int i = ...;...)
  • Операторы обновления в цикле for, то есть i += 2 в выражении for(int i = ...;...; i += 2)

Используйте поле максимальное количество операторов в методе, чтобы указать допустимое количество операторов в методе.
