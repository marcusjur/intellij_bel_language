Сообщает о классах, в которых не объявлен логгер.

Обеспечение каждого класса выделенным логгером является важным шагом для реализации единообразного логования в приложении. Эта проверка не распространяется на интерфейсы, перечисления, аннотации, внутренние классы и абстрактные классы.

Например:


  public class NoLoggerDeclared {
    int calculateNthDigitOfPi(int n) {
      // todo
      return 1;
    }
  }

Имена классов логгеров можно указать в таблице в разделе Options. Проверка будет сообщать о классах, в которых не объявлено поле одного из указанных типов.
