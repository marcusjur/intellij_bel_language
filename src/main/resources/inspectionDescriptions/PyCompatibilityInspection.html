Сообщает о несовместимости с указанной версией Python. Если вам нужно обеспечить совместимость вашего кода с различными версиями Python (например, при разработке библиотеки), включите эту проверку.

Чтобы определить диапазон проверяемых версий Python, выберите соответствующие флажки в разделе Параметры.

Подробную информацию о поддерживаемых ИСР версиях Python см. в веб-справке.
