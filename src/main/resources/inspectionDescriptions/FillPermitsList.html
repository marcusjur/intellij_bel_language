Сообщает о классах sealed без явного списка permits. Для sealed-классов, имеющих подклассы только в том же файле, явный список permits не требуется. По умолчанию эта проверка не подсвечивается в редакторе, а только предлагает исправление.

Пример:


  sealed class A {}
  final class B extends A {}

После применения быстрого исправления:


  sealed class A permits B {}
  final class B extends A {}

Новое в версии 2020.3
