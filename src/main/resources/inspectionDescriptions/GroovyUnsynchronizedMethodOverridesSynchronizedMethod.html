Обнаружен несинхронизованный метод, который переопределяет synchronized метод.

Пример:


  abstract class Base {
      synchronized void foo() {
          // ...
      }
  }
  class Derived extends Base {
      @Override
      void foo() {
          super.foo()
            // ...
        }
    }

В данном случае несинхронизованный метод foo() в классе Bar переопределяет synchronized метод.
