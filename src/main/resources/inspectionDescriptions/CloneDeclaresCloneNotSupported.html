Сообщает о методе clone(), который не объявляет throws CloneNotSupportedException.

Если не объявлен throws CloneNotSupportedException, подклассы этого метода не смогут запретить клонование стандартным способом. Эта проверка не выдает предупреждения для методов clone(), объявленных как final, и методов clone() внутри final классов.

Настройка проверки:

Используйте опцию Предупреждать только для 'protected' методов клонования, чтобы проверка выдавала предупреждения только для методов protected clone(). В книге Effective Java (2-е и 3-е издания) рекомендуется опускать объявление CloneNotSupportedException для public методов, так как методы, не выбрасывающие проверяемые исключения, проще в использовании.

Пример:


  public class Example implements Cloneable {
    // Метод не объявляет 'throws CloneNotSupportedException'
    protected Object clone() {
        try {
            return super.clone();
        } catch (CloneNotSupportedException e) {
            return null;
        }
    }
  }
