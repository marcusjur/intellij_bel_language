Отчёт о классах, использующих определённые аннотации JSF без реализации необходимых интерфейсов:

  • @FacesComponent должен расширять <javax/jakarta>.faces.component.FacesComponent
  • @FacesRenderer должен расширять <javax/jakarta>.faces.render.Renderer
  • @FacesConverter должен расширять <javax/jakarta>.faces.convert.Converter
  • @FacesValidator должен расширять <javax/jakarta>.faces.validator.Validator

Пример:


@FacesConverter(value="...")  // класс должен расширять 'javax.faces.convert.Converter'
public class FooConverter {
}
@FacesRenderer(rendererType="..", componentFamily="..")  // класс должен расширять 'javax.faces.render.Renderer'
public class FooConverter {
}
@FacesRenderer(rendererType="..", componentFamily="..")  // класс должен расширять 'javax.faces.validator.Validator'
public class FooValidator {
}
