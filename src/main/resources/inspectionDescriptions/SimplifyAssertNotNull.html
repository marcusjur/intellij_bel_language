Сообщает о вызовах assert, которые проверяют, что объявленная переменная не равна null.

Код можно упростить, используя операторы !! или ?:.

Быстрое исправление заменит assert в инициализаторе переменной на операторы !! или ?:.

Пример:


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

После применения быстрого исправления:


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }
