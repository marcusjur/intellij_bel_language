Обнаружена метка, не указывающая на функцию.

Объявление меток, не относящихся к функциям, запрещено.

Быстрое исправление удалит метку.

Пример:


  fun testValLabelInReturn() {
      L@ val fn = { return@L }
      fn()
  }

После применения быстрого исправления:


  fun testValLabelInReturn() {
      L@ val fn = { return }
      fn()
  }

Эта проверка выполняется только если уровень языка проекта или модуля 1.4 или выше.
