Правило проверки, которое сообщает о операторах switch или выражениях switch с избыточными блоками кода.

Пример:


  String s = switch (n) {
    case 1 -> { yield Integer.toString(n); }
    default -> "default";
  };

После применения быстрого исправления:


  String s = switch (n) {
    case 1 -> Integer.toString(n);
    default -> "default";
  };

Новое в версии 2019.1
