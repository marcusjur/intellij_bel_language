Сообщает о циклах while, которые выполняют итерацию по коллекции и могут быть заменены улучшенным циклом for (синтаксис foreach).

Пример:


  Iterator it = c.iterator();
  while(it.hasNext()) {
    Object obj = it.next();
    System.out.println(obj);
  }

Может быть заменено на:


  for (Object obj : c) {
    System.out.println(obj);
  }
