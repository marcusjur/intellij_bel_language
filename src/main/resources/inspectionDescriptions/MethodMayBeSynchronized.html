Сообщает о методах, тело которых содержит только один оператор synchronized. Для методов экземпляра выражение блокировки в операторе synchronized должно быть равно this; для статических методов оно должно быть равно [ClassName].class.

Для улучшения читаемости таких методов можно удалить оболочку synchronized и пометить сам метод как synchronized.

Пример:


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

После применения быстрого исправления:


  public synchronized int generateInt(int x) {
    return 1;
  }
