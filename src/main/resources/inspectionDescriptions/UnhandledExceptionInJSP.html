Сообщение об исключении, выброшенном из верхнеуровневого JSP Scriptlet.

Пример:


  <%!
  String getFormattedDate() {
    ...
    throw new IllegalArgumentException(...); // ошибка
  }
  %>
  <html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title>Hello, JSP!</title>
  </head>
  <body>
  <i>Today is <%= getFormattedDate() %></i>
  </body>
  </html>

Примечание: Такие JSP-страницы компилируются, поскольку servlet-контейнер оборачивает весь JSP-код в блок try {} catch() {}, однако они вызывают исключения времени выполнения при развертывании на сервере.
