Сообщает об инициализаторах экземпляров, которые могут быть объявлены как static.

Инициализатор экземпляра может быть статическим, если он не ссылается на какие-либо нестатические члены своего класса. Статические инициализаторы выполняются после разрешения класса, тогда как инициализаторы экземпляров выполняются для каждого экземпляра этого класса.

Эта проверка не распространяется на пустые инициализаторы экземпляров и инициализаторы в анонимных классах.

Пример:


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

После применения быстрого исправления:


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // теперь инициализуется только один раз для класса
     }
  }
