Рекомендуется использовать команду ng add для установки зависимостей.

ng add использует пакетный менеджер для загрузки и вызывает схемы, которые могут обновлять проект путем изменения конфигурации, добавления дополнительных зависимостей (например, полифилов) или создания начального кода, специфичного для пакета.
