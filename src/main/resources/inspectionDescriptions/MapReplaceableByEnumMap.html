Сообщает об инициализации объекта java.util.Map, где тип ключа является перечислением. Такой объект java.util.Map может быть заменен объектом java.util.EnumMap.

Реализация java.util.EnumMap более эффективна, так как использует простой массив в качестве базовой структуры данных.

Пример:


  Map<MyEnum, String> myEnums = new HashMap<>();

После применения быстрого исправления:


  Map<MyEnum, String> myEnums = new EnumMap<>(MyEnum.class);
