Отчёт не проверяет выделение больших массивов для java.lang.OutOfMemoryError. В средах с ограниченной памятью рекомендуется проверять выделение больших объектов данных на предмет исчерпания памяти.

Эта проверка применима для Java ME и других сред с сильно ограниченными ресурсами. Бездумное применение этой проверки может негативно повлиять на читаемость кода и его архитектуру.

Используйте эту опцию, чтобы указать максимальное количество элементов, допустимое в непроверяемых выделениях массива.
