Сообщает о вызовах java.lang.Thread.sleep() внутри цикла.

Такие вызовы указывают на "активное ожидание" (busy wait). Активное ожидание обычно неэффективно и может привести к непредвиденным взаимоблокировкам, поскольку поток в активном ожидании не освобождает заблокированные ресурсы.

Пример:


  class X {
     volatile int x;
     public void waitX() throws Exception {
      while (x > 0) {
        Thread.sleep(10);// Предупреждение: вызов 'Thread.sleep()' в цикле может указывать на активное ожидание
      }
    }
  }
