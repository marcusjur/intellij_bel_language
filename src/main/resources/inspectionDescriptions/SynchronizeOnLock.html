Сообщает о блоках synchronized, блокирующих экземпляры java.util.concurrent.locks.Lock. Такая синхронизация почти наверняка является непреднамеренной и должна быть заменена на соответствующие версии .lock() и .unlock().

Пример:


  final ReentrantLock lock = new ReentrantLock();
  public void foo() {
    synchronized (lock) {}
  }
