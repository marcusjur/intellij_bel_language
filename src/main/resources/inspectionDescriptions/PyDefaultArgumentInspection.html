Обнаружена проблема с использованием изменяемых значений в виде списков или словарей в качестве значений по умолчанию для аргументов функции.
Значения аргументов по умолчанию вычисляются только один раз при определении функции, что означает, что изменение значения по умолчанию повлияет на все последующие вызовы этой функции.

Пример:


def func(s, cache={}):
    cache[s] = None

После применения быстрого исправления код станет:


def func(s, cache=None):
    if cache is None:
        cache = {}
    cache[s] = None
