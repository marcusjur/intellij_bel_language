Сообщает о выражениях доступа к массиву или указателю, где индекс может выходить за пределы массива или выделенного буфера.

Пример:


  void foo() {
      int buffer[100];
      for (int i = 0; i <= 100; i ++)
          buffer[i] = 0; // buffer overflow when i is equal to 100
  }
