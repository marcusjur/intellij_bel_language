Отчёт может пропускать модификатор видимости public для свойств, имеющих указанную пониженную видимость set. Если не указано иное, свойства PHP неявно доступны для публичного чтения.

PHP 8.4 и более поздние версии поддерживают асимметричную видимость. Для получения дополнительной информации см. PHP RFC: Асимметричная видимость v2.
