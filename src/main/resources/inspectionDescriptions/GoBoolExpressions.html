Сообщает о частях булевых выражений, которые всегда возвращают true, всегда возвращают false или являются избыточными. Такие булевы выражения можно упростить для улучшения читаемости кода. В некоторых случаях это также может указывать на другие проблемы.

Пример:

func isNonZero(x, y int) bool {
  // Второе сравнение либо всегда true,
  // либо никогда не выполняется
  return x > 0 && x > 0
}

Вы можете применить быстрое исправление Упростить выражение к части x > 0 && x > 0. После применения быстрого исправления выражение будет выглядеть так: x > 0.
