Обнаружены модификаторы, не соответствующие порядку, рекомендованному в руководстве по стилю.

Быстрое исправление порядка модификаторов доступно для автоматического исправления кода.

Пример:


  private inline fun correctOrder(f: () -> Unit) {} // <== Ok
  infix private fun Int.wrongOrder(expr: Int) {} // <== Неверный порядок, быстрое исправление изменит на "private infix"
