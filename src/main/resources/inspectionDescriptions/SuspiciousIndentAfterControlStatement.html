Сообщать о подозрительных отступах в операторах, следующих за управляющими конструкциями без фигурных скобок.

При таком отступе оператор может выглядеть как часть управляющей конструкции, хотя на самом деле он будет выполняться безусловно после неё.

Пример:


  class Bar {
      void foo(int i) {
          if (i == 0)
              System.out.println("foo");
              System.out.println("bar"); // предупреждение
          if (i == 1);
              System.out.println("great"); // предупреждение
          if (i == 42)
              System.out.println("answer");
                  System.out.println("question"); // предупреждение
      }
  }
