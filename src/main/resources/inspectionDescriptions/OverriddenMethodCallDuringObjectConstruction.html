Сообщает о любых вызовах переопределённых методов текущего класса во время конструирования объекта. Это происходит внутри:
  • конструкторов
  • нестатических инициализаторов экземпляров
  • нестатических инициализаторов полей
  • clone()
  • readObject()
  • readObjectNoData()

Такие вызовы могут привести к труднообнаруживаемым ошибкам, поскольку нет гарантии, что объект был инициализован до вызова метода.

Пример:


  abstract class Parent {
    void someMethod() { }
  }
  class Child extends Parent {
    Child() {
      someMethod();
    }
    @Override
    void someMethod() { }
  }

Эта проверка имеет общую функциональность со следующими проверками:

  • Проверка вызовов абстрактных методов во время конструирования объекта
  • Проверка вызовов переопределяемых методов во время конструирования объекта

Следует включать только одну из этих проверок, чтобы избежать дублирования предупреждений.
