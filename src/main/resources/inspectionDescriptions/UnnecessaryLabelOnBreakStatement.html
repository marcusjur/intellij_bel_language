Сообщение о наличии оператора break с необязательной меткой. Такие метки не влияют на поток управления, но усложняют чтение кода.

Пример:


  label:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break label;
    //doSmth
  }

После применения быстрого исправления:


  label:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break;
    //doSmth
  }
