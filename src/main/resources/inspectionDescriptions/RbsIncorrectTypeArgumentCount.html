Сообщает о ссылках на обобщенные типы с неправильным количеством аргументов типа. Использование обобщенного типа без передачи правильного количества аргументов типа является ошибкой.

Количество передаваемых аргументов типа можно исправить с помощью быстрого исправления.

Пример:

class Example[A]
end
# Передано слишком много аргументов типа
type example = Example[untyped, untyped]

После применения быстрого исправления:

class Example[A]
end
type example = Example[untyped]
