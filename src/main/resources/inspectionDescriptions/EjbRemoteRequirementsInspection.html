Обнаружены следующие нарушения спецификации EJB:
  • Метод не объявляет, что выбрасывает java.rmi.RemoteException
  • Тип возвращаемого значения или аргументы метода несовместимы с RMI/IIOP
  • Интерфейс конечной точки EJB содержит объявления констант

Пример:


public interface EntityRemote extends javax.ejb.EJBObject{
  public int businessExample(String value) throws java.lang.RuntimeException; // Ошибка: следует объявить 'RemoteException'
}
