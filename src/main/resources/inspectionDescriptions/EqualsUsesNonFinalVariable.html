Сообщает о реализации метода equals(), в которой происходит обращение к нефинальным (final) переменным. Такой доступ может привести к тому, что метод equals() будет возвращать разные результаты в разные моменты жизненного цикла объекта, что может вызвать проблемы при использовании стандартных коллекций.

Пример:


  public class Person {
    private String lastName;
    @Override
    public boolean equals(Object obj) {
      ...
      Person other = (Person) obj;
      if (lastName == null) {
        if (!lastName.equals(other.lastName)) {
        return false;
      ...
      }
    }
  }
