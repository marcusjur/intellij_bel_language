Сообщает об использовании выражений instanceof или getClass() == SomeClass.class с проверкой this.

Такие выражения указывают на неудачный объектно-ориентированный дизайн и должны быть заменены полиморфной конструкцией.

Пример:


class Super {
  void process() {
    if (this instanceof Sub) { // предупреждение
      doSomething();
    } else {
      doSomethingElse();
    }
  }
}
class Sub extends Super {}

Для исправления проблемы используйте переопределение метода:


class Super {
  void process() {
    doSomethingElse();
  }
}
class Sub extends Super {
  @Override
  void process() {
    doSomething();
  }
}
