Сообщение о неправильной конфигурации контекста Spring.

  • Неразрешённые файлы и каталоги в атрибуте locations и соответствующих псевдонимах мета-аннотации @ContextConfiguration
  • Отсутствие файлов контекста приложения по умолчанию

Для получения дополнительной информации см. @ContextConfiguration.

Пример:


  @ContextConfiguration(locations = "classpath:META-INF/unknown-context.xml") // Сообщение "Не удаётся разрешить файл 'unknown-context.xml'"
  class MyTests {...}
