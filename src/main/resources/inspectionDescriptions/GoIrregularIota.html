Сообщает о нерегулярном использовании iota в объявлениях констант.

Идентификатор iota сбрасывается для каждого объявления констант и автоматически увеличивается для каждой спецификации константы. В пределах спецификации константы идентификатор сохраняет свое значение. Явное указание iota не сбрасывает счетчик.

Подробнее см. раздел Iota в спецификации Go.

Эта проверка срабатывает, если две спецификации констант имеют текстуально идентичные списки выражений, содержащие хотя бы одну ссылку на iota, и между этими спецификациями находятся только спецификации констант без списков выражений.

Рекомендуется либо опустить избыточные списки выражений, либо всегда выписывать списки выражений полностью.

Пример:

const (
  a = iota  // 0
  b         // 1
  c = iota  // 2
)

Проверка срабатывает из-за избыточного использования iota в определении c.

Пример:

const (
  a, aa = iota, iota       // 0, 0
  b, bb                    // 1, 1
  c, cc = iota + 40, iota  // 42, 2
)

Проверка не срабатывает, так как ни один из списков выражений не является избыточным.
