Вызов с позиционными аргументами после именованных аргументов. Применимо для PostgreSQL, Oracle и Db2.

Пример (в PostgreSQL):

CREATE FUNCTION foo(a int, b int, c int) RETURNS int
    LANGUAGE plpgsql AS
$$
BEGIN
    RETURN a + b + c;
END
$$;
SELECT foo(a => 1, b => 2, c => 3);
  -- `3` после именованных аргументов
SELECT foo(1, b => 2, 3);
  -- `1` и `3` после именованных аргументов
SELECT foo(b => 2, 1, 3);
