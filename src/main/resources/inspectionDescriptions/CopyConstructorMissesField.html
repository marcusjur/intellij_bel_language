Сообщает о конструкторе копирования, который не копирует все поля класса.

Поля с модификатором final с инициализаторами и поля transient считаются необязательными для копирования.

Пример:


  class Point {
    private int x;
    private int y;
    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }
    Point(Point other) {
      // Поля x и y не инициализованы
    }
  }
