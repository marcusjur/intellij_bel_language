Сообщает об использовании модификатора видимости protected внутри final класса. В этом случае protected члены доступны только внутри самого класса, что фактически делает их private.

Пример:


  class FinalClass {
      protected fun foo() {}
  }

После применения быстрого исправления:


  class FinalClass {
      private fun foo() {}
  }
