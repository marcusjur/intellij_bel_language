Сообщает о присваивании null переменной типа Optional или возврате null из метода, возвращающего Optional.

Рекомендуется использовать Optional.empty() (или Optional.absent() для Guava) для представления пустого значения.

Пример:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : null;
  }

После применения быстрого исправления:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : Optional.empty();
  }

Настройка проверки:

Опция Сообщать о сравнении Optional с null также позволяет обнаруживать сравнения вида optional == null. Хотя в редких случаях (например, для полей Optional с отложенной инициализацией) это может быть корректным, обычно переменные Optional не должны быть null, и вместо этого следует использовать optional.isPresent().
