Сообщает о полях, объявленных без модификаторов доступа (также известных как package-private).

Не сообщает о константах (то есть полях, помеченных как static и final).

Пример:


  public class A {
    Object object;             // предупреждение
    final static int MODE = 0; // константа, без предупреждения
  }
