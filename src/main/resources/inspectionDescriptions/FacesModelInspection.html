Сообщает о некорректных элементах в файле Java Server Faces faces-config.xml:

  • Ссылки на неинстанцируемые классы
  • Ссылки на классы, не расширяющие требуемые классы
  • Ссылки на классы с неправильной областью видимости
  • Пустые теги и значения атрибутов
  • Теги и значения атрибутов, не соответствующие требуемой схеме (например, Java-идентификаторы)
  • Теги без обязательных дочерних тегов или атрибутов
  • Теги, определяющие объекты с повторяющимися именами

Пример:


  <faces-config >
     <application>
        <action-listener>java.lang.String</action-listener>  <!-- 'java.lang.String' не может быть присвоен 'javax.faces.event.ActionListener'  -->
    </application>
    <navigation-rule>
        <from-view-id>myView.xhtml</from-view-id>
        <navigation-case>
            <to-view-id>unknown.xhtml</to-view-id>  <!-- Невозможно разрешить 'unknown.xhtml'  -->
        </navigation-case>
    </navigation-rule>
  </faces-config>
