Сообщает об использовании выражений доступа к статическим членам трейта. Такой прямой доступ считается устаревшим начиная с PHP 8.1. Вместо этого следует обращаться к статическим членам через класс, использующий трейт.

Для получения дополнительной информации см. Устаревшие возможности в PHP 8.1: доступ к статическим членам трейтов (php.net).
