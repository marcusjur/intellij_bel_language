Класс JUnit 4 параметризованного теста с аннотацией @RunWith(Parameterized.class), который либо не содержит метод-поставщик данных с аннотацией @Parameterized.Parameters, либо имеет неправильную сигнатуру этого метода. Такой тестовый класс не может быть выполнен. Метод-поставщик данных должен быть public и static и иметь возвращаемый тип Iterable или Object[].

Рекомендуется создать пустой метод-поставщик параметров или исправить сигнатуру существующего метода-поставщика данных.

Пример:


  @RunWith(Parameterized.class)
  public class ImportantTest {
    private int input;
    private int expected;
    ImportantTest(int input, int expected) {
      this.input = input;
      this.expected = expected;
    }
    // ...тестовые случаи
  }

После применения быстрого исправления:


  @RunWith(Parameterized.class)
  public class ImportantTest {
    private int input;
    private int expected;
    ImportantTest(int input, int expected) {
      this.input = input;
      this.expected = expected;
    }
    @Parameters
    public static Iterable<Object[]> parameters() {
      return null;
    }
    // ... test cases
  }
