Сообщает об упрощаемых цепочках сравнения.

Пример:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if x >= xmin and x <= xmax:
          pass

ИСР предлагает упростить if x >= xmin and x <= xmax. После применения быстрого исправления код становится:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if xmin <= x <= xmax:
          pass
