Сообщает о ссылках на внутренние классы, которые не используют полное имя включающего класса.

Пример:


  import foo.Foo.Bar;
  class Foo {
    class Bar {}
  }
  class Baz {
    void f(Bar bar) {}
  }

После применения быстрого исправления:


  class Foo {
    class Bar {}
  }
  class Baz {
    void f(Foo.Bar bar) {}
  }

В настройках проверки можно игнорировать ссылки на внутренние классы в пределах того же класса, поэтому для них импорт не требуется.
