Сообщает о нестандартных проверках типа is для объектов.

Рекомендуется заменить такие проверки на сравнение ссылок.

Пример:


  object Foo
  fun foo(arg: Any) = when {
      arg is Foo -> ...
      arg !is Foo -> ...
  }

После применения быстрого исправления:


  object Foo
  fun foo(arg: Any) = when {
      arg === Foo -> ...
      arg !== Foo -> ...
  }
