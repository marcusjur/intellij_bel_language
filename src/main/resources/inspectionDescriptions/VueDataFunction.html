Свойство data не является функцией Vue компонента. Рекомендуется обернуть объектный литерал в функцию.

При определении компонента, data должно быть объявлено как функция, возвращающая объект с начальными данными, поскольку будет создано несколько экземпляров с одним и тем же определением. Если data останется обычным объектом, этот объект будет использоваться по ссылке и совместно использоваться всеми созданными экземплярами! С функцией data при создании каждого нового экземпляра она будет вызываться для возврата новой копии начальных данных.
