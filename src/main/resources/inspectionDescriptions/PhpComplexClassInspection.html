Обнаружение чрезмерно сложных классов.

Сложность класса оценивается по трем метрикам:

  • TCC (Tight Class Cohesion) — мера связности класса (чем выше, тем лучше).
  • WMC (Weighted Method Count) — сумма цикломатической сложности всех методов класса (чем ниже, тем лучше).
  • ATFD (Access To Foreign Data) — количество свойств других классов, к которым обращается текущий класс (чем меньше, тем лучше).
