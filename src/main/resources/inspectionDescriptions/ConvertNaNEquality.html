Сообщает о проверках на равенство с использованием Float.NaN или Double.NaN (следует заменить на проверку isNaN()).

Согласно стандарту IEEE 754, проверки на равенство с NaN всегда возвращают false, даже для NaN == NaN. Следовательно, такая проверка, вероятно, ошибочна.

Быстрое исправление заменит сравнение на проверку isNaN(), использующую другую технику сравнения, которая корректно обрабатывает значения NaN.

Пример:


  fun check(value: Double): Boolean {
      return Double.NaN == value
  }

После исправления:


  fun check(value: Double): Boolean {
      return value.isNaN()
  }
