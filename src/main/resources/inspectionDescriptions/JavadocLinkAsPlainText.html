Обнаружение простых текстовых ссылок в комментариях Javadoc.

Быстрое исправление предложит обернуть ссылку в тег <a>.

Пример:


  class Main {
   /**
    * https://en.wikipedia.org/
    */
    void foo() {}
  }

После применения быстрого исправления:


  class Main {
   /**
    * <a href="https://en.wikipedia.org/">https://en.wikipedia.org/</a>
    */
    void foo() {}
  }
