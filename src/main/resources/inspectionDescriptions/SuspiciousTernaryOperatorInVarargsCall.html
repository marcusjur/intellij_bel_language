Сообщает о вызове vararg-метода с использованием тернарного оператора, содержащего как массивы, так и не-массивы в своих ветвях.

Во время компиляции обе ветви оборачиваются в массивы. В результате ветвь с массивом становится двумерным массивом, что может указывать на проблему.

Быстрое исправление обернет не-массив в массив, чтобы предотвратить преобразование компилятором.

Пример:


    static void bar(boolean flag) {
        Object[] a = {1, 2};
        Object b = "hello";
        foo(flag ? a : b);
    }
    static void foo(Object... obj) {
    }

После применения быстрого исправления:


    static void bar(boolean flag) {
        Object[] a = {1, 2};
        Object b = "hello";
        foo(flag ? a : new Object[]{b});
    }
    static void foo(Object... obj) {
    }
