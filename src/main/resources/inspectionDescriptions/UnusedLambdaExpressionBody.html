Сообщает о вызовах функций, где возвращаемое значение лямбда-выражения с телом-выражением не используется.

Если между заголовком функции и блоком тела присутствует =, код внутри функции не будет выполнен, что может привести к некорректному поведению.

Удаление токена = из объявления функции доступно как быстрое исправление.

Пример:


  fun printHello() = { println("Hello") }
  fun main() {
      printHello() // эта функция ничего не выведет
  }

После применения быстрого исправления:


  fun printHello() { println("Hello") }
  fun main() {
      printHello()
  }
