Сообщает о создании экземпляра Throwable или его подкласса, когда созданный Throwable фактически не был выброшен.

Отмеченный код указывает на ошибки, которые сложно обнаружить при тестовании.

Кроме того, эта проверка сообщает о вызовах методов, возвращающих экземпляр Throwable или его подкласса, когда полученный экземпляр Throwable не выбрасывается.

Пример:


  fun check(condition: Boolean) {
      if (!condition) /* throw is missing here */ IllegalArgumentException("condition is not met");
  }
  fun createError() = RuntimeException()
  fun foo() {
      /* throw is missing here */ createError()
  }
