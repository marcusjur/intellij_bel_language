Обнаружение избыточных круглых скобок в выражениях и типах.

Пример:

func _(x (int), y ((string))) {
}
func _() {
 _ = (1 + 1)
 _ = (((1 + 1)))
 _ = (((1 + 1))) + (((2 + 2)))
}

После применения быстрого исправления Удалить лишние скобки:

func _(x int, y string) {
}
func _() {
 _ = 1 + 1
 _ = 1 + 1
 _ = (1 + 1) + (2 + 2)
}
