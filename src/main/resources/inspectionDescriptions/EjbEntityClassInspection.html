Обнаружение следующих нарушений спецификации EJB уровня ошибки:
  • Методы Finder и Selector определены некорректно
  • Методы доступа к полям CMP и CMR определены некорректно

Пример:


public class SessionEJB implements javax.ejb.SessionBean {
  // Ошибка: метод 'ejbCreateXXX' должен быть public
  protected void ejbCreateExample() { ... }
  // Ошибка: метод 'ejbCreateAnotherExample' должен возвращать 'void'
  public int ejbCreateAnotherExample() { ... }
}
