Сообщает о несоответствии имен переменных в деструктурирующем объявлении с именами свойств класса данных.

Пример:


  data class Foo(val a: String, val b: Int, val c: String)
  fun bar(f: Foo) {
      val (a, c) = f
  }

Быстрое исправление изменит имена переменных, чтобы они соответствовали именам полей класса:


  data class Foo(val a: String, val b: Int, val c: String)
  fun bar(f: Foo) {
      val (a, b) = f
  }
