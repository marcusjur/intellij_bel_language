Сообщение о TestNG аннотации org.testng.annotations.Configuration.

Рекомендуется использовать современные аннотации @BeforeXXX/@AfterXXX для конфигурации тестового набора.

Пример:


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

После применения быстрого исправления:


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
