Сообщает о переменных, значения которых не используются после присваивания.
Рекомендуется удалить неиспользуемые переменные для сокращения кода и избежания избыточных присваиваний.

Будут отмечены следующие случаи:

  • Значение переменной никогда не считывается после присваивания.
  • Значение переменной всегда перезаписывается новым присваиванием перед следующим считыванием.
  • Инициализация переменной избыточна (по одной из вышеуказанных причин).
