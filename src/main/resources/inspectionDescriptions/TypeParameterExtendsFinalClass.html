Сообщает о параметрах типа, объявленных для расширения класса с модификатором final.

Поскольку невозможно расширить класс с модификатором final, рекомендуется заменить параметр типа на тип указанного класса final.

Пример:


  void foo() {
    List<? extends Integer> list; // предупреждение: класс Integer является final
  }

После применения быстрого исправления:


  void foo() {
    List<Integer> list;
  }
