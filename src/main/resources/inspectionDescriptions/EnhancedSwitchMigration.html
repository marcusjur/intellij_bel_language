Инспекция может автоматически заменить оператор switch на расширенный оператор или выражение switch.

Пример:


  double getPrice(String fruit) {
    // Оператор Switch можно заменить на расширенный 'switch'
    switch (fruit) {
      case "Apple":
        return 1.0;
      case "Orange":
        return 1.5;
      case "Mango":
        return 2.0;
      default:
        throw new IllegalArgumentException();
    }
  }

После применения быстрого исправления:


  double getPrice(String fruit) {
    return switch (fruit) {
      case "Apple" -> 1.0;
      case "Orange" -> 1.5;
      case "Mango" -> 2.0;
      default -> throw new IllegalArgumentException();
    };
  }

  • Используйте опцию показывать предупреждение только при возможности преобразования в выражение, чтобы не получать предупреждения о преобразовании в оператор switch.
  • Используйте опцию максимальное количество операторов в ветке для преобразования в switch-выражение, чтобы получать предупреждения о преобразовании в выражение только когда количество операторов в каждой ветке меньше указанного значения.

Новое в версии 2019.1
