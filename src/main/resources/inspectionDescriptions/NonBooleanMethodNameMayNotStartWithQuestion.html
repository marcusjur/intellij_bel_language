Сообщает о небулевых методах, названия которых начинаются с вопросительных слов. Такие названия методов могут вводить в заблуждение.

Эта проверка игнорирует небулевые методы, которые переопределяют методы библиотек.

Пример: 


  public void hasName(String name) {
    assert names.contains(name);
  }

Быстрое исправление для переименования таких методов доступно только в редакторе.

Настройка проверки:

  • Используйте список префиксов булевых методов, чтобы указать вопросительные слова, которые должны использоваться только для булевых методов.
  • Используйте опцию игнорировать методы с возвращаемым типом 'java.lang.Boolean', чтобы игнорировать методы, возвращающие java.lang.Boolean.
  • Используйте опцию игнорировать методы, переопределяющие/реализующие методы суперкласса, чтобы игнорировать методы, имеющие методы суперкласса.
