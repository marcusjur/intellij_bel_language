Сообщает об использовании строковых литералов с национальными символами без префикса N.

Без префикса N строка будет преобразована в кодование базы данных по умолчанию. Это кодование по умолчанию может не распознавать некоторые символы. Для получения дополнительной информации см. страницу "nchar и nvarchar" (Transact-SQL) на docs.microsoft.com.

Пример (Microsoft SQL Server):

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

В выражении SELECT 'абвгд' AS c; отсутствует префикс N, поэтому часть 'абвгд' будет подсвечена.
