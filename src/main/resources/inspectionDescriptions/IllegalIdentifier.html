Обнаружены идентификаторы проекта Android, которые не поддерживаются средой выполнения Android. Например, имена методов, содержащие пробелы.

Пример:


  class Test {
      var `a b c` = 3
  }

Для исправления этой проблемы измените имена идентификаторов в соответствии с требованиями среды выполнения Android.
