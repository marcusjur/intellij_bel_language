Сообщает о выражениях типа char, используемых в операциях сложения или вычитания.

Такой код не обязательно является проблемой, но может привести к ошибкам (например, если требуется строка).

Пример: int a = 'a' + 42;

После применения быстрого исправления: int a = (int) 'a' + 42;

Для контекста String: 

int i1 = 1;
int i2 = 2;
System.out.println(i2 + '-' + i1 + " = " + (i2 - i1));

После применения быстрого исправления:

System.out.println(i2 + "-" + i1 + " = " + (i2 - i1));
