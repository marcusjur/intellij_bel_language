Сообщает о вызовах определённых функций и методов, результаты которых не обрабатываются.

API таких функций подразумевает, что основной целью вызова функции должно быть получение и обработка возвращаемого значения, а не побочные эффекты. Вызовы без обработки результата могут указывать на неправильное использование API.

Пример:

fmt.Errorf("error: %s", reason) // Сформированная ошибка будет проигнорирована

После применения быстрого исправления Ввести локальную переменную:

err := fmt.Errorf("error: %s", reason)
