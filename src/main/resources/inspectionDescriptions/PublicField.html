Сообщать о полях с модификатором public. Не сообщать о константах (полях, помеченных как static и final).

Пример:


  class Main {
    public String name;
  }

После применения быстрого исправления:


  class Main {
    private String name;
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
  }

Настройка проверки:

  • Используйте список Игнорировать при наличии аннотаций для указания аннотаций, которые следует игнорировать. Проверка будет пропускать поля с любой из указанных аннотаций.
  • Используйте опцию Игнорировать поля 'public final' в перечислениях, чтобы пропускать поля public final в типах enum.
