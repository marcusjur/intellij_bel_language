Сообщает о параметрах типа, которые имеют то же имя, что и видимые типы в текущей области видимости. Такие имена параметров типа могут вызывать путаницу.

Пример:


  abstract class MyList<T> extends AbstractList<T> {
    private List<T> elements;
    // Параметр типа 'T' скрывает параметр типа 'T'
    public <T> T[] toArray(T[] array) {
        return elements.toArray(array);
    }
}
