Отчёт содержит таблицу с двумя столбцами с автоинкрементом. В диалектах MySQL, Microsoft SQL Server и Db2 таблица может иметь только одно поле с опцией автоинкремента, и это поле должно быть ключом.

Пример (MySQL):

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT,
    c2 INT AUTO_INCREMENT,
);

Ограничение AUTO_INCREMENT для c2 будет подсвечено, так как c1 уже имеет это ограничение. Чтобы исправить предупреждение, вы можете установить id как первичный ключ и удалить AUTO_INCREMENT для c2.

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT PRIMARY KEY,
    c2 INT,
);
