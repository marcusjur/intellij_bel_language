Рекомендуется заменить for(Entry<?,?> entry : map.entrySet()) {...} или map.entrySet().forEach(entry -> ...) на map.forEach((key, value) -> ...).

Пример


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

После применения быстрого исправления:


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

При включенной опции Не сообщать о циклах будут отображаться только случаи с entrySet().forEach(). Однако быстрое исправление также доступно для циклов for.
