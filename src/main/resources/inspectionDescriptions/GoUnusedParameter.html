Сообщать о неиспользуемых параметрах функций. 
func main() {
 printAll(
  42,
  "bird",
 )
}
func printAll(
 i int,
 s string,
) {
 fmt.Println(i)
}

Мы вызываем функцию printAll, передавая 42 и bird в качестве аргументов. Функция printAll принимает два параметра типов int и string, но использует только первый параметр. Поэтому s string будет отображаться серым цветом.
