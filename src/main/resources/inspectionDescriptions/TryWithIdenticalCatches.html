Обнаружение одинаковых блоков catch в одном операторе try.

Объединение таких блоков в один блок multi-catch позволяет уменьшить дублирование кода и предотвратить ситуации, когда один блок catch обновляется, а другой остается без изменений.

Пример:


    try {
        doSmth();
    }
    catch (IOException e) {
        LOG.error(e);
    }
    catch (URISyntaxException e) {
        LOG.error(e);
    }

С помощью быстрого исправления код можно сделать более компактным:


    try {
        doSmth();
    }
    catch (IOException | URISyntaxException e) {
        LOG.error(e);
    }
