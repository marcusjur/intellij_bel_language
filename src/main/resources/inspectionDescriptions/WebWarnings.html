Повторяющийся welcome-file в дескрипторе развертывания, используемый для развертывания веб-модуля на сервере.

Пример:



  <?xml version="1.0" encoding="UTF-8"?>
  <!DOCTYPE web-app PUBLIC "-//Sun Microsystems, Inc.//DTD Web Application 2.3//EN" "http://java.sun.com/dtd/web-app_2_3.dtd">
  <web-app>
    <welcome-file-list>
      <welcome-file>
        Welcome1.jsp
      </welcome-file>
      <welcome-file>
        /Welcome1.jsp // Ошибка: повторяющийся welcome-file
      </welcome-file>
      <welcome-file>
        Welcome3.jsp
      </welcome-file>
    </welcome-file-list>
  </web-app>
