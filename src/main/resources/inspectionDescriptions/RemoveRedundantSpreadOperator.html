Сообщает о случаях использования избыточного оператора расширения с вызовами функции arrayOf.

Используйте быстрое исправление "Удалить избыточный оператор расширения" для очистки кода.

Пример:


  fun foo(vararg s: String) { }
  fun bar(ss: Array<String>) {
      foo(*arrayOf("abc"))       // Для обоих вызовов 'foo' создание массива
      foo(*arrayOf(*ss, "zzz"))  // и последующее выделение избыточны
  }

После применения быстрого исправления:


  fun foo(vararg s: String) { }
  fun bar(ss: Array<String>) {
      foo("abc")
      foo(*ss, "zzz")
  }
