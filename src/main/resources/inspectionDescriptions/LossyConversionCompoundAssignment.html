Сообщает о составном присваивании, если тип правого операнда несовместим с типом переменной.

В таких составных присваиваниях происходит неявное приведение типов, что может привести к потере данных.

Пример:


  long c = 1;
  c += 1.2;

После применения быстрого исправления:


  long c = 1;
  c += (long) 1.2;

Новое в версии 2023.2
