Анализ пакетов

Используйте встроенный плагин Package Checker для проверки проектов Maven и Gradle на наличие уязвимых зависимостей. В главном меню выберите Код | Анализовать код, затем нажмите Показать уязвимые зависимости.

Результаты анализа будут показаны в отдельном окне инструмента, где каждая найденная зависимость будет сопровождаться подробным описанием.
