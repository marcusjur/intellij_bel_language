Сравнение Jupyter Notebook

Чтобы сравнить два Jupyter Notebook, выберите один из них в окне инструмента Проект, щёлкните правой кнопкой мыши по нему и выберите Сравнить с из контекстного меню. В файловой системе выберите Notebook для сравнения.

IntelliJ ИСРA отобразит оба Notebook и результаты сравнения в средстве просмотра различий.
