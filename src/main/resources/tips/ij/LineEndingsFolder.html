Изменение разделителей строк

Вам не нужно открывать файл в редакторе, чтобы изменить стиль разделителей строк. Вместо этого используйте окно инструментов Проект: выберите один или несколько файлов или папок, выберите Файл | Свойства файла | Разделители строк в главном меню, затем выберите нужный стиль окончания строк.

Для директорий новые разделители строк будут применены рекурсивно.
