Управление запросами на включение изменений

Управляйте входящими запросами на включение изменений GitHub прямо из IntelliJ ИСРA: выберите Git | GitHub | Просмотр запросов на включение изменений в главном меню. IntelliJ ИСРA позволяет назначать и объединять запросы на включение изменений, просматривать временную шкалу и встроенные комментарии, комментарии к коммитам и проверки, а также принимать изменения, не выходя из ИСР.
