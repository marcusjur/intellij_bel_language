Точка останова с логованием

Чтобы записывать состояние программы во время отладки, используйте точки останова без приостановки выполнения. Выберите выражение для логования, удерживайте клавишу Shift и нажмите на область слева от строки, где находится выражение для логования.

В этом примере при достижении строки 25 будет записано выражение let person = new Person(firstName, lastName, age, eyeColor);.
