Интерактивная консоль отладчика

При отладке клиентского JavaScript-кода вы можете выполнять фрагменты кода во вкладке интерактивной консоли. При вводе команд консоль отображает подсказки и предлагает автодополнение кода.

Консоль показывает предварительный просмотр вывода команд, который вы можете развернуть при необходимости.
