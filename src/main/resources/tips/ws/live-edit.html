Live Edit

Во время сессии отладки Live Edit позволяет отслеживать изменения в коде без обновления страницы. Знаете ли вы? Теперь вы можете использовать Live Edit без расширения JetBrains ИСР Support для Chrome! Откройте диалоговое окно Настройки &shortcut:ShowSettings;, перейдите в раздел Сборка, выполнение, развёртывание | Отладчик | Live Edit и установите флажок Обновлять приложение в Chrome при изменениях.
