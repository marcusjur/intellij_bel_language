Просмотр быстрой документации при наведении

Нет необходимости явно вызывать быструю документацию &shortcut:QuickJavaDoc;. Окно быстрой документации появится автоматически при наведении указателя мыши на нужный символ.

Эта функция включена по умолчанию. Чтобы отключить её, откройте диалоговое окно Settings (&shortcut:ShowSettings;), перейдите в раздел Editor | Code Editing и снимите флажок Показывать быструю документацию при наведении.
