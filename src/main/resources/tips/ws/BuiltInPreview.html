Встроенный предпросмотр

Встроенный предпросмотр WebStorm отображает результат HTML-кода в отдельной вкладке редактора. WebStorm сохраняет все изменения, внесенные в HTML-файл или связанные CSS и JavaScript файлы, и автоматически обновляет предпросмотр.

Чтобы открыть предпросмотр, наведите курсор на HTML-код для появления всплывающего окна браузера, затем нажмите на значок WebStorm.
