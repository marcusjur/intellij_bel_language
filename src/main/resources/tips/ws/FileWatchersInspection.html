File Watchers

Проверка доступных File Watcher вызывается для каждого файла, определённого как цель для применения предустановленных File Watcher (SASS, LESS, SCSS, TypeScript, Jade, Dart2JS или CoffeeScript). Если с текущим проектом не связан ни один применимый предустановленный File Watcher, WebStorm предложит добавить его.
