Использование нескольких текстовых курсоров

В WebStorm можно использовать несколько текстовых курсоров. Удерживайте Shift+Alt (в Windows и Linux) / Shift+Alt (в macOS) и кликайте в разных местах, чтобы установить дополнительные текстовые курсоры в редакторе.

Затем можно использовать клавиши со стрелками влево или вправо с нажатой клавишей Shift для выделения нескольких фрагментов текста.

Нажмите &shortcut:EditorEscape;, чтобы удалить все текстовые курсоры, кроме последнего добавленного.
