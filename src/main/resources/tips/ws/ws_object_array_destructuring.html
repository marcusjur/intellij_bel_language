Деструктуризация

Деструктуризация позволяет легко распаковать значения из массивов и объектов в переменные. Этот функционал имеет очень лаконичный синтаксис и часто используется при передаче данных в приложении.

В WebStorm вы можете вызвать деструктуризацию с помощью действия намерения (&shortcut:ShowIntentionActions;). Используйте действие Заменить на деструктуризацию объекта/массива, чтобы удалить исходное присваивание. Чтобы сохранить присваивание, используйте Внедрить деструктуризацию объекта/массива.
