Управление запросами на слияние

Управляйте входящими запросами на слияние GitHub прямо из RustRover: выберите Git | GitHub | Просмотр запросов на слияние в главном меню. RustRover позволяет назначать и объединять запросы на слияние, просматривать временную шкалу и встроенные комментарии, оставлять комментарии и проверять изменения, не выходя из ИСР.
