Переход между ошибками

Нажмите &shortcut:GotoNextError; / &shortcut:GotoPreviousError; для перехода между подсвеченными синтаксическими ошибками.

Нажмите &shortcut:PreviousOccurence; / &shortcut:NextOccurence; для перехода между сообщениями об ошибках или результатами поиска.

Чтобы пропустить предупреждения, щёлкните правой кнопкой мыши на панели валидации/маркеров и выберите Переходить только к проблемам высокого приоритета.
