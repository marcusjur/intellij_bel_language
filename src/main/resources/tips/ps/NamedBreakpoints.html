Добавление описания точки останова

Вы можете добавить краткое описание к любой точке останова для удобства последующего поиска.

Нажмите &shortcut:ViewBreakpoints;, чтобы просмотреть все существующие точки останова, затем щёлкните правой кнопкой мыши по точке останова, к которой хотите добавить описание, и выберите Редактировать описание:
