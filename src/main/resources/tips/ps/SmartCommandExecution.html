Запуск функций ИСР из терминала

Если вы видите подсвеченную команду в терминале, не выполняйте её сразу. Нажмите Control+Enter (в Windows и Linux)/Control+Enter (в macOS), чтобы активировать соответствующую функцию ИСР.

Например, вы можете использовать git log, чтобы открыть вкладку Лог в окне инструмента Git, или попробовать использовать "Показать различия", чтобы открыть файл в просмотрщике различий.
