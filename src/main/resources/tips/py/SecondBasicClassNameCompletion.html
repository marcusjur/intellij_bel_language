Автодополнение кода

Автодополнение предлагает варианты, доступные из текущей позиции текстового курсора. Нажмите &shortcut:CodeCompletion; дважды, чтобы расширить список предложений.

Если вы используете автодополнение для поиска имени класса, и этот класс еще не импортован, оператор import будет добавлен автоматически.
