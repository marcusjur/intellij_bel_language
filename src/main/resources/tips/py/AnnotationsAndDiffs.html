Просмотр аннотаций Git Blame

Аннотации Git показывают подробную информацию о происхождении каждой строки кода (щёлкните правой кнопкой мыши по области отступов и выберите Аннотировать с помощью Git Blame).

Щёлкните правой кнопкой мыши по аннотации и выберите Показать изменения, чтобы просмотреть различия между текущей и предыдущей версиями файла.
