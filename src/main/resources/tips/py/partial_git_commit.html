Частичная фиксация изменений

Чтобы разделить изменения одного файла между разными коммитами, нажмите Коммит в окне инструмента &shortcut:ActivateCommitToolWindow;, затем нажмите Различия. Отметьте флажками измененные или новые блоки кода, которые хотите зафиксировать, и нажмите Коммит. Неотмеченные изменения останутся в текущем списке изменений.
