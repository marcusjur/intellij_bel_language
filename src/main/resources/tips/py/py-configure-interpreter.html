PyCharm позволяет настроить интерпретатор Python на разных этапах разработки:

  • При создании проекта (Файл | Новый проект, или Экран приветствия с кнопкой Создать новый проект).

  • В существующем проекте, используя виджет Python интерпретатор в строке состояния, или выбрав Проект | Python интерпретатор в настройках проекта.
