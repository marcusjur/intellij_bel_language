Сравнение файлов и папок

При сравнении файлов или папок можно использовать V-образные кнопки ( ) для применения различий между панелями в средстве просмотра различий

Чтобы добавить содержимое текущей панели к другой панели, удерживайте Ctrl. Кнопки изменятся на .
