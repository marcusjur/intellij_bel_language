Извлечение метода Spark

Если в вашем проекте есть классы Scala, вы можете извлечь фрагмент кода из заметки Zeppelin в метод Scala и добавить его в этот класс. Щёлкните правой кнопкой мыши по выражению, выберите Рефакторинг | Извлечь метод Spark, затем используйте открывшийся мастер для указания имени метода и его видимости.
