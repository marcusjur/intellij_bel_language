Извлечение Spark задачи

Если в вашем Zeppelin Notebook есть фрагмент кода, содержащий Spark задачу, вы можете быстро извлечь его в отдельный Scala файл в виде Scala класса.

Для этого просто щёлкните правой кнопкой мыши по Spark задаче и выберите Рефакторинг | Извлечь Spark задачу из контекстного меню.
