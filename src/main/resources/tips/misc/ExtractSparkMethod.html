Если в вашем проекте есть класс Scala, вы можете извлечь методы Spark из блокнота Zeppelin в этот класс.

  1. Выберите фрагмент кода метода Spark в блокноте Zeppelin.
  2. Щёлкните правой кнопкой мыши по выбранному коду и выберите Рефакторинг | Извлечь метод Spark из контекстного меню.
  3. В открывшемся диалоговом окне укажите класс Scala (его можно выбрать в дереве проекта).
  4. Объявите метод и нажмите Рефакторинг. Код выбранного метода будет добавлен в целевой класс Scala.
