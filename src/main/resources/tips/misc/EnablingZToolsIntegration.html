С помощью экспериментальной функции ZTools вы можете просматривать локальные переменные текущей сессии Zeppelin.

Чтобы включить интеграцию ZTools, нажмите Ctrl+Alt+S (Windows, Linux) или Cmd+, (macOS), перейдите в Инструменты | Подключения Big Data Tools, выберите нужную конфигурацию Zeppelin, затем установите флажок Включить интеграцию ZTools (экспериментальная функция).
