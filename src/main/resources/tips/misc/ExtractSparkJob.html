С помощью плагина Big Data Tools вы можете извлечь Spark-задачи из блокнота Zeppelin в отдельные файлы.

  1. Выберите фрагмент кода Spark-задачи в блокноте.
  2. Щёлкните правой кнопкой мыши по выбранному коду и выберите Рефакторинг | Извлечь Spark-задачу из контекстного меню.
  3. Сохраните файл Scala, который откроется в отдельной вкладке редактора.
