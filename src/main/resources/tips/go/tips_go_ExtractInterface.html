Рефакторинг: Извлечение интерфейса

Вы можете создать интерфейс на основе выбранного типа. Диалоговое окно Извлечение интерфейса отображает список методов и спецификаций, доступных для создания новой спецификации методов.

Чтобы создать интерфейс, поместите текстовый курсор на тип, интерфейс или метод, который хотите использовать в качестве основы для нового интерфейса. Выберите Рефакторинг | Извлечь | Интерфейс. В диалоговом окне Извлечение интерфейса выберите элементы для извлечения вместе с интерфейсом и нажмите Рефакторинг.
