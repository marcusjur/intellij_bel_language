Открытие и просмотр мини-дампов Windows

Файлы мини-дампов Windows записывают состояние программы во время её работы или сбоя. Для чтения файлов мини-дампов вам потребуются исполняемые файлы приложения и файлы дампа.

Мини-дампы можно открывать и исследовать в GoLand. Чтобы открыть мини-дамп, перейдите в Выполнить | Открыть дамп ядра. Укажите пути к исполняемому файлу и файлу дампа. Нажмите ОК.

Примечание: Создавать мини-дампы Windows можно только в операционной системе Microsoft Windows. Однако анализовать и отлаживать ваше приложение можно во всех поддерживаемых операционных системах.
