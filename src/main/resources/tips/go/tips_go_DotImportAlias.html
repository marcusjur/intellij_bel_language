Псевдонимы точечного импорта

Вы можете включить или отключить псевдонимы точечного импорта для пакета с помощью действий намерения.

Щёлкните на оператор импорта, нажмите &shortcut:ShowIntentionActions;, затем выберите Добавить псевдоним точечного импорта. Чтобы удалить псевдоним, вызовите действие намерения и выберите Удалить псевдоним точечного импорта.
