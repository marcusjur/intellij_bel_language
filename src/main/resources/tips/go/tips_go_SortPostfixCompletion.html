Автодополнение суффикса: добавление метода sort перед массивами и срезами

Для выполнения сортировки используйте шаблон автодополнения sort. Шаблон .sort преобразует выражение массива или среза, добавляя перед ним подходящий метод sort в зависимости от типа элементов. Если тип элементов пользовательский, GoLand будет использовать метод sort.Slice.
