Вложенные файлы '_test'

Чтобы лучше структурировать проект, используйте правила вложения файлов для группировки файлов в кластеры. Например, можно настроить все файлы _test как дочерние по отношению к основным файлам.

Чтобы определить правило, нажмите на значок Показать меню параметров в окне инструмента Проект, выберите Вложение файлов, затем введите правило в поле Суффикс дочерних файлов (например, _test.go).
