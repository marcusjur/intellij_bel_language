Запуск всех тестов в проекте

Чтобы запустить все тесты в проекте, создайте конфигурацию запуска/отладки для директории.

Нажмите Выполнить | Изменить конфигурации. Нажмите кнопку Добавить новую конфигурацию и выберите Go тест. В поле Директория укажите путь к директории, содержащей файлы приложения и тестов. Убедитесь, что Тип теста установлен как Директория. Нажмите Применить и ОК.

Чтобы запустить созданную конфигурацию, нажмите Выполнить | <go_test_configuration_name>.
