Добавление пользовательского словаря для проверки орфографии

Когда в предустановленном словаре отсутствуют определённые слова, вы можете создать свой собственный словарь. Пользовательский словарь — это текстовый файл с расширением .dic, где каждое слово находится на новой строке.

Чтобы добавить новый словарь, нажмите &shortcut:ShowSettings;, чтобы открыть настройки, затем перейдите в раздел Редактор | Естественные языки | Орфография.
