Перемещение объявления элемента в другой пакет или файл

Чтобы выполнить рефакторинг перемещения, выберите идентификатор для рефакторинга и нажмите &shortcut:Move;. В диалоговом окне Переместить выберите все необходимые объявления для перемещения вместе с идентификатором. Укажите путь к целевой директории, имя файла и название пакета.
