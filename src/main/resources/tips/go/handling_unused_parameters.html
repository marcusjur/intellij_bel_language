Быстрое исправление: Удаление имени параметра

Для неиспользуемых параметров GoLand подсвечивает имя и тип параметра (если можно удалить оба). Если тип используется для реализации спецификации метода, подсвечивается только имя.

В последнем случае можно щелкнуть по имени и нажать &shortcut:ShowIntentionActions;, чтобы выбрать между следующими быстрыми исправлениями: Удалить имя параметра или Переименовать в _. Такое же поведение доступно для функций, используемых в качестве типов функций.
