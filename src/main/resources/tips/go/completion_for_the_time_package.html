Автодополнение кода для пакета 'time'

Пакет time использует предопределённые шаблоны для функций Time.Format и time.Parse.

В GoLand при нажатии &shortcut:CodeCompletion; вы увидите плейсхолдеры для элементов времени и даты, соответствующих стандарту ISO-8601. Автодополнение также предлагает стандартные форматы времени вне строковых литералов.
