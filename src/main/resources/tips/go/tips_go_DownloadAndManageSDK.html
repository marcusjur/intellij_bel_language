Загрузка Go SDK и управление установленными Go SDK

Чтобы загрузить Go SDK, нажмите Добавить SDK рядом с полем GOROOT, затем выберите Загрузить. Выберите версию SDK из списка Версия и нажмите OK.

Чтобы выбрать локальную копию Go SDK, нажмите Добавить SDK рядом с полем GOROOT, затем выберите Локальный. В файловом менеджере перейдите к версии SDK на вашем жестком диске и нажмите Открыть.
