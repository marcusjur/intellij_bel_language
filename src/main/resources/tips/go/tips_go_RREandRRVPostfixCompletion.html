Обработка ошибок с помощью проверки 'nil'

Шаблон автодополнения с суффиксом rr генерирует код, который проверяет, не является ли переменная ошибки nil, а затем выходит из функции и возвращает значение по умолчанию. Вы можете использовать шаблон автодополнения с суффиксом rr для обработки ошибок. Этот шаблон объединяет функциональность двух существующих шаблонов rre и rrv.
