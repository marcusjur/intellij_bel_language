Список функций, применимых к значениям определённого типа

Вы можете просматривать и использовать автодополнение для функций, подходящих для значений определённого типа. Например, если у вас есть переменная t строкового типа, то при вводе t и нажатии Ctrl+Space+Space, вы увидите список функций, принимающих строковое выражение в качестве первого аргумента.
