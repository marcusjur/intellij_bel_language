Запуск 'gofmt' после операции переформатования

Можно настроить запуск gofmt при каждом переформатовании кода (например, при нажатии &shortcut:ReformatCode;). Сначала форматование выполняется встроенным форматтером GoLand, а затем gofmt.

Чтобы включить эту функцию, откройте настройки, нажав &shortcut:ShowSettings;, затем перейдите в Редактор | Стиль кода | Go. Перейдите на вкладку Другое и установите флажок При выполнении операции "Переформатовать код".
