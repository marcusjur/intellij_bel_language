Выполнение ячеек

Используйте следующие умные сочетания клавиш для быстрого выполнения ячеек кода:

  • Ctrl + Enter: Выполнить текущую ячейку.

  • Shift + Enter: Выполнить текущую ячейку и выбрать ячейку ниже.

  • Ctrl + Shift + Alt + Enter: Выполнить все ячейки кода в блокноте.

Дополнительные параметры доступны в меню Выполнить.
