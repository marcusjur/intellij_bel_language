Сравнение Jupyter Notebook

Чтобы сравнить два Jupyter Notebook, выберите один из них в окне инструмента Рабочая область, щёлкните по нему правой кнопкой мыши и выберите Сравнить с из контекстного меню. В файловой системе выберите Notebook для сравнения.

DataSpell отобразит оба Notebook и результаты сравнения в средстве просмотра различий.
