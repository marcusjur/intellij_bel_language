Добавление ячеек в Jupyter Notebook

Добавить новую ячейку можно одним из следующих способов:

  • Выберите Ячейка в меню, затем Вставить ячейку выше или Вставить ячейку ниже.

  • Нажмите кнопку + на панели инструментов блокнота.

  • Наведите курсор мыши на верхнюю границу ячейки, чтобы увидеть встроенные команды для добавления ячеек кода и Markdown.
