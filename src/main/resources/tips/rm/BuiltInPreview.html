Встроенный предпросмотр

Встроенный предпросмотр RubyMine отображает вывод HTML-кода в отдельной вкладке редактора. RubyMine сохраняет все изменения, внесенные в HTML-файл или связанные CSS и JavaScript файлы, и автоматически обновляет предпросмотр.

Чтобы открыть предпросмотр, наведите курсор на HTML-код для появления всплывающего окна браузера, затем нажмите на значок RubyMine.
