Логирующая точка останова

Чтобы записывать состояние программы во время отладки, используйте неприостанавливающие точки останова. Выберите выражение, которое нужно записать, удерживая клавишу Shift, затем щёлкните на полях строки, где находится выражение для записи.

В этом примере значение user_params будет записано при достижении строки 21.
