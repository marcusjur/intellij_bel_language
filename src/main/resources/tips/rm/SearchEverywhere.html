Поиск везде

Нажмите дважды Shift, чтобы искать файлы, действия, символы, элементы интерфейса, ветки Git и комментарии в проекте. Нажмите дважды Shift еще раз, чтобы расширить поиск на элементы вне проекта.

Используйте вкладки или прямые сочетания клавиш: &shortcut:GotoClass; для поиска классов, &shortcut:GotoFile; для файлов, &shortcut:GotoSymbol; для символов и &shortcut:GotoAction; для действий, чтобы сузить область поиска.
