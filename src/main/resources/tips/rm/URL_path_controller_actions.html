Встроенные подсказки для URL-путей

Нажмите на встроенную подсказку над каждым действием контроллера, чтобы открыть список доступных операций с URL:

  • Перейти к объявлению или использованию: переход к объявлению в файле routes.rb.

  • Создать запрос в HTTP-клиенте: создание нового запроса в HTTP-клиенте.

  • Показать все конечные точки модуля: отображение сводного представления API модуля в окне инструмента "Конечные точки".
