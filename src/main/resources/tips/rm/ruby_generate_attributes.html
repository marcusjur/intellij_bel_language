Создание методов доступа

В RubyMine вы можете быстро добавить методы доступа для переменных экземпляра. Поместите текстовый курсор в класс, нажмите &shortcut:Generate;, затем в всплывающем окне Создать выберите Reader, Writer или Accessor. Затем в появившемся диалоговом окне выберите нужные переменные экземпляра и нажмите OK.
