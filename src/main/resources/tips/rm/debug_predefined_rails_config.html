Отладка с использованием предустановленной конфигурации Rails

Для проектов Rails, RubyMine автоматически создает конфигурацию запуска/отладки development с соответствующим окружением Rails. Чтобы начать отладку приложения с помощью конфигурации запуска/отладки development, дважды нажмите Control для вызова диалогового окна Выполнить что угодно. Затем начните вводить development, выберите его из списка, удерживайте клавишу Shift (заголовок диалога изменится на Отладка), и нажмите &shortcut:EditorEnter;.
