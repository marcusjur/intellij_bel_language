Docker Compose в качестве удалённого интерпретатора

Чтобы настроить Docker Compose как удалённый интерпретатор, откройте диалоговое окно Настройки &shortcut:ShowSettings;, перейдите на страницу Языки и фреймворки | Ruby SDK и Gem. Нажмите и выберите Новый удалённый. В появившемся диалоговом окне выберите Docker Compose и укажите необходимые настройки.
