Включение режима только для чтения для подключения

Чтобы защитить вашу базу данных от случайных изменений, вы можете включить режим только для чтения для подключения.

Чтобы включить режим только для чтения для подключения, нажмите Файл | Источники данных и выберите нужный источник данных в списке Источники данных. На вкладке Параметры установите флажок Только для чтения.
