Консоль базы данных

Консоль базы данных — это SQL-файл, прикрепленный к источнику данных. Консоль запросов создаётся автоматически при создании источника данных. Чтобы открыть консоль, щёлкните правой кнопкой мыши по источнику данных и выберите Навигация | Перейти к консоли запросов | Открыть консоль по умолчанию.
