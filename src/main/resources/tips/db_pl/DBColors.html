Использование цветов для источников данных

Вы можете использовать цвета для различения источников данных и их элементов. Чтобы установить цвет для источника данных или его объектов, щёлкните правой кнопкой мыши по элементу в окне инструмента База данных (Окно | Инструменты | База данных), затем выберите Инструменты | Установить цвет.
