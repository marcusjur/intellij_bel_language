Генерация шаблонного кода с помощью автодополнения

Вы можете генерировать шаблонный код, такой как геттеры и сеттеры, а также реализовывать методы интерфейса с помощью автодополнения кода.

Просто начните вводить возможное имя метода, например введите gn для генерации getName(), или введите ct для реализации compareTo().
