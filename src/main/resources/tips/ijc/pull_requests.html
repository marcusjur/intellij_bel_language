Управление запросами на слияние

Управляйте входящими запросами на слияние GitHub прямо из IntelliJ ИСРA Community: выберите Git | GitHub | Просмотр запросов на слияние в главном меню. IntelliJ ИСРA Community позволяет назначать и объединять запросы на слияние, просматривать временную шкалу и встроенные комментарии, комментарии к коммитам и проверки, а также принимать изменения, не выходя из ИСР.
