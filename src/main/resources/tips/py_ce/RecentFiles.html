Просмотр последних файлов

Нажмите &shortcut:RecentFiles; (Вид | Последние файлы) для просмотра списка недавно открытых файлов.

Вы также можете просмотреть результаты недавнего поиска использований. Для этого используйте ту же комбинацию клавиш &shortcut:RecentFiles; в окне инструмента Поиск, затем выберите нужный результат поиска во всплывающем окне Последние поиски использований.
