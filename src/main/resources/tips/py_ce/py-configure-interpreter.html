PyCharm Community позволяет настраивать интерпретатор Python на разных этапах разработки:

  • При создании проекта (Файл | Новый проект, или Экран приветствия с кнопкой Создать новый проект).

  • В существующем проекте, используя виджет Интерпретатор Python в строке состояния, или выбрав Проект | Интерпретатор Python в настройках проекта.
