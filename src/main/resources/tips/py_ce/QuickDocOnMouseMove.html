Быстрая документация при наведении

Нет необходимости явно вызывать быструю документацию &shortcut:QuickJavaDoc;. Окно быстрой документации появится автоматически при наведении указателя мыши на нужный символ.

Чтобы включить эту функцию, установите флажок Показывать быструю документацию при наведении на странице Редактор | Редактирование кода в диалоговом окне Настройки (&shortcut:ShowSettings;).
