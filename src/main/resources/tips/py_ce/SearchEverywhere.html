Поиск везде

Нажмите дважды Shift, чтобы искать файлы, действия, символы, элементы интерфейса, ветки Git и комментарии в проекте. Повторное двойное нажатие Shift расширит поиск на элементы вне проекта.

Используйте вкладки или прямые сочетания клавиш: &shortcut:GotoClass; для классов, &shortcut:GotoFile; для файлов, &shortcut:GotoSymbol; для символов и &shortcut:GotoAction; для действий, чтобы сузить область поиска.
