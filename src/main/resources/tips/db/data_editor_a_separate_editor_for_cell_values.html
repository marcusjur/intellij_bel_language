Вы можете форматовать однострочные XML и JSON строки для значений ячеек в отдельном редакторе.

Кроме того, в редакторе значений можно просматривать изображения. Обратите внимание, что размер LOB ограничен 1024 байтами. Вы можете изменить это значение в разделе База данных | Редакторы и средства просмотра данных.
