Установка плагинов

Вы можете установить плагины для интеграции с Git, HTTP-клиентом, Docker или добавить другие функции. Чтобы установить плагин, нажмите &shortcut:ShowSettings; для открытия настроек, затем перейдите в раздел Плагины. Введите название плагина в поле поиска и нажмите &shortcut:EditorEnter;.
