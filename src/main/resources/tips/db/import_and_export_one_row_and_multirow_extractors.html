Экстракторы для однострочных и многострочных SQL-вставок

Экстрактор One-Row копирует столбцы в строку, разделенную запятыми. Его можно использовать для вставки набора значений в условие IN.

Экстрактор SQL-Insert-Multirow генерирует один оператор INSERT, который может вставить несколько новых строк.
