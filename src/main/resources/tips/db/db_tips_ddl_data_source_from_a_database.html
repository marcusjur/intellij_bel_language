Выгрузка в DDL-источник данных

DDL-источник данных — это виртуальный источник данных со схемой, основанной на нескольких SQL-скриптах. Хранение этих файлов в системе контроля версий (СКВ) — это способ сохранения базы данных под контролем версий.

Чтобы создать DDL-источник данных, щёлкните правой кнопкой мыши по источнику данных в Обозревателе баз данных и выберите Импорт/Экспорт | Выгрузка в DDL-источник данных.
