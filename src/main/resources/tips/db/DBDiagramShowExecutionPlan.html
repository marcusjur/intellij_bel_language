План выполнения запроса

Вы можете визуализовать набор шагов, используемых для доступа к данным в базе данных.

Чтобы создать план выполнения, щёлкните правой кнопкой мыши по запросу в консоли запросов и выберите Explain Plan. Если вы хотите создать диаграмму для запроса, нажмите на значок Показать визуализацию.
