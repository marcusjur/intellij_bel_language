Выполнение пользовательских запросов

Вы можете настроить параметры выполнения запросов, когда курсор находится внутри или вне запроса, а также для выделенной области. Чтобы установить эти параметры, откройте настройки ИСР &shortcut:ShowSettings; и перейдите в раздел База данных | Выполнение запросов.
