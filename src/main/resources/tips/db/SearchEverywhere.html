Поиск везде

Дважды нажмите Shift для поиска файлов, действий, символов, элементов интерфейса, веток Git и комментариев в проекте. Повторное двойное нажатие Shift расширит поиск на элементы вне проекта.

Используйте вкладки или прямые сочетания клавиш для сужения результатов поиска: &shortcut:GotoClass; для объектов базы данных, &shortcut:GotoFile; для файлов, &shortcut:GotoSymbol; для символов и &shortcut:GotoAction; для действий.
