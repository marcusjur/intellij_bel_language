Консоли баз данных, временные файлы и временные буферы

Консоль базы данных — это SQL-файл, привязанный к источнику данных. Консоль базы данных создаётся автоматически при создании источника данных. Чтобы открыть консоль, щёлкните правой кнопкой мыши по источнику данных и выберите Навигация | Перейти к консоли запросов | Открыть консоль по умолчанию. Чтобы создать новую консоль, щёлкните правой кнопкой мыши по источнику данных и выберите Создать | Консоль запросов.

Временные файлы — это SQL-файлы, не привязанные к источнику данных. Временные файлы можно использовать как черновики или заметки для кода вне контекста проекта. Чтобы создать временный файл, перейдите в Файл | Создать | Временный файл (или нажмите &shortcut:NewScratchFile;).

Временные буферы — это простые текстовые файлы без функций помощи при написании кода. Временные буферы можно использовать для простых списков задач и заметок для себя. Нет специального пункта меню для создания новых временных буферов, но вы можете использовать всплывающее окно Найти действие (&shortcut:GotoAction;) и выполнить действие Создать временный буфер.
