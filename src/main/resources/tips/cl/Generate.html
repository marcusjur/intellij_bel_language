В любом месте кода нажмите &shortcut:Generate; или выберите Код | Сгенерировать, чтобы увидеть все доступные варианты генерации кода. С помощью этого меню можно быстро создавать конструкторы/деструкторы, геттеры/сеттеры, различные операторы и реализовывать/переопределять функции.

При вызове из файла Google Test меню Сгенерировать также включает опции для тестовых и фикстурных классов.
