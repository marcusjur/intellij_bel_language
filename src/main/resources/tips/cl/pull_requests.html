Управление pull request'ами

Управляйте входящими pull request'ами GitHub прямо из CLion: выберите Git | GitHub | Просмотр pull request'ов в главном меню. CLion позволяет назначать и объединять pull request'ы, просматривать временную шкалу и встроенные комментарии, оставлять комментарии и проверять изменения, не выходя из ИСР.
