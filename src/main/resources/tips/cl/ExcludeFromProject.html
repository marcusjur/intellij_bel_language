Исключение файлов

Вы можете исключить любой файл из проекта, чтобы он игнорировался при индексации, проверках и автодополнении кода.

В окне инструмента Проект, щёлкните правой кнопкой мыши по файлу, который нужно исключить, выберите Переопределить тип файла из контекстного меню, затем выберите Обычный текст из списка Выбор типа файла.

При необходимости вы всегда можете восстановить исходный тип файла, выбрав Сбросить переопределение типа файла из контекстного меню.
